Запрос документов

Примечание. Функционал доступен только при значении константы LawActUseRequestDoc равном "Истина".

Запрос документов происходит аналогично описанному в разделе Запрос документов для неопределенных судебных дел в модуле Юридический департамент, только для одного дела.

1.Войдите в модуль Карточка судебной работы в судебном производстве.

2.Нажмите кнопку Запросить документы.

3.Статус дела меняется на "Запрос документов".

4.Далее осуществляется формирование настроенных запланированных действий.

5.Осуществляется протоколирование.
