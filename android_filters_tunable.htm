Настраиваемые фильтры

Show/Hide Hidden Text

Форма настраиваемых фильтров включает в себя две части:

·Основные фильтры;

·Дополнительные фильтры.

Основные фильтры отображаются сразу после нажатия кнопки Фильтр. Они состоят из основных, заранее определенных фильтров. Для каждой экранной формы, поддерживающей фильтрацию, они различны.

Дополнительные фильтры при входе на форму фильтрации скрыты. Они открываются по нажатию на кнопку . В дополнительных фильтрах дается возможность фильтрации по всем данным, по которым строится отображение данных на экранной форме. Для разных форм дополнительные фильтры различаются. На приведенном рисунке показан развернутый дополнительный фильтр на форме "Выезды" (Рисунок 11‑1).

Примечание. Для всех полей, поддерживающих выбор из множества значений (словарных), предусматривается возможность выбора также пустого значения.

Фильтр формы "Выезды"

Основные фильтры

·Флаг "Выезды на сегодня" — если флаг включен, выводятся только выезды, назначенные на текущую дату, по умолчанию флаг включен;

·Дата выезда — интервальный список ("С … по …"), поле выводится только при выключенном флаге "Выезды на сегодня";

·Тип адреса — выбор типа адреса из списка (словарь № 5);

·Балл № 1 — выбор из списка (словарь № 101).

·Балл № 2 — выбор из списка (словарь № 102);

·Балл № 3 — выбор из списка (словарь № 244);

·Балл № 4 — выбор из списка (словарь № 245).
