Регистрация обращения

Регистрация обращений из карточки судебной работы происходит аналогично регистрации обращений из карточки должника.

Примечание. Для доступа к механизму обращений используется константа UseAppeal логического типа. Если ее значение равно "Истина", работа с обращениями доступна. Если ее значение равно "Ложь", обращения недоступны.

1.Войдите в карточку судебной работы.

2.Нажмите кнопку Зарегистрировать обращения.

3.Выберите в выпадающем меню пункт "Зарегистрировать обращение".

4.Откроется форма Обращение для добавления обращения. Заполните поля:

·Дата обращения — по умолчанию установлена текущая дата. Для изменения имеется возможность выбрать дату при помощи календаря. Обязательно для заполнения.

·Тип обращения — см. словарь № 238 "Тип обращения". Обязательно для заполнения.

·Подтип обращения — см. словарь № 262 "Подтип обращения". Необязательно для заполнения.

·Номер обращения — текст, ввод пользователем вручную. Обязательно для заполнения.

·Наименование органа — см. словарь № 239 "Орган". Необязательно для заполнения.

·Характер обращения — введите дополнительную информацию, касающуюся характера обращения. Необязательно для заполнения.

Примечание. Нажмите кнопку Сохранить, чтобы стали доступны следующие действия:

·Ответить на обращение;

·Прикрепить документ;

·Редактировать документ;

·Удалить документ;

·Выгрузить документ.

5.Нажмите OK.

Обращение будет доступно из модуля "Обращения".
