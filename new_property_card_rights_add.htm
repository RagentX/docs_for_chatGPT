Добавление правовой характеристики имущества

Добавление правовых характеристик имущества доступно только пользователям, в роли которых доступ № 745 PROPERTY_CARD_LEGAL_SPECIFICATION_ADD имеет значение "TRUE".

Действие включает в себя добавление правовой характеристики и сроков ее действия.

Для того чтобы добавить правовую характеристику для имущества (связь имущества с правом на объект), выполните следующее:

1.Откройте форму "Карточка имущества" для нужного имущества.

2.Активируйте вкладку "Характеристики", подчиненную вкладку "Правовые характеристики".

3.Нажмите кнопку Добавить. Откроется форма "Добавить". На форме отображается список видов права, которые еще не были добавлены данному имуществу.

4.Выберите запись в списке видов права.

5.Заполните даты в следующих полях:

·С — дата, с которой действует право.

·По — дата, по которую действует право.

6.Нажмите ОК.

Запись о праве будет добавлена.

Поля списка видов права на форме "Добавить":

·ID права.

·Вид права.

·Дата регистрации права.

·Регистрационный номер права.

·Владелец права.

Действия со списком записей о правах

Добавить право

Добавление записи о праве доступно только пользователям, в роли которых доступ № 807 PROPERTY_CARD_PROPERTY_LAW_ADD имеет значение TRUE.

Для того, чтобы добавить запись о праве в список доступных на форме добавления связи имущества с правом на объект "Добавить", выполните следующее:

1.Откройте форму "Карточка имущества" для нужного имущества.

2.Активируйте вкладку "Характеристики", затем подчиненную вкладку "Правовые характеристики".

3.Нажмите кнопку Добавить. Откроется форма "Добавить".

4.На форме нажмите кнопку Добавить. Откроется форма "Право" для добавления записи о праве. Заполните следующие поля:

·Вид права — выберите вид права, см. словарь № 352 "Вид права на объект имущества" (поле обязательно для заполнения);

·Дата регистрации права — укажите дату регистрации права;

·Регистрационный номер права — введите регистрационных номер права;

·Владелец права — выберите владельца права (см. словарь № 353 "Владелец права на объект имущества");

·Дата С — укажите дату, с которой действует право;

·Дата ПО — укажите дату, по которую действует право;

·Размер доли в праве, % — введите значение размера доли в праве в процентах.

5.Нажмите ОК.

Запись о праве будет добавлена.

Редактировать право

Редактирование записи о праве доступно только пользователям, в роли которых доступ № 808 PROPERTY_CARD_PROPERTY_LAW_EDIT имеет значение TRUE.

Для того, чтобы отредактировать запись о праве в списке доступных на форме добавления связи имущества с правом на объект "Добавить", выполните следующее:

1.Откройте форму "Карточка имущества" для нужного имущества.

2.Активируйте вкладку "Характеристики", затем подчиненную вкладку "Правовые характеристики".

3.Нажмите кнопку Добавить. Откроется форма "Добавить".

4.На форме нажмите кнопку Редактировать. Откроется форма "Право" для редактирования права с заполненными полями. Измените нужные значения полей.

5.Нажмите ОК.

Запись о праве будет изменена.

Удалить право

Доступна только при 809 PROPERTY_CARD_PROPERTY_LAW_DELETE

Удаление записи о праве доступно только пользователям, в роли которых доступ № 809 PROPERTY_CARD_PROPERTY_LAW_DELETE имеет значение TRUE.

Для того, чтобы удалить запись о праве в списке доступных на форме добавления связи имущества с правом на объект "Добавить", выполните следующее:

1.Откройте форму "Карточка имущества" для нужного имущества.

2.Активируйте вкладку "Характеристики", затем подчиненную вкладку "Правовые характеристики".

3.Нажмите кнопку Добавить. Откроется форма "Добавить".

4.На форме нажмите кнопку Удалить.

5.Откроется форма подтверждения удаления. Нажмите Yes.

Запись о праве будет удалена.
