Вкладка "Долги"

Вкладка Долги в карточке должника предназначена для просмотра и редактирования данных о долгах и относящихся к ним обещаниям оплаты и совершенным платежам. Вкладка содержит следующие элементы:

·список долгов;

·кнопки для работы со списком долгов;

· — Добавление долга;

· — Редактирование долга;

· — Перевод долга в проблемные/поиск информации/без перспектив/пользовательский статус;

· — Просмотр выписки по долгу;

· — Завершение работы с долгом;

· — Установка предыдущего оператора для закрытых долгов и Установка предыдущего портфеля;

· — Добавить долг в группу;

· — Формирование произвольных отчетов по долгу;

· — Ручная постановка задачи по долгу;

· — Судебная работа;

· — Настройка персональных ограничений коммуникации;

· — Окраска записи о долге.

·вкладка Платежи, содержащая:

·список платежей должника по выбранному долгу;

·кнопки для работы со списком платежей;

·вкладка Обещания, содержащая:

·список обещаний оплаты по выбранному долгу;

·кнопки для работы со списком обещаний;

·вкладка Графики платежей, содержащая:

·историю графиков платежей по выбранному долгу;

·кнопки для работы со графиками платежей.

Кнопка  Поставить задачу, относящаяся к списку долгов и кнопка  Поставить задачу, относящаяся к списку платежей, связаны с функционалом Организатор. В настоящее время функционал не поддерживается. Глобальное включение/выключение функционала производится при помощи коммерческой константы @Organizer и регулируется доступами:

·151 ORGANIZER_VIEW;

·152 ORGANIZER_FULL;

·153 DEBT_CARD_TASK_DEBT;

·154 DEBT_CARD_TASK_PAY.

Если константа @Organizer выключена, то кнопка не видна.
