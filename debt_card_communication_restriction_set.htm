       Описание формы "Персональные ограничения"

Форма Персональные ограничения включает следующие элементы:

·Таблица со списком добавленных данному долгу (или персоне) схем коммуникации;

·Кнопки для работы со схемами коммуникации:

·Добавить — для добавления схемы коммуникации к персональным ограничениям коммуникации;

·Редактировать — для изменения схемы коммуникации из персональных ограничений;

·Удалить — удаление одной или более выбранных схем коммуникации из персональных ограничений.

·Таблица со списком периодов выбранной схемы из персональных ограничений коммуникации — доступна только для просмотра. Изменение периодов схемы коммуникации осуществляется в модуле Словари на вкладке Схемы коммуникации.

       Добавление ограничения коммуникации

Для добавления ограничения коммуникации выполните следующие действия:

1.В модуле Карточка должника активируйте вкладку "Долги" и выберите долг.

2.При необходимости перейдите на вкладку "Имущество" и выберите запись об имуществе или перейдите на вкладку "Поручители" и выберите запись о поручителе.

3.Нажмите кнопку . Откроется форма Персональные ограничения.

4.Нажмите кнопку .

5.В открывшейся форме заполните следующие поля:

·Схема коммуникации — выберите схему коммуникации из настроенных в модуле Словари на вкладке Схемы коммуникации. Поле обязательно для заполнения;

·Дата начала действия — укажите дату начала действия выбранной схемы. Поле обязательно для заполнения;

·Дата окончания действия — укажите дату окончания действия выбранной схемы либо оставьте поле пустым. Поле не обязательно для заполнения;

·Комментарий — при необходимости введите дополнительную информацию. Поле не обязательно для заполнения;

·Действует на весь договор — включите данный флаг, если вы хотите наложить ограничение на коммуникации со всеми персонами, связанными с данным долгом, иначе для каждой персоны настраивайте ограничения по отдельности.

Примечание. Флаг Действует на весь договор присутствует только на форме Настройка схемы коммуникации, открытой из вкладки "Долги". Если форма открыта из вкладки Имущество или Поручители, то этот флаг отсутствует.

6.Нажмите ОК.

Внимание! Важно! Если добавляемая схема коммуникации входит в противоречие с одной из уже добавленных схем (то есть для определенного периода для одного и того же типа контакта добавляется более одной схемы коммуникации), появляется предупреждение:

Для указанного периода уже задано ограничение по некоторым типам контактов.

и схема не добавляется.

       Редактирование ограничения коммуникации

1.В модуле Карточка должника активируйте вкладку "Долги" и выберите долг.

2.При необходимости перейдите на вкладку "Имущество" и выберите запись об имуществе или перейдите на вкладку "Поручители" и выберите запись о поручителе.

3.Нажмите кнопку . Откроется форма Персональные ограничения.

4.Выберите запись в верхней таблице.

5.Нажмите кнопку .

6.В открывшейся форме измените нужные атрибуты.

7.Нажмите ОК.

       Удаление ограничения коммуникации

1.В модуле Карточка должника активируйте вкладку "Долги" и выберите долг.

2.При необходимости перейдите на вкладку "Имущество" и выберите запись об имуществе или перейдите на вкладку "Поручители" и выберите запись о поручителе.

3.Нажмите кнопку . Откроется форма Персональные ограничения.

4.Выберите запись в верхней таблице.

5.Нажмите кнопку .

6.Подтвердите удаление схемы.
