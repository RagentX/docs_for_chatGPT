Вкладка "Данные о ходе ИП (Кворум)"

Вкладка доступна только в том случае, если в Системе включена интеграция с сервисом ФССП "Кворум".

Примечание. Данный функционал включается по согласованию с вендором. Обратитесь в техническую поддержку. Коммерческая константа @Quorum.Use.

Вкладка Данные о ходе ИП (Кворум) включает следующие элементы:

·Кнопка Запрос информации о ходе ИП — для отправки запроса о ходе данного ИП в сервис "Кворум";

·Таблица Документы исполнительного производства с данными по текущему ИП со следующими колонками:

·ID документа — quorum_fssp_exeс_proc_docs_inf.doc_id;

·Номер документа — quorum_fssp_exeс_proc_docs_inf.doc_number;

·Наименование документа — quorum_fssp_exeс_proc_docs_inf.doc_type_name;

·Дата документа — quorum_fssp_exeс_proc_docs_inf.doc_dt;

·СПИ — quorum_fssp_exeс_proc_docs_inf.spi_fio.

·Таблица Расчеты по исполнительному производству с данными по текущему ИП со следующими колонками:

·Дата проводки — quorum_fssp_exeс_proc_acct_rec.acc_date;

·Тип регистра — quorum_fssp_exeс_proc_acct_rec.reg_type;

·Тип назначения — quorum_fssp_exeс_proc_acct_rec.dest_type;

·Сумма — quorum_fssp_exeс_proc_acct_rec.amount.

·Таблица Сведения об имущественном положении с данными по текущему ИП со следующими колонками:

·Тип имущества — quorum_fssp_exeс_proc_prop_inf.property_type;

·Статус ареста имущества — quorum_fssp_exeс_proc_prop_inf.status;

·Описание — quorum_fssp_exeс_proc_prop_inf.property_type.dsc.

В таблице Сведения об имущественном положении выводятся данные по документу, выбранному на текущий момент в списке в таблице Документы исполнительного производства.
