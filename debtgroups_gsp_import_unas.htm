Инструкция по импорту вкладки, настроенной вендором

1)Для настройки вкладки, созданной вендором, получите у службы технической поддержки два файла:

·файл  вида "*.std", содержащий сохраненный шаблон структуры;

·файл  вида "*.qtd", содержащий сохраненный запрос, созданный с использованием данной структуры.

2)Выполните импорт структуры:

1.В форме Конструктор структур нажмите кнопку Импорт.

2.В открывшемся окне выберите файл.

Примечание. Файл структуры должен иметь расширение "*.std".

3.Нажмите Открыть (Open). Структура будет открыта.

3)Нажмите Сохранить изменения.

4)Нажмите Сохранить данные. Подтвердите сохранение.

5)В открывшейся форме введите заголовок для сохраняемого шаблона структуры.

6)Нажмите OK.

Форма для сохранения шаблона структуры

7)Выполните импорт запроса:

1.В форме Конструктор запросов нажмите кнопку Создать запрос.

2.Выберите шаблон структуры.

3.Нажмите кнопку Импорт.

4.В открывшемся окне выберите файл.

Примечание. Файл запроса должен иметь расширение "*.qtd".

5.Нажмите Открыть (Open). Запрос будет открыт.

8)Нажмите кнопку — Выполнитьзапрос.

9)Нажмите кнопку — Сохранить запрос. Подтвердите сохранение.

10)В открывшейся форме введите заголовок и описание запроса и нажмите кнопку ОК. Запрос будет сохранен.

Сохранение запроса

11)Выполните настройку доступов, как описано в разделе Назначение доступа к группам свободного поиска.
