Настройка ограничений доступа к долгам по оператору

Для того чтобы изменить ограничения доступа оператора к долгам, выполните следующие действия:

1.В форме Настройка списка должников активируйте вкладку Ограничения по оператору.

2.Выберите запись об операторе, для которого требуется настроить ограничения доступа к долгам, и выполните двойной щелчок мышью по ней.

3.В открывшейся форме Ограничивающие условия настройте следующие ограничения:

·Статус — выберите статус долгов, к которым оператор будет иметь доступ. Если ничего не выбрано — ограничения не накладываются.

·Стадия — в раскрывающемся списке выберите нужную стадию должника. Оператор сможет работать только с долгами, находящимися на выбранной стадии. Если ничего не выбрано — ограничения не накладываются.

Примечание. Для того чтобы очистить настроенные ограничения в поле Статус или Стадия — очистите поле с помощью клавиши Backspace и затем нажмите OK в форме Огрничивающие условия.

·Кредитор  — в раскрывающемся списке установите флаги для тех кредиторов, с долгами которых сможет работать оператор. Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Портфели — в раскрывающемся списке установите флаги для тех портфелей, с долгами которых сможет работать оператор. Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются. Данное ограничение доступно, если константа OperatorScope.Portfolio.Use имеет значение "Истина".

·КА — в раскрывающемся списке установите флаги для банков, с исходящими портфелями которых сможет работать  оператор. Также доступен выбор "Без КА" для долгов, не имеющих исходящего портфеля. Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Регион — в раскрывающемся списке установите флаги для нужных регионов (см. словарь № 39), к которым относятся долги. Оператор сможет работать только с долгами, относящимися к выбранным регионам. Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Филиал — в раскрывающемся списке установите флаги для нужных филиалов (см. словарь № 61), к которым относятся долги. Оператор сможет работать только с долгами, относящимися к выбранным филиалам. Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Балл № 1 — в раскрывающемся списке установите флаги для нужных значений поля (см. словарь № 101, Скоринговые баллы). Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Балл № 2 — в раскрывающемся списке установите флаги для нужных значений поля (см. словарь № 102, Скоринговые баллы). Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Балл № 3 — в раскрывающемся списке установите флаги для нужных значений поля (см. словарь № 244, Скоринговые баллы). Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

·Балл № 4 — в раскрывающемся списке установите флаги для нужных значений поля (см. словарь № 245, Скоринговые баллы). Поддерживается множественный выбор. Если ничего не выбрано — ограничения не накладываются.

Примечание. Для того чтобы очистить настроенные ограничения в полях, включающих флаги для множественного выбора — в раскрывающемся списке снимите выбор со всех включенных флагов. Это касается полей: Кредитор, КА, Регион, Филиал, Балл № 1, Балл № 2, Балл № 3, Балл № 4.

·Диапазон важности — укажите начальное и конечное значение диапазона важности должника, с долгами которого сможет работать оператор. Если диапазон не указан — ограничения не накладываются.

·Диапазон кол-ва дней просрочки — укажите начальное и конечное значение диапазона количества дней просрочки по долгам, с которыми сможет работать оператор. Если диапазон не указан — ограничения не накладываются.

·Диапазон суммы долга — укажите начальное и конечное значение диапазона сумм долгов, с которыми сможет работать оператор. Если диапазон не указан — ограничения не накладываются.

·Диапазон порядочности — укажите начальное и конечное значение диапазона важности должника, с долгами которого сможет работать оператор. Если диапазон не указан — ограничения не накладываются.

·Комментарий — дополнительная информация.

·Пустой список — признак того, что оператор заблокирован и никакие долги ему недоступны.

4.Нажмите кнопку ОК, чтобы сохранить данные.
