История изменений

В Системе логируются изменения групп (как автогрупп, так и простых групп), автофильтров и связей автофильтров с группами.

Логи доступны для промотора только тем пользователям, в роли которых доступ GROUP_DEBT_LOG_VIEW имеет значение TRUE.

Окно лога представляет собой форму История изменений, открывается при нажатии на кнопку .

Данная кнопка располагается в следующих местах:

·На вкладке Автогруппы под списком групп — см. Просмотр истории изменений выбранной группы;

·На вкладке Простые группы под списком групп — см. Просмотр истории изменений выбранной группы;

·На вкладке Автогруппы на вложенной вкладке Все автофильтры под списком автофильтров — см. Просмотр истории изменений автофильтра;

·На вкладке Автогруппы на вложенной вкладке Связи автофильтров под списком автофильтров — см. Просмотр истории изменений связи автофильтра с группой.

Описание формы "История изменений"

Форма История изменений содержит следующие элементы:

·Область данных Версии, содержащая список всех редакций данной группы;

·Область данных Изменения, содержащая подробности выбранной редакции.

Типы изменений:

·I (insert) — присваивается при первой редакции группы, то есть ее создании;

·U (update) — присваивается при изменениях атрибутов группы;

·D (delete) — присваивается при удалении группы (недоступно для просмотра в данной форме).

Для просмотра выбранного поля "Старое значение" или "Новое значение" в области данных Изменения щелчком мыши можно открыть область, содержащую значение этого поля, размеры которой доступны для изменения. Это используется в тех случаях, когда, например, значение поля имеет длину, превышающую ширина столбца.
