Возбуждение исполнительного производства/отказ в возбуждении исполнительного производства

Регистрация возбуждения исполнительного производства или отказа в возбуждении исполнительного производства доступна для дел в статусах:

·В работе;

·Возобновление;

·Подано.

Возбуждение исполнительного производства

Для установки отметки о возбуждении исполнительного производства выполните следующие действия:

1.В карточке исполнительного производства на вкладке Общая информация нажмите кнопку ИП и в раскрывшемся списке выберите Возбуждено.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.Откроется форма Возбуждение ИП. Заполните следующие поля:

·Дата возбуждения — поле обязательно для заполнения; по умолчанию поле заполнено датой возбуждения, указанной в текущей карточке исполнительного производства;

·№ исп. производства — поле обязательно для заполнения; по умолчанию поле заполнено значением этого поля, указанным в текущей карточке исполнительного производства.

4.Нажмите кнопку ОК.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.После этого дело будет переведено в статус "Возбуждено". Действие протоколируется.

Далее можно перейти к регистрации погашения задолженности или завершению работы с должником в связи с невозможностью взыскания долга.

Примечание. При возбуждении исполнительного производства осуществляется расчет плановой даты окончания ИП в соответствии со значением константы LawExecDaysForCompletePlanDate.

·Если в значении константы имеется запись вида (<ID банка>:<количество дней>), причем, <ID банка> соответствует банку, в портфеле которого находится долг, то плановая дата окончания ИП рассчитывается путем прибавления к дате возбуждения ИП указанного в данной записи количества дней;

·Если предыдущее условие не выполнено, но  в значении константы имеется запись вида (0:<количество дней>), то плановая дата окончания ИП рассчитывается путем прибавления к дате возбуждения ИП указанного в данной записи количества дней;

·Если предыдущие условия не выполняются, то расчет плановой даты окончания ИП не осуществляется;

Если расчет плановой даты окончания ИП осуществлен, его значение отображается в поле Плановая дата окончания ИП.

Отказ в возбуждении исполнительного производства

Для установки отметки об отказе в возбуждении исполнительного производства выполните следующие действия:

1.В карточке исполнительного производства на вкладке Общая информация нажмите кнопку ИП и в раскрывшемся списке выберите Отказ в возбуждении.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.Откроется форма Отказ в возбуждении ИП. Заполните следующие поля:

·Дата отказа в возбуждении — поле обязательно для заполнения;

·Причина отказа в возбуждении — см. словарь № 125; поле обязательно для заполнения;

·Комментарий — необязательное поле для дополнительной информации.

4.Нажмите кнопку ОК.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.После этого дело будет переведено в статус "Отказ". Действие протоколируется.

Для продолжения работы необходимо будет заново подать дело в ФССП.
