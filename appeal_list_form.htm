Форма "Обращения"

Show/Hide Hidden Text

Форма "Обращения" содержит следующие элементы:

·поля мультифильтров:

·Портфели — входящие портфели долгов;

·Операторы — операторы, ответственные за работу с должниками;

·вкладки, на которых сгруппированы данные об обращениях должников:

·Активные обращения — содержит обращения, по которым еще не направлен ответ;

·Архив обращений — содержит обращения, по которым ответ уже направлен;

·вкладка Обращения третьих лиц — см. Работа с обращениями третьих лиц.

Примечание. Право на использование функционала имеют те пользователи, в роли которых доступ № 535 INNER_CALL_APPEAL_REG имеет значение TRUE.

При открытии формы списки на вкладках отображаются пустыми. Нажмите кнопку Найти, чтобы вывести в список все записи с учетом значений в фильтрах, заданных по умолчанию, или сначала настройте условия фильтра для отбора записей (см. Мультифильтры, Сложный фильтр).

Для того чтобы перейти к форме "Обращение" (карточке обращения), выполните двойной щелчок левой клавишей мыши на записи о нужном обращении. Подробнее о работе с формой см. описание Форма "Обращение".
