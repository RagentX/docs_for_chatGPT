Регистрация контакта

Обновлено до 50 релиза 03.11.2020.

Модуль Регистрация контактов предназначен для регистрации результатов контактов с должниками. Система позволяет регистрировать контакты следующих типов:

·исходящий звонок;

·выезд;

·входящий звонок;

·посещение офиса должником;

·специальное;

·письмо;

·SMS-сообщение;

·Email-сообщение.

Дерево результатов (в тексте для результата контакта используется также синоним "исход") по каждому типу контактов и параметры исходов настраиваются в модуле Настройка дерева контактов.

Модуль вызывается из других модулей и форм, например, из модулей, "Список должников", "Карточка должника" и др. при помощи кнопки .

Право для входа в модуль имеют только те пользователи, в роли которых доступ CONTACT_REG_VIEW имеет значение TRUE.
