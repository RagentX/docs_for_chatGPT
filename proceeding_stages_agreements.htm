Заключение мирового соглашения

Мировое соглашение между сторонами может быть заключено на любом этапе после подачи иска в суд.

Заключение мирового соглашения фиксируется в два этапа:

1.Отметка о заключении мирового соглашения.

2.Получение исполнительного листа на мировое соглашение.

Отметка о заключении мирового соглашения

В текущей версии Системы поддержаны два режима заключения мирового соглашения: стандартный и расширенный. Расширенный режим включает дополнительно выбор типа мирового соглашения и возможность создания графика платежей по мировому соглашению. Включение расширенного режима заключения мировых соглашений осуществляется вендором. Если вам требуется расширенный режим, обратитесь в техническую поддержку.

Расширенный режим включается установкой технической константы Law.Settlement.Mode = 2.

Стандартный режим

Для того чтобы отметить заключение мирового соглашения с должником, выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Мировое соглашение и в раскрывшемся списке выберите Мировое соглашение.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить заключение мирового соглашения с должником.

3.В открывшейся форме заполните поля:

·Сумма Мир. соглашения — сумма, которую должник должен выплатить по мировому соглашению. Поле обязательно для заполнения;

·Дата Мир. соглашения — дата заключения мирового соглашения. Поле обязательно для заполнения;

·Гос. пошлина — сумма госпошлины;

·Итоговая сумма — итоговая сумма, которую должен выплатить должник, рассчитывается автоматически.

Примечание. Запрещен ввод отрицательных сумм в поля формы.

4.Нажмите кнопку ОК.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.После этого иск будет переведен в статус "Мировое соглашение". Далее можно перейти к получению исполнительного листа на мировое соглашение или возобновлению дела.

8.Действие протоколируется.

Расширенный режим

Для того чтобы отметить заключение мирового соглашения с должником, выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Мировое соглашение и в раскрывшемся списке выберите Мировое соглашение.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить заключение мирового соглашения с должником.

3.В открывшейся форме заполните поля:

·Тип мирового соглашения — см. словарь № 337 "Тип мирового соглашения"; поле обязательно для заполнения; по умолчанию указывается термин № 1 словаря: "Нематериальный";

Примечание. При выборе типа мирового соглашения "Материальный" форма уточнения итоговой суммы мирового соглашения меняет свой вид — к ней добавляется область данных для создания графика платежей.

·Сумма Мир. соглашения — сумма, которую должник должен выплатить по мировому соглашению. Поле обязательно для заполнения;

·Дата Мир. соглашения — дата заключения мирового соглашения. Поле обязательно для заполнения;

·Гос. пошлина — сумма госпошлины;

·Итоговая сумма — итоговая сумма, которую должен выплатить должник, рассчитывается автоматически, к сумме мирового соглашения прибавляется сумма госпошлины;

Если выбран тип мирового соглашения Материальный, то создайте график платежей мирового соглашения.

Примечание. Создание графика платежей осуществляется аналогично созданию графика платежей в карточке должника (см. Графики платежей и работа с ними), за исключением функционала "Расчет по формуле". После создания графика платежей по мировому соглашению будут доступны действия по добавлению, редактированию и удалению платежей в графике.

Примечание. Запрещен ввод отрицательных сумм в поля формы.

4.Нажмите кнопку ОК.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.После этого иск будет переведен в статус "Мировое соглашение". Далее можно перейти к получению исполнительного листа на мировое соглашение или возобновлению дела.

8.Действие протоколируется.

Получение исполнительного листа на мировое соглашение

Для того чтобы отметить получение исполнительного листа на мировое соглашение, выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Мировое соглашение и в раскрывшемся списке выберите Получить ИЛ на МС.

2.В открывшемся диалоговом окне подтвердите перевод дела в статус "Получение ИЛ на МС", нажав кнопку Yes.

3.Выполните прикрепление документа.

4.Далее осуществляется формирование настроенных запланированных действий.

5.После этого иск будет переведен в статус "Получение ИЛ на МС". Информацию об исполнительных листах на мировые соглашения можно внести на вкладке Мировые соглашения.

6.Действие протоколируется.

Возобновление на ИП

Если дело находится в статусе "Получение ИЛ на МС" и по делу существует исполнительное производство, становится доступным действие Возобновление на ИП.

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Мировое соглашение и в раскрывшемся списке выберите Возобновление на ИП.

2.В открывшемся диалоговом окне нажмите кнопку Yes.

3.Выполните прикрепление документа.

4.Далее осуществляется формирование настроенных запланированных действий.

5.После этого иску будет присвоен статус "Перевод на исполнительное производство", и дело будет переведено в архив судебной работы. Связанное с делом ИП получит статус "Возобновление".

6.Действие протоколируется.
