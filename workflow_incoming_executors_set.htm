Назначение исполнителей входящего документа

Назначение исполнителей доступно пользователям, в роли которых доступ № 672 DOCFLOW_EXECUTOR_CHANGE имеет значение TRUE, при этом доступно назначение исполнителей одновременно для нескольких выбранных документов.

При назначении исполнителей открывается окно Выбор оператора, которое поддерживает множественный выбор.

Если для выбранных документов уже назначены исполнители, пользователю предлагается выбор, следует ли переназначать исполнителей или нет.

Порядок действий для назначения исполнителей входящего документа

Для того чтобы назначить исполнителей документов, выполните следующие действия:

1.В модуле Документооборот активируйте вкладку Входящие.

2.Сформируйте выборку, используя при необходимости фильтр по дате и сложный фильтр. В выборку попадают операторы, у которых указан департамент (в структуре предприятия).

3.Выберите одну или более записей, при необходимости используя клавиши SHIFT и/или CTRL.

4.Нажмите кнопку Назначить исполнителей.

5.Если хотя бы один из выбранных документов уже имеет исполнителей, то появится форма подтверждения с текстом:

Внимание! Среди выбранных документов уже имеются документы с назначенным исполнителем (исполнителями). Переназначить исполнителей у таких документов (при этом старые исполнители будут откреплены)?

Если вы хотите изменить назначенных исполнителей, нажмите Yes. Если вы не хотите менять исполнителей, нажмите No, а затем вернитесь к п. 3 для коррекции выбранных документов.

6.Откроется форма Выберите исполнителей со списком операторов. Форма работает аналогично форме мультифильтра и поддерживает множественный выбор. Выберите исполнителей, а затем нажмите ОК.

7.После выполнения операции будет показано уведомление с текстом:

Новые исполнители назначены.

Нажмите ОК.

Форма "Выберите исполнителей".
