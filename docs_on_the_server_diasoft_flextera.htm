Все полученные документы в рамках данного запроса сохраняются в БД, в таблицу doc_attach с предварительной проверкой на существование такого файла по doc_attach.ext_id = ATTACHMENTID:

·Если документ с таким идентификатором не найден, то создается новый документ. 

·Если документ с таким идентификатором найден, то его атрибуты полностью обновляются.

·Если в doc_attach есть документ, полученный из Flextera, а в списке пришедших документов его нет, то документ удаляется из doc_attach

При сохранении документа параметру doc_attach.save_mode присваивается значение "3".

В качестве результата выполнения online – запроса пользователю выводится информация о количестве произведенных действий с документами в следующем виде:

Отчет по долгу "Получение документов (Flextera)"

На базе Функции получения списка документов по долгу из сервиса Diasoft Flextera создан произвольный отчет по долгу для запуска из модуля "Карточка должника" по кнопке  "Отчеты" из вкладки "Долги".

Название отчета: "Получение документов (Flextera)".

Доступ к данному отчету настроен для следующих ролей:

·СУПЕРВИЗОР_БАНК;

·АДМИНИСТРАТОР ИТ;

·БИЗНЕС АДМИНИСТРАТОР;

·РУКОВОДИТЕЛЬ ГРУППЫ;

·АНАЛИТИК;

·СПЕЦИАЛИСТ ПО АУТСОРСИНГУ (БАНК);

·КОЛЛЕКТОР (БАНК).
