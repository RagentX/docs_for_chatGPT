Сохранить выборку (Выгрузка данных в Excel, XML, CSV)

Show/Hide Hidden Text

В модуле Исходящее информирование сохранение выборки разрешено только пользователям, в роли которых доступ № 221 INFO_DEBT_EXPORT имеет значение TRUE.

Сохранение выгрузки доступно на всех вкладках.

 

Поддерживается сохранение выборки в следующие форматы:

·Форматы MS Excel:

·Excel 97-2003;

·Excel 2007-2013;

·Формат XML;

·Формат CSV:

·Разделитель - точка с запятой;

·Разделитель - запятая.

Для сохранения данных в выборки выполните следующие действия:

1.Активируйте нужную вкладку в модуле.

2.Сформируйте выборку записей. Экспортироваться будут все записи, которые отображаются в полученном списке.

3.Выберите в контекстном меню (вызывается правой кнопкой мыши) пункт Сохранить выборку или нажмите сочетание клавиш CTRL+S.

Примечание. В модуле Платежи  пункт Сохранить выборку отсутствует. Используйте кнопку .

4.Откроется стандартное диалоговое окно Windows для сохранения файла.

5.Выберите формат сохранения файла:

·Файлы XML (*.xml);

·Файлы Excel 97-2003 (*.xls);

·Файлы CSV (разделитель - точка с запятой)(*.csv);

·Файлы CSV (разделитель - запятая)(*.csv);

·Файлы Excel 2007-2013 (*.xls).

6.Укажите путь и имя файла, в который будут выгружены данные.

7.Нажмите кнопку Сохранить (Save). По окончании выгрузки будет выдано сообщение о завершении создания файла.

8.Нажмите кнопку ОК, чтобы завершить процедуру. Файл с выборкой будет сохранен в указанном формате.

Примечание. Данное действие сохраняет всю выборку, которая была отображена по нажатию кнопки "Найти" (а не только выделенные записи). При сохранении больших объемов данных могут возникнуть проблемы в работе программы MS Excel.

       Как открыть файл формата CSV в программе MS Excel

Файл формата CSV можно открыть для просмотра и редактирования в программе MS Excel в качестве листа в книге.

Для того чтобы открыть файл CSV, выполните следующие действия:

1.Откройте существующий Excel-файл или создайте новый.

2.В меню Данные выберите пункт Из текста.

3.Откроется форма выбора файла. Прежде чем выбрать файл, укажите возможные файлы: Все файлы (*.*).

4.Выберите файл с расширением CSV и нажмите Открыть (Open).

5.Откроется форма Мастер текстов (импорт) - шаг 1 из 3. В ней:

·Укажите формат данных — выберите значение: с разделителями;

·Начать импорт со строки — выберите 1;

·Формат файла — выберите 1251: Кириллица (Windows).

6.Нажмите Далее.

7.Откроется форма Мастер текстов (импорт) - шаг 2 из 3. В ней:

·Символом-разделителем является — выберите точка с запятой или запятая, в зависимости от того, с каким разделителем был сохранен этот файл.

8.Нажмите Далее.

9.Откроется форма Мастер текстов (импорт) - шаг 3 из 3. В ней при необходимости определите форматы данных столбцов.

10.Нажмите Готово.

11.Откроется форма Импорт данных. Выберите, куда следует поместить данные: на имеющийся лист (если открыт пустой файл) или на новый лист.

12.Нажмите ОК. Данные будут открыты в виде листа в книге Excel.
