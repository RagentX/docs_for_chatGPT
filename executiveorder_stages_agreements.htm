Заключение мирового соглашения

Мировое соглашение между сторонами может быть заключено на любом этапе после подачи дела в ФССП.

В текущей версии Системы поддержаны два режима заключения мирового соглашения: стандартный и расширенный. Расширенный режим включает дополнительно выбор типа мирового соглашения и возможность создания графика платежей по мировому соглашению. Включение расширенного режима заключения мировых соглашений осуществляется вендором. Если вам требуется расширенный режим, обратитесь в техническую поддержку.

Расширенный режим включается установкой технической константы Law.Settlement.Mode = 2.

Стандартный режим

Для того чтобы отметить заключение мирового соглашения с должником, выполните следующие действия:

1.В карточке исполнительного производства на вкладке Общая информация нажмите кнопку Мировое соглашение.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить заключение мирового соглашения с должником.

3.Откроется форма уточнения параметров мирового соглашения. Заполните поля:

·Сумма Мир. соглашения — сумма мирового соглашения. Поле обязательно для заполнения;

·Дата Мир. соглашения — дата заключения мирового соглашения. Поле обязательно для заполнения;

4.Нажмите кнопку ОК.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.После этого дело будет переведено в статус "Мировое соглашение". Действие протоколируется.

При необходимости можно возобновить дело в исполнительном производстве.

Расширенный режим

Для того чтобы отметить заключение мирового соглашения с должником, выполните следующие действия:

1.В карточке исполнительного производства на вкладке Общая информация нажмите кнопку Мировое соглашение.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить заключение мирового соглашения с должником.

3.Откроется форма уточнения параметров мирового соглашения. Заполните поля:

Поля для мирового соглашения нематериального типа:

·Тип мирового соглашения — см. словарь № 337 "Тип мирового соглашения"; поле обязательно для заполнения; по умолчанию указывается термин № 1 словаря: "Нематериальный";

·Дата Мир. соглашения — дата заключения мирового соглашения. Поле обязательно для заполнения;

·Комментарий — дополнительная информация.

Поля для мирового соглашения материального типа:

Примечание. При выборе типа мирового соглашения Материальный форма уточнения итоговой суммы мирового соглашения меняет свой вид — к ней добавляется область данных для создания графика платежей.

·Тип мирового соглашения — см. словарь № 337 "Тип мирового соглашения"; поле обязательно для заполнения; по умолчанию указывается термин № 1 словаря: "Нематериальный";

·Сумма Мир. соглашения — сумма, которую должник должен выплатить по мировому соглашению. Поле обязательно для заполнения;

·Дата Мир. соглашения — дата заключения мирового соглашения. Поле обязательно для заполнения;

·Комментарий — дополнительная информация.

4.Если выбран тип мирового соглашения Материальный, то создайте график платежей мирового соглашения.

Примечание. Создание графика платежей осуществляется аналогично созданию графика платежей в карточке должника (см. Графики платежей и работа с ними), за исключением функционала "Расчет по формуле". После создания графика платежей по мировому соглашению будут доступны такие же действия по добавлению, редактированию и удалению платежей в графике, как при создании графика платежей мирового соглашения в карточке судебной работы. См. Действия с графиком платежей при заключении мирового соглашения_2

5.Нажмите кнопку ОК.

6.Выполните прикрепление документа.

7.Далее осуществляется формирование настроенных запланированных действий.

8.После этого дело будет переведено в статус "Мировое соглашение". Действие протоколируется.

При необходимости можно возобновить дело в исполнительном производстве.
