Предварительная настройка констант

Константа RepMainInfoContactLogSuccessResultShowMode определяет режим вывода в выездном листе блока контактов по результатам контактов. В зависимости от ее значения:

·  1 — отображаются все выезды и все звонки;

·  2 — отображаются все выезды и только успешные звонки;

·  3 — отображаются только успешные выезды и только успешные звонки;

·  4 — отображаются только успешные выезды и все звонки.

Константа RepMainInfoContactLogShowMode определяет режим вывода в выездном листе блока контактов по долгам. В зависимости от ее значения:

·1 — отображаются по всем долгам должника;

·2 — отображаются по текущему долгу должника.

Константа RepMainInfoRandomValueShowMode определяет режим вывода в выездном листе блока произвольных атрибутов по долгам. В зависимости от ее значений:

·1 — отображаются по всем долгам должника;

·2 — отображаются по текущему долгу должника.

Право на печать выездных листов имеют пользователи, в роли которых включен доступ № 30 REP_MAIN_INFO.

Порядок действий

Для того чтобы сформировать в печатном виде выездные письма должникам, выполните следующие действия:

1.В форме Список должников выберите нужные записи. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL.

2.Нажмите кнопку Отчет "Версия для печати". По выбранным долгам будут сформированы отчеты и открыты в режиме предварительного просмотра в приложении Fast Report.

Примечание. В настоящей версии Системы список поддерживаемых типов графических файлов для генерации и печати в FastReport (шаблоны писем, выездные листы, произвольные отчеты и т.д.) задается пользователем. Для этого создана константа FastReportSupportedImageType.

Отображение данных на выездном листе

Выездной лист включает разделы:

·Общая информация:

·ФИО должника;

·Дата рождения;

·Место работы;

·Должность;

·Важность;

·Порядочность;

·Комментарий;

·Телефоны — отображаются все телефоны должника и связанных персон (подробнее — см. далее, свернутый список);

·Адреса — отображаются все адреса должника и связанных персон (подробнее — см. далее, свернутый список);

·Информация о долге;

·Последние контакты.

Отображение последних контактов зависит от значения константы RepMainInfo.ContactLog.PhoneAddress.Show. Если константа имеет значение "Истина", то для каждой записи отображается также адрес/телефон, по которому был осуществлен контакт (как на приведенном скриншоте).

Если константа RepMainInfo.ContactLog.ShowSmsLetters.Use имеет значение "Истина", то в выездном листе в разделе «Информация о долге» добавляются следующие строки:

·Количество доставленных СМС.

·Количество отправленных писем.
