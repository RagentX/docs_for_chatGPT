Описание словаря

Список типов дерева контакта задается в словаре № 257 "Тип дерева контактов".

Первым термином в словаре должно быть описано основное дерево контактов. Дополнительные деревья описываются в терминах с номерами больше 1. Если словарь № 257 отсутствует или имеет только один термин, то используется только основное дерево контактов.

Для пользователей выбор дерева контактов осуществляется при помощи формулы, номер которой указан в значении константы RegContactTreeTypFormula. Если значение константы равно нулю или пусто, то для всех пользователей используется основное дерево контактов.
