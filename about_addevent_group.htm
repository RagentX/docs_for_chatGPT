Доступность групп объектов для групповых мероприятий

Для основной части мероприятий в списке доступных групп выводятся только группы с объектом "Долги". См. Объекты для группировки

Мероприятия, использующие другие объекты для групп:

·Смена оператора (код 15), Постановка задач (код 17), Снятие закрепления (код 27) — доступны группы с объектами:

·Долги;

·Судебная работа;

·Исполнительное производство;

·Переформирование группы (код 20) — доступны группы всех настроенных объектов;

·Вызов процедуры (код 21) — доступность групп зависит от специфики каждой процедуры;

·Создание банкротства (код 29) — доступны группы с объектом Должники;

·Блокировка телефонов (код 30) — доступны группы с объектом Телефоны;

·Блокировка адресов (код 31) — доступны группы с объектом Адреса;

·Смена статуса задачи (код 32), Уведомление по задаче (код 33)  — доступны группы с объектом Планирование (задачи);

·Установка приоритетного значения атрибута (код 34) — доступны группы с объектом Должники;

·Создание ИП (код 35) — доступны группы с объектами:

·Долги;

·Судебная работа.
