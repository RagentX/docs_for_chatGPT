Кворум. Запросить информацию о ходе ИП

Действие доступно только в том случае, если в Системе включена интеграция с сервисом ФССП "Кворум".

Примечание. Данный функционал включается по согласованию с вендором. Обратитесь в техническую поддержку. Коммерческая константа @Quorum.Use.

Запрос информации о ходе ИП в сервис Кворум выполняется из контекстного меню вкладки Исполнительное производство.

Действие доступно только пользователям, в роли которых доступ № 725 LAW_ACT_EXECUTIVE_PROCEEDINGS_INFO_REQUEST_QUORUM имеет значение "TRUE".

Порядок действий

Для того чтобы отправить запрос о ходе ИП, выполните следующее:

1.В модуле Юридический департамент активируйте вкладку Исполнительное производство.

2.Сформируйте выборку, используя фильтры.

3.Выберите нужные записи, при необходимости используя клавиши SHIFT и/или CTRL.

4.Активируйте контекстное меню, нажав правую кнопку мыши, и выберите пункт Кворум. Запросить информацию о ходе ИП.

5.Если не все из выбранных дел ИП имеют номер исполнительного производства, то будет показано сообщение:

Не все выбранные записи имеют номер ИП. По таким ИП не будет выполнен запрос на получение информации о ходе ИП. Продолжить?

с вариантами "Да", "Нет". Выберите "Да".

6.Выбранные дела будут отправлены в очередь на формирование запроса в сервис "Кворум" (кроме тех из них, которые уже находятся в этой очереди на данный момент). В дальнейшем встроенный в Систему сервис автоматически осуществит отправку запроса (см. описание Сервис для интеграции с сервисом ФССП "Кворум").
