Автогруппы

Show/Hide Hidden Text

На вкладке Автогруппы формы Группы долгов содержится список автогрупп, т.е. групп, использующих автофильтры.

Доступ № 104 GROUP_DEBT_FULL дает право на работу со всеми группами, если доступ выключен или отсутствует — пользователь может ограниченно работать только со своими группами.

Доступ № 539 GROUP_DEBT_SQL_EDIT дает право на редактирование поля SQL, а также на работу с автофильтрами.

Порядок действия при создании автогруппы объектов

1.Сначала создайте новую автогруппу. Автогруппа будет создана без автофильтров и не включающая объекты.

2.Создайте необходимое количество автофильтров (см. Создание, редактирование и удаление автофильтра).

3.Проверьте созданные автофильтры (см. Проверка автофильтров).

4.Свяжите автофильтры с созданной автогруппой (см. Связывание групп с автофильтрами ). Теперь группа имеет правила, по которым в нее могут добавляться долги.

5.Сформируйте группу из автофильтров (см. Формирование группы из автофильтров).

6.Просмотреть список долгов, входящих в группу, можно на вкладке "Долги в группе" (см. Долги в группе).

7.При необходимости повторять формирование группы, используйте мероприятие Переформирование групп модуля Расписание.

Форма редактирования автогруппы

·Название — поле обязательно для заполнения.

·Объект группы — Данное поле актуально только для автогрупп. Определяет, какого типа объекты будут включаться в группу. Список. Настройка списка объектов групп: в технической константе GroupDebtObjectForGroup перечислить идентификаторы соответствующих терминов словаря № 117. Поле обязательно для заполнения.

Примечание. Если группа создается из вкладки Простые группы, то поле Объект группы отсутствует. Объект такой группы всегда Долги. Если сохранить такую группу с включенным флагом Автофильтр, то группа появится в списке на вкладке Автогруппы, и для нее будет возможность изменять объект группы.

·Комментарий — дополнительная информация.

·SQL — в данное поле можно ввести SQL-запрос для автоматического формирования группы долгов. Запрос должен возвращать идентификатор долга, названный как debt_id.  

Ограничения на запрос:

·Запрос должен начинаться с select, недопустимо описание переменных (declare);

·Обязательно должно быть поле debt_id;

·Некорректно использовать select в select;

·Не использовать group by более одного раза;

·Нельзя в названии полей, употребленных в "order by" использовать цифры и точки. Например "order by fld_1" или "order by a.fld".

Например: select id as debt_id, contract, account from debt where debt_sum=0. Если SQL-запрос не указан, то после создания группы необходимо сформировать список входящих в нее долгов.

Примечание. Если у пользователя отсутствует доступ № 539 GROUP_DEBT_SQL_EDIT, ему запрещено редактировать поле SQL, и данное поле заблокировано.

·Флаг очистки группы — см. словарь № 371 "Флаг очистки группы"; используется для очистки группы перед ее формированием — наполнением долгами;

Примечание. При переформировании группы, в зависимости от выбранного термина словаря № 371 "Флаг очистки группы", очистка данных в группе и ее наполнение долгами будут происходить следующим образом:

·0. Не очищать предыдущие данные — при переформировании в группу добавляются долги, удовлетворяющие условиям группы.

·1. Очищать предыдущие данные — группа полностью очищается, а затем в нее добавляются долги, удовлетворяющие условиям группы.

·2. Очищать только данные, не попавшие повторно — долги, которые не момент формирования группы не удовлетворяют условиям, удаляются. А для тех долгов, которые были в группе до ее переформирования и остались после ее переформирования, не изменяется дата попадания в группу.

·Флаг Автофильтр  — если флаг включен, группа будет отнесена к автогруппам, иначе – к простым группам. Поле доступно только при редактировании группы. При создании группы поле отсутствует

·Флаг Разрешить добавление долгов в группу вручную — разрешает или запрещает ручное добавление долга в группу из списочных РМ.

Создание автогруппы

Для того чтобы создать автогруппу долгов, выполните следующие действия:

1.В форме Группы долгов на вкладке Автогруппы нажмите кнопку .

2.В открывшейся форме заполните  поля.

Примечание. Поле Владелец (в форме не отображается) автоматически будет заполнено данными оператора, создавшего новую группу должников.

3.Нажмите кнопку ОК.

Примечание. Если у пользователя отсутствует доступ № 539 GROUP_DEBT_SQL_EDIT, ему запрещено редактировать поле SQL, и данное поле заблокировано..

Редактирование автогруппы

·Если у пользователя есть доступ № 104 GROUP_DEBT_FULL, то он может редактировать все группы:

·Если у пользователя есть доступ № 539 GROUP_DEBT_SQL_EDIT, то он может редактировать поле "SQL";

·Если у пользователя нет доступа № 539 GROUP_DEBT_SQL_EDIT, то он не может редактировать поле "SQL";

·Если у пользователя нет доступа № 104 GROUP_DEBT_FULL, то он может редактировать только свои группы:

·Если у пользователя есть доступ № 539 GROUP_DEBT_SQL_EDIT, то он может редактировать поле "SQL";

·Если у пользователя нет доступа № 539 GROUP_DEBT_SQL_EDIT, то он не может редактировать поле "SQL".

Для того чтобы отредактировать атрибуты группы долгов, выполните следующие действия:

1.В форме Группы долгов на вкладке Автогруппы выберите нужную группу и нажмите кнопку  или выполните клик (двойной щелчок левой клавишей мыши) по выбранной записи.

2.В форме редактирования атрибутов группы долгов измените необходимые атрибуты.

3.Нажмите кнопку OK, чтобы сохранить изменения.

Превращение автогруппы в простую группу

Автогруппа, имеющая в качестве объекта группы "Долги", может быть превращена в простую группу путем выключения флага Автофильтр в форме редактирования группы.

Для того чтобы превратить автогруппу в простую группу, выполните следующие действия:

1.В форме Группы долгов на вкладке Автогруппы выберите нужную группу (с объектом "Долги") и нажмите кнопку  или выполните клик (двойной щелчок левой клавишей мыши) по выбранной записи.

2.В форме редактирования атрибутов группы выключите флаг Автофильтр.

3.Нажмите кнопку OK, чтобы сохранить изменения.

После этого группа исчезнет из вкладки Автогруппы и появится на вкладке Простые Группы.

Примечание. Автогруппа с объектом, отличным от "Долги" не может быть превращена в простую группу.

Удаление группы

Примечание. Прежде чем удалить автогруппу, удалите все ее автофильтры.

·Если у пользователя есть доступ № 104 GROUP_DEBT_FULL, то он может удалить любую группу;

·Если у пользователя нет доступа № 104 GROUP_DEBT_FULL, то он может удалить только свою группу.

Для того чтобы удалить группу долгов, выполните следующие действия:

1.В форме Группы долгов на вкладке Простые группы для простой группы или на вкладке Автогруппы для автогруппы выберите нужную группу и нажмите кнопку .

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление. Запись о группе и всех входящих в нее долгах будет удалена.
