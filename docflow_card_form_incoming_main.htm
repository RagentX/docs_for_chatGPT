Вкладка "Основная" входящего документа

Вкладка Основная содержит основную информацию по документу.

Поля вкладки "Основная" входящего документа

·Входящий номер — номер входящего документа (доступность редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE; при включенном доступе № 674 дополнительно — доступом № 825 DOCFLOW_FIELD_BAN_EDIT) (docflow.document_incoming_number).

·Получатели — см. Изменение получателей входящего документа.

·Тип документа — тип документа (см. словарь № 332 "Тип документа") (docflow.document_type).

·Форма — форма документа (см. словарь № 331 "Форма документа") (docflow.document_form).

·Отправитель — см. Изменение отправителя входящего документа.

·Исходящий номер — (при включенном доступе № 674 доступность редактирования определяется доступом № 825 DOCFLOW_FIELD_BAN_EDIT) docflow.document_outgoing_number.

·Дата исходящего — docflow.document_dt.

·Дата оригинала — docflow.original_dt.

·Способ доставки — способ доставки (см. словарь № 333 "Способ доставки") (docflow. delivery_method).

·Исполнители — см. Изменение исполнителей входящего документа.

·Приоритет — см. словарь № 334 "Приоритет отработки документа".

·Подписной лист — дата для подписного листа (docflow.subscription_list_dt).

·Регистратор — пользователь, который создал документ (поле недоступно для редактирования) (docflow.r_reg_user_id).

·Дата регистрации — дата регистрации документа (доступность редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE; при включенном доступе № 674 дополнительно — доступом № 825 DOCFLOW_FIELD_BAN_EDIT) (docflow.reg_dt).

·Статус — статус документа (см. словарь № 329 "Статус документа") (docflow.status).

·Флаг Без конверта — docflow.cover_flag.

·Содержание — описание документа (доступность для редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE) (docflow.dsc).

·Комментарий — docflow.comment.

·Комментарий исполнителя — комментарий исполнителя (поле доступно для редактирования только исполнителю) (docflow.operator_comment).

·Адрес отправителя — адрес отображается только при включенной константе DocflowCard.SenderAddress.Use. При этом для отображения выбирается адрес с типом, указанным в значении константы DocflowCard.SenderAddress.Typ. Если таких адресов несколько, то выбирается тот, у которого id наименьший. Поле не доступно для редактирования.
