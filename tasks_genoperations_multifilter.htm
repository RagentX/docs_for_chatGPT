Применение мультифильтров

На вкладках формы Постановка задач доступны мультифильтры по следующим параметрам:

Примечание. Разным вкладкам модуля соответствуют различные наборы мультифильтров.

·Портфель — входящий портфель, в состав которого входит долг;

·Операторы — операторы, ответственные за работу с должниками;

·Регион — регион, к которому относится долг (см. словарь № 39);

·Реестр КА — исходящий портфель, в состав которых входит долг;

·Филиал — филиал организации, к которому относится долг (словарь № 61);

·Балл № 1 — см. словарь № 101;

·Балл № 2 — см. словарь № 102;

·Балл № 3 — см. словарь № 244;

·Балл № 4 — см. словарь № 245;

·Группы — значения определяются константой GroupDebtForFilter.

·Акция — акции, в который участвуют долги (только на вкладке Акции).

Подробнее см. раздел Мультифильтры.

Примечание. В модуле Настройка списка должников накладываются ограничения для вывода списков в следующих мультифильтрах:

·Портфели;

·Регион;

·Филиал;

·Реестр КА;

·Балл № 1;

·Балл № 2;

·Балл № 3;

·Балл № 4.

Примечание. В модуле Структура предприятия накладываются ограничения для вывода списков в мультифильтре Операторы (в списочных модулях, кроме модуля Группы свободного поиска).

Примечание. Во всех списочных модулях, кроме модуля Юридический департамент, в мультифильтре Группы отображаются только группы с объектом "Долги".
