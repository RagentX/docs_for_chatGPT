Вкладка "Заявления"

В карточке судебной работы на вкладке Заявления отображается список заявлений. Вкладка содержит следующие элементы:

·Область данных Заявления, включающая:

·список заявлений со следующими полями:

o№ — номер заявления;

oДата подачи;

oСтатус письма — колонка отображается только при включенном функционале интеграции с почтой России Коммерческая константа @PostRussia.Use. Выводится текст термина по словарю № 350 "Почта России. Статусы писем" с кодом post_russia_letter.letter_status записи, для которой:

·post_russia_letter.obj_id=108;

·post_russia_letter.r_id= law_act_solicitation.id;

oДата статуса письма — колонка отображается только при включенном функционале интеграции с почтой России Коммерческая константа @PostRussia.Use. Выводится текст термина по словарю № 350 "Почта России. Статусы писем" с кодом post_russia_letter.letter_status записи, для которой:

·post_russia_letter.obj_id=108;

·post_russia_letter.r_id= law_act_solicitation.id;

oТип заявления — см. словарь № 33;

oСтатус — см. словарь № 34;

oДата рассмотрения;

oОписание;

·кнопки для работы со списком заявлений.

·Область данных Файлы, включающая:

·список документов, прикрепленных к выбранному заявлению;

·кнопки для работы со списком документов, прикрепленных к выбранному заявлению.

См. Заявления в суд и работа с ними.
