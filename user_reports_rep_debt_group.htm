Отчет по группам должников

Отчет "Отчет по группам должников" выводит агрегированные данные о долгах и платежах по ним в выбранных портфелях по  месяцам за выбранный период в Excel.

Отчет "Отчет по группам должников" доступен только пользователям, в роли которых доступ REP_DEBT_GROUP имеет значение "Истина".

Порядок действий

Для того чтобы сформировать отчет, выполните следующее:

1.Выберите в главном меню пункт Отчеты, затем выберите отчет "Отчет по группам".

2.Откроется форма "Отчет по группам должников". Заполните следующие поля:

·Год с: — выберите год начала периода (по умолчанию указан текущий год).

·Месяц с: — выберите месяц начала периода (по умолчанию указан текущий месяц).

·Год по: — выберите год окончания периода (по умолчанию указан текущий год).

·Месяц по: — выберите месяц окончания периода (по умолчанию указан текущий месяц).

·Портфели — откроется форма для множественного выбора портфелей.

3.Нажмите кнопку "Запустить". Откроется стандартное окно операционной системы для указания расположения и названия файла с последующим сохранением.

4.Сохраните полученный файл. В процессе сохранения файла будет отображаться индикатор выполнения. По окончании будет показано соответствующее системное сообщение.

Описание файла отчета

·Возрастная группа.

·Пол.

·Образование.

·Банк.

·Название продукта.

·Тип продукта.

·Регион.

·Область.

·Район.

·Город.

·Филиал.

·Сумма кредита.

·Кредитная группа.

·Начальный долг.

·Группа по начальному долгу.

·Остаток к погашению.

·Группировка по остатку.

·Основной долг.

·Просроченный основной долг.

·Проценты.

·Просроченные проценты.

·Штрафы.

·Пени.

·Комиссия.

·Гос.пошлина.

·Количество.

·Сумма.

·№ месяца в работе.
