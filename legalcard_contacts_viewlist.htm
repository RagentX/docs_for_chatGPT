Вкладка "Контакты"

В карточке судебной работы на вкладке Контакты отображается список контактов с должником. При выборе того или иного контакта в поле Комментарий в нижней части формы отображается комментарий к выбранному контакту.

Для того чтобы отфильтровать список по типу контактов, выберите тип в поле в верхней части списка. В списке будет отображаться информация только о контактах выбранного типа.

Доступен просмотр сохраненных писем (аналогично Просмотру письма должнику в карточке должника).

Примечание.

·Если в роли пользователя доступ № 637 CONTACT_BLOCK_PHONE_VIEW  имеет значение TRUE, то в списках контактов и комментариев отображаются данные о контактах по заблокированным телефонам.

·Если в роли пользователя доступ № 638 CONTACT_BLOCK_ADDRESS_VIEW имеет значение TRUE, то в списках контактов и комментариев отображаются данные о контактах по заблокированным адресам.

Поля списка контактов

·Название продукта — указывается название продукта, по которому возник долг;

·Дата — фактическая дата регистрации контакта;

·Тип — тип контакта (см. словарь № 3);

·Направление — номер телефона, по которому был установлен контакт;

·Оператор — оператор, зарегистрировавший контакт;

·Результат — результат контакта (см. Настройка дерева контактов);

·Обещ. дата — обещанная дата (отображается также и для неверифицированных обещаний);

·Комментарий — дополнительная информация о контакте;

·Флаг Поручитель — включен, если контакт был зарегистрирован из каротчки поручителя;

·Флаг Залогодатель — включен, если контакт был зарегистрирован из карточки залогодателя;

·Направление email — поле для контактов типа "Email". Может иметь значение "Исходящее" (для отправленных email) или "Входящее" (для полученных email).
