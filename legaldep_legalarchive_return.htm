Возврат дел из архива судебной работы

Возврат дел из архива судебной работы доступен только для тех пользователей, в роли которых доступ LAW_ACT_RETURN_FROM_ARCHIVE имеет значение TRUE.

Для того чтобы возвратить дела из архива в работу:

1.Войдите в модуль Юридический департамент, активируйте вкладку Архив судебной работы.

2.Создайте выборку при помощи мультифильтров или сложного фильтра. Выберите одну или несколько записей.

3.Вызовите контекстное меню нажатием правой кнопки мыши, выберите пункт Возврат из архива.

4.В появившемся окне предупреждения нажмите OK.

5.Выбранные дела будут возвращены в работу. После обновления данных на вкладке (кнопка "Найти") вы сможете убедиться в том, что возвращенные в работу дела в ней больше не отображаются.

6.Возвращенные приказы станут доступны из вкладки Приказы, возвращенные иски – из вкладки Иски, правопреемственные — из вкладки Правопреемство. Дела получат статус "Возобновление".

7.Действие будет запротоколировано. В протокол попадет запись:

·для приказа: с действием  "Приказное производство: дело возобновлено" (согласно словарю № 26) и с комментарием "Возобновление приказа";

·для правопреемства: с действием "Правопреемство: возобновление" и с комментарием (согласно словарю № 26) и с комментарием "Правопреемство: возобновление";

·для иска: с действием "Исковое производство: дело возобновлено" (согласно словарю № 26) и с комментарием "Возобновление иска".

8.В зависимости от настроек Системы возможно формирование настроенных запланированных действий для типа судебной работы Иск, Правопреемство или Приказ и типа действия Возобновить.
