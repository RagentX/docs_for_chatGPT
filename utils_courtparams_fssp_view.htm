Вкладка "ФССП и приставы"

Список ФССП и судебных приставов отображаются в форме Настройка судов и ФССП на вкладке ФССП и приставы. Вкладка содержит следующие элементы:

·список ФССП, содержащий следующие поля:

·ID — порядковый номер ФССП в системе;

·Название — название ФССП;

·Тип — тип исполнительного органа;

·Район, Телефон, Email, Веб-сайт, Адрес — данные о ФССП;

·Комментарий;

·Флаг Не используется — если для ФССП данный флаг включен, то она не отображается в списке при Выборе ФССП по делу в исполнительном производстве;

·флаг Показать все — включите флаг для отображения в списке также записей с признаком Не используется;

·кнопки для добавления, редактирования, удаления данных о ФССП;

·кнопка Загрузка ФССП для запуска утилиты загрузки ФССП из Excel-файла;

·Вложенная вкладка Приставы, отображающая список судебных приставов, относящихся к выбранной ФССП. Он содержит следующие поля:

·ID — порядковый номер пристава в системе;

·ФИО — фамилия, имя отчество судебного пристава;

·Категория, Телефон, Email, Веб-сайт, Кабинет — данные о приставе;

·Комментарий;

·Флаг Не используется — если для пристава данный флаг включен, то он не отображается в списке приставов при Внесение данных о судебном приставе по делу в исполнительном производстве;

·флаг Показать все — включите флаг для отображения в списке также записей с признаком Не используется;

·кнопки для добавления, редактирования, удаления данных о судебных приставах;

·Вложенная вкладка Настройки для автозакрепления — см. Привязка оператора к ФССП.
