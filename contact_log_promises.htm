Обещания оплаты долгов

Show/Hide Hidden Text

Список обещаний оплаты долгов, зарегистрированных в результате контактов с должниками, отображается на вкладке Обещания. При открытии формы список отображается пустым. Нажмите кнопку Найти, чтобы вывести в список все записи за установленный по умолчанию период, или предварительно настройте условия фильтра для отбора записей (см. Применение мультифильтров в списочных модулях, Сложный фильтр).

       Описание полей списка

Информация об обещаниях отображается в следующих полях списка:

·Дата — дата регистрации обещания;

·Оператор, взявший обещание;

·Обещ. дата — обещанная дата оплаты долга (отображается также и для неверифицированных обещаний);

·Обещ. сумма — обещанная сумма (отображается также и для неверифицированных обещаний); выводится сумма значений по выборке;

·Подтв. платежи — сумма подтвержденных платежей по долгу; выводится сумма значений по выборке;

·Неподтв. платежи — сумма неподтвержденных платежей по долгу; выводится сумма значений по выборке;

·Статус обещания — см. Статусы обещаний;

·Модуль регистрации — указывается одно из значений:

·Регистрация контакта — для контактов, зарегистрированных при помощи модуля "Регистрация контакта";

·Android — для контактов, зарегистрированных при помощи мобильной версии приложения.

В нижней части списка под столбцами Обещанная сумма и Неподтвержденный платеж отображается общая сумма по всем записям списка.

       Подтверждение обещаний, направленных на согласование руководителю

Право на подтверждение обещаний имеют пользователи, в роли которых доступ 18 DEBT_CARD_PROM_REG имеет значение TRUE.

Подтвердить можно только обещания, требующие верификации (поле Статус имеет значение "Не верифицировано"). Если

Для того чтобы подтвердить обещания, выполните следующие действия:

1.В списке обещаний на вкладке Обещания сформируйте выборку, используя фильтры.

2.Выберите одно или более обещаний в статусе "Не верифицировано".

3.Нажмите кнопку .

4.Если все выделенные обещания находятся в статусе "Не верифицировано", то будет показана форма подтверждения. Нажмите кнопку Yes, чтобы подтвердить обещания.

5.Если в списке выделенных обещаний хотя бы одно находится в статусе, отличном от "Не верифицировано", то будет показано сообщение об ошибке. Нажмите ОК, чтобы закрыть форму, а затем сформируйте список заново.

       Удаление обещаний

Для того чтобы удалить обещание, выполните следующие действия:

1.В списке обещаний на вкладке Обещания сформируйте выборку, используя фильтры.

2. Выберите одно или более обещаний.

3.Нажмите кнопку .

4.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление обещаний. Обещания будут удалены.
