Вкладка "Основная" исходящего документа

Вкладка Основная содержит основную информацию по документу.

Поля вкладки "Основная" исходящего документа

Вкладка Основная исходящего документа включает следующие поля:

·Исходящий номер — номер исходящего документа (доступность редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE; при включенном доступе № 674 дополнительно — доступом № 825 DOCFLOW_FIELD_BAN_EDIT) (docflow.document_outgoing_number).

·Составитель — пользователь системы сформировавший исходящий документ (docflow.r_former_user_id). При вводе данных с клавиатуры выполняется автоматический поиск совпадений.

·Тип документа — тип документа (см. словарь № 332 "Тип документа") (docflow.document_type).

·Способ доставки — docflow.delivery_method.

·Наименование документа — данное поле может быть скрыто по согласованию с технической поддержкой. Значение выводится из поля docflow.dict_value_1. Если у данного column_spec стоит признак is_not_use = 1, то требуется скрывать поле с формы.

·Дата регистрации — дата регистрации документа (доступность редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE; при включенном доступе № 674 дополнительно — доступом № 825 DOCFLOW_FIELD_BAN_EDIT) (docflow.reg_dt).

·Регистратор — пользователь, который создал документ (поле недоступно для редактирования) (docflow.r_reg_user_id).

·Получатели — см. Изменение получателей исходящего документа. Поля списка получателей:

·ФИО/Название;

·Комментарий ОД;

·Адрес — адрес отображается только при включенной константе DocflowCard.SenderAddress.Use. При этом для отображения выбирается адрес с типом, указанным в значении константы DocflowCard.SenderAddress.Typ. Если таких адресов несколько, то выбирается тот, у которого id наименьший.

·Содержание — описание документа (доступность для редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE) (docflow.dsc).

·Комментарий — docflow.comment.
