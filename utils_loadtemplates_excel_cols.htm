Настройка столбцов формата

Настройка столбцов формата заключается в выборе названий и разделителей данных для столбцов файлов MS Excel, из которых будут загружаться данные. Разделители используются в процессе преобразования данных для разделения исходных столбцов таблицы, ячейки в которых содержат два или более элементов данных, на соответствующее количество преобразованных столбцов

Для того чтобы настроить список столбцов в исходном файле MS Excel для выбранного формата загрузки данных, выполните следующие действия:

1.В форме Настройка шаблонов загрузки выберите нужный формат в списке.

2.Нажмите кнопку , относящуюся к списку столбцов формата.

3.В открывшейся форме заполните следующие поля:

·Название столбца — поле обязательно для заполнения;

·Разделитель первого типа — основной разделитель, который в процессе преобразования данных будет использоваться для разделения исходных столбцов таблицы, ячейки в которых содержат два или более элементов данных, на соответствующее количество преобразованных столбцов;

Пример. Столбец "Паспортные данные" может содержать данные в формате "<серия>, <номер>, <кем выдан>" (например, "45 97, 176081, выдан 09.02.1999 5ОМ З РУВД УВД ЦАО"). Разделителем в этом случае будет знак ",".

В результате преобразования столбец будет разбит на три столбца: "<серия>", "<номер>", "<кем выдан>" ("45 97", "176081", "выдан 09.02.1999 5ОМ З РУВД УВД ЦАО").

Примечание: для полных адресов разделитель "," автоматически используется по умолчанию.

·Разделитель второго типа — дополнительный разделитель, используется так же, как и основной разделитель;

Примечание. В качестве второго разделителя для полных адресов не следует использовать символ пробела, который может входить в состав названий некоторых городов и улиц (например, улица "Ивана Ярыгина"), что приводит к ошибкам при преобразовании адреса;

·Ширина колонки в пикселях — не используется.

4.Нажмите кнопку ОК. Столбец будет добавлен в список. Далее следует настроить связь столбца с полями таблиц БД.

5.Повторите шаги 2-4, чтобы добавить все столбцы формата. При этом существуют следующие ограничения:

·телефоны и адреса должника должны быть указаны до данных о поручителях;

·при задании ФИО первого поручителя последующие телефоны и адреса относятся к нему;

·при задании ФИО второго поручителя последующие телефоны и адреса относятся к нему и т. д.

Для того чтобы изменить выбранный столбец, нажмите кнопку , измените данные в открывшейся форме, как описано выше, и нажмите кнопку ОК для сохранения.

Для того чтобы удалить выбранный столбец из списка, нажмите кнопку и подтвердите удаление в диалоговом окне. Будет удален столбец и все его связи.

Для перемещения столбца в списке на одну позицию вниз или вверх, нажмите соответственно кнопку или .
