Геолокация (отслеживание местоположения пользователей мобильного устройства)

Внимание! Функционал геолокации включается по согласованию с вендором.

Функционал геолокации дает возможность отслеживания местоположения пользователей мобильного приложения. Для корректной работы функционала на устройстве должно быть включено определение Геолокации. Чтобы прекратить отслеживание, достаточно отключить геолокацию на устройстве.

Если функционал включен, при входе в Систему появляется информационное сообщение (Рисунок 22‑1). Для продолжения работы нажмите OK.

Далее во время работы в Системе осуществляется автоматическое отслеживание местоположения пользователей мобильного устройства. Параметры процесса задаются указанными константами в основной (десктопной) версии Системы.

Система получает данные о положении устройства. Запись данных о времени и координатах мобильного устройства в соответствующую таблицу происходит с интервалом времени, заданным константой Android.GeoTracking.DeviceSensorPollingPeriod или при превышении расстояния, указанного в значении константы Android.GeoTracking.FlushDistanceThreshold. Передача новых строк таблицы из устройства на основную БД осуществляется с интервалом времени, заданным константой Android.GeoTracking.FlushTimeThreshold.
