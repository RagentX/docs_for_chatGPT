Хранение адреса ФССП как отдельной сущности БД

Для включения функционала хранения адреса ФССП как отдельной сущности БД обратитесь в техническую поддержку.

Функционал включается с помощью технической константы LawСourt.AddressAsObject.

Таблица obj_address_link, которая  является развязочной таблицей между адресами и сущностями, которой они принадлежат:

Функционал хранения адреса ФССП как отдельной сущности БД позволяет работать с адресами ФССП так же, как с адресами персон в Системе. Кроме того, в этом случае доступна регистрация контакта типа "Выезд в ФССП".

Форма "Информация о ФССП"

Если включен функционал хранения адреса ФССП как отдельной сущности БД, то форма Информация о ФССП имеет вид:

Описание формы

Вид формы отличается только областью адреса.

Таблица со списком адресов, расположенная в разделе Адреса, включает следующие поля:

·ID адреса (address.id) — по умолчанию колонка скрыта;

·Адрес (address.full_adr) — полный адрес в текстовом виде;

·Комментарий (address.dsc) — комментарий к адресу;

·Блокирован (address.block_flag) — признак блокировки в виде флага;

·Статус (address.status) — статус адреса по словарю № 63 "Статус адреса должника";

·Дата блокировки (address.block_dt) — дата последней блокировки адреса;

·Причина блокировки (address.block_reason) — по словарю № 314 "Причина блокировки адреса".

Действия на форме "Информация о ФССП"

На форме Информация о ФССП доступны следующие действия:

Добавить адрес

Данная операция доступна пользователям, в роли которых включен доступ LAW_JUDGE_SET_ADDR_ADD.

Добавление адреса ФССП доступно, если у данной ФССП отсутствуют активные (незаблокированные) адреса. Если активный адрес есть, добавление адреса не доступно, и при попытке добавить адрес будет показано соответствующее предупреждение.

Для того чтобы добавить адрес ФССП выполните следующее:

1.В форме Информация о ФССП нажмите кнопку Добавить адрес.

2.Откроется форма добавления адреса. Поля формы совпадают с формой добавления адреса должника, за исключением ряда полей. Заполните поля формы и нажмите OK.

3.Адрес будет добавлен в список.

Изменить адрес

Данная операция доступна пользователям, в роли которых включен доступ LAW_JUDGE_SET_ADDR_EDIT.

Для того чтобы изменить атрибуты адреса выполните следующее:

1.В форме Информация о ФССП выберите адрес и нажмите кнопку Изменить адрес.

2.Откроется форма, аналогичная форме добавления адреса ФССП, но с заполненными полями. Измените нужные значения полей и нажмите OK.

3.Измененный адрес будет сохранен.

Заблокировать адрес

Данная операция доступна пользователям, в роли которых включен доступ LAW_JUDGE_SET_ADDR_BLOCK.

Заблокировать можно только активный адрес ФССП.

Для того чтобы заблокировать адрес выполните следующее:

1.В форме Информация о ФССП в разделе Адреса выберите активный адрес и активируйте контекстное меню правой кнопкой мыши.

2.В открывшемся списке выберите пункт Заблокировать адрес.

3.Откроется окно подтверждения. Нажмите Yes.

4.Откроется форма выбора причины блокировки адреса. Выберите причину из списка.

5.Нажмите OK. Адрес будет заблокирован.

Разблокировать адрес

Данная операция доступна пользователям, в роли которых включен доступ LAW_JUDGE_SET_ADDR_ UNBLOCK.

Разблокировать заблокированный адрес можно только в том случае, если у данной ФССП отсутствует активный адрес.

Для того чтобы разблокировать адрес выполните следующее:

1.В форме Информация о ФССП в разделе Адреса выберите заблокированный адрес и активируйте контекстное меню правой кнопкой мыши.

2.В открывшемся списке выберите пункт Разблокировать адрес.

3.Адрес будет разблокирован.
