Разговор

Show/Hide Hidden Text

Сценарии телефонных разговоров используются в качестве подсказок оператору при регистрации контакта с должником в модуле "Регистрация контакта". При необходимости можно настроить несколько сценариев, которые будут использоваться в зависимости от ситуации и характеристик должника. Если номер сценария будет указан в поле Номер для автосценария какого-либо узла дерева контактов (см. Добавление результата контактов), то при регистрации контакта при выборе данного узла оператору отобразится именно этот сценарий. Если значение для узла не задано, то по умолчанию в качестве ID сценария берется значение, указанное в константе FormulaAutoMessage.

Вкладка "Разговор"

       Описание вкладки "Разговор"

·список сценариев разговоров, содержащий следующие поля:

·ID — идентификатор сценария, формируется автоматически;

·Номер — номер сценария;

·Название — название сценария;

·Текст — текст сценария разговора;

·кнопки для добавления, редактирования, удаления сценариев.

Форма добавления/редактирования сценария разговора

       Описание формы добавления/редактирования сценария разговора

·Название — название сценария. Поле обязательно для заполнения;

·Номер — номер сценария, который может использоваться для привязки сценария к узлу дерева контактов (см. Добавление результата контактов);

·Текст — текст сценария разговора, который будет видеть оператор. Поле может содержать как статичный текст, так и переменные, вместо которых при выполнении сценария будут подставлены конкретные значения (см. Подстановка переменных в текст сообщения).

В нижней части формы расположены кнопки управления внешним представлением сценария разговора. Например, для того чтобы изменить цвет определенного фрагмента текста, выделите его в поле Текст, выберите цвет в раскрывающемся списке, а затем нажмите кнопку . Так же можно изменить тип шрифта (жирный — кнопка , курсив — кнопка , подчеркнутый — кнопка ).

       Создание сценария разговора

1.В форме Сценарии сообщений для должников на вкладке Разговор нажмите кнопку .

2.В открывшейся форме добавления сценария разговора заполните нужные поля настройте внешнее представление сценария.

3.Нажмите кнопку ОК. Сценарий будет добавлен в список.

       Редактирование сценария разговора

1. В форме Сценарии сообщений для должников на вкладке Разговоры выберите нужную запись и нажмите кнопку или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2. В форме редактирования атрибутов сценария измените необходимые атрибуты.

3. Нажмите кнопку ОК, чтобы сохранить изменения.

       Удаление сценария разговора

1.В форме Сценарии сообщений для должников на вкладке Разговор выберите нужную запись.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.
