Открытие исходного файла и формирование таблицы данных

Для того чтобы открыть файл MS Excel с данными для загрузки, выполните следующие действия:

1.В форме Утилита определения ID долга нажмите кнопку Открыть файл.

2.В открывшемся стандартном диалоговом окне выберите файл с данными и нажмите кнопку Open (Открыть). В процессе открытия файла в нижней строке окна формы будет отображаться тип операции (открытие файла), информация о проценте обработанных записей, директория и имя файла.

По окончании открытия файла данные будут выведены в таблицу формы. Нумерация строк таблицы и установка ширины каждого столбца с учетом объема данных также производится автоматически. Данные в каждой ячейке таблицы идентичны данным в соответствующего элемента в файле MS Excel. При этом автоматически удаляются левые и правые пробелы, в ячейках столбца ФИО должника лишние прописные буквы автоматически заменяются строчными. Доступна возможность корректировки данных в любой ячейке таблицы и ручное изменение ширины любого столбца.

Если количество заполненных столбцов в открытом файле MS Excel отличается от шести, то будет выдано сообщение: "Файл <директория и имя файла> не соответствует требуемому формату: Банк, Префикс портфеля, ФИО должника, Номер договора, Номер счета, ID в системе клиента". В этом случае кнопка Открыть файл останется доступной, а кнопки Обработать и Отменить — недоступны.

Если в исходном файле имеется ошибочная ячейка, не позволяющая открыть файл, то будет выдано сообщение: "Строка №<номер строки> столбец №<номер столбца>, ошибка при открытии ячейки: <наименование ошибки>". В этом случае кнопка Открыть файл останется доступной, а кнопки Обработать и Отменить — недоступны.

Если файл открыт успешно, станут доступны кнопки Обработать и Отменить, и можно переходить к поиску долгов.
