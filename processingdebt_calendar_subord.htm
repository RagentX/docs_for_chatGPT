Просмотр календаря подчиненных

Если в роли пользователя есть доступ RING_UP_SUBORD, установленный в значение TRUE, то он может просматривать действия, которые необходимо совершить всем его подчиненным. Для этого выполните следующие действия:

1.На вкладке Календарь нажмите кнопку . Откроется форма для выбора операторов. В левой части формы находится список еще не выбранных значений, в правой — список выбранных значений.

Примечание: в левом списке значений доступны функции отбора и сортировки записей. При необходимости их можно использовать, чтобы найти нужные записи.

2.Перенесите записи о нужных значениях из правого списка в левый. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL. Для переноса записей между списками используйте следующие кнопки:

· — переместить выбранные записи из левого списка в правый;

· — переместить все записи из левого списка в правый;

· — переместить выбранные записи из правого списка в левый;

· — переместить все записи из правого списка в левый.

Перенести одну выбранную запись между списками можно с помощью двойного щелчка левой клавишей мыши.

3.Нажмите кнопку ОК. В календаре будут отображаться действия всех выбранных операторов.

Начальник видит календарь подчиненного
