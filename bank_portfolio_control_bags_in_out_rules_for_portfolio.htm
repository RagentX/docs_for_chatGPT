Правило расчета остатков по долгу

В блоке Портфели при пункта контекстного меню Правило расчета остатков по долгу вы можете настроить применение разных правил расчетов остатков для долгов, входящих в выбранный портфель. Также вы можете снять ранее настроенное правило или запретить применение правил расчета остатков к выбранным долгам.

Данное действие доступно как для входящих, так и для исходящих портфелей.

Порядок действий

Для того, чтобы назначить некоторым долгам, входящим в портфель, правило расчета остатков, выполните следующее:

1.В форме Клиенты и портфели в блоке Контрагенты выберите запись о контрагенте.

2.В блоке Портфели выберите портфель.

3.Активируйте контекстное меню правой кнопкой мыши и выберите пункт Правило расчета остатков по долгу.

4.Откроется форма Настройка правила расчета остатков по долгам. В форме отображается список незакрытых долгов (в статусах кроме (6,7,8)), принадлежащих этому портфелю. Для поиска нужных долгов воспользуйтесь фильтрацией и/или сложным фильтром.

5.Выберите в списке один или более долгов.

6.Нажмите кнопку Выбор правила расчета. Откроется форма Выбор правила расчета.

Примечание. Если для всех выбранных долгов уже настроено одно и то же правило, то в поле Правило отображается это правило.

7.В поле Правила расчета доступны все настроенные правила расчета. Выберите нужный вариант действия:

·Правило — к выбранным долгам вместо правила, установленного всему портфелю, будет применяться данное правило.

·Вариант "не заполнено" — к выбранным долгам будет применяться правило, установленное для портфеля.

·Вариант "без списания" — к выбранным долгам не будет применяться никаких правил расчета остатков.

8.Нажмите ОК.

Выбранным долгам будет назначено указанное правило расчета остатков или выбранный вариант.
