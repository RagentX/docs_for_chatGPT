Дополнительные атрибуты банка

Настройка дополнительных атрибутов для банка доступна по кнопке , расположенной на форме  редактирования записи о контрагенте Клиент.

Примечание. Для создания новых дополнительных атрибутов банка обратитесь к сопровождению. Если дополнительные атрибуты банка в системе сопровождением не созданы, то кнопка на форме Клиент будет отсутствовать.

Информация для поддержки. Как настроить атрибут для клиентов

В настоящей версии Системы поддержаны дополнительные атрибуты следующих объектов:

·Пользователь — в модуле Пользователи;

·Контрагент — в модуле Клиенты и портфели;

·Портфель — в модуле Клиенты и портфели;

·Адрес.

 Важно! Если в словаре № 278 нет ни одного термина, имеющего в качестве родительского соответствующий термин словаря № 117, то кнопка настройки дополнительных атрибутов не видна.

1.В словарь № 278  "Произвольный атрибут объекта системы (obj_attr_typ)" добавьте название атрибута. При этом в поле "Термин родительского словаря" укажите:

·для пользователей — "Пользователи системы" (термин № 36 словаря № 117);

·для банков — "Информация о банке" (термин № 68 словаря № 117);

·для портфелей — "Данные о портфелях" (термин № 49 словаря № 117);

·для адресов — "Адреса (термин № 2 словаря № 117)".

2.В таблицу obj_attr_typ добавьте запись об атрибуте. Поля таблицы:

·OBJ_ID — код термина родительского словаря № 117 "Объект системы (table_spec)", который относится к данному атрибуту;

·Code — код термина, соответствующего атрибуту, в словаре № 278;

·Name — название атрибута (текст термина в словаре № 278);

·TYP — тип атрибута (код словаря № 8);

·DEFAULT_VALUE — значение атрибута по умолчанию;

·R_DIC_ID — для атрибутов типа 6 "Словарь" укажите номер словаря.

Форма "Список атрибутов объекта"

Пример дополнительных атрибутов

Описание формы

Форма "Список атрибутов объекта" включает следующие элементы:

·Таблицу со списком дополнительных атрибутов:

·ID — идентификатор атрибута;

·Значение — значение атрибута;

·Атрибут — название атрибута;

·Оператор — ФИО оператора, добавившего данный атрибут;

·Дата изменения — дата и время последнего изменения атрибута;

·Комментарий — дополнительная информация;

·Кнопки для работы с записями списка:

· — Добавить;

· — Редактировать;

· — Удалить.

Настройка дополнительных атрибутов банка

Добавление дополнительного атрибута банка

Для того чтобы настроить дополнительные атрибуты банка, выполните следующие действия:

1.Выберите контрагента.

2.Нажмите кнопку Изменить или выполните двойной клик левой кнопкой мыши на выбранной записи.

3.Откроется форма Клиент. Нажмите кнопку Дополнительные атрибуты.

4.Откроется форма Список атрибутов объекта.

5.Нажмите кнопку Добавить атрибут. Откроется форма Атрибут. Заполните следующие поля:

·Атрибут — выберите из списка (поле обязательно для заполнения);

·Значение — выберите из списка (поле обязательно для заполнения);

·Комментарий — при необходимости введите дополнительную информацию.

6.Нажмите OK.

Редактирование дополнительных атрибутов банка

Для того чтобы изменить дополнительный атрибут банка, выполните следующие действия:

1.Выберите контрагента.

2.Нажмите кнопку Изменить или выполните двойной клик левой кнопкой мыши на выбранной записи.

3.Откроется форма Клиент. Нажмите кнопку Дополнительные атрибуты.

4.Откроется форма Список атрибутов объекта.

5.Выберите нужный атрибут. Нажмите кнопку Изменить атрибут.

6.Откроется форма Атрибут. Измените значения нужных полей и нажмите ОК.

Удаление дополнительного атрибута банка

Для того чтобы удалить дополнительный атрибут банка, выполните следующие действия:

1.Выберите банка.

2.Нажмите кнопку Изменить или выполните двойной клик левой кнопкой мыши на выбранной записи.

3.Откроется форма Клиент. Нажмите кнопку Дополнительные атрибуты.

4.Откроется форма Список атрибутов объекта.

5.Выберите нужный атрибут. Нажмите кнопку Удалить атрибут.

6.В окне подтверждения нажмите ОК.
