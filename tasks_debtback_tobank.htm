Подготовка долгов к передаче клиенту

Операция перевода долгов в статус 10 "К возврату Клиенту" доступна на вкладках Все должники, Проблемные, Поиск  информации. Перевод может быть выполнен:

·для всех долгов из выборки, сформированной с учетом фильтров;

·для всех долгов, выделенных в текущей выборке с помощью клавиш SHIFT и/или CTRL.

Для того чтобы перевести долги в статус 10 "К возврату Клиенту", выполните следующие действия:

1.В форме Постановка задач перейдите на вкладку Все должники, Проблемные или Поиск информации.

2.При необходимости сформируйте выборку долгов с помощью фильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Вернуть клиенту и в раскрывающемся списке выберите вариант выполнения операции: По всей выборке или Только выделенные.

Примечание: если константа WorkTaskAllMode имеет значение "Ложь", то возможность вернуть долги клиенту по всей выборке недоступна.

4.В открывшемся диалоговом окне подтвердите выполнение операции, нажав кнопку Yes. Откроется форма для указания причины возврата долгов клиенту и комментария.

5.В форме в поле Причина укажите причину возврата долгов клиенту, в поле Комментарий введите поясняющую информацию. Оба поля являются обязательными для заполнения. Перечень возможных причин возврата долгов клиенту формируется в словаре № 27.

Примечание: если поясняющая информация отсутствует, можно установить флаг Выполнить без ввода комментария.

6.Нажмите кнопку ОК.

Долги будут переведены в статус 10 "К возврату Клиенту". Информация о них будет доступна на вкладке К передаче Клиенту. При этом долги будут откреплены от оператора, и операторы получат уведомления о снятии долгов.

Есть возможность настроить Систему так, чтобы при переводе долга в статусы (любая комбинация указанных статусов):

·6 — "Завершено";

·7 — "Аннулировано";

·8 — "Отозвано клиентом";

·10 — "К возврату клиенту"

автоматического освобождения оператора не происходило. Для этого обратитесь в техническую поддержку.

Для того чтобы при переходе долга в один из этих статусов долг не откреплялся от оператора, исключите код этого статуса из значения технической константы WorkTask.ClearOperator.DebtStatusList.
