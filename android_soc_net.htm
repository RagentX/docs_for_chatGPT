Социальные сети

Внимание! Функционал социальных сетей включается по договоренности с вендором. Для этого включите константу Android.SocialNetwork.Use.

Доступы для работы с социальными сетями

Работать с социальными сетями может пользователь, в роли которого имеют значение TRUE следующие доступы:

·608 SOCIAL_NETWORK_ADD — Социальные сети. Право на добавление информации о социальной сети;

·609 SOCIAL_NETWORK_EDIT — Социальные сети. Право на изменение информации о социальной сети.

·610 SOCIAL_NETWORK_DELETE — Социальные сети. Право на удаление информации о социальной сети.

Действия с социальными сетями

Добавить социальную сеть

Добавлять записи о социальной сети могут пользователи, в роли которых доступ № 608 SOCIAL_NETWORK_ADD имеет значение TRUE.

Для того чтобы добавить ссылку на аккаунт должника в социальной сети, выполните следующие действия:

1.В карточке должника нажмите кнопку "Действия".

2.В открывшемся меню выберите пункт "Атрибуты".

3.В списке атрибутов коснитесь заголовка "Социальные сети". Раздел развернется.

4.Нажмите кнопку .

5.В открывшейся форме "Добавить социальную сеть" (Рисунок 20‑1) заполните следующие поля:

·Тип — выберите тип социальной сети из списка, обязательное поле;

·URL — введите ссылку на аккаунт должника, обязательное поле;

·Комментарий — дополнительная информация, обязательное поле.

6.Нажмите ОК.

Изменить запись о социальной сети

Изменять записи о социальной сети могут пользователи, в роли которых доступ № 609 SOCIAL_NETWORK_EDIT имеет значение TRUE.

Для того чтобы изменить запись об аккаунте должника:

7.В карточке должника нажмите кнопку "Действия".

8.В открывшемся меню выберите пункт "Атрибуты".

9.В списке атрибутов коснитесь заголовка "Социальные сети". Раздел развернется.

10.В списке аккаунтов коснитесь той строки, которую вы хотите изменить.

11.Откроется форма "Добавить социальную сеть" (Рисунок 20‑1).

12.Измените нужные поля и нажмите ОК.

Открыть ссылку

Для того чтобы открыть ссылку на аккаунт должника:

13.В карточке должника нажмите кнопку "Действия".

14.В открывшемся меню выберите пункт "Атрибуты".

15.В списке атрибутов коснитесь заголовка "Социальные сети". Раздел развернется.

16.Выберите запись и нажмите справа от нее.

17.В открывшемся меню (Рисунок 20‑2) выберите пункт "+ Открыть ссылку".

Удалить запись

Удалять записи о социальной сети могут пользователи, в роли которых доступ № 610 SOCIAL_NETWORK_DELETE имеет значение TRUE.

Для того чтобы удалить запись об аккаунте должника:

18.В карточке должника нажмите кнопку "Действия".

19.В открывшемся меню выберите пункт "Атрибуты".

20.В списке атрибутов коснитесь заголовка "Социальные сети". Раздел развернется.

21.Выберите запись и нажмите справа от нее.

22.В открывшемся меню (Рисунок 20‑2) выберите пункт "- Удалить".

Запись будет удалена без предупреждения.
