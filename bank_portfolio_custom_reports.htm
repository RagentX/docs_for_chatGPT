Формирование произвольных отчетов по портфелю

Для формирования произвольных отчетов по портфелю на форме Клиенты и портфели расположена кнопка Отчеты , дающая доступ к заранее определенным произвольным отчетам.

Кнопка доступна для пользователей, в роли которых доступ № 574 BANK_CUSTOM_REPORTS имеет значение "Истина" только в том случае, если в Системе настроен хотя бы один произвольный отчет по портфелю.

Максимально возможное количество отчетов — 10.

Примечание. Если доступ № 574 включен, но кнопка на форме отсутствует, значит в Системе не настроены отчеты по портфелю.

Для настройки произвольных отчетов по портфелю зарегистрируйте заявку в техническую поддержку.

 Для того чтобы настроить отчеты по портфелю:

1.Убедитесь, что доступ № 574 для соответствующих пользователей включен.

2.Создайте нужные отчеты в модуле "Произвольные отчеты".

3.Идентификаторы отчетов присвойте техническим константам:

·BankPortfolio.RequestReport1;

·BankPortfolio.RequestReport2;

·BankPortfolio.RequestReport3;

·BankPortfolio.RequestReport4;

·BankPortfolio.RequestReport5;

·BankPortfolio.RequestReport6;

·BankPortfolio.RequestReport7;

·BankPortfolio.RequestReport8;

·BankPortfolio.RequestReport9;

·BankPortfolio.RequestReport10.

 Примечание. Если значение константы равно 0, то она игнорируется.

4.Названия отчетов укажите в таблице strings:

·name — "BankPortfolio.RequestReportX",где X соответствующий номер константы;

·text — название отчета.
