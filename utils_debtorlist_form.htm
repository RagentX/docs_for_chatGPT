Форма "Настройка списка должников"

Форма Настройка списка должников предназначена для настройки ограничений доступа к долгам как для операторов, так и для ролей. Данные ограничения используются в многих модулях Системы — см. Применение ограничений доступа к долгам.

Форма Настройка списка должников включает следующие вкладки:

·Ограничения по оператору — для настройки ограничений любому оператору:

·Вкладка включает список всех операторов и заданных для них ограничений доступа к долгам.

·Первая колонка "Ограничения по оператору" — ФИО (фамилия, имя, отчество оператора).

·Ограничения по роли — для настройки ограничений любой роли.

·Вкладка включает список всех ролей и заданных для них ограничений доступа к долгам.

·Первая колонка вкладки "Ограничения по роли" — Роль.

Остальные колонки на вкладках идентичны:

·Блок — признак того, что оператор или все операторы, имеющие указанную роль заблокирован и не имеет доступа к долгам (соответствует полю Пустой список в форме настройки ограничений).

·Статус — ограничение по статусу долгов, с которыми может работать оператор или все операторы, имеющие указанную роль.

·Кредитор — ограничение по клиентам (банкам) (см. модуль "Клиенты и портфели"), с долгами которых может работать оператор или все операторы, имеющие указанную роль.

·Портфели — ограничение по портфелям (см. модуль "Клиенты и портфели"), с долгами которых может работать оператор или все операторы, имеющие указанную роль. Данное ограничение доступно, если константа OperatorScope.Portfolio.Use имеет значение "Истина".

·Регион — ограничение по региону, в котором выдан кредит (см. словарь № 39) для оператора или всех операторов, имеющих указанную роль.

·Стадия — ограничение по стадии должника (Soft, Hard, Legal) для оператора или всех операторов, имеющих указанную роль.

·Важность — ограничение по уровню важности должников, с долгами которых может работать оператор или все операторы, имеющие указанную роль.

·Порядочность — ограничение по уровню порядочности должников, с долгами которых может работать оператор или все операторы, имеющие указанную роль.

·Сумма долга — ограничение по сумме долгов, с которыми может работать оператор или все операторы, имеющие указанную роль.

·Кол-во дней проср. — ограничение по количеству дней просрочки долга для оператора или всех операторов, имеющих указанную роль.

·Филиал — ограничение по филиалу, в котором выдан кредит (см. словарь № 61) для оператора или всех операторов, имеющих указанную роль.

·Балл № 1 — ограничение по значению Балла № 1 долга (см. словарь № 101, Скоринговые баллы) для оператора или всех операторов, имеющих указанную роль.

·Балл № 2 — ограничение по значению Балла № 2 долга (см. словарь № 102, Скоринговые баллы) для оператора или всех операторов, имеющих указанную роль.

·Балл № 3 — ограничение по значению Балла № 3 долга (см. словарь № 244, Скоринговые баллы) для оператора или всех операторов, имеющих указанную роль.

·Балл № 4 — ограничение по значению Балла № 4 долга (см. словарь № 245, Скоринговые баллы) для оператора или всех операторов, имеющих указанную роль.

·КА — ограничение по списку банков, с исходящими портфелями которых может работать оператор или все операторы, имеющие указанную роль.

Пустое поле ограничения в записи означает, что по данному критерию ограничений нет.
