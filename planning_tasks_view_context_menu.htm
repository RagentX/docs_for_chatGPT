Контекстное меню информационной вкладки "Задачи"

Контекстное меню открывается правой кнопкой мыши.

В контекстном меню информационной вкладки "Задачи" доступны следующие кнопки:

·Массовая смена приоритета;

·Массовая смена статуса;

·Открыть карточку;

·Копировать строки — осуществляется копирование выделенных строк в буфер обмена;

·Копировать ячейку — осуществляется копирование текста выделенной ячейки в буфер обмена.

Массовая смена приоритета

Изменяется приоритет задачи на указанный.

Для изменения приоритета выполните следующие действия:

1.В списке  задач выберите одну или более задач, при необходимости воспользовавшись клавишами SHIFT и/или CTRL.

2.Активируйте контекстное меню, нажав на правую кнопку мыши.

3.Выберите пункт Смена приоритета.

4.В открывшемся списке выберите:

·Все — для выбора всех задач в списке;

·Только выделенные — для выбора только выделенных задач.

5.В открывшейся форме Изменение приоритета заполните следующие поля:

·Приоритет — выберите значение из списка (см. словарь № 235 Приоритеты задач). Поле обязательно для заполнения.

6.Нажмите Изменить. Приоритет выбранных задач изменится на указанное значение.

Массовая смена статуса

Изменяется статус задачи на указанный. См. Статусы и жизненный цикл задачи.

Для того чтобы изменить статус, выполните следующие действия:

1.В списке  задач выберите одну или более задач, при необходимости воспользовавшись клавишами SHIFT и/или CTRL.

2.Активируйте контекстное меню, нажав на правую кнопку мыши.

3.Выберите пункт Смена статуса.

4.В открывшемся списке выберите:

·Все — для выбора всех задач в списке;

·Только выделенные — для выбора только выделенных задач.

5.В открывшейся форме Изменение статуса заполните следующие поля:

·Статус — выберите значение из списка (см. словарь № 275 Статус задачи (Новое планирование)). Поле обязательно для заполнения;

·Причина — причина смены статуса. Поле доступно только при выборе статусов, подразумевающих причину смены. Для каждого статуса причина смены выбирается из соответствующего словаря. Поле не обязательно для заполнения.

6.Нажмите Изменить. Статус выбранных задач изменится на указанное значение для тех из выбранных задач, для которых такое изменение возможно — см. Статусы и жизненный цикл задачи. Будет показано информационное окно с результатом выполнения операции.

Открыть карточку

Открывается карточка объекта задачи.

В зависимости от стадии взыскания:

·Досудебная стадия — открывается карточка должника;

·Судебная стадия — открывается карточка судебной работы;

·Исполнительная стадия — открывается стадия исполнительного производства.

Открыть карточку должника в новом окне

Данный пункт доступен только для задач досудебной стадии и только если используется функционал открытия карточки должника в режиме агента.

Открывается карточка должника в новом окне.
