Вкладка "Автогруппы"

Вкладка Автогруппы для работы с автогруппами долгов и содержит следующие элементы:

·список автогрупп долгов, содержащий следующие поля:

·ID группы — идентификатор группы долгов;

·Название группы — все автоматически созданные группы имеют название $auto_group_№;

·Очищать предыдущие — признак того, что при формировании автогруппы все долги будут удалены из группы, затем автогруппа будет сформирована заново в соответствии с условиями автофильтра;

·Владелец — оператор, создавший автогруппу;

·Дата формирования — дата и время последнего формирования списка долгов автогруппы;

·SQL — текст SQL-запроса для формирования списка долгов, входящих в автогруппу (размер поля 15000 символов);

·Комментарий — дополнительная информация;

·Объект группы — объекты, которые включаются в группу; Настройка списка объектов групп: в технической константе GroupDebtObjectForGroup перечислить идентификаторы соответствующих терминов словаря № 117.

·Разрешено добавление вручную — признак, разрешается ли ручное добавление объектов в группу

·кнопки для создания, редактирования, удаления автогрупп и привязки автогрупп к автофильтрам;

·кнопка История изменений — для просмотра истории изменений выбранной группы;

·кнопка ГСП, для перехода к конструкторам запросов, структур и настройке доступа пользователей к вкладкам модуля Группы свободного поиска;

·Область Автофильтры содержит следующие элементы:

·Вкладка Связи автофильтров — список автофильтров, с которыми связана выбранная автогруппа.

·Вкладка Все автофильтры — все автофильтры, созданные в системе.

·Кнопки для добавления, редактирования и удаления автофильтра, обеспечивающие работу с автофильтрами.

·Флаг "Группы текущего пользователя" — если он выключен, в списке групп отображаются все группы; если он включен, то в списке групп отображаются только те группы, владельцем которых является текущий пользователь;

·кнопка История изменений:

oна вкладке Связи автофильтров — для просмотра истории изменений связи автофильтра с группой;

oна вкладке Все автофильтры — для просмотра истории изменений автофильтра;

·Кнопка Источники для настройки источников данных.

На вкладке Автогруппы доступны следующие действия:

·Создание автогруппы;

·Редактирование автогруппы;

·Превращение автогруппы в простую группу;

·Удаление группы;

·Связывание групп с автофильтрами;

·Формирование группы из автофильтров;

·Просмотр истории изменений выбранной группы.
