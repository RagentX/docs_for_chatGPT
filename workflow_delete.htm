Удаление документов

Для удаления документов используется кнопка Удалить документы. Кнопка присутствует на всех вкладках модуля. Доступность кнопки определяется доступом № 670 DOCFLOW_DELETE.

Порядок действий при удалении документов

Для того чтобы удалить документы, выполните следующие действия:

1.В модуле Документооборот активируйте любую вкладку.

2.Сформируйте выборку, используя при необходимости фильтр по дате и сложный фильтр.

3.Выберите одну или более записей. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL.

4.Нажмите кнопку Удалить документы.

5.Откроется окно подтверждения с текстом:

Вы действительно хотите удалить выделенные документы?

Нажмите Yes.

6.Выделенные документы будут удалены. По окончании действия будет показано уведомление о его результатах с текстом:

Документы удалены.

Нажмите ОК.
