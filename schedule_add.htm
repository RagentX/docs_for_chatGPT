Создание, редактирование и удаление графика рабочего времени

Создание графика рабочего времени

Для того чтобы создать новый график рабочего времени, выполните следующие действия:

1.В форме Графики рабочего времени нажмите кнопку . Откроется форма редактирования нового графика рабочего времени. Состав полей и управляющих элементов формы различается в зависимости от типа графика (поле Тип графика) (см. Понедельный график работы и Периодический график работы).

Примечание. Первоначальные данные для нового графика подготовлены по умолчанию. Для первых пяти дней недели или периода (см. поле Тип графика) количество интервалов по умолчанию равно двум, для остальных дней — нулю. Значения интервалов для каждого из первых пяти дней по умолчанию равны "09:00 — 13:00" и "14:00 — 18:00". Все интервалы — основные (полное рабочее время совпадает с основным). Контроль прихода на работу, отработки за день, отработки за неделю (период) по умолчанию предусмотрен.

2.В форме заполните атрибуты графика рабочего времени.
О заполнении формы для понедельного графика см. Понедельный график работы.
О заполнении формы для периодического графика см. Периодический график работы.

3.Нажмите кнопку ОК, чтобы сохранить график рабочего времени.

Редактирование графика рабочего времени

Для того чтобы отредактировать график рабочего времени, выполните следующие действия:

1.В форме Графики рабочего времени выберите нужный график в раскрывающемся списке и нажмите кнопку . В зависимости от типа графика (понедельный/периодический) откроется форма редактирования понедельного графика рабочего времени или форма редактирования периодического графика рабочего времени.

2.В открывшейся форме измените необходимые атрибуты (см. Понедельный график работы или Периодический график работы).

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление графика рабочего времени

Удалить график рабочего времени можно только в том случае, если он не назначен ни одному оператору. Для того чтобы удалить график рабочего времени, выполните следующие действия:

1.В форме Графики рабочего времени выберите нужный график.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите Yes, чтобы подтвердить удаление графика рабочего времени.

Если график рабочего времени назначен каким-либо операторам, будет выдано сообщение о невозможности выполнения операции. Нажмите кнопку ОК, чтобы закрыть его.
