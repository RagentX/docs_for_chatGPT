Форма "Список должников"

Форма Список должников содержит следующие элементы:

·вкладки, на которых отображаются списки должников и их долгов, сгруппированных по различным признакам (см. Описание вкладок модуля);

·поля мультифильтров и кнопка Найти для применения условий мультифильтров;

·поле, в котором отображается информация о датах, способах и результатах контактов с должником, информация о переводе долга на аутсорсинг и возврата из аутсорсинга (отображается, если константа DebtListContactHistory имеет значение "Истина", при этом в списке не отображаются контакты тех пользователей, чьи идентификаторы перечислены в значении константы ContactHistory.ContactLog.IgnoreUsersList);

·кнопки добавления, редактирования, аннулирования карточки должника;

·кнопка для регистрации контакта с выбранным должником;

·кнопка для регистрации входящего звонка;

·кнопка для выгрузки данных в формат MS Excel;

·кнопка для вывода данных на печать;

·кнопка для вызова утилит загрузки данных;

·кнопка для выгрузки и печати документов и писем по долгу;

·индикатор, показывающий количество выбранных записей и общее количество записей в списке.

Отдельные области формы можно скрыть/отобразить с помощью специальных элементов управления.

Записи о долгах могут быть окрашены, цвет записи о долге задается в модуле Карточка должника.

Примечание. В списке отображаются только те долги, к которым у текущего пользователя настроен доступ в модуле Настройка списка должников.

Для работы со списком должников также используется контекстное меню, которое вызывается щелчком правой клавиши мыши в области списка.
