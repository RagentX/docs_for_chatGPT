Просмотр данных о принятых мерах

Данные о принятых мерах в рамках исполнительного производства доступны на вкладке Принятые меры.

На вкладке доступны:

·мультифильтры по портфелю и ответственному оператору (см. Применение мультифильтров в списочных модулях), а также

·возможность выбора интервала времени, за который нужно просмотреть информацию о принятых мерах. Для выбора интервала укажите дату и время в полях С и По. По умолчанию начало интервала — это первый день текущего месяца (время 00:00:00), конец интервала — последний день текущего месяца (время 23:59:59). Для применения фильтра нажмите кнопку Найти.

При осуществлении двойного клика на одной из найденных записей открывается соответствующая карточка исполнительного производства.

Действия с принятыми мерами доступны в Карточке исполнительного производства на вкладке Принятые меры — см. Принятые меры.

Вкладка Принятые меры модуля Юридический департамент содержит следующие поля:

·№ дела — уникальный идентификатор (порядковый номер) дела в системе;

·ID Должника — идентификатор должника, связанного с делом;

·ID Долга — идентификатор долга, связанного с делом;

·Предмет — предмет, на который распространяется действие, см. словарь № 46. Поле обязательно для заполнения;

·ФИО — фамилия, имя, отчество ответчика по делу (должника или поручителя);

·Портфель;

·Банк;

·Факт. дата меры — фактическая дата меры;

·Тип действия — см. словарь № 47. Поле обязательно для заполнения;

·Дата регистрации;

·Сумма — сумма, полученная в результате предпринятого действия;

·Рег. номер;

·Год выпуска/рег.;

·Комментарий — дополнительная информация;

·Название;

·Оператор;

·ID Оператора.
