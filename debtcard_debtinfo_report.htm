Отчет коллектора по работе с долгом

Для того чтобы добавить отчет коллектора по работе с долгом, выполните следующие действия:

1.В карточке должника на вкладке Долги выберите нужную запись в списке долгов и нажмите кнопку или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.В форме редактирования данных о долге введите информацию в поле Отчет коллектора.

3.Для того чтобы выбрать один из предварительно настроенных комментариев, перечисленных в словаре № 108, нажмите кнопку , расположенную рядом с полем Отчет коллектора и выберите комментарий, затем нажмите ОК. Комментарий будет добавлен в поле Отчет коллектора.

Примечание. История комментариев Отчет коллектора отображается при просмотре истории комментариев.
