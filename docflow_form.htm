Форма "Документооборот"

Форма Документооборот включает следующие элементы:

·Фильтр С: и по: — для задания интервала времени;

·Кнопка Сложный фильтр — для вызова инструмента Сложный фильтр;

·Кнопка Найти — для применения заданных условий фильтрации;

·Вкладки:

·Входящие — см. описание отображения данных на вкладке "Входящие", Действия на вкладке "Входящие" ;

·Исходящие — см. описание отображения данных на вкладке "Исходящие", Действия на вкладке "Исходящие" ;

·Архив — см. описание отображения данных на вкладке "Архив";

·Кнопки для действий с документами:

·На вкладке Входящие:

oСоздать документ с подпунктами:

·Простое создание — см. Простое создание входящего документа;

·По шаблону — см. Создание входящего документа по шаблону;

·Настройка шаблонов — см. Настройка шаблонов документов;

oУдалить документ — см. Удаление документов;

oНазначить исполнителей — см. Назначение исполнителей;

oОткрепить исполнителей — см. Открепление исполнителей;

·На вкладке Исходящие:

oСоздать документ с подпунктами:

·Простое создание — см. Простое создание исходящего документа;

·По шаблону — см. Создание исходящего документа по шаблону;

·Настройка шаблонов — см. Настройка шаблонов документов;

oУдалить документ — см. Удаление документов;

·На вкладке Архив:

o Удалить документ — см. Удаление документов;

·Контекстное меню, которое вызывается нажатием правой кнопки мыши на области списка (активно только на вкладках Входящие и Архив), включающее пункт:

·Сменить статус документа — см. Смена статуса документов.
