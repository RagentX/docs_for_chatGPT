Отчет по платежам (за месяцы)

Отчет "Отчет по платежам (за месяцы)" выводит агрегированные данные о платежах должников за месяцы в Excel.

Отчет "Отчет по платежам (за месяцы)" доступен только пользователям, в роли которых доступ REP_PAY_MONTH имеет значение "Истина".

Порядок действий

Для того чтобы сформировать отчет, выполните следующее:

1.Выберите в главном меню пункт Отчеты, затем выберите отчет "Отчет по платежам (за месяцы)".

2.Откроется форма "Отчет по платежам (за месяцы)". Заполните следующие поля:

·Период:

·С: — выберите месяц и год начала периода (по умолчанию в поле указан первый месяц текущего года).

·По: — выберите месяц и го окончания периода (по умолчанию в поле указан текущий месяц текущего года).

·Панель "Параметры":

·Портфели — откроется форма для множественного выбора портфелей.

·Флаг "Учитывать платежи закрытых" — в отчет попадают также платежи по долгам в закрытых статусах.

·Флаг "Показывать нулевые":

·Если флаг включен, то  в отчет попадают все должники: те должники, по которым платежей не было, попадают в отчет с нулевыми показателями по платежам.

·Если флаг выключен, то в отчет попадают только те должники, по которым за период был хотя бы один платеж.

3.Нажмите кнопку "Запустить". Начнется формирование отчета. В процессе будет отображаться индикатор выполнения. Затем откроется стандартное окно операционной системы для указания расположения и названия файла с последующим сохранением.

4.Сохраните полученный файл. По окончании сохранения файла будет показано соответствующее системное сообщение.

Логика формирования отчета

Для каждого должника, удовлетворяющего условиям отбора по кредитору, просматриваются все платежи, попавшие в период отчета. Далее считается количество и сумма этих платежей за каждый месяц отчетного периода.

Все денежные суммы выводятся в рублях. При этом платежи конвертируются на дату принятия банком, а начальный долг и остаток к погашению по курсу на текущий день.

Описание файла отчета

Отчет "Отчет по платежам (за месяцы)" представляет собой таблицу Excel со следующими полями:

·№ должника.

·№ долга.

·№ Договора.

·Лицевой счет.

·ФИО.

·Возраст.

·Продукт.

·Банк.

·Портфель.

·Дата смены портфеля.

·Регион.

·Тип продукта.

·Начальный долг.

·Остаток к погашению.

·Основной долг.

·Просроченный основной долг.

·Проценты.

·Просроченные проценты.

·Штрафы.

·Пени.

·Комиссия.

·Гос. пошлина.

·Дата подписания реестра.

·DPD банковское.

·DPD фактическое.

·Текущее состояние.

Далее следуют по три колонки для каждого месяца отчетного периода: первая колонка содержит количество, вторая сумму, третья - комиссию, например:

·Кол-во (Август 2020).

·Сумма (Август 2020).

·Комиссия (Август 2020).
