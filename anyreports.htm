Произвольные отчеты

Обновлено до 46 релиза 24.03.2020.

Модуль Произвольные отчеты предназначен для того, чтобы пользователи могли формировать произвольные отчеты на основании данных системы и выгружать их в форматы Microsoft Excel, в файл CSV или выводить на экран.

Произвольный отчет представляет собой табличную форму с данными, отобранными по заданным критериям. Критерии задаются в виде SQL-запроса.

Произвольный отчет может иметь входные параметры, значения которых пользователь вводит непосредственно при формировании отчета и которые затем поступают в SQL-запрос.

Право для входа в модуль имеют только те пользователи, в роли которых доступ CUSTOM_REPORTS_VIEW имеет значение TRUE.
Право для выполнения любых действий в модуле имеют те пользователи, в роли которых доступ CUSTOM_REPORTS_FULL имеет значение TRUE.

Примечание: если константа CustomReportsPermit имеет значение "Истина", то доступ к отчетам определяется настройками в модуле Роли и доступы. Если константа имеет значение "Ложь", то все произвольные отчеты доступны.

Для того чтобы перейти к настройке произвольных отчетов, выберите в главном меню пункт Отчеты а Произвольные отчеты.
