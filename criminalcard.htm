Карточка уголовного преследования

Обновлено до 50 релиза 03.11.2020.

Модуль "Карточка уголовного преследования" предназначен для работы с делами по уголовному преследованию в отношении должников. Карточки уголовного преследования создаются и открывается из модуля "Уголовное преследование".

Открыть карточку судебного преследования может пользователь, в роли которого доступ № 382 PROSECUTION_CARD_VIEW имеет значение TRUE.

Карточка «Уголовное преследование» - доступ к кнопкам ЖЦ. В скобках указывается код статуса по словарю 145, двоеточие и через запятую перечисляются статусы, в которые можно перевести дело. Для дел в статусах, для которых в значении доступа вообще не указаны соответствия, нет никаких ограничений. Если значение доступа ALL, никаких ограничений не действует. На данный момент в системе поддержана возможность настройки ограничений только для статуса «Направлено в ОВД». Пример значения доступа: (2:3,4)
