Добавление и редактирование термина словаря

Функции добавления, редактирования, удаления терминов в словаре доступны для пользователей с соответствующими правами. При этом действуют следующие ограничения:

·для пользователей, имеющих право работы со словарными терминами только в режиме чтения, кнопки добавления, изменения, удаления термина недоступны;

·для пользователей, имеющих право работы только с общими словарными терминами, кнопки добавления, изменения, удаления термина доступны, если выбран общий словарный термин. В этом случае тип термина автоматически устанавливается в значение "Общий" без возможности его изменения на "Системный".

Добавление термина в словарь

Для того чтобы добавить термин в словарь, выполните следующие действия:

1.В форме Работа со словарями выберите нужный словарь. При этом в списке Словарные термины будут отображаться термины, уже включенные в словарь.

2.Нажмите кнопку , относящуюся к списку терминов. Откроется форма редактирования атрибутов термина.

3.В форме заполните следующие поля:

·Код термина — системный номер термина. Поле обязательно для заполнения. Код можно ввести вручную или сформировать автоматически.

Если поле не заполнено вручную, код будет сформирован автоматически при сохранении термина.

Если при ручном заполнении ввести в поле уже существующий код, будет выдано соответствующее предупреждение. Чтобы присвоить значение автоматически, нажмите в нем кнопку Yes. Чтобы повторно ввести код вручную, нажмите No.

·Название термина — термин словаря;поле обязательно для заполнения;

 Примечание. Запрещено использовать в терминах словарей символы ";" и "|", так как они являются системными.

·Термин родительского словаря — для иерархических словарей в данном поле указывается термин родительского словаря, которому подчинен создаваемый термин. Значение выбирается в списке терминов родительского словаря;

·Тип термина — группа, к которой относится термин (системный/общий). Поле обязательно для заполнения.

·Флаг "Термин не используется" — включают для терминов, использование которых прекращено.

4.Нажмите кнопку ОК, чтобы сохранить термин в списке.

Поддержка мультиязычности

В режиме поддержки мультиязычности, то есть при включенной константе @UseMultiLanguage, в окне "Введите словарный термин" (fmCardDict) поле "Название термина" (edName) скрыто. 

Вместо него присутствуют поля для ввода названия на каждом из поддерживаемых языков системы. Перечень поддерживаемых языков системы хранится в константе MultiLanguage.SupportedLanguageList. 

Каждое из этих полей обязательно для заполнения.

Редактирование термина

Для того чтобы отредактировать термин в словаре, выполните следующие действия:

1.В форме Работа со словарями выберите нужный словарь.

2.В списке Словарные термины выберите термин.

3.Нажмите относящуюся к списку словарных терминов кнопку  или выполните двойной щелчок левой клавишей мыши по выбранной записи.

4.В форме редактирования термина включите или выключите флаг Термин не используется или измените  необходимые атрибуты (см. Добавление термина).

5.Нажмите кнопку ОК, чтобы сохранить изменения.

Поддержка мультиязычности

В режиме поддержки мультиязычности, то есть при включенной константе @UseMultiLanguage, в окне "Введите словарный термин" (fmCardDict) поле "Название термина" (edName) скрыто. Вместо него присутствуют поля с названием термина на каждом из поддерживаемых языков системы. Перечень поддерживаемых языков системы хранится в константе MultiLanguage.SupportedLanguageList. 

Изменение следующих полей ведет к их соответствующему изменению для термина на всех языках:

·Код термина;

·Термин родительского словаря;

·Тип термина;

·Термин не используется.

 Например, имеем термин с кодом 12 на двух языках:

·Русский – Пользователь;

·Английский – User.

Если оператор изменяет код с 12 на 15, то код для термина на обоих языках должен стать 15.

Изменение полей с названием ведет к их соответствующему изменению для термина на каждом конкретном языке.

 Т.е., на предыдущем примере, если оператор меняет название на английском на USER, то изменение должно произойти только у термина на английском языке.

Удаление термина

Для того чтобы удалить термин, выполните следующие действия:

1.В форме Работа со словарями выберите нужный словарь.

2.В списке терминов выберите нужный термин.

3.Нажмите кнопку .

4.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление. Термин будет удален из списка.

Поддержка мультиязычности

В режиме поддержки мультиязычности, то есть при включенной константе @UseMultiLanguage, в случае удаления термина данный термин удаляется на всех языках.
