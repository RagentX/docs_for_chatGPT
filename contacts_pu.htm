Ввод показаний ПУ при регистрации контакта

Если для исхода контакта настроено поле Ввод показаний, то на форме ввода данных о результате контакта будет отображаться панель Показания ПУ. Отображение данной панели выполняется по согласованию с технической поддержкой(Техническая константа RegContact.MeterDisplay.Use).

Панель Показания ПУ включает следующие элементы:

·Прибор учета — выпадающий список; в поле отображается выбранный прибор учета, связанный с данным долгом.

Для каждого прибора учета отображаются показания в соответствии с количеством тарифных зон:

·Тарифная зона — название тарифной зоны.

·Дата снятия показания — форма выбора даты; по умолчанию в поле выводится дата, указанная в форме Выберите оператора в начале регистрации контакта. При этом поле Дата снятия показания не доступна для редактирования.

·Значение показания — значение показания в настроенном формате. Введите новые значения показаний.

Если ранее уже были введены показания, они отображаются в правой части панели Показания ПУ:

·Последние значения — отображаются последние введенные значения, далее в скобках выводится тип показаний по словарю № 368 "Тип показания".

Если новые показания, введенные при регистрации контакта, меньше последних значений, пользователю отображается соответствующее информационное сообщение.

Для ввода доступно только поле Значения показания. Обязательность ввода данных регулируется значением поля Ввод показаний в модуле Настройка дерева контактов.

Примечание. Если в форме Выберите оператора в начале регистрации контакта не была указана дата, то в форме регистрации контакта на панели Показания ПУ поле Дата снятия показания не заполнена и доступна для редактирования.
