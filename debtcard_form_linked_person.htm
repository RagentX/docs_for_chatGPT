Вкладка Связанные лица в карточке должника предназначена для просмотра и действий со списком лиц, связанных с активным долгом выбранного должника.

Вкладка Связанные лица содержит следующие элементы:

·Поле Долг с раскрывающимся списком для выбора долга. Выберите долг в этом поле, чтобы просмотреть информацию о поручителях по нему. Список долгов содержит следующие поля:

·Долг — название продукта;

·Договор — номер договора.

·Список связанных лиц, со следующими полями:

·ФИО;

·Тип лица;

·Тип связи;

·Серия и номер паспорта;

·Дата выдачи;

·Кем выдан;

·Комментарий.

·Кнопки для действий со связанными лицами:

· Добавить участника — для добавления связанного лица;

· Изменить участника — для редактирования связанного лица;

· Удалить участника — для удаления связанного лица.
