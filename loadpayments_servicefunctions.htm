Сервисные функции

Show/Hide Hidden Text

В модуле "Загрузка платежей" доступны следующие сервисные функции:

       Поиск ошибочной ячейки

Для того чтобы выполнить поиск ошибочной ячейки, в форме Загрузка платежей в контекстном меню (вызывается правой кнопкой мыши) выберите пункт Поиск проблемной ячейки или нажмите клавишу F3. Будет выделена первая ошибочная ячейка. Если ошибочные ячейки отсутствуют, будет выделена левая верхняя ячейка таблицы.

Для последовательного просмотра ошибочных ячеек используйте клавишу F3. По окончании просмотра всех ошибочных ячеек в таблице данных будет выделена левая верхняя ячейка таблицы.

Примечание: если проверка данных не производилась, то выбор пункта меню Поиск проблемной ячейки, а также нажатие клавиши F3 игнорируется.

       Удаление текущей строки в таблице данных

Для того чтобы удалить строку в таблице данных, в форме Загрузка платежей в контекстном меню (вызывается правой кнопкой мыши) выберите пункт Удаление строки или нажмите клавишу F8, затем подтвердите удаление в открывшемся диалоговом окне, нажав кнопку Yes. После удаления строки автоматически запускается проверка данных в таблице.

       Просмотр легенды (информация об окраске фона ячеек таблицы по результатам проверки)

Для того чтобы открыть форму, содержащую информацию об окраске фона ячеек таблицы по результатам проверки, в форме Загрузка платежей в контекстном меню (вызывается правой кнопкой мыши) выберите пункт Легенда или нажмите клавишу F11.

Функции доступны из контекстного меню модуля, которое вызывается щелчком правой кнопки мыши в области таблицы данных, или при нажатии соответствующих "горячих" клавиш.
