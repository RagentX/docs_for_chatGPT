Удаление из группы

Функционал удаления долга из группы в модуле Группы свободного поиска включается по согласованию с технической поддержкой. Для этого специалисты техподдержки особым образом настраивают определенные вкладки модуля.

(Примечание для поддержки: Дополнительно для вкладки ГСП должно быть sql_builder_structure.is_by_group = 8.)

Для вкладки, на которой поддержан функционал удаления из группы, настройте нужным пользователям доступ к функционалу удаления долга из группы путем включения флага элемента управления ГСП Работа с долгом. Удаление из группы в модуле Группы должников.

Если элемент управления ГСП Работа с долгом. Удаление из группы включен, то на такой вкладке доступен просмотр списка долгов, входящих в выбранную группу и удаление долгов из группы. На вкладке мультифильтр Группа работает по аналогии с РМ Группы должников вкладка "Долги в группе". При нажатии на элемент поля Группа пользователю отображается форма для выбора группы. На форме выводятся все группы долгов. Выбрать можно не более одной группы. В меню кнопки Работа с долгом на такой вкладке дополнительно доступен пункт Удалить из группы.

Просмотр долгов, входящих в группу

Для того чтобы просмотреть список долгов, входящих в группу, выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, для которой настроен доступ к функционалу удаления из группы.

2.Нажмите на элемент  поля Группа.

3.Откроется форма Выберите группу, на которой отображаются все группы долгов. Выберите нужную группу, а затем нажмите кнопку ОК.

4.На вкладке отобразится список долгов, входящих в выбранную группу.

Удаление долгов из группы

Для того чтобы удалить долги из группы выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, для которой настроен доступ к функционалу удаления из группы, и выполните действия для отображения списка долгов, входящих в нужную группу (см. описание выше).

2.В списке долгов выберите один или более долгов.

3.Нажмите кнопку Работа с долгом и в открывшемся списке выберите пункт Удалить из группы. Затем выберите режим выполнения операции: "Для всех" или "Только выделенные".

4.Откроется форма подтверждения удаления долгов. Нажмите Yes.

5.Долги будут удалены из выбранной группы.
