Освободить оператора

Show/Hide Hidden Text

В меню кнопки Работа с оператором в пункте Освободить оператора доступны следующие подпункты:

·ОДВ — Оператор Досудебного Взыскания (оператор, ответственный за долг на досудебной стадии взыскания. Назначается и освобождается также в модуле Постановка задач);

·ОСВ — Оператор Судебного Взыскания (оператор, отвечающий за дело на судебной стадии. Назначается в модуле Юридический департамент путем группового изменения атрибута — Ответственный, на судебных вкладках);

·ОИП —  Оператор Исполнительного Производства (оператор, отвечающий за дело на стадии исполнительного производства. Назначается в модуле Юридический департамент путем группового изменения атрибута — Ответственный, на вкладках, относящихся к исполнительному производству);

Подпункт Назначить оператора виден, если  активен один из подпунктов:

·ОДВ — видно, если на вкладке есть полеID долга, есть доступ WORK_TASK_SET и в РМ Группы должников было предоставлено право Работа с оператором. Освободить оператора ОДВ;

·ОСВ — видно, если на вкладке есть поле ID СВ (судебного взыскания), есть доступ LAW_ACT_CHANGE_OPERATOR и в РМ Группы должников было предоставлено право Работа с оператором. Освободить оператора ОСВ;

·ОИП — видно, если на вкладке есть поле ID ИП (исполнительного производства), есть доступ LAW_EXEC_CHANGE_OPERATOR и в РМ Группы должников было предоставлено право Работа с оператором. Освободить оператора ОИП.

Освобождение оператора досудебного взыскания

Для того, чтобы освободить оператора досудебного взыскания, выполните следующие действия:

1.В модуле Группы свободного поиска, активируйте вкладку, на которой доступен данный функционал.

2.Создайте выборку долгов с помощью фильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Работа с оператором.

4.В открывшемся меню включенный флаг Только выделенные означает выполнение операции только для выделенных записей, выключенный — для всей выборки.

5.Выберите пункт Освободить оператора, далее выберите подпункт ОСВ.

6.В открывшемся диалоговом окне подтвердите выполнение операции, нажав кнопку Yes.

7.Появится информационное окно с указанием количества обработанных долгов. Нажмите ОК.

Освобождение оператора судебного взыскания

Если для выбранных долгов не созданы судебные дела, будет показано сообщение: "Среди выбранных строк нет подходящих для обработки." Никаких действий произведено не будет.
