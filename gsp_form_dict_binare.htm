Бинарные словари

Задача в Организаторе № 25142.

Отображение словарных полей по их коду в ГСП

Цель доработки

В настоящий момент в ГСП отображение полей, ссылающихся на словарь, реализовано только в виде текста в самом SQL-запросе путем join’а с таблицей dict. Требуется это исправить, чтобы путем настройки силами сопровождения, можно было указывать, какие поля SQL-запроса содержат в себе коды терминов конкретного словаря, и на вкладке в ГСП отображение таких полей требуется реализовать через LookUpComboBox. Причем требуется поддержать также отображение значений по бинарным словарям – кодами терминов таких словарей является число 2 в некоторой степени (1, 2, 4, 8, 16, 32, 64 и т.д. Максимум 31 значение). Примером такого словаря в нашей системе является словарь 90 "Цели выезда" (visit_address.purpose).

Новое поле таблицы sql_builder_structure

Таблицу sql_builder_structure требуется расширить новым полем:

-field_properties [nvarchar(MAX) / CLOB] --свойства полей

В данном поле будет храниться описание словарных полей вкладки ГСП в следующем виде:

<root>

 <field name="…" dict_code="…" is_binary="…"/>

 …

</root>

 

Атрибуты тега field

Атрибут

Описание

name

Уникальное имя поля

Содержится в поле sql_builder_structure.data (см. аналогичное свойство поля FieldNames)

dict_code

Код словаря, на который ссылается поле

is_binary

Признак бинарности словаря

Указывается 0 или 1. Если 1, то значит, что используется бинарный словарь.

 

Наполнение данного поля на текущий момент будет поддержано только путем вставки значений непосредственно SQL-запросом. Т.е. в интерфейсе программы "Контакт" при создании или редактировании вкладки ГСП это сделать будет невозможно.  

Доработка РМ "Группы свободного поиска"

Требуется доработать логику построения колонок на вкладках ГСП с учетом нового поля field_properties для вкладок:

По name определяется, какое поле требуется смотреть:

·Если dict_code непустой, то для колонки данного поля требуется добавить Properties = LookUpComboBox для отображения значений по указанному в dict_code словарю.

·Если dict_code непустой и is_binary=1, то для колонки данного поля требуется добавить Properties = CheckComboBox для отображения списка значений через точку с запятой (;) по указанному в dict_code бинарному словарю. Для этого в Items требуется добавить термины данного словаря в следующем виде:

o0 – %термин_словаря_с_кодом_1%

o1 – %термин_словаря_с_кодом_2%

o2 – %термин_словаря_с_кодом_4%

o3 – %термин_словаря_с_кодом_8%

o4 – %термин_словаря_с_кодом_16%

o5 – %термин_словаря_с_кодом_32%

o6 – %термин_словаря_с_кодом_64%

oИ т.д. максимум до 31
