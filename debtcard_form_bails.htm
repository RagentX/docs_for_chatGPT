Вкладка Поручители в карточке должника предназначена для просмотра и редактирования данных о поручителях (созаемщиках) должника. Вкладка содержит следующие элементы:

·поле Долг с раскрывающимся списком для выбора долга. Выберите долг в этом поле, чтобы просмотреть информацию о поручителях по нему. Список долгов содержит следующие поля:

·долг — название продукта;

·договор — номер договора.

Примечание: при наличии у пользователя доступа DEBT_CARD_SHOW_CLOSE_GUARANTOR в список будут попадать все долги, в том числе закрытые, даже если у пользователя нет доступа DEBT_CARD_SHOW_CLOSE, и будет доступна информация о поручителях по закрытым долгам.

·список поручителей должника по кредиту, по которому возник долг;

·кнопки для работы со списком поручителей.

По кнопке осуществляется настройка персональных ограничений коммуникации.
