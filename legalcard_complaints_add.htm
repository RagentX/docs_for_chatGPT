Добавление жалобы

Для того чтобы добавить данные о жалобе, выполните следующие действия:

Для того чтобы добавить жалобу, выполните следующие действия:

1.Войдите в модуль Карточка судебной работы. Активируйте вкладку Жалобы.

2.Нажмите кнопку Добавить. Откроется форма Жалоба.

3.Заполните следующие поля:

·Направление жалобы — см. словарь № 121, поле обязательно для заполнения;

·Дата подачи жалобы — на данном этапе кнопка заблокирована;

·Факт. дата жалобы — фактическая дата жалобы, поле обязательно для заполнения;

·Вид жалобы — см. словарь № 30 (если в словаре есть термины, имеющие в качестве родительского термин № 46 "Судебная работа", то выводятся только эти термины, иначе выводятся все термины), поле обязательно для заполнения;

·Тип жалобы — см. словарь № 31, поле обязательно для заполнения;

·Дата заседания — дата заседания по рассмотрению жалобы;

·Результат рассмотренной жалобы — результат заседания по рассмотрению жалобы; первое поле см. словарь № 119 (если в словаре есть термины, имеющие в качестве родительского термин № 46 "Судебная работа", то выводятся только эти термины, иначе выводятся все термины; если в словаре № 119 отсутствуют термины, то первое поле не отображается); второе поле см. словарь № 120;

·Файл — файл, содержащий текст жалобы. Поле обязательно для заполнения. Для прикрепления файла нажмите кнопку , выберите файл в стандартном диалоговом окне Windows и нажмите кнопку Open (Открыть).

4.Нажмите OK.

5.Выполните Прикрепление документа к действию. Прикрепленный документ будет доступен на вкладке Вложения.

6.Далее осуществляется Формирование настроенных запланированных действий.

В результате жалоба добавится в список, и статус ее будет "Не подана". Становится доступной кнопка Отметить подачу жалобы.

Протоколирование: на вкладке Протокол появится новая запись со следующими значениями полей:

·Дата — дата совершения действия;

·Действие — "Добавление жалобы";

·Оператор — оператор, совершивший действие;

·Документ — включено;

·Комментарий — "Добавление жалобы. №<идентификационный номер жалобы>. Вид жалобы: <вид жалобы>. Тип жалобы: <тип жалобы>".

Форма "Жалоба"

Если константа LawAct.Appeal.SessionAfterCreate.Use имеет значение "Истина", и поле Дата заседания было заполнено, то при создании жалобы, сразу после сохранения новой жалобы автоматически открывается форма Карточка заседания для внесения данных о заседании, причем дата заседания заполнена.

Для того чтобы внести данные о заседании, заполните карточку заседания и нажмите «ОК». При выходе из формы без сохранения внесенных данных, заседание не добавляется.
