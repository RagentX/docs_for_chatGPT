Создать исполнительный документ

Создание исполнительных документов по делам судебной работы поддержано в Системе только при значении константы UseLawExecDoc равном "Истина". Действие доступно только пользователям, в роли которых доступ № 512 LAW_EXEC_DOC_CREATE имеет значение TRUE.

Порядок действий

Для того чтобы создать исполнительный документ по иску, выполните следующее:

1.В карточке судебной работы в в исковом производстве нажмите кнопку Исполнительная стадия и в открывшемся списке выберите пункт Создать исп. док-т.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить перевод.

3.Осуществляется проверка статуса дела. Для создания исполнительных документов доступны иски только в статусах 8 "Удовлетворение иска", 11 "Решение Суда обжаловано", 16 "Получение ИЛ на МС", 39 "Апелляция удовлетворена", 40 "Кассация удовлетворена", 41 "Надзор удовлетворен", 50 "Частичное удовлетворение иска" (см. словарь № 25). Если дело находится в статусе, для которого не доступно создание исполнительных документов, то выполнение действия будет прервано, будет показано сообщение:

Перевод на исполнительное производство осуществляется только после удовлетворения иска..

4.Если в Системе настроена константа LawAct.TransferToExec.Ban.DebtStatusList, то осуществляется проверка статуса долга, связанного с данным делом. Если долг находится в одном из статусов, перечисленных в значении константы, то создание ИП не разрешено. Выполнение действия будет прервано, будет показано сообщение:

Связанный с делом долг имеет статус, для которого запрещен перевод на ИП.

5.Создается исполнительное производство:

·ему присваивается статус "Не создано";

·поле Дата возбуждения ИП не заполняется;

·статус дела судебной работы, по которому создается исполнительный документ, не изменяется.

6.Действие протоколируется.

7.Осуществляется прикрепление документа.

8.Осуществляется формирование настроенных запланированных действий.

9.По окончании перевода будет выдано сообщение о результатах выполнения операции. Нажмите кнопку ОК, чтобы закрыть его.

10.Созданный исполнительный документ доступен на вкладке Исполнительные документы.
