Произвольные атрибуты должника

Show/Hide Hidden Text

Произвольные атрибуты должника предназначены для хранения дополнительной информации о должнике. Перечень всех возможных  произвольных атрибутов хранится в словаре № 53. Настройка произвольных атрибутов (тип данных произвольного атрибута, значение по умолчанию) осуществляется в модуле Произвольные атрибуты.

Список произвольных атрибутов должника содержится в карточке должника на вкладке Произвольные атрибуты.

Право на просмотр произвольных атрибутов долга определяется для пользователя значением доступа № 128 DEBT_CARD_RANDVAL_VIEW.

Право на действия с произвольными атрибутами определяется для пользователя значением доступа 129 DEBT_CARD_RANDVAL_FULL, а также настройками на вкладке Произвольные атрибуты в модуле Роли и доступы для его роли.

       Добавление произвольного атрибута долга

Пользователь имеет право на добавление произвольных атрибутов, только если в его роли доступ № 129 DEBT_CARD_RANDVAL_FULL имеет значение TRUE. При этом для добавления доступны только атрибуты, перечисленные на вкладке Произвольные атрибуты в модуле Роли и доступы для его роли. (Если список атрибутов на вкладке Произвольные атрибуты для роли пользователя пуст, то ограничения не накладываются.)

Для того чтобы добавить произвольный атрибут долга, выполните следующее:

1.В карточке должника на вкладке Произвольные атрибуты нажмите кнопку Добавить атрибут.

2.Откроется форма добавления произвольного атрибута. Заполните следующие поля:

·Долг — выберите долг должника, к которому относится произвольный атрибут;

·Атрибут — выберите произвольный атрибут (см. словарь № 53);

·Значение — значение произвольного атрибута вводится с учетом его типа;

·Комментарий — дополнительная информация.

3.Нажмите кнопку ОК, чтобы сохранить данные.

Произвольный атрибут будет добавлен в список.

       Редактирование произвольного атрибута

Редактирование произвольного атрибута долга заключается в изменении его значения и/или комментария.

Пользователь имеет право на редактирование произвольных атрибутов, только если в его роли доступ № 129 DEBT_CARD_RANDVAL_FULL имеет значение TRUE. При этом для редактирования доступны только атрибуты, перечисленные на вкладке Произвольные атрибуты в модуле Роли и доступы для его роли. (Если список атрибутов на вкладке Произвольные атрибуты для роли пользователя пуст, то ограничения не накладываются.)

Для того чтобы изменить значение или комментарий произвольного атрибута долга, выполните следующее:

1.В карточке должника на вкладке Произвольные атрибуты выберите нужную запись и нажмите кнопку Изменить атрибут или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.Откроется форма редактирования произвольного атрибута. Она аналогична форме добавления произвольного атрибута, но для изменения доступны только следующие поля:

·Значение — значение произвольного атрибута;

·Комментарий — дополнительная информация.

3.Измените нужные значения в доступных полях.

4.Нажмите кнопку ОК, чтобы сохранить изменения.

Примечание. Если в системе поддерживается версионность данного атрибута, то при редактировании его значения будет добавлена новая версия атрибута (см. Версионность произвольных атрибутов долга). Датой начала жизни версии будет дата и время ввода нового значения.

       Удаление произвольного атрибута

Пользователь имеет право на удаление произвольных атрибутов, только если в его роли доступ № 129 DEBT_CARD_RANDVAL_FULL имеет значение TRUE. При этом для удаления доступны только атрибуты, перечисленные на вкладке Произвольные атрибуты в модуле Роли и доступы для его роли. (Если список атрибутов на вкладке Произвольные атрибуты для роли пользователя пуст, то ограничения не накладываются.)

Для того чтобы удалить произвольный атрибут долга, выполните следующее:

1.В карточке должника на вкладке Произвольные атрибуты выберите нужную запись.

2.Нажмите кнопку Удалить атрибут.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.

Примечание. Если в системе поддерживается версионность данного атрибута, то он не будет удален безвозвратно. Его и его историю версий можно будет просмотреть при включении флага Показывать удаленные.

       Просмотр удаленных атрибутов (работает только при поддержке версионности)

Этот пункт работает только при поддержке версионности произвольных атрибутов долга. В технической константе RandomValue.Version.CodeList перечислены атрибуты, для которых поддерживается версионность. Если значение константы равно нулю, то версионность произвольных атрибутов не поддерживается.

Для того чтобы просмотреть удаленные атрибуты, для которых поддерживается версионность, включите флаг Показывать удаленные.

По каждому из удаленных атрибутов доступна история версий.
