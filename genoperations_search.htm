Быстрый поиск должника

Show/Hide Hidden Text

Функционал быстрого поиска должника доступен из всех списочных модулей Системы. При нажатии комбинации клавишей CTRL + F открывается форма Поиск для задания критериев поиска.

Список модулей, для которых доступен быстрый поиск

·Список должников.

·Юридический департамент.

·Обработка должников.

·Протокол контактов.

·Постановка задач.

·Исходящее информирование.

·Платежи.

·Статистика.

·Мониторинг.

·Группы свободного поиска (функционал отличается, см. Поиск по долгу).

Внимание! Важно! Константа FastSearch.InfoDrop.Mode в модуле "Группы свободного поиска" не работает!

Порядок действий для осуществления поиска

1.Откройте нужный модуль. Активируйте нужную вкладку.

2.Создайте выборку, используя мультифильтры, сложный фильтр, фильтрацию по полям таблицы. Дальнейший поиск будет происходить внутри сформированной выборки.

3.Нажмите комбинацию клавиш CTRL + F. Откроется форма Поиск.

4.В окне задайте один или несколько параметров поиска. В текстовых полях формы (таких, ка: ФИО, ID клиента, № договора, Лицевой счет) можно указать несколько символов, которые должны содержаться в соответствующем атрибуте искомого должника. Регистр символов при поиске не учитывается.

5.Нажмите кнопку Найти далее. Будет выполнен переход к первой найденной записи списка, которая удовлетворяет заданным условиям.

6.Для продолжения поиска нажимайте повторно кнопку Найти далее. Каждый раз в списке будет выделяться следующая по порядку запись, удовлетворяющая условиям.

7.При необходимости перейдите на другую вкладку модуля и сформируйте выборку. Продолжайте поиск в ней.

8.Для того, чтобы стереть введенные в поля формы Поиск критерии, нажмите кнопку Очистить.

Особенности работы функционала быстрого поиска

·Если константа FastSearch.InfoDrop.Mode имеет значение «1», то при закрытии формы Поиск и при переходе между вкладками одного модуля ее поля очищаются. Если константа FastSearch.InfoDrop.Mode имеет значение «0», то введенные значения сохраняются.

·При переключении вкладки открытое окно поиска может изменяться следующим образом:

·Если на вкладке доступен поиск не по всем полям, то в окне отображаются только те поля, по которым поиск возможен на данной вкладке;

·Если на вкладке доступен поиск не по всем полям, то ранее введенные в недоступные поля данные игнорируются при поиске на этой вкладке;

·Поиск осуществляется только по отобранным и отфильтрованным записям (выборке).

Список полей, по которым может осуществляться поиск

·ФИО;

·ID клиента;

·Договор;

·Лицевой счет;

·Остаток или Размер требования (для модуля Юридический департамент):

·От;

·До;

·Статус (долга).
