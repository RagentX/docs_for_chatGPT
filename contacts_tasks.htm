Выбор автоматических задач

После регистрации контакта с исходом, для которого настроены задачи, осуществляется добавление соответствующих задач долгу. Для этого открывается форма "Группы задач".

Задача добавляются автоматически, без уведомлений, в тех случаях, когда:

·Для задачи не установлена связь с группой;

·Задача является единственной в группе, при этом группа имеет признак обязательности выбора.

В остальных случаях выводится окно выбора задач. При этом:

·Для групп с множественным выбором отображается чекбокс со списком задач группы (на примере — группа "Судебная стадия");

·Для групп с альтернативным выбором отображаются радиокнопки со списком задач группы (на примере — группа "Взаимодействие с  должником", в данном случае группа имеет включенный признак обязательности выбора);

·Для групп с выключенным признаком обязательности и единичным выбором добавляется еще один пункт — "Не создавать", позволяющий отказаться от создания задач из данной группы (на примере — группа "Дополнительно").

Если осуществляется попытка выйти из окна без добавления обязательных задач, выдается предупреждение.

Пример

В примере, описанном выше (см. Автоматическая постановка задач — Пример), окно выбора задач имеет вид:

·Группа "Судебная стадия" включает четыре задачи, не имеет признака обязательности и имеет признак множественности, поэтому для задач в этой группе отображается чекбокс (флаги) с возможностью выбрать любое количество из четырех задач.

·Группа "Дополнительно" включает две задачи, не имеет признака обязательности и не имеет признака множественности, поэтому задачи в ней отображаются радиокнопками с возможностью выбрать одну из двух задач, а также дополнительно отображается радио-кнопки "Не создавать";

·Группа "Взаимодействие с должником" включает две задачи, имеет признак обязательности выбора и не имеет признака множественности, поэтому в ней нет дополнительной радио-кнопки "Не создавать", есть возможность выбрать только одну задачу из двух, и запрещено закрывать форму "Группы задач" без выбора задачи в данной группе.

·не имеет признака множественности, поэтому список задач группы отображается радио-кнопками, с возможностью выбрать только одну задачу. При этом группа имеет признак обязательности выбора, поэтому.

·Задача № 260 "Обязательная задача" не включена ни в одну группу, поэтому она не отображается. При этом добавление задачи осуществляется всякий раз при регистрации контакта с соответствующим исходом.
