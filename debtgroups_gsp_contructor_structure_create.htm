Создание шаблона структуры БД

Для того чтобы создать новый шаблон, выполните следующие действия:

1.Войдите в модуль Группы должников.

2.Нажмите кнопку ГСП. В открывшемся списке выберите Конструктор структур. Откроется форма Конструктор структур.

3.Нажмите кнопку Создать запрос.

4.Добавьте таблицы или представления.

5.Настройте связи.

6.Настройте поля шаблона.

7.После каждого добавления информации в шаблон нажимайте кнопку Сохранить изменения.

8.По окончании создания нажмите кнопку Сохранить данные.

9.В открывшемся окне введите заголовок шаблона.

10.Нажмите OK.
