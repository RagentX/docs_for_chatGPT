Список пользовательских функций

Список доступных пользовательских функций приведен в таблице.

Функция

Описание

Пример использования

INCDAY(<дата>,<целое число>)

Увеличивает дату на указанное количество дней

Увеличение даты кредита на 12 дней:

[INCDAY(<frxDBRep1."credit_date">,12)]

ROUNDTO(<действительное число>,<целое число>)

Округляет число до указанного разряда. При отрицательном значении второго аргумента функция округляет до указанного знака после запятой

Если сумма кредита составляет 52406,54 руб., то функция с параметрами

[ROUNDTO(<frxDBRep1."debt_sum">,3)]

округлит значение до 52000 руб., а функция с параметрами

[ROUNDTO(<frxDBRep1."debt_sum">,-1)] округлит до 52406,5 руб.

PRINTMONEY(<действительное число>)

Функция для печати денежных сумм. Округляет и приводит копейки к стандартному формату.

Примечание. Для корректного отображения лучше использовать эту функцию в объекте Memo.

[PRINTMONEY(<frxDBRep1."debt_sum">)]

GETMOBALL(<ID персоны>)

Выводит через запятую номера всех мобильных телефонов для должника с указанным ID

[GETMOBALL(<frxDBRep1."p_id">)]

PRINTMONEYTEXT(<действительное число>)

Выводит сумму словами

[PRINTMONEYTEXT(<frxDBRep1."debt_sum">)]

RANDOMVALUE(<ID долга>,<название произвольного атрибута>)

Выводит значение произвольного атрибута для выбранного долга по названию

[RANDOMVALUE(<frxDBRep1."d_id">, 'Статус')]

GETADDRESS(<ID персоны>,<код тип адреса>)

Выводит адрес выбранного типа (см. словарь № 5)

[GETADDRESS(<frxDBRep1."p_id">, 1)]

GETPHONE(<ID персоны>,<код тип телефона>)

Выводит телефон выбранного типа (словарь № 4)

[GETPHONE(<frxDBRep1."p_id">, 1)]

GETGUARANTORADDRESS (<ID поручителя>,<код тип адреса>)

Выводит адрес поручителя выбранного типа (словарь № 5)

Адрес регистрации поручителя:
[GETGUARANTORADDRESS(<frxDBDebtGuarantorRep."id">,101)]

Фактический (домашний) адрес поручителя:

[GETGUARANTORADDRESS(<frxDBDebtGuarantorRep."id">,102)]

GETLASTOUTNUM(<ID долга>)

Выводит исходящий номер последнего зарегистрированного письма

[GETLASTOUTNUM(<frxDBRep1."d_id">)]

GETPERSONPROPRTY(<ID параметра имущества>, <ID долга>)

Функция является устаревшей и оставлена для совместимости.

Отображает параметры имущества. Посмотреть ID параметра имущества можно, в модуле "Параметры имущества" (Утилиты -> Параметры Имущества), раздел Параметры имущества, столбец ID

[GETPERSONPROPRTY(1, <frxDBRep1."d_id">)]

GETPERSONPROPRTY2(<ID параметра имущества>, <ID имущества>)

Отображает параметры имущества. Посмотреть ID имущества и ID параметра имущества можно в модуле "Параметры имущества" (Утилиты -> Параметры Имущества)

[GETPERSONPROPRTY2(1, 1)]

CALCSUM (<ID долга>)

Выводит сумму всех подтвержденных и неотмененных платежей для указанного долга

[CALCSUM (<frxDBRep1."d_id">)]

CoverSumToRUR (<сумма>)

Отображает сумму в рублях по текущему курсу валюты долга

[CoverSumToRUR (<frxDBRep1."debt_sum">)]

DEBTSUM (<ID персоны>)

Сумма остатков всех активных договоров персоны

[DEBTSUM (<frxDBRep1."p_id">)]

DEBTSUMFIELD(<ID персоны>, <название поля>)

Поддерживаются следующие поля: debt_sum, basic_sum, percent_sum, peni_sum, start_sum, total_sum, exp_basic_sum , exp_percent_sum, exp_commission_sum, penalty_sum

Сумма значений указанного поля по всем долгам персоны.

Отображение суммы всех кредитов должника:

[DEBTSUMFIELD(<frxDBRep1."p_id">, 'total_sum')]

 

GETCONTRACTS (<ID персоны>)

Выводит номера всех активных договоров заданного должника через запятую

[GETCONTRACTS (<frxDBRep1."p_id">)]

DICT(<Код словаря>, <Код термина>)

 

Выводит название термина из словаря

Вывод региона должника:

[DICT(39, <frxDBRep1."region">)]

Вывод филиала:

[DICT(61, <frxDBRep1."filial">)]

GETLASTLETTERDT (<ID долга>; <код словаря № 36 (последняя дата определенного письма, -1 — это последняя дата из всех писем)>)

Выводит дату последнего письма

Вывод даты последнего из всех писем должнику:

[GETLASTLETTERDT(<frxDBRep1."d_id">), -1]

GETMOB (<ID> - ID Должника,<S:строка> - Комментарий телефона (У некоторых клиентов записывались "№ договора")

Выводит мобильный номер должника

[GETMOB(<frxDBRep1.”p_id”>, “123456789”)] – в результате выводится один мобильный номер должника

GETMOBMGTS (<ID> -  ID Должника)

Выводит домашний номер должника

[GETMOBMGTS(<frxDBRep1.”p_id”>)] – в результате выводится домашний номер должника

GETMOBX (<ID> - ID Должника, <TYP>> - Тип телефона (из словаря № 4))

Выводит один номер должника указанного типа

[GETMOBX (<frxDBRep1.”p_id”>, 1)] – в результате выводится один мобильный (т.к. <TYP> = 1) номер должника

LASTCALCDATE (<DEBT_ID> - ID долга)

Выводится последняя дата подтвержденного платежа

[LASTCALCDATE(<frxDBRep1.”d_id”>)]

LASTCALCSUM (<DEBT_ID> - ID долга)

Выводится последняя сумма подтвержденного платежа

[LASTCALCSUM(<frxDBRep1.”d_id”>)]
