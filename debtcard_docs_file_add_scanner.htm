Прикрепление документов с использованием сканера

В настоящей версии Системы есть возможность добавления вложений, полученных в результате сканирования. Вопросы о возможности поддержки механизма сканирования решайте путем обращения в техническую поддержку.

Функционал использования сканера регулируется коммерческой константой @DocAttach.Scanner.Use.

Примечание. Система поддерживает только twain-совместимые сканирующие устройства. Обязательно наличие библиотек EZTwain в папке пользователя. Если указанные библиотеки отсутствуют в сетевой папке, обратитесь в техническую поддержку.

Если ваша версия Системы поддерживает сканирование, то на форме добавления вложения есть кнопка Отсканировать документ.

Функционал прикрепления вложений используется в следующих модулях:

·Карточка должника, вкладка "Вложения", область данных "Файлы";

·Карточка судебной работы, вкладка "Вложения";

·Карточка исполнительного производства, вкладка "Вложения";

·Документооборот — на карточке документа (входящего или исходящего).

Порядок действий для сканирования

Для того прикрепить в качестве вложения отсканированный документ, выполните следующее:

1.В нужном модуле активируйте нужную вкладку.

2.Нажмите кнопку Добавить.

3.Откроется форма добавления вложения Документ. Нажмите кнопку Отсканировать документ.

4.Откроется форма Сканирование и распознавание. Нажмите кнопку Сканировать.

5.Откроется форма настройки сканирования. Укажите нужные настройки и нажмите кнопку Scan (Для уточнения правильности настройки параметров вы можете воспользоваться режимом "Preview").

6.По окончании процесса сканирования вы вновь окажетесь в форме Сканирование и распознавание, при этом в рабочем окне будет отображен отсканированный документ. Вы можете изменить его, воспользовавшись следующими действиями:

·Кнопка Настройка сканера открывает форму для настройки параметров сканирования;

·В поле Имя файла отображается имя, с которым будет сохранен файл;

·Кнопки и для листания многостраничного документа;

·Кнопки и для поворота изображения;

·Поле Масштаб — для изменения масштаба просмотра.

7.Нажмите кнопку Сохранить. Документ будет сохранен в расширении и под именем, указанным на форме.
