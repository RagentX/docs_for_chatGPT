Настройка автораспределения по многоугольникам

Форма для настройки автораспределения по многоугольникам

Форма включает в себя две области:

·Область с картой для построения многоугольника;

·Форма со списком операторов для автораспределения по многоугольникам.

Примечание. На показанном изображении была нажата кнопка Показать адреса на карте для отображения адресов.

Область с картой для построения многоугольника

Область включает в себя следующие элементы:

·Кнопки раздела Рисование многоугольника:

·Добавить;

·Удалить;

·Завершение редактирования;

·Сохранить многоугольник;

·Кнопка раздела Геолокация:

·Сохранить результат;

·Карта;

·Область, где отображаются числовые значения координат многоугольника после завершения его редактирования (координаты отображаются только после завершения редактирования многоугольника при его настройке).

Форма со списком операторов для автораспределения по многоугольникам

1.Разверните список операторов.

2.Используйте кнопки , и для добавления, редактирования и удаления записи из списка операторов, для которых настроено автораспределение.

3.Кнопка "Обновить" используется для обновления информации, если другие пользователи Системы внесли изменения.

4.Кнопка "Показать адреса на карте" используется для отображения адресов на карте.

5.Флаг "Отображать многоугольники" - в случае его включения отображаются многоугольники.

6.Флаг "Отображать все адреса" - если он включен, отображаются все адреса должников в соответствии с типизацией. Если флаг выключен, для каждого должника отображается только один адрес, имеющий тип "Регистрация". Если у должника в базе имеется более одного адреса указанного типа, из них выбирается тот, что имеет наименьший идентификационный номер.

Порядок действий для настройки автораспределения по многоугольнику

Рекомендуется сначала добавить оператора, а затем для него настроить многоугольник. Если вы хотите изменить многоугольник автораспределения для уже настроенного оператора, то выберите нужную строку в списке операторов и перейдите к настройке многоугольника.

Добавить оператора

1.Разверните список операторов или используйте кнопки, относящиеся к настройке операторов на форме.

2.Нажмите кнопку , относящуюся к настройке операторов.

3.Откроется форма добавления оператора.

4.Заполните поля:

·Мнемокод – идентификатор записи;

·Пользователь – выберите пользователя, для которого настроена геолокация;

·Порядок выполнения – приоритет;

·Многоугольник – не заполняйте эту вкладку, на ней будут отражены координаты многоугольника после его настройки;

·Примечание – дополнительная информация.

5.Нажмите OK. В созданной записи поле "Многоугольник" будет пусто.

Настройка  многоугольника автораспределения

1.Установите курсор на нужную запись в списке операторов (при необходимости). Дальнейшие действия осуществляются в области, содержащей карту.

2.Нажмите кнопку "Добавить", относящуюся к рисованию многоугольника.

3.Создайте на карте многоугольник. Если результат вас не устраивает, нажмите кнопку "Удалить", а затем "Добавить" и начните сначала. По окончании редактирования нажмите кнопку "Завершить редактирование".

4.Нажмите кнопку "Сохранить многоугольник". Сохранный многоугольник будет добавлен в выбранную запись списка операторов.
