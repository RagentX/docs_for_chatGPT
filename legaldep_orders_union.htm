Объединение судебных дел

Данное действие осуществляет объединение выбранных судебных дел по одному долгу в одно, с переносом данных в объединенное дело.

Функционал объединения судебных дел доступен в Системе, если константа LawActList.UnionLawAct.Use имеет значение "Истина".

Функционал объединения судебных дел доступен пользователям, в роли которых доступ LAW_ACT_LIST_UNION_LAW_ACT имеет значение TRUE.

Логика объединения судебных дел

1.Для объединения выбираются дела, созданные по одному долгу. Запрещено объединять дела, созданные по разным долгам.

2.Указанное пользователем дело считается основным, остальные дела из списка объединяемых, переводятся в архив. Остальные объединяемые дела считаются объединенными. Они переносятся в архив и не могут быть повторно объединены.

3.Производится расчет суммы объединенного судебного дела.

4.Производится перенос всех периодов объединяемых судебных дел в основное дело.

Порядок действий

Для того чтобы объединить судебные дела, выполните следующее:

1.В модуле Юридический департамент активизируйте вкладку Приказы или Иски.

2.Сформируйте выборку при помощи фильтров.

3.Выберите дела для объединения. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL.

4.Активируйте контекстное меню, нажав на правую кнопку мыши. В открывшемся списке выберите пункт Перевод между стадиями, затем — Объединение дел.

5.Будет показана форма подтверждения с текстом:

Вы действительно хотите объединить N дел(а)?

Нажмите Yes, чтобы объединить дела.

Примечание.

Если выбранные дела созданы по разным долгам, то будет показано сообщение об ошибке:

Операция не выполнена. Объединение дел, созданных по разным долгам, невозможно.

Если хотя бы одно из выбранных дел является объединенным (то есть оно было перенесено в архив после объединения), то будет показано сообщение об ошибке:

Операция не выполнена. Некоторые судебные дела ранее уже были объединены, повторное объединение недоступно.

В этих случаях объединение дел не будет осуществлено.

6.Откроется форма для выбора основного дела. Выберите дело и нажмите ОК.

7.Дела будут объединены. Активным будет выбранное основное дело, в которое будут перенесены требования из остальных объединенных дел. Дела, отличные от основного будут перенесены в архив и станут объединенными и недоступными для повторных объединений.
