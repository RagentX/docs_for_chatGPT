Форма "Произвольные отчеты" (список отчетов, полей и входных параметров)

Список произвольных отчетов отображается в форме Произвольные отчеты, которая включает в себя следующие элементы:

·список произвольных отчетов, содержащий следующие атрибуты:

·ID — идентификатор произвольного отчета, заполняется автоматически;

·Название — название отчета;

·Комментарий;

·кнопки для работы со списком произвольных отчетов:

· Добавить отчет — Создание нового произвольного отчета;

· Изменить отчет — Изменение произвольного отчета;

· Удалить отчет — Удаление произвольного отчета;

· FastReport — просмотр или формирование отчета в программе FastReport;

· Выгрузить формат — Выгрузка данных об отчете в файл SQL;

·список полей (столбцов) отчета на вкладке Описание полей, содержащий следующие атрибуты:

·ID — идентификатор поля, заполняется автоматически;

·Название поля — название столбца в отчете;

·Порядковый номер — порядковый номер столбца в отчете;

·Маска — см. Использование масок ввода для полей отчета;

·Ширина столбца;

·Системное название — название поля в БД;

·кнопки для работы со списком полей отчета на вкладке Описание полей;

·список входных параметров отчета на вкладке Входные параметры со следующими полями:

·ID — идентификатор входного параметра, заполняется автоматически;

·Название — название входного параметра;

·Тип — тип входного параметра;

·Признак обязательности — признак обязательности заполнения входного параметра;

·Системное название — название поля в БД;

·Порядковый номер — порядковый номер входного параметра, используется при формировании отчетов.

·кнопки для работы со списком входных параметров отчета на вкладке Входные параметры.
