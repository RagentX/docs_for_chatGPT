Реестр на аутсорсинг

Модуль отчета "Реестр на аутсорсинг" предназначен для формирования реестра долгов в формате файла MS Excel.

Право для входа в модуль имеют только те пользователи, в роли которых доступ REP_OUTSORSING_VIEW имеет значение TRUE.

Для того чтобы сформировать отчет, выполните следующие действия:

1.Выберите в главном меню пункт Отчеты/Реестр на аутсорсинг.

2.В открывшейся форме укажите значения входных параметров отчета:

·Портфели — портфели, в которые должны входить долги (подробнее см. Мультифильтры). Если не выбрать портфели, то будет произведен отбор долгов по всем портфелям;

·Операторы — операторы, ответственные за работу с долгами (подробнее см. Мультифильтры). Если не выбрать операторов, то будет произведен отбор долгов по всем операторам;

·Регион — регион, в которому принадлежат долги (см. словарь № 39);

·Статус — статус долгов для включения в отчет;

·Флаг Сортировка по ФИО — если установить данный флаг, то записи в отчете будут отсортированы по фамилии, имени, отчеству должника;

·Флаг Не показывать закрытых — если установить данный флаг, то в отчет не будут включены долги в статусах 6 "Завершено", 8 "Отозван клиентом" или 7 "Аннулировано".

3.Нажмите кнопку Запустить.

4.В открывшемся диалоговом окне Windows укажите путь и имя файла, в который будут выгружены данные, и нажмите кнопку Save (Сохранить). Будет сформирован файл MS Excel, содержащий следующие столбцы:
