Загрузка документов

Право на загрузку документов из модуля Список должников имеют пользователи, в роли которых доступ DEBT_LIST_LOAD_DOC имеет значение TRUE.

Требования к загружаемым документам

Документы, предназначенные для загрузки, должны лежать в одной папке (директории). Имя каждого файла должно включать номер договора в качестве первых символов. Если требуется включить в название файла другие символы, то после номера договора следует поставить символ "_".

Имя файла может иметь вид (примеры):

·10084007;

·10084007_Андреев_МВ_исполнительный_лист,

где 10084007 – номер договора. По номеру договора определяется ID долга.

Причины невозможности загрузить документ

Файл не может быть загружен, если:

·ID долга не определен по номеру договора;

·ID долга определен неоднозначно;

·размер файла превышает ограничения, установленные в константе DocAttachMaxSize.

Порядок загрузки документов

Для того чтобы осуществить загрузку документов:

1.Войдите в модуль Список должников.

2.Нажмите на кнопку Утилиты загрузки. В открывшемся списке выберите пункт Загрузка документов.

3.Откроется форма Загрузка документов из директории. Заполните следующие поля:

·Тип документа — см. словарь № 100; для выбора доступны только те типы вложений, доступ к которым настроен для роли пользователя в модуле Роли и доступы на вкладке Типы вложений.

·Путь к файлам — используйте кнопку для поиска папки.

4.Нажмите кнопку Загрузить документы. Откроется форма Результаты проверки загружаемых файлов. В таблице в поле Загрузка указано, какие файлы могут быть загружены, а какие нет. Если файл не может быть загружен, то в поле Причина указывается причина невозможности загрузки файла.

5.Для продолжения процесса нажмите кнопку Загрузить документы.

6.Будет показано сообщение с вопросом: "Сформировать файл ошибочных записей?". Если вы ответите Yes, то файл ошибочных записей будет сформирован. Если вы ответите No, то файл не будет сформирован. В любом случае документы, которые могут быть загружены, будут загружены.

7.Если файл ошибочных записей был сформирован, вы можете просмотреть его.
