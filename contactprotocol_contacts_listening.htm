Прослушивание и сохранение записи телефонного разговора

Внимание! Действия с записями разговоров доступны только если в Системе настроена интеграция с телефонией GENESYS.

Право на данный функционал определяется доступом № 235 CONTACT_LOG_CALL_RECORD. Только пользователи, в роли которых данный доступ имеет значение TRUE, могу прослушивать и сохранять записи телефонных разговоров.

Наличие записи телефонного разговора маркируется флагом Запись в строке, соответствующей данному телефонному контакту.

Прослушивание записи разговора

Для того чтобы прослушать запись разговора выполните следующее:

1.В модуле Протокол контактов активируйте вкладку Контакты.

2.Создайте выборку и найдите нужный контакт (строка, для которой флаг Запись включен).

3.В контекстном меню (открывается правой кнопкой мыши) выберите пункт Прослушать разговор.

4.В открывшемся списке выберите Прослушать.

Сохранение записи телефонного разговора

Для того чтобы сохранить запись разговора выполните следующее:

1.В модуле Протокол контактов активируйте вкладку Контакты.

2.Создайте выборку и найдите нужный контакт (строка, для которой флаг Запись включен).

3.В контекстном меню (открывается правой кнопкой мыши) выберите пункт Прослушать разговор.

4.В открывшемся списке выберите Сохранить.

5.Откроется стандартная форма сохранения файла. Укажите путь для сохранения и нажмите OK.

Примечание. Если для разговора выключен флаг Запись, то пункт Прослушать разговор в контекстном меню отсутствует.
