Ведение аудиозаписи разговоров при личных встречах и автоматическая аудиозапись телефонных переговоров

Мобильная версия Системы осуществляет следующие виды аудиозаписи:

·Ведение аудиозаписи разговоров при личных встречах;

·Автоматическая аудиозапись телефонных переговоров.

Ведение аудиозаписи разговоров при личных встречах

Для ведения аудиозаписи разговора выполните следующие действия:

1.Активируйте форму "Должники".

2.Выберите должника и откройте карточку должника.

3.Разверните раздел "Документы".

4.Нажмите кнопку "Добавить новый", которая относится к списку документов.

5.В открывшемся списке "Прикрепить файл" выберите тип файла (Рисунок 10‑18) "Новая запись".

6.Откроется окно с доступом к функциям звукозаписи (Рисунок 10‑20). Нажмите кнопку .

7.В окне выбора типа файла (Рисунок 10‑19) нажмите OK, так как по умолчанию для документа установлен тип "Аудиозапись".

8.Звукозапись включится. Приложение вернется в карточку должника. Запись будет идти в фоновом режиме.

9.По окончании разговора выполните остановку записи и ее сохранение. Для этого активируйте список уведомлений на устройстве, найдите информацию о текущей записи и нажмите кнопку остановки записи. Затем нажмите кнопку сохранения записи.

10.В список документов должника добавится файл с сохраненной записью.

Примечание. Чтобы увидеть добавленную запись в списке документов, выйдите из карточки должника и войдите в нее повторно.

Автоматическая аудиозапись телефонных переговоров

Запрет выхода из карточки должника без регистрации контакта

По умолчанию в Системе настроен запрет выхода оператора из карточки должника после телефонного разговора без регистрации контакта. При попытке выйти из карточки или закрыть приложение будет показано сообщение:

Невозможно закрыть карточку. Пожалуйста, зарегистрируйте контакт.

Доступна настройка разрешения на выход оператора из карточки должника после совершения звонка без регистрации контакта. В этом случае при выходе оператора из карточки должника после звонка, но без регистрации контакта, Система зарегистрирует контакт автоматически. Результат такого контакта должен быть настроен заранее, а комментарий будет иметь вид:

Создан автоматически по факту звонка.

Обратитесь в техническую поддержку, если вам нужна такая настройка.

Описание для поддержки

В настоящей версии Системы реализована связь каждой записи разговора, которая сохраняется в таблицу mobile_call_record, с результатом контакта. Для этого осуществлены следующие доработки:

1.При совершении вызова из мобильного приложения формируется уникальный идентификатор записи разговора.

2.При совершении вызова из мобильного приложения автоматически создается предварительный («технический») контакт на Клиенте. 

3.Производится проверка технической константы Android.DebtCard.DenyCloseAfterCall:

·Если значение константы «Истина», то возможность после завершения звонка пользователю не разрешается выходить из карточки должника без регистрации контакта по данному номеру.   При попытке закрытия карточки открывается сообщение с текстом:

Невозможно закрыть карточку. Пожалуйста, зарегистрируйте контакт.

·Если значение константы = «Ложь» (или константа не найдена), то ограничения не накладываются.

4.Запись разговора связывать со всеми зарегистрированными пользователем контактами по данному номеру, которые были созданы до момента первого выхода из карточки должника.

5.При передаче контактов на сервер передается уникальный идентификатор записи разговора, созданный в п.1. Связь записи разговора с зарегистрированным контактом реализуется следующим образом:

6.Если пользователем был зарегистрирован хотя бы один контакт с привязкой аудиозаписи, ранее созданный предварительный контакт (из п.2) передавать не требуется.

7.Иначе, если пользователь не зарегистрировал контакт вручную (например, завершил работу приложения), то предварительный контакт (из п.2) передается на сервер, как реальный контакт с результатом, код которого указан в значении технической константы Android.SaveCallRecord.DefaultCodeResult. По такому контакту автоматически формируется комментарий:

Создан автоматически по факту звонка.
