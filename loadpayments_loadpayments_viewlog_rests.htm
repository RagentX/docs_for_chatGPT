Данные в логе о загрузке остатков погашения

Данные в файле лога о загрузке остатков погашения формируются в следующей последовательности:

1.Данные о корректировке полей debt_sum, debt_dt, debt_exp_dt в таблице БД debt для записей с ненулевым остатком.

2.Данные о корректировке поля debt_dt в таблице БД debt для записей с пустым остатком при наличии правила расчета.

3.Данные об установке статуса 11 ("Возврат в работу") в таблице БД debt для записей с ненулевым остатком.

4.Данные об установке статуса 6 ("Завершено") в таблице БД debt для записей с нулевым остатком.

Перед корректировкой в таблице БД debt для записей с ненулевым остатком в файле лога формируется сообщение: "SaveCalc. Корректировка полей debt_sum, debt_dt, debt_exp_dt для записей с ненулевым остатком, количество корректируемых записей в таблице БД debt <количество записей>". Если константа LoadCalcBulkInsert имеет значение "Ложь", то в файле лога дополнительно формируются:

·сообщение "SaveCalc. Дополнительные данные о корректировке:";

·сообщения по каждой записи с ненулевым остатком вида "запись №<номер записи>, id debt = <ID долга>".

Перед корректировкой в таблице БД debt для записей с пустым остатком при наличии правила расчета в файле лога формируется сообщение: "SaveCalc. Корректировка поля debt_dt для записей с пустым остатком при наличии правила расчета, количество корректируемых записей в debt <количество записей>". Если константа LoadCalcBulkInsert имеет значение "Ложь", то в файле лога дополнительно формируются:

·сообщение "SaveCalc. Дополнительные данные о корректировке:";

·сообщения по каждой записи с пустым остатком при наличии правила расчета вида "запись <№номер записи>, id debt = <ID долга>".

Перед установкой статуса 11 ("Возврат в работу") в таблице БД debt для записей с ненулевым остатком в файле лога формируется сообщение: "SaveCalc. Установка статуса 11 для записей с ненулевым остатком, количество корректируемых записей в debt и work_task <количество записей>". Если константа LoadCalcBulkInsert имеет значение "Ложь", то в файле лога дополнительно формируются:

·сообщение "SaveCalc. Дополнительные данные о корректировке:";

·сообщения по каждой записи с ненулевым остатком вида "запись <№номер записи>, id debt = ID долга, id work_task = <id work_task>".

Перед установкой статуса 6 ("Завершено") в таблице БД debt для записей с нулевым остатком в файле лога формируется сообщение: "SaveCalc. Установка статуса 6 для записей с нулевым остатком, количество корректируемых записей в debt и work_task <количество записей>". Если константа LoadCalcBulkInsert имеет значение "Ложь", то в файле лога дополнительно формируются:

·сообщение "SaveCalc. Дополнительные данные о корректировке:";

·сообщения по каждой записи с нулевым остатком вида "запись №<номер записи>, id debt = <ID долга>, id work_task = <id work_task>".
