Использование данных из файла в качестве критериев фильтрации

В качестве критериев фильтрации могут быть заданы данные из файлов.

Примечание. Файл для фильтрации обязательно должен содержать шапку. В качестве эталонных значений для фильтрации всегда выбираются данные из первой колонки таблицы, начиная со второй строки и до первой пустой ячейки. Подходят файлы Excel и Excel 97-2003.

Функционал доступен в модулях:

·Список должников;

·Постановка задач;

·Юридический департамент;

·Группы свободного поиска.

Для фильтрации с использованием данных из файла выполните следующие действия:

1.Войдите в один из указанных модулей и активируйте нужную вкладку.

2.Нажмите на кнопку сложного фильтра .

3.Откроется форма Фильтр. Активируйте вкладку Загрузка из файла.

4.Откроется форма Загрузка из файла (Рисунок "Сложный фильтр, вкладка «Загрузка из файла»").

5.В первом поле условия выберите критерий фильтрации (Рисунок "Выбор критерия сортировки").

Примечание. Выбор критерия всегда должен предшествовать выбору файла.

6.Во втором поле — выберите файл при помощи стандартного окна выбора файла. Если Системе удалось прочитать данные, в поле SQL текст появится текст SQL-запроса (Рисунок "Одно условие задано").

7.Если вам необходимо более одного условия, нажмите кнопку Добавить. Появится вторая панель условия. Повторите для нее действия, описанные в пунктах 5 и 6 (Рисунок "Два условия заданы").

Примечание. Обратите внимание, что если вы используете более одного условия, то логическим оператором, который их связывает, будет И (AND).

8.Вы можете использовать столько условий, сколько вам нужно. Чтобы удалить условие, нажмите кнопку , расположенную справа от него.

9.Нажмите OK.
