Телефоны

Раздел Телефоны находится в карточке должника на вкладке Общая, вложенная вкладка Телефоны, и содержит следующие элементы:

·Список контактных телефонов должника с полями:

·ID телефона — уникальный идентификатор номера телефона должника в базе данных (по умолчанию поле скрыто);

·Тип — тип телефонного номера (см. словарь № 4);

Примечание: если включена константа CardAllPhoneShow, отображается информация по телефонам поручителей и залогодателей.

·Номер — номер телефона;

·Статус — статус телефонного номера (см. ниже);

·Посл. звонок — дата и время последнего успешного звонка должнику по данному номеру;

·Контактное лицо — фамилия, имя, отчество контактного лица;

·Перевод на автоинформатор;

·Комментарий — дополнительная информация;

·Блокирован — см. Блокирование/разблокирование контактного телефона;

·Регион — регион, к которому относится телефон (см. словарь № 39);

·Активен — признак активности (активным считается телефон, у которого этот флаг включен либо NULL);

·Код качества — поле может принимать значение (см. словарь № 114) в зависимости от последнего результата, зарегистрированного по данному номеру. Код качества может меняться при регистрации новых контактов. Связь результата с кодом качества настраивается в модуле "Настройка дерева контактов". Коды качества используются при выгрузке списка телефонов во внешний автодайлер для сегментации;

·Номер существует;

·Связан;

·Верифицирован — см. Верификация контактных данных;

·Дата блокировки — заполняется только для заблокированных телефонов;

·Причина блокировки — заполняется только для заблокированных телефонов.

·Кнопки:

· Добавить — добавление телефона;

· Изменить — изменение телефона;

· Удалить телефон — удаление телефона;

· Отправить SMS — формирование SMS должнику.

· Установить связь с долгом — см. Связь телефонов с долгом (долгами).

·Флаг Верификация контактных данных — см. Ручная верификация данных.

Окраска телефонов в зависимости от статуса

Строки в списке телефонов выделены цветом фона в зависимости от статуса контактного телефона (см. Статусы телефонов):

·Не звонили ни разу — серый цвет.

·Результата не было — белый цвет.

·Неверный номер — красный цвет.

·Результат — зеленый цвет .

·Последний результат — голубой цвет.

·Автоинформатор — розовый цвет.

Если у пользователя есть доступ 127 DEBT_CARD_ALL_PHONE, то в карточке должника он может видеть заблокированные номера телефонов. Такие записи в списке телефонов выделены серым шрифтом на белом фоне.

Доступные действия

·Добавление, редактирование, удаление номера — см. Ведение списка контактных телефонов должника.

·Блокирование / разблокирование номера — см. Блокирование/разблокирование контактного телефона.

·Просмотр, добавление или удаление связи телефона с долгом — см. Связь телефонов с долгом (долгами).

·Копирование телефонного номера или строки списка телефонов

·Для копирования выбранного номера телефона в буфер обмена выберите в контекстном меню (вызывается правой кнопкой мыши) пункт Копировать номер. При этом скопированный номер сохраняется в буфер обмена.

·Для копирования всей выбранной строки в буфер обмена выберите в контекстном меню (вызывается правой кнопкой мыши) пункт Копировать строку. При этом в буфер обмена сохраняется вся строка таблицы списка телефонов.
