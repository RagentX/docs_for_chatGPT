Единая форма "Имущество"

Форма включает следующие элементы:

·Основная информация о залогодателе:

·ФИО залогодателя — фамилия, имя, отчество залогодателя ;

·Тип — словарь № 54;

·ID Клиента — идентификатор клиента, который является залогодателем;

·Вкладка с общей информация о залогодателе в зависимости от его типа:

·Физическое лицо — для физических лиц с полями:

oДата рождения;

oПол;

oСемейное положение;

oКомпания;

oДолжность;

oОбразование;

oПаспортные данные;

oМесто рождения;

·Юридическое лицо — для юридических лиц с полями:

oИНН;

oКПП;

oФИО ген. директора;

oПлатежные реквизиты;

·Юридическое лицо — для ИП с полями:

oИНН;

oФИО;

oПлатежные реквизиты;

·Вкладка Финансовая информация с полями:

·Номер договора обеспечения;

·Дата заключения — дата заключения договора обеспечения;

·Дата окончания — дата окончания действия договора обеспечения;

·Дата завершения;

·Залоговая стоимость — залоговая стоимость имущества. Поле обязательно для заполнения;

·Рыночная стоимость — рыночная стоимость имущества;

·Оценочная стоимость — стоимость имущества, установленная в процессе оценки, самостоятельно не редактируется;

·Валюта — валюта, в которой выражена стоимость имущества;

·Коэффициент качества — коэффициент качества;

·Тип имущества — словарь № 85;

·Статус — словарь № 86;

·Наименование объекта — наименование объекта залога в произвольном виде;

·Область данных с телефонами, адресами и комментарием:

·Телефоны — телефоны залогодателя. Для работы со списком телефонов залогодателя используются те же функции, что и для списка телефонов должника (см. Ведение списка контактных телефонов залогодателя);

·Адреса — адреса залогодателя. Для работы со списком адресов залогодателя используются те же функции, что и для списка адресов должника (см. Ведение списка адресов залогодателя);

·Email — адреса электронной почты залогодателя. Для работы со списком адресов электронной почты залогодателя используются те же функции, что и для списка адресов электронной почты должника (см. Ведение списка адресов электронной почты залогодателя);

·Комментарий — дополнительная информация.
