Судебная работа по связанным лицам

Судебная работа по связанным лицам выполняется по кнопке Судебная работа. Данная кнопка отображается при открытии карточки связанного лица для редактирования. При создании связанного лица эта кнопка не отображается.

Меню кнопки Судебная работа в карточке связанного лица включает следующие пункты:

·Создать суд. дело (предв.) — создается карточка неопределенного судебного дела по выбранномк связанному лицу (подробнее см. Создание неопределенного судебного дела). Пункт доступен пользователям, в роли которых доступ № 813 LAW_ACT_LINKED_PERSON_CREATE и доступ № 58 LAW_ACT_CREATE имеют значение "Истина".

·Создать судебный иск — создается карточка судебного иска по выбранномк связанному лицу (подробнее см. Карточка судебной работы в исковом производстве). Пункт доступен пользователям, в роли которых доступ № 813 LAW_ACT_LINKED_PERSON_CREATE и доступ № 58 LAW_ACT_CREATE имеют значение "Истина".

·Создать судебный приказ — создается карточка судебного приказа по выбранномк связанному лицу (подробнее см. Карточка судебной работы в приказном производстве). Пункт доступен пользователям, в роли которых доступ № 813 LAW_ACT_LINKED_PERSON_CREATE и доступ № 58 LAW_ACT_CREATE имеют значение "Истина".

·Создать правопреемство — создается карточка правопреемства по выбранномк связанному лицу (подробнее см. Карточка судебной работы в правопреемстве). Пункт доступен пользователям, в роли которых доступ № 813 LAW_ACT_LINKED_PERSON_CREATE и доступ № 58 LAW_ACT_CREATE имеют значение "Истина", если константа LawActUseSuccession равна "Истина".

·Исполнительное производство — создается карточка исполнительного производства по выбранномк связанному лицу (подробнее см. Карточка исполнительного производства). Пункт доступен пользователям, в роли которых доступ № 814 LAW_EXEC_LINKED_PERSON_CREATE, а также доступы №№ 167 и 623  имеют значение "Истина".

·Зарегистрировать входящий иск — создается входящий иск. Пункт доступен пользователям, в роли которых доступ № 815 LAW_ACT_LINKED_PERSON_CREATE_INNER имеет значение "Истина".

Подробнее см. описание Судебная работа.
