Загрузка ФССП из Excel-файла

Условия доступности

Кнопка Загрузка ФССП видна на вкладке ФССП и приставы и доступна пользователям, в роли которых доступ № 663 LAW_JUDGE_SET_LAW_COURT_LOAD имеет значение TRUE.

Порядок действий при загрузке ФССП

Для того чтобы осуществить загрузку судов, выполните следующие действия:

1.Войдите в модуль Настройка судов и ФССП. Активируйте вкладку ФССП и приставы.

2.Нажмите кнопку Загрузка ФССП. Откроется стандартное окно для выбора файла.

3.Выберите файл и нажмите кнопку Открыть (Open).

4.Будет осуществлена проверка соответствия выбранного файла шаблону загрузки ФССП. При этом проверяется количество столбцов и наличие данных в файле:

·в случае несовпадения формата файла с шаблоном пользователю выводится сообщение следующего вида:

Файл <NAME> не соответствует требуемому формату. Количество столбцов не совпадает с шаблоном,

где <NAME> — полное имя файла;

·в случае отсутствия в файле данных для загрузки пользователю выводится сообщение:

Список ФССП для загрузки пуст

5.Если выбранный файл не соответствует шаблону, его загрузка не осуществляется, действие прекращается.

6.Затем открывается окно Предварительный просмотр с кнопками:

·Загрузка в БД — изначально неактивна (для загрузки судов в БД);

·Проверить — активна (для проверки записей перед загрузкой в БД);

·Excel — изначально неактивна (для сохранения excel-файла с ошибочными записями).

7.Нажмите кнопку Проверить. Будет осуществлен поиск ошибочных записей в загружаемом файле.

8.Если ошибочных записей нет, то появится сообщение:

Проверка возможности загрузки судов прошла успешно.

Нажмите ОК. В форме Предварительный просмотр станет доступна кнопка Загрузка в БД. Нажмите на нее, и загрузка осуществится. По ее результату появится сообщение:

Записи, прошедшие проверку успешно загружены в БД.

9.Если в результате проверки найдены ошибочные записи, то будет показано сообщение:

Найдены строки, которые не подлежат загрузке: подробнее см. столбец "Причина невозможности загрузки".

Нажмите ОК. После этого в форме Предварительный просмотр станут доступны кнопки Загрузка в БД и Excel. При этом в таблице формы будут добавлены две колонки:

·Возможность загрузки — в столбце отображается "Да", если запись не является ошибочной, иначе — "Нет";

·Причина невозможности загрузки — в столбце отображается соответствующая причина только для тех записей, для которых в столбце "Возможность загрузки" выводится "Нет" (см. список ошибок при загрузке ФССП далее).

10.Нажмите кнопку "Загрузка в БД". При этом появится сообщение:

Загружены будут только записи, прошедшие проверку. Сформировать файл ошибочных записей?

Если вы хотите сформировать файл ошибочных записей, нажмите Yes, иначе нажмите No.
