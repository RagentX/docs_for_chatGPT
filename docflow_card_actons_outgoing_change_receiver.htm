Изменение получателей исходящего документа

Show/Hide Hidden Text

Изменение получателей исходящего документа доступно только пользователя, в роли которых доступ № 679 DOCFLOW_CARD_OUTGOING_RECEIVER_CHANGE имеет значение TRUE.

получатель исходящего документа и отправитель входящего документа может быть выбран из списка должников, судов, ФССП, а также из списка обычных физических и юридических лиц (не должников) с возможностью создания новых лиц.

Изменение получателей документа осуществляется при помощи кнопки Изменить получателей, расположенной справа от списка получателей. После нажатия кнопки открывается форма выбора получателей.

Описание формы выбора получателей исходящего документа

Форма выбора получателей включает следующие элементы:

·Кнопка Добавить со следующими подпунктами:

·Суд — для добавления суда. При выборе открывается форма добавления суда. После добавления суд будет доступен также в модуле Настройка судов и ФССП, а также в других модулях Системы;

·ФССП — для добавления ФССП. При выборе открывается форма добавления ФССП. После добавления ФССП будет доступна также в модуле Настройка судов и ФССП, а также в других модулях Системы;

·Физ. / Юр. лицо — для добавления персоны: физического или юридического лица, которое не является должником;

·Расшир.поиск физ. / юр. лица — для расширенного поиска персоны: должника или не должника, физического или юридического лица;

·Поле ввода — для поиска получателя;

·Кнопка Очистить — для очистки поля ввода;

·В левой части формы находится таблица со списком возможных получателей;

·В правой части формы находится таблица с результирующим списком получателей;

·Кнопка Добавить — для добавления получателя в результирующий список;

·Кнопка Удалить — для удаления получателя из результирующего списка;

·Кнопка ОК — для сохранения созданного списка получателей. При нажатии на нее в документе сохраняется список получателей, отображенный в таблице с результирующим списком.

Выбор получателей из списка возможных получателей

Выберите запись в области списка возможных получателей и нажмите кнопку Добавить или выполните двойной клик мышью. Выбранная запись будет добавлена в результирующий список.

Удаление получателя из списка получателей

Выберите запись в списке получателей и нажмите кнопку Удалить. Запись будет удалена из списка получателей.

Поиск получателя с помощью поля ввода

При вводе символов в поле ввода Система осуществляет поиск возможного получателя, полное имя или название которого включают введенные символы. Поиск ведется среди судов, ФССП, персон в списке возможных получателей. В результате поиска в списке возможных получателей остаются только записи, удовлетворяющие условиям поиска.

Далее выполните выбор получателей из списка возможных получателей.
