Предварительная настройка констант

Перед началом работы необходимо в модуле "Значения констант" настроить константы, влияющие на работу модуля. В случае отсутствия констант в БД при входе в форму Утилита определения ID долга будут выданы соответствующие предупреждения: "Отсутствует константа <имя константы>".

Название константы

Описание настройки

UtilGetIdSelectDebtMode

Числовая константа, может принимать значение "0" или "1". Константа определяет способ выбора альтернативного долга.

В случае значения 0 выбирается долг с наименьшим ID.

В случае значения 1 выбор долга осуществляется в два этапа. Вначале выполняется поиск активного долга с наименьшим ID (под активным понимается не завершенный, не отозванный и не аннулированный долг). При отсутствии активных долгов выбирается долг с наименьшим ID

UtilGetIdOutputFormat

Числовая константа, задает формат выгрузки результата определения ID долга. Может принимать значения:

·0 – результат выгружается в два различных файла для успешных и неуспешных записей (установлено по умолчанию);

·1 – результат выгружается в один файл.
