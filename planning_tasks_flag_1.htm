Работа флага "Показывать завершенные задачи"

Если флаг Показывать завершенные задачи включен, то в списке отображаются все задачи во всех статусах, доступные текущему пользователю и удовлетворяющие условиям, заданным в мультифильтрах.

По умолчанию флаг выключен.

Если флаг выключен, то в списке отображаются только задачи в активных статусах. Обычно активными статусами для оператора и постановщика считаются следующие статусы:

·Для задач, в которых текущий пользователь обладает полномочиями оператора, следующие статусы являются активными:

§Поставлена;

§В работе;

§Отложена;

§Возврат в работу.

·Для задач, в которых текущий пользователь обладает полномочиями постановщика, следующие статусы являются активными:

§Поставлена;

§В работе;

§Отклонена;

§Отложена;

§Готова к проверке;

§Возврат в работу.

Самостоятельная настройка активных статусов задач

Существует возможность настройки активных статусов. Для этого настройте их в значении константы DebtTask.ActiveStatusList. Формат: код уровня доступа (1 - оператор, 2 - постановщик), двоеточие, список статусов задач через запятую по словарю № 275 и затем полученный текст заключается в круглые скобки.

Для стандартной реализации, описанной выше, использовано значение этой константы по умолчанию: (1:1,2,5,7)(2:1,2,3,5,6,7).
