Создание, редактирование и удаление словарей

Новые словари создаются, как правило, при установке и настройке системы. Словарь может быть системным или общим. Системные словари создаются разработчиком. Общие словари — создаются и заполняются пользователями.

 Внимание! Не рекомендуется самостоятельно менять системные словари.

Создание словаря

В форме Работа со словарями нажмите кнопку , относящуюся к списку словарей. Откроется форма редактирования атрибутов словаря.

1.В форме заполните следующие поля:

·Код словаря — системный номер словаря. Поле обязательно для заполнения. Код можно ввести вручную или сформировать автоматически. 
Если поле не заполнено вручную, код будет сформирован автоматически при сохранении созданного словаря. 
Если при ручном заполнении ввести в поле уже существующий код, будет выдано соответствующее предупреждение. Чтобы присвоить значение автоматически, нажмите в нем кнопку Yes. Чтобы повторно ввести код вручную, нажмите No.

·Название словаря — поле обязательно для заполнения;

·Родительский словарь — родительский словарь, которому будет подчинен создаваемый словарь. Используется при создании иерархического словаря.

·Тип словаря — группа, к которой относится словарь (системный/общий). Поле обязательно для заполнения.

2.Нажмите кнопку ОК, чтобы сохранить словарь в списке.

3.Сформируйте перечень терминов для созданного словаря (см. Управление списком терминов словаря).

Редактирование словаря

1.В форме Работа со словарями выберите нужную запись в списке словарей и нажмите относящуюся к нему кнопку  или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.В форме редактирования атрибутов словаря измените необходимые атрибуты (см. Создание словаря).

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Для того чтобы изменить перечень терминов словаря, см. Управление списком терминов словаря.

Удаление словаря

 Внимание! Удаление системных словарей может привести к нарушению функционирования системы.

Для того чтобы удалить словарь, выполните следующие действия:

1.В форме Работа со словарями выберите запись в списке словаре и нажмите относящуюся к нему кнопку .

2.В открывшемся диалоговом окне нажмите в кнопку Yes, чтобы подтвердить удаление.
