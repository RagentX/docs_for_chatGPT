Создание нового шаблона входящего документа

Для создания нового шаблона входящего документа выполните следующие действия:

1.В модуле Документооборот активируйте вкладку Входящие.

2.Нажмите кнопку Создать документ и в открывшемся меню выберите пункт Настройка шаблонов.

3.Откроется форма Шаблоны документов. В этой форме нажмите кнопку Создать новый шаблон.

4.Откроется форма Шаблон входящего документа. Заполните нужные поля:

·Название — введите название шаблона (поле обязательно для заполнения);

Остальные поля соответствуют полям вкладки Основная формы Карточка входящего документа и являются необязательными для заполнения. Подробнее — см. Вкладка "Основная" входящего документа:

·Получатели;

·Наименование документа — см. словарь № 10002.

Примечание. Поле Наименование документа отображается по согласованию с технической поддержкой.

Словарь № 10002 "Наименование документа" имеет в качестве родительского словарь № 332 "Тип документа". После того, как в поле Наименование документа выбрано значение, поле Тип документа автоматически заполняется термином, который является родительским по отношению к выбранному термину словаря № 10002. При заполнении поля Тип документа ранее введенное значение в поле Наименование документа сбрасывается. Если у данного column_spec (id = 2067) стоит признак is_not_use = 1, то требуется скрывать поле с формы (работа поля аналогично данному полю в карточке документа).

·Тип документа;

·Форма;

·Способ доставки;

·Флаг Без конверта;

·Отправитель;

·Приоритет;

·Дата исходящего;

·Дата подписного листа;

·Дата оригинала;

·Содержание;

·Комментарий.

5.Нажмите ОК.

Форма "Шаблон входящего документа".
