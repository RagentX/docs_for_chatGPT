Добавление и редактирование вложений доступно пользователям, в роли которых доступ № 15 DEBT_CARD_DOC_ADD_EDIT имеет значение "Истина".

Если в роли пользователя доступ № 15 DEBT_CARD_DOC_ADD_EDIT имеет значение "Ложь", то право на добавление и редактирование вложений определяется следующими доступами:

·Добавление вложений разрешено пользователям, в роли которых доступ № 829 DEBT_CARD_ATTACH_ADD имеет значение "Истина".

·Редактирование вложений разрешено пользователям, в роли которых доступ № 830 DEBT_CARD_ATTACH_EDIT имеет значение "Истина".

Если в роли пользователя доступ № 823 DEBT_CARD_ATTACH_ADD_TO_CLOSED имеет значение "Истина", то пользователь может добавить файл к долгу, который находится в закрытом статусе 6,7 или 8. Если доступ имеет значение "Ложь", то добавление файлов к закрытым долгам запрещено.

Удаление вложений доступно пользователям, в роли которых доступ № 267 DEBT_CARD_DOC_DEL имеет значение "Истина".

Примечание. Если в системе включен функционал работы с документами "По папкам", то внешний вид формы и список доступных действий отличаются. Подробности — см. описание Работа с документами "По папкам". Функционал включается по согласованию с вендором! Обратитесь в поддержку.

Форма добавления документа (карточка документа)

Карточка документа

Добавление файла документа

В модуле Карточка должника можно прикреплять файлы документов, размер которых не превышает величину, заданную константой DocAttachMaxSize.

Если в вашей версии поддерживается механизм сканирования, то доступно также Прикрепление документов с использованием сканера (Вопросы о возможности поддержки механизма сканирования решайте путем обращения в техническую поддержку).

Для того чтобы прикрепить файл документа, выполните следующие действия:

1.В модуле Карточка должника на вкладке "Вложения" нажмите кнопку .

2.В открывшейся форме заполните следующие поля:

·Название — название документа. Поле обязательно для заполнения. См. Примечание ниже.

Для выбора документа нажмите кнопку . В открывшемся диалоговом окне Windows выберите файл и нажмите кнопку Open (Открыть).

·Прикрепить к — выберите объект, к которому будет прикреплен документ. Данное поле отображается, если включен ф-л прикрепления файлов к объектам Системы. Подробнее — см. Логика прикрепления вложений и ссылок к объектам Системы.

·Тип документа — см. словарь № 100 (для выбора доступны только те типы вложений, доступ к которым настроен для роли пользователя в модуле Роли и доступы на вкладке Типы вложений). Если значение константы DebtCardUseDocAttachMandatoryType равно "Истина", то поле является обязательным для заполнения;

·Номер — номер документа;

·Техническое поле типа "Словарь" 1 — дополнительное поле типа "Словарь". Данное поле отображается по согласованию с технической поддержкой, так же, как и словарь, который соответствует данному полю. Наименование поля также согласовывается с поддержкой.

·Техническое поле типа "Дата" 1 — дополнительное поле типа "Дата". Данное поле отображается по согласованию с технической поддержкой. Наименование поля также согласовывается с поддержкой.

3.Нажмите кнопку ОК. Документ будет загружен в систему и добавлен в список.

Примечание. Названия для прикрепляемых файлов.

1.Если в Системе настроено автоматическое именование файлов вложений, то поле Название не доступно для редактирования. Подробнее см. Логика прикрепления вложений и ссылок к объектам Системы, раздел Правило именования файлов.

2.Если автоматическое именование файлов не настроено, то:

·Если при нажатии на кнопку "Добавить документ" поле Название пусто, то в него автоматически вводится название файла.

·Если при нажатии на кнопку "Добавить документ" поле Название заполнено, то введенное название сохраняется.

Примечание. Значение атрибута Стадия при прикреплении документа или ссылки на документ заполняется автоматически терминами словаря № 132 Стадия взыскания в зависимости от того, в каком модуле было осуществлено прикрепление документа:

·Карточка должника — термин № 1 "Досудебная";

·Юридический департамент, вкладки судебной работы — термин № 2 "Судебная работа";

·Карточка судебной работы —  термин № 2 "Судебная работа";

·Карточка исполнительного производства — термин № 3 "Исполнительное производство".

Прикрепление документов к долгу или должнику:

Прикрепление документов к долгу или к должнику определяется значением константы DebtCardAttachDocsToDebt:

·Если значение  равно "Истина", то в форме добавления документа по умолчанию выбран объект "Долги" — активный долг.

·Если значение  равно "Ложь", то в форме добавления документа по умолчанию выбран объект "Должники" — данный должник.

Просмотр файла документа

1.В модуле Карточка должника на вкладке "Вложения" в списке файлов документов выполните двойной щелчок левой клавишей мыши по нужной записи. Откроется файл документа.

2.По окончании просмотра закройте файл средствами соответствующего приложения.

Примечание: при попытке редактирования файла изменения в нем сохранены не будут. Порядок редактирования файлов описан в разделе Редактирование файла документа.

Прослушивание аудиозаписей

Документы, являющиеся аудиозаписями и имеющие тип "Аудиозапись", доступны для прослушивания стандартными средствами операционной системы.

Редактирование файла документа

Открытый документ может быть изменен. Для этого исправленный документ следует сохранить под другим именем.

Для изменения документа выполните следующие действия:

1.В модуле Карточка должника на вкладке "Вложения" в списке файлов документов выполните двойной щелчок левой клавишей мыши по нужной записи. Откроется файл документа.

2.В приложении выполните команду Сохранить как и сохраните файл на компьютер пользователя.

3.Отредактируйте сохраненный файл документа, сохраните и закройте его средствами соответствующего приложения.

4.Заново добавьте файл в систему, как описано в разделе Добавление файла документа.

Редактирование атрибутов файла документа

Для того чтобы изменить атрибуты прикрепленного файла документа, выполните следующие действия:

1.В модуле Карточка должника на вкладке "Вложения" выберите нужную запись и нажмите кнопку .

Примечание: двойной щелчок мыши по выбранной записи приведет к открытию файла документа, а не к открытию карточки документа.

Для выбора другого документа нажмите кнопку . В открывшемся диалоговом окне Windows выберите файл и нажмите кнопку Open (Открыть).

2.В открывшейся карточке документа измените необходимые атрибуты.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление файла документа

1.В модуле Карточка должника на вкладке "Вложения" нажмите выберите нужную запись в списке файлов документов.

2.Нажмите кнопку , относящуюся к списку файлов документов.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.

Выгрузить файл

1.В модуле Карточка должника на вкладке "Вложения" в списке файлов документов выберите документ.

2.Нажмите кнопку Выгрузить файл.

3.В открывшемся диалоговом окне укажите путь и имя файла, в который будут выгружены данные, и нажмите кнопку Save (Сохранить). По окончании выгрузки будет выдано сообщение о завершении создания файла.

4.Нажмите кнопку ОК, чтобы завершить процедуру.
