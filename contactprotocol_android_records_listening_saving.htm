Прослушивание и сохранение записей телефонных разговоров, совершенных с помощью мобильной версии

Если для контакта типа "Звонок" на вкладке "Android" включен флаг "Запись", то в контекстном меню (вызывается правой кнопкой мыши) для такого контакта доступен пункт Прослушать разговор с подпунктами:

·Сохранить;

·Прослушать.

Примечание. Данный пункт доступен только если выбрана строго одна запись в списке.

Прослушивание записи

При выборе подпункта Прослушать открывается файл записи соответствующего разговора. Файл воспроизводится во внешнем приложении, установленном по умолчанию для данного типа файлов.

Если соответствующая запись разговора не найдена, то выводится окно ошибки с текстом:

Ошибка при получении файла для прослушивания.

Сохранение записи

При выборе подпункта Сохранить сначала Система ищет соответствующий файл. Если файл найден, то открывается стандартная форма операционной системы для сохранения файла в указанную директорию.

Если файл не найден, то выводится окно ошибки с текстом:

Ошибка при получении файла для прослушивания.
