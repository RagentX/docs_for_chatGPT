Понедельный график работы

Форма редактирования атрибутов графика рабочего времени открывается при его создании (см. Создание графика рабочего времени). Ниже представлена форма редактирования понедельного графика работы. Выполните следующие действия, чтобы сформировать понедельный график работы:

1.Выберите день недели в списке.

2.В поле Название укажите название графика рабочего времени. Поле обязательно для заполнения.

3.В поле Тип графика укажите значение "Понедельный".

4.Сформируйте список интервалов рабочего времени для каждого дня недели. Для этого:

a.Выберите день недели в списке.

b.Задайте количество интервалов рабочего времени в выбранном дне. По умолчанию для каждого дня отображается два интервала: 9.00-13.00 и 14.00-18.00, каждый интервал является основным. Для добавления интервала нажмите значок , для удаления последнего интервала в списке нажмите значок . Максимальное количество интервалов — пять.

c.Задайте время начала (поле С:) и окончания (поле По:) для каждого интервала.

d.Установите флаг Доп. для тех интервалов, где рабочее время является дополнительным (не основным).

e.Повторите шаги a-d для каждого дня недели. При переходе к редактированию следующего для, а также при сохранении графика осуществляется автоматический контроль значений интервалов. В случае ошибочного задания интервала или пересечения соседних интервалов (начальное время последующего интервала превышает конечное время предыдущего интервала) в поле Итого: выводится предупреждение вида: "Понедельник, интервал 2. Конечное время не превышает начальное" или "Вторник. Интервалы 1 и 2 пересекаются".

5.При необходимости установите флаги контроля рабочего времени операторов:

·Контроль прихода на работу — если установлен этот флаг, то будет выполняться контроль прихода на работу;

·Контроль отработки за день — если установлен этот флаг, то будет выполняться контроль отработанного времени за день;

·Контроль отработки за неделю — если установлен этот флаг, то будет выполняться контроль отработанного времени за неделю.
