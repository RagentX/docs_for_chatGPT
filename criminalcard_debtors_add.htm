Добавление должника, связанного с уголовным преследованием

Для того чтобы добавить данные о должнике, связанном с уголовным преследованием, выполните следующие действия:

1.На вкладке Договоры нажмите кнопку .

2.В открывшейся форме в разделе Поиск задайте условия поиска (подробнее см. Поиск должника) и нажмите кнопку . Общее количество найденных записей будет выведено внизу раздела, а сами найденные записи будут отображаться в списке Должники. Для выполнения нового поиска можно очистить значения в разделе Поиск, нажав кнопку .

3.В списке Должники выберите нужную запись и нажмите кнопку ОК. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL. Выбранная запись будет добавлена на вкладку Договоры. Ответственному оператору и его прямому руководителю будет направлено соответствующее письмо.
