Печать документов юридического департамента

Печать документов юридического департамента доступна в форме Юридический департамент пользователям, в роли которых доступ № 116 REP_LAW_EXEC_APPEL.

Документы судебной работы доступны на вкладках:

·Приказы;

·Правопреемство;

·Иски;

·Банкротство;

·Судебные дела (предв.);

·Архив судебной работы.

Для вкладок судебной работы открывается список документов судебной работы, созданных в модуле Конструктор документов судебной работы и относящихся к форме Судебная работа (см. словарь № 79). Из полного списка (см. словарь № 33) пользователю доступны для выбора только те печатные формы документов судебной работы, которые удовлетворяют следующим условиям:

·доступ к документам настроен в роли пользователя в модуле Роли и доступы на вкладке Печатные формы (Суд);

·если константа Law.PrintForm.BankRestriction имеет значение "Истина", то по долгу доступны для формирования только те документы судебной работы, которые привязаны к банку, которому принадлежит этот долг.

Документы исполнительного производства доступны на вкладках:

·Исполнительные документы;

·Исполнительное производство;

·Архив ИП.

Для вкладок исполнительного производства открывается список документов исполнительного производства, созданных в модуле Конструктор документов судебной работы и относящихся к форме Исп. производство (см. словарь № 79). Из полного списка (см. словарь № 48) пользователю доступны для выбора только те печатные формы документов исполнительного производства, доступ к которым настроен в его роли в модуле Роли и доступы на вкладке Печатные формы (ИП).

Для печати документов выполните следующие действия:

1.В форме Юридический департамент на нужной вкладке выберите дело. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL.

2.В контекстном меню (вызывается правой кнопкой мыши) выберите пункт Сформировать документы по шаблону. В открывшемся меню выберите вариант выполнения действия: По всей выборке или Только выделенные.

3.Откроется форма Параметры запуска. Заполните следующие поля:

·Тип документа — выберите шаблон документа в раскрывающемся списке.

Примечание. Размер формы раскрывающегося списка может быть увеличен. Для этого потяните за элемент расположенный в правом нижнем углу списка.

·Тип адреса — установите флажки для типов адресов, на которые необходимо сформировать документы (см. словарь № 5).

·Флаг Регистрировать письмо — если флаг включен, документ будет зарегистрирован в базе данных, запись о нем будет доступна на вкладке Контакты в карточке судебной работы или исполнительного производства соответственно, а также в модуле Карточка должника на вкладке Контакты.

Примечание. Значение данного флага по умолчанию определяется:

·для документов судебной работы — константой LawActRegLetterDefaultCheckState.

·для документов исполнительного производства — константой LawExecRegLetterDefaultCheckState.

·Флаг Исключить дублирование адресов — признак определяет, будет ли осуществляться исключение дублирования адресов. По умолчанию флаг включен.

·Флаг Сохранять копию письма — если этот флаг включен, то копия письма будет сохранена в БД.

·Флаг Использовать адрес подачи:

·Данный флаг выводится на форме только если константа LawAddressLinkUseMode принимает значение отличное от 0.

·При активации данного режима работы выполняется следующее:

·Заблокирована возможность выбора адреса в блоке Тип адреса.

·Заблокирована возможность выбора режима Исключить дублирование адресов.

·Значение чекбокса по умолчанию Отключено, однако последнее пользовательское значение записывается в пользовательские настройки.

·Если выбран данный режим работы, то для каждого выбранного дела в качестве адреса для печати используется адресе подачи. Если для дела не удалось определить адрес, то он исключается из выборки.

·Флаг Автовыбор шаблона:

·При активации данного режима работы заблокирована возможность выбора шаблона в выпадающем списке Тип документа.

·Значение чекбокса по умолчанию Отключено, однако, последнее пользовательское значение записывается в пользовательские настройки.

·После указания параметров, для каждого выбранного дела определяется печатный шаблон.

·Если для дела не удалось определить шаблон, то дело исключается из выборки.

·По оставшимся делам формируются документы.

Примечание. Флаг Автовыбор шаблона выводится на форме по согласованию с технической поддержкой. Флаг Автовыбор шаблона выводится на форме только если техническая константа Law.PrintForm.AutoSetType принимает значение Истина.

4.Нажмите кнопку ОК. Документы будут сформированы и открыты в режиме предварительного просмотра в приложении Fast Report.

5.Чтобы распечатать документ, нажмите кнопку в верхней части формы с документом (в приложении Fast Report).

Примечание. документ представляет собой шаблон, в который вставлены данные из выделенной строки. Перечень шаблонов формируется в модуле Конструктор документов. Если выбранный шаблон отсутствует в базе данных, то появится соответствующее сообщение.
