Добавление запроса

Для того чтобы добавить данные о запросе, выполните следующие действия:

1.В карточке исполнительного производства на вкладке Запросы нажмите кнопку .

2.В открывшейся форме заполните следующие поля:

·Тип запроса — см. словарь № 44. Поле обязательно для заполнения;

·Предмет — предмет, к которому относится запрос, см. словарь № 46 (доступность поля определяется подчиненостью соотвествующих словарей);

·Ответ на запрос — ответ на запрос судебного пристава, см. словарь № 45 (доступность поля определяется подчиненостью соотвествующих словарей);

·Фактич. дата запроса — фактическая дата запроса;

·Дата регистрации запроса — по умолчанию указана текущая дата;

·Дата ответа;

·Название — уточняющее название предмета запроса (например, марка машины). Поле обязательно для заполнения;

·Сумма — примерная стоимость предмета запроса;

·Трек-номер — поле отображается, если включена интеграция с почтой России, особенности обработки поля — см. далее);

·Регистрационный номер, Год выпуска — регистрационный номер и год выпуска, если в качестве предмета выбрана машина;

·Краткая характеристика — краткая характеристика предмета запроса.

Примечание. В зависимости от типа и предмета запроса, поля формы могут изменяться.

Примечание. По согласованию с технической поддержкой доступно переименование полей "Фактич. дата запроса" и "Дата регистрации".

Стринги:

·Фактич. дата запроса — law_exec_request.request_dt.

·Дата регистрации запроса — law_exec_request.dt.

3.Нажмите кнопку ОК.

4.Выполните прикрепление документа.

Примечание. Если при создании запроса вы прикрепите к запросу один из существующих документов, этот документ не будет отображаться в списке документов запроса.

Если вы прикрепите новый документ, то он прикрепится к должнику или к долгу (в зависимости от значения DebtCardAttachDocsToDebt) и будет отображаться как в списке документов запроса, так и на вкладке Вложения всех карточек, связанных с данным должником или долгом.

Примечание. Реализована возможность устанавливать обязательность прикрепления документа к выбранным типам запросов.

5.Далее осуществляется формирование настроенных запланированных действий.

6.Запись о запросе будет добавлена в список.

Обработка трек-номера при добавлении запроса

Поле Трек-номер отображается на форме, если в Системе включена интеграция с почтой России (Коммерческая константа @PostRussia.Use).

При нажатии кнопки ОК при добавлении запроса, если поле Трек-номер непустое, для него выполняются следующие проверки:

·если в поле Трек-номер введено значение, содержащее буквенные символы (т.е. не только цифры) или введенная строка меньше 14 символов, пользователю выводится сообщение с текстом:

Трек-номер не соответствует внутрироссийскому формату почтового идентификатора. Продолжить?

с вариантами "Да", "Нет".

·при нажатии кнопки "Да" введенное значение сохраняется в новой записи таблицы, хранящей письма и их статусы.

·при нажатии кнопки "Нет" пользователь "возвращается" на форму "Запрос".

·Иначе, т.е. если в поле Трек-номер строка из 14 цифр, то введенное значение сохраняется в новой записи таблицы, хранящей письма и их статусы.

Действие протоколируется. Дополнительно сохраняется номер запроса.

Особенности доступности полей "Тип запроса", "Предмет" и "Ответ на запрос" в зависимости от подчиненности соответствующих словарей

Доступность полей определяется подчиненностью словарей, содержащих значения для этих полей.

·Если словари 46, 45 являются дочерними по отношению к словарю 44, то поля «Предмет» и «Ответ на запрос» становились доступными одновременно после заполнения поля «Типа запроса».

·Если словарь 46 является дочерним по отношению к словарю 44, а словарь 45 является дочерним по отношению к словарю 46, то:

·пока не выбран «Тип запроса», в выпадающих списках «Предмет» и «Ответ на запрос» нет вариантов для выбора;

·после выбора «Типа запроса» в выпадающем списке «Предмет» для выбора выводятся дочерние термины выбранного «Типа запроса» по словарю 46;

·после выбора «Предмета» в выпадающем списке «Ответ на запрос» для выбора выводятся дочерние термины выбранного «Предмета» по словарю 45.

Если словарь 45 является дочерним по отношению к словарю 44, а словарь 46 является дочерним по отношению к словарю 45, то:

·пока не выбран «Тип запроса», в выпадающих списках «Предмет» и «Ответ на запрос» нет вариантов для выбора;

·после выбора «Типа запроса» в выпадающем списке «Ответ на запрос» для выбора выводятся дочерние термины выбранного «Типа запроса» по словарю 45;

·после выбора «Ответа на запрос» в выпадающем списке «Предмет» для выбора выводятся дочерние термины выбранного «Ответа» по словарю 46.
