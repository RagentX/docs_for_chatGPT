Описание формы "Карточка должника"

Форма "Карточка должника" (Рисунок 10‑1) включает следующие элементы:

·Кнопка — для возвращения на предыдущую форму;

·Название формы "Карточка должника";

·Кнопка — Меню (Рисунок 10‑2);

·Кнопка — дополнительная информация, при прикосновении к ней отображается комментарий (Рисунок 10‑3);

·Фамилия, имя и отчество должника;

·Идентификатор активного долга, Статус долга, Банк, Статус контакта на текущий момент;

·Кнопка — отмена всех выездов;

·Заголовок раздела "Долги";

·Заголовок раздела "Адрес";

·Заголовок раздела "Номер";

·Заголовок раздела "Документы";

·Кнопка — действия на форме "Карточка должника" (Рисунок 10‑4).

Раздел "Долги"

Прикосновение ("tap") к заголовку раздела "Долги" приводит к разворачиванию списка долгов (Рисунок 10‑5). Повторное прикосновение — к сворачиванию.

Каждая запись в списке долгов содержит:

·Идентификатор долга;

·Сумма долга (с указанием валюты);

·Статус долга (см. словарь № 6).

Записи в списке долгов снабжены переключателями (radio button). Если должник имеет несколько долгов, то выбранный элемент указывает на активный долг. Если у должника долг один, то он является выбранным. Для смены активного долга коснитесь строки с неактивным долгом.

Раздел "Адрес"

Прикосновение ("tap") к заголовку раздела "Адрес" приводит к разворачиванию списка адресов (Рисунок 10‑5). Повторное прикосновение — к сворачиванию.

Каждая запись в списке адресов содержит:

·Адрес;

·Тип адреса;

·Статус выезда по этому адресу (см. словарь № 270).

·Блокирован — Признак блокировки, если указано "Нет", то адрес не блокирован, если указано "Да", то адрес блокирован;

Примечание. Если в роли пользователя доступ № 280 DEBT_CARD_ALL_ADR имеет значение TRUE, то в списке адресов будут отображаться заблокированные адреса.

·Выезд — статус выезда по этому адресу (см. словарь № 270);

·Комментарий.

Окраска адреса в зависимости от статуса

Шрифт записей в списке адресов может быть окрашен в зависимости от статуса адреса. Для этого настройте константу Android.AddressStatus.Color. При этом блокированные записи всегда будут отображаться с серым цветом шрифта.

В разделе "Адреса" доступны следующие действия:

Показать ближайшие

Для того чтобы просмотреть на карте адреса ваших должников, расположенные близко к выбранному адресу, выполните следующие действия:

1.Выберите адрес в списке адресов.

2.Нажмите на значок справа от выбранного адреса.

3.В открывшемся меню выберите пункт "Показать ближайшие". Откроется форма "Местоположение" (Рисунок 10‑6).

На форме "Местоположение" будет отображаться карта с маркерами, указывающими на адреса должников, ближайшие к выбранному, в соответствии с предварительными настройками фильтра. Карта поддерживает все возможности выбранного картографического сервиса.

Цветовая индикация маркеров отображаемых адресов

Значки отображаемых адресов различаются по цвету следующим образом:

·Основной адрес (который был выбран в карточке) остается красным цветом.

·Адреса, по которым назначены выезды, отмечаются зеленым цветом.

·Все остальные адреса — синим цветом.

Просмотр информации по ближайшему адресу

Для просмотра подробной информации по одному из отображенных на карте адресов коснитесь его ("tap"). Будет открыто информационное окно (Рисунок 10‑7).

Доступные действия:

·Назначить выезд — Откроется форма выбора даты выезда (Рисунок 10‑9). Выберите дату и нажмите ОК. Выезд будет назначен (см. Назначение выезда);

·Отмена — нажмите для выхода из формы:

·Карточка должника — откроется карточка должника.

Фильтрация списка ближайших

Фильтр позволяет вывести записи, отвечающие иным критериям, чем заданные на форме по умолчанию.

Настройка фильтров для списка ближайших — см. описание настраиваемых фильтров (Фильтр формы "Местоположение").

Просмотр адреса на карте

Для того чтобы просмотреть адрес на карте:

1.Выберите адрес в списке адресов.

2.Нажмите на значок справа от выбранного адреса.

3.В открывшемся меню выберите пункт "На карте". Откроется форма "Адрес должника" (Рисунок 10‑8).

На форме "Адрес должника" отображается карта с маркером, указывающим на адрес должника. Карта поддерживает все возможности выбранного картографического сервиса.

Назначение выезда

Для того чтобы назначить выезд по адресу:

1.Выберите адрес в списке адресов.

2.Нажмите на значок справа от выбранного адреса.

3.В открывшемся меню выберите пункт "+ Назначить выезд".

4.Откроется форма выбора даты выезда (Рисунок 10‑9). Выберите дату и нажмите ОК. Выезд будет назначен.

Отмена выезда

Для того чтобы отметить выезд по адресу:

1.Выберите адрес в списке адресов.

2.Нажмите на значок справа от выбранного адреса.

3.В открывшемся меню выберите пункт "- Отменить выезд". Если выездов по данному адресу не назначено, будет показано предупреждение.

4.Если выезд по данному адресу назначен, его можно будет отменить.

Регистрация выезда

Для того чтобы зарегистрировать выезд:

1.Выберите адрес и нажмите на соответствующую строку на экране устройства;

2.Появится список вариантов согласно настройке дерева контактов;

3.Пройдите по списку до пункта "Регистрация";

4.Если требуется повторный выезд, следует задать его дату;

5.Во вкладке "Контакты" добавится запись с зарегистрированным выездом.

В зависимости от настройки выбранного результата контакта осуществляется смена статуса адреса.

Добавление адреса

Для того чтобы добавить адрес нажмите на кнопку "Добавить новый", которая относится к списку адресов.

В открывшейся форме (Рисунок 10‑10) заполните поля и нажмите ОК.

Примечание. При работе в локальном режиме (при отсутствии интернета либо при локальной авторизации) добавление адреса недоступно.

Раздел "Номер"

Прикосновение ("tap") к заголовку раздела "Номер" приводит к разворачиванию списка номеров телефонов (Рисунок 10‑11). Повторное прикосновение — к сворачиванию.

Каждая запись в списке телефонов содержит:

·Имя;

·Номер

·Тип телефона;

·Статус телефона должника (см. словарь № 37).

·Блокирован — Признак блокировки, если указано "Нет", то телефон не блокирован, если указано "Да", то телефон блокирован.

Примечание. Если в роли пользователя доступ № 127 DEBT_CARD_ALL_PHONE имеет значение TRUE, то в списке номеров будут отображаться блокированные номера телефонов.

Окраска телефона в зависимости от статуса

Шрифт записей в списке номеров телефонов может быть окрашен в зависимости от статуса телефона. Для этого настройте константу Android.PhoneStatus.Color. При этом блокированные записи всегда будут отображаться с серым цветом шрифта.

В разделе "Телефоны" доступны следующие действия:

Звонок

Для того чтобы позвонить по телефону:

1.Выберите номер в списке телефонных номеров.

2.Нажмите на значок справа  от выбранного номера.

3.В открывшемся меню выберите "Call".

Внимание! Важно! По окончании телефонного разговора звонка зарегистрируйте контакт! В Системе настроен запрет на выход из карточки должника без регистрации контакта после телефонного разговора. (подробности и возможность отключения данного запрета — см. здесь).

Примечание. Осуществляется автоматическая аудиозапись телефонных переговоров и сохранение их в базу данных Системы.

Отправка SMS

Для того чтобы отправить SMS:

1.Выберите номер в списке телефонных номеров.

2.Нажмите на значок справа от выбранного номера.

3.В открывшемся меню выберите "Send SMS".

4.В открывшейся форме (Рисунок 10‑12) выберите шаблон или введите текст сообщения.

5.Нажмите ОК.

Регистрация контакта

Поддержаны следующие типы контакта:

·Звонок — исходящий звонок пользователя должнику;

·Входящий звонок — звонок должника пользователю.

Для того чтобы зарегистрировать контакт:

1.Выберите номер в списке телефонных номеров и нажмите на соответствующую строку на экране устройства.

2.Выберите тип контакта (Рисунок 10‑13);

3.Появится список вариантов согласно настройке дерева контактов (Рисунок 10‑14).

Примечание. Отображается дерево контактов, настроенное в основной версии Системы, соответствующее выбранному типу контакта.

4.Пройдите по списку до пункта "Регистрация";

5.Во вкладке "Контакты" добавится запись с зарегистрированным контактом.

В зависимости от настройки выбранного результата контакта осуществляется:

·смена статуса телефона;

·перезакрепление долга за оператором, зарегистрировавшим контакт, если у ответственного оператора включен флаг Авт. перезакреп. (если в настройках исхода выставлен флаг “перераспределение на оператора").

Добавление нового номера

Для того чтобы добавить номер телефона нажмите на кнопку "Добавить новый", которая относится к списку телефонов.

В открывшейся форме (Рисунок 10‑15) заполните поля и нажмите ОК.

Примечание. При работе в локальном режиме (при отсутствии интернета либо при локальной авторизации) добавление телефонного номера недоступно.

Раздел "Документы"

Прикосновение ("tap") к заголовку раздела "Документы" приводит к разворачиванию списка документов (Рисунок 10‑16). Повторное прикосновение — к сворачиванию.

Список документов включает файлы и ссылки.

 Примечание. Механизм ссылок может быть не поддержан в Системе. Для включения/отключения механизма ссылок обратитесь в техподдержку. Ссылки включаются с помощью константы Android.DocLinks.Use.

Каждая запись в списке документов содержит:

·Тип документа (см. словарь № 100);

·Название файла;

·Дата загрузки;

·Значок, отображающий состояние документа:

· — для документов, сохраненных на устройстве;

· — для документов, не закачанных на устройство.

Скачивание документа с сервера на устройство

Для того чтобы скачать документ, выполните следующие действия:

1.Выберите документ, помеченный значком .

2.Прикоснитесь ("tap") к соответствующей строке списка документов.

3.Появится окно подтверждения (Рисунок 10‑17). Нажмите "Скачать".

4.Документ будут сохранен на устройство. Во время сохранения будет отображаться окно с индикацией хода процесса.

5.По окончании загрузки документа на устройство значок изменится на .

Просмотр документа

Для просмотра доступны только документы. Загруженные на устройство.

Для просмотра документа прикоснитесь ("tap") к строке, соответствующей выбранному документу.

Добавление документа

Для того чтобы добавить документ:

1.Нажмите кнопку "Добавить новый", которая относится к списку документов.

2.В открывшемся списке "Прикрепить файл" выберите тип файла (Рисунок 10‑18):

·Новое фото;

·Галерея;

·Новая запись;

·Новый документ.

3.В зависимости от выбранного типа файла последующие действия различаются. По окончании действия соответствующий файл будет добавлен к списку документов.

Новое фото:

· Устройство перейдет в режим фотосъемки;

· Сделайте фотографию;

·Выберите тип сохраняемого документа (Рисунок 10‑19). По умолчанию для документа установлен тип "Фото". Список типов документов см. словарь № 100.

·Нажмите ОК.

Галерея:

· Откроется доступ к файлам, хранящимся на устройстве;

· Выберите файл.

·Выберите тип сохраняемого документа (Рисунок 10‑19). По умолчанию для документа установлен тип "Фото". Список типов документов см. словарь № 100.

·Нажмите ОК.

Новая запись:

·Откроется окно с доступом к функциям звукозаписи (Рисунок 10‑20).

·Нажмите кнопку .

·Выберите тип сохраняемого документа (Рисунок 10‑19). По умолчанию для документа установлен тип "Аудиозапись". Список типов документов см. словарь № 100.

·Нажмите ОК.

·Звукозапись включится. Приложение вернется в карточку должника. Запись будет идти в фоновом режиме.

·Для остановки записи активируйте список уведомлений на устройстве, найдите информацию о текущей записи и нажмите кнопку остановки записи.

·Затем нажмите кнопку сохранения записи.

Новый документ:

·Откроется доступ к файлам, хранящимся на устройстве;

·Выберите файл;

·Выберите тип сохраняемого документа (Рисунок 10‑19). Список типов документов см. словарь № 100.

·Нажмите ОК.

Меню карточки должника

Меню карточки должника (Рисунок 10‑2) включает следующие пункты:

·История контактов;

·Настройки;

·Информация;

Действия карточки должника

По кнопке (Рисунок 10‑4) доступны следующие действия:

·График платежей;

·Платежи;

·Контакты;

·Атрибуты;

График платежей

Выбор действия "График платежей" приводит к открытию графика платежей.

Платежи

Выбор действия "Платежи" приводит к открытию формы "Платежи" (Рисунок 10‑22).

Форма "Платежи" содержит следующие элементы:

·Кнопка — для возвращения на предыдущую форму;

·Название формы "Карточка должника";

·Кнопка — Меню (Рисунок 10‑2);

·Список платежей;

·Кнопка — действия на форме "Карточка должника" (Рисунок 10‑4).

Каждая запись списка платежей содержит следующие поля:

·Сумма;

·Статус платежа (см. словарь № 20);

·Дата платежа.

Список платежей отсортирован по убыванию даты.

Контакты

Выбор действия "Контакты" приводит к открытию формы "Контакты" (Рисунок 10‑23).

Форма "Контакты" содержит следующие элементы:

·Кнопка — для возвращения на предыдущую форму;

·Название формы "Карточка должника";

Атрибуты

Форма изменения атрибутов (Рисунок 10‑24) содержит следующие разделы:

·Классификаторы;

·Произвольные атрибуты;

·Социальные сети.

Прикосновение к заголовку раздела приводит к разворачиванию раздела. Повторное прикосновение — к сворачиванию.

Классификаторы

Для изменения доступны настроенные скоринговые баллы (см. словарь № 101, 102, 244, 245).

Произвольные атрибуты

Для настройки доступны произвольные атрибуты, заданные в модуле "Произвольные атрибуты" основной (десктопной) версии Системы.

Социальные сети

Работа с функционалом социальных сетей — см. Социальные сети.
