·Prosecution.RequestReport5;

Для настройки отчета, доступного из карточки уголовного преследования, выполните следующие действия:

1.Создайте отчет в модуле Произвольные отчеты.

2.Назначьте идентификатор отчета в качестве значения одной из соответствующих констант.

3.Введите в поле text соответствующих записей таблицы strings названия пунктов меню кнопки .

Если в системе не настроены отчеты, доступные для формирования из модуля Карточка уголовного преследования, то кнопка будет не видна.

Примечание. По поводу настройки отчетов, доступных для формирования из карточки уголовного преследования, обратитесь к специалистам технической поддержки.

Для формирования отчета выполните следующие действия:

1.В модуле Карточка уголовного преследования активируйте вкладку Договоры.

2.Нажмите кнопку и в открывшемся списке выберите нужный отчет.

3.При необходимости введите входные параметры отчета и нажмите OK.

4.Отчет будет выведен на экран с возможностью сохранения.
