Удаление связи адреса с долгом на форме "Информация по связям"

Удаление связи адреса с долгами доступно пользователям, в роли которых доступ № 92 DEBT_CARD_PHONE_ADR_EDIT имеет значение "TRUE".

Для удаления связи адреса с долгом в форме "Информация по связям", выполните следующее:

1.В карточке должника на вкладке "Общая", на подчиненной вкладке "Адреса" выберите адрес.

2.Активируйте контекстное меню правой кнопкой мыши. В открывшемся списке выберите пункт "Связи с долгом".

3.Откроется форма "Информация по связям".

4.В форме отображаются все активные долги должника, с которыми установлена связь для данного адреса. Для отображения закрытых долгов, с которыми связан адрес, включите флаг "Показывать завершенные".

5.Выберите долг, с которым вы хотите удалить связь.

6.Нажмите кнопку "Удалить".

7.Откроется форма подтверждения. Нажмите "Yes".

Для выбранного адреса будет удалена связь с выбранным долгом.
