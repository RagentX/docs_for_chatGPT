Вход в систему и регистрация внутреннего номера телефона пользователя

Запуск системы выполняется стандартными для ОС Windows способами. После запуска системы откроется окно авторизации. Введите в нем логин и пароль пользователя и нажмите кнопку ОК.

Примечание. Уточнить логин и пароль можно у системного администратора.

Примечание. Если в системе включен контроль сложности пароля (т.е. константа ComplexPassword имеет значение "Истина"; пароль должен содержать как минимум одну заглавную, одну строчную букву, одну цифру и быть не менее 8 символов) и действующий пароль не отвечает этим требованиям, то при входе необходимо будет его сменить.

Примечание. Если в системе установлен срок действия пароля (константа PasswordExpirationPeriod имеет значение, отличное от 0) и он истёк, то при входе необходимо будет сменить пароль.

Примечание. Возможность смены пароля зависит от значения доступа LOGIN_CHANGE_USER_PASSWORD.

Примечание. Запрещено входить в Систему под одним пользователем несколько раз одновременно. При попытке повторного входа в Систему пользователю будет показано сообщение:

        Вход под данным пользователем уже осуществлен.

Если вход выполнен успешно, откроется информационное окно с предупреждением, что пользователь получает доступ к конфиденциальной информации. Нажмите кнопку Yes, чтобы продолжить работу в системе.

Примечание. Отображение данного окна зависит от константы LoginConfirmation.

Если у пользователя в роли есть доступ LOGIN_PHONE, имеющий значение TRUE, то при входе в систему будет отображаться диалоговое окно для уточнения внутреннего номера телефона.

Примечание. Внутренний номер должен быть назначен пользователю для того, чтобы он мог использовать функции телефонии. Кроме того, в модуле "Пользователи" для него должен быть прописан параметр AGENT_ID.

В диалоговом окне необходимо ввести внутренний добавочный номер (extension) телефона или программного телефона (softphone) для рабочей станции пользователя и нажать кнопку ОК. После этого будет открыто главное меню системы.

Примечание. Уточнить свой добавочный номер телефона можно у системного администратора.

Ввод неверного пароля и блокировка пользователя

Если пользователем введен неверный пароль, то пользователю отображается сообщение, а форма авторизации блокируется на 15 секунд.

Вывод произвольного отчета после успешной авторизации пользователя

Если в роли у текущего пользователя в значении доступа № 537 LOGIN_CUSTOM_REPORT указано число, то сразу после его успешной авторизации в Системе на экран выводится произвольный отчет, имеющий в качестве идентификатора значение данного доступа и в качестве единственного входного параметра идентификатор пользователя.

Если в роли текущего пользователя значение данного доступа имеет значение NULL, то отчет не выводится.
