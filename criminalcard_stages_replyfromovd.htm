Получение ответа из ОВД

При получении ответа из ОВД возможны следующие варианты:

·Отказ в возбуждении уголовного дела;

·Возбуждение уголовного дела.

Для архивных дел в статусе "Получен ОВУД" (см. словарь № 145) доступны действия:

·Возбуждение уголовного дела;

·Направлена жалоба.

Отказ в возбуждении уголовного дела

Для того чтобы зафиксировать отказ в возбуждении уголовного дела, выполните следующие действия:

1.На вкладке Уголовное преследование в разделе Передано в суд заполните поля:

·Дата возбуждения — дата отказа в возбуждении уголовного дела (обязательное поле);

·Номер УД — номер отказа в возбуждении уголовного дела (необязательное поле).

2.Нажмите кнопку Ответ из ОВД и в раскрывшемся списке выберите Отказ в возбуждении УД.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

Если какие-либо обязательные поля не заполнены, будет выдано информационное сообщение. Нажмите в нем кнопку ОК, после чего произойдет возврат в карточку уголовного преследования для внесения данных.

4.В правом верхнем углу нажмите кнопку сохранения изменений, затем подтвердите сохранение, нажав кнопку Yes. Дело будет переведено в конечный статус 3 "Получен ОВУД" и затем автоматически перенесено в архив.

Действие логируется.

Возбуждение уголовного дела

Для того чтобы зафиксировать факт возбуждения уголовного дела, выполните следующие действия:

1.На вкладке Уголовное преследование в разделе Передано в суд заполните поля:

·Дата возбуждения — дата возбуждения уголовного дела (обязательное поле);

·Номер УД — номер уголовного дела (необязательное поле);

·Статья УК — статья Уголовного кодекса РФ, по которой возбуждено уголовное дело.

2.Нажмите кнопку Ответ из ОВД и в раскрывшемся списке выберите Возбуждено УД.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

Если какие-либо обязательные поля не заполнены, будет выдано информационное сообщение. Нажмите в нем кнопку ОК, после чего произойдет возврат в карточку уголовного преследования для внесения данных.

4.Откроется форма Корректировка данных. Заполните поля и нажмите ОК.

5.В правом верхнем углу нажмите кнопку сохранения изменений, затем подтвердите сохранение, нажав кнопку Yes. Дело будет переведено в статус 4 "Возбуждено УД".

Действие логируется.

Направлена жалоба

Для того чтобы зафиксировать направление жалобы, выполните следующие действия:

1.Выберите архивное дело в статусе 3 "Получен ОВУД".

2.На вкладке Уголовное преследование нажмите кнопку Ответ из ОВД и в раскрывшемся списке выберите Направлена жалоба.

3.В открывшемся диалоговом окне нажмите Yes, чтобы подтвердить выполнение операции.

5.В правом верхнем углу нажмите кнопку сохранения изменений, затем подтвердите сохранение, нажав кнопку Yes. Дело будет переведено в статус 2 "Направлено в ОВД" и затем автоматически возвращено из архива.

Действие логируется.
