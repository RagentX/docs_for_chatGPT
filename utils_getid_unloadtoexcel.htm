Выгрузка данных о результатах поиска в итоговые файлы

Результаты поиска можно выгрузить в файлы MS Excel или CSV. Положительные результаты поиска выгружаются в один файл, отрицательные — в другой. Для того чтобы выгрузить данные, выполните следующие действия:

1.В форме Утилита определения ID долга нажмите кнопку Выгрузить файлы Excel. Если требуется выгрузить результаты в файлы CSV, нажмите кнопку Выгрузить файлы Csv.

2.В открывшемся стандартном диалоговом окне выберите директорию и укажите имя файла, в который будут выгружены положительные результаты поиска. Нажмите кнопку Save (Сохранить).

3.По окончании выгрузки данных будет выдано сообщение: "Создание файла положительных результатов поиска <директория и имя файла> закончено". Нажмите кнопку ОК, чтобы закрыть его.

4.В следующем открывшемся стандартном диалоговом окне выберите директорию и укажите имя файла, в который будут выгружены отрицательные результаты поиска. Нажмите кнопку Save (Сохранить).

5.По окончании выгрузки данных будет выдано сообщение: "Создание файла отрицательных результатов поиска <директория и имя файла> закончено". Нажмите кнопку ОК, чтобы закрыть его.

На время выгрузки данных кнопки Обработать, Выгрузить файлы Excel, Выгрузить файлы Csv, Отменить, а также возможность ручного редактирования ячеек таблицы становятся недоступны. В процессе открытия файла в нижней строке окна формы будет отображаться тип операции (выгрузка результатов поиска), информация о проценте выполнения операции.

По окончании выгрузки кнопки Обработать, Выгрузить файлы Excel, Выгрузить файлы Csv и Отменить, а также возможность ручного редактирования ячеек таблицы вновь становятся доступными.
