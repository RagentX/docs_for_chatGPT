Ведение списка расчетных счетов должника

Добавление, редактирование и удаление счета доступно пользователям, в роли которых доступ № 556 DEBT_CARD_ACCOUNT_EDIT имеет значение TRUE.

Форма добавления/редактирования атрибутов расчетного счета должника

Добавление расчетного счета

Данные о расчетных счетах вводятся на вкладке Счета. Для того чтобы добавить расчетный счет в список, выполните следующие действия:

1.В модуле Карточка должника активируйте вкладку Счета.

2. Нажмите кнопку Добавить.

3.В открывшейся форме форме добавления атрибутов расчетного счета заполните следующие поля:

·Банк — введите наименование банка, в котором открыт расчетный счет;

·Расчетный счет — введите номер расчетного счета. Поле обязательно для заполнения;

·БИК — введите банковский идентификационный код;

·Кор. счет — введите корреспондентский счет банка;

·Остаток на счете — введите сумму остатка на счете;

·        Комментарий — при необходимости введите дополнительную информацию.

4.Нажмите кнопку ОК, чтобы сохранить данные.

Редактирование расчетного счета

Для того чтобы отредактировать атрибуты расчетного счета должника, выполните следующие действия:

1. На вкладке Счета выберите нужную запись и нажмите кнопку Изменить или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2. В форме редактирования атрибутов расчетного счета измените необходимые данные.

3. Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление расчетного счета

Для того чтобы удалить расчетный счет из списка, выполните следующие действия:

1.На вкладке Счета выберите нужную запись.

2.Нажмите кнопку Удалить.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.
