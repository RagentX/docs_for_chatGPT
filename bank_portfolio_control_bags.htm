Управление портфелями

Направление портфеля

Направление портфеля задается системным словарем № 140 Направления портфеля и может принимать значения:

·Входящий;

·Исходящий.

Перечень статусов портфелей хранится в словаре № 57, который является подчиненным для словаря № 140.

Статусы входящих портфелей

Для входящих портфелей статус может принимать следующие значения (изменение статуса выполняется вручную путем редактирования статуса портфеля):

·Загружается — портфель находится на этапе формирования;

·В работе — портфель долгов находится в работе;

·Закрыт — работа с портфелем завершена.

Статусы исходящих портфелей

Для исходящих портфелей статус может принимать следующие значения (изменение статуса выполняется вручную путем редактирования статуса портфеля):

·Новый — если портфель находится в данном статусе, в него можно добавлять новые долги.

·Сформирован — после того как все долги добавлены в портфель, для него выполняется операция формирования исходящего портфеля, и портфель переводится в статут "Сформирован". Для портфелей в этом статусе добавление долгов становится невозможным.

·Передан — после того как портфель сформирован, для него выполняется операция передачи на аутсорсинг, и портфель переводится в статус "Передан". Для портфелей в этом статусе ни добавление, ни удаление долгов становится невозможным до его отзыва из аутсорсинга;

·Отозван — после того как работа в аутсорсинге завершена, для портфеля выполняется операция отзыва из аутсорсинга, и портфель переводится в статус "Отозван", а все долги из портфеля — в статус 16 "Возврат из аутсорсинга";

·Окончание работ — портфель автоматически переводится в данный статус при наступлении даты завершения работы с портфелем. Все долги из этих портфелей переводятся в статус 16 "Возврат из аутсорсинга".

Примечание. Возможность выполнять те или иные действия в портфеле не зависит от его статуса. Если портфель имеет статус "Закрыт", то в мультифильтрах можно установить/отключить флаг Отображать закрытые портфели.

См. далее:

ØУправление портфелями обоих направлений:

üФормирование названия портфеля;

üПросмотр списка долгов, входящих в портфель;

üПеренос портфеля между контрагентами;

üФормирование списка документов портфеля;

ØУправление исходящими портфелями.
