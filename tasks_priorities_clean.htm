Очистка приоритетов обзвона должников

Операция очистки приоритетов обзвона должников доступна в форме Постановка задач на вкладке Распределение приоритета. Операция может быть выполнена:

·для всех долгов из выборки, сформированной с учетом фильтров;

·для всех долгов, выделенных в текущей выборке с помощью клавиш SHIFT и/или CTRL.

Для того чтобы очистить приоритеты, выполните следующие действия:

1.В форме Постановка задач перейдите на вкладку Распределение приоритета.

2.При необходимости сформируйте выборку долгов с помощью фильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Очистить приоритет и в раскрывающемся списке выберите вариант выполнения операции: По всей выборке или Только выделенные. Приоритеты будут очищены. Операторы не будут видеть таких должников в модуле "Обработка должников".
