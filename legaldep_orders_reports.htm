Формирование отчетов по судебной работе

Отчеты по судебной работе формируются в модуле на следующих вкладках:

·Приказы;

·Правопреемство;

·Иски;

·Банкротство;

·Судебные дела (предв.);

·Архив судебной работы.

Кроме того, отчеты по судебной работе формируются в карточке банкротства, в разделе Дела судебной работы по списку дел, выделенных

Кнопка для доступа к отчетам по судебной работе имеет вид .

Отчеты по судебной работе должны удовлетворять следующим условиям:

·Отчет содержится в списке произвольных отчетов;

·Отчет имеет один из входных параметров — Список Excel;

·Тип ID для списочных РМ этого параметра — Судебная работа.

Если в Системе отсутствуют такие отчеты, то кнопка Произвольные отчеты будет отсутствовать. Если такие отчеты есть, то кнопка появляется.

Для того чтобы сформировать произвольный отчет:

1.В модуле Юридический департамент откройте нужную вкладку из указанных выше.

2.Сформируйте выборку.

3.Выберите одну или более записей в списке.

4.Нажмите кнопку "Произвольные отчеты". В открывшемся меню вы можете снять или включить флаг "Только выделенные", если это необходимо. После этого потребуется еще раз нажать кнопку .

5.В открывшемся меню выберите пункт "Сформировать". Откроется форма "Выбор отчета". Выберите отчет.
