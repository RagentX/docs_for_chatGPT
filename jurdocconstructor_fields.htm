Примечание.

·Для документов судебной работы используются поля frxRep1.court_*. При этом поля frxRep1.fssp_* не используются.

·Для документов исполнительного производства работают следующие правила:

·Если ИП имеет связь с делом судебной работы, то для такого ИП используются только поля frxRep1.court_*;

·Если ИП не имеет связи с делом судебной работы, то для такого ИП используются только поля frxRep1.fssp_*.
