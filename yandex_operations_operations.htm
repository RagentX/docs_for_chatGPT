Операции с выделенными адресами

После того, как адреса выделены, вам становятся доступны операции с ними.

Доступны следующие операции:

·Установить статус выезда;

·Закрепить за оператором (доступно только из модуля Постановка задач);

·Печать выездных листов;

·Печать уведомлений.

Установка статуса выезда

Доступна установка следующих статусов:

·Необходим выезд;

·Требуется выезд;

·Подготовлен выезд;

·Выполнен успешно;

·Выполнен неуспешно.

Для того чтобы установить статус выезда для выделенных адресов:

1.Выберите пункт Установить статус выезда.

2.В открывшемся списке выберите нужный статус.

3.Появится окно подтверждения. Нажмите Yes.

4.После выполнения действия будет показано информационное окно с сообщением о результатах. Нажмите OK.

Закрепление за оператором

Внимание! Данный функционал доступен только в модуле Постановка задач! При открытии карт из других модулей данный пункт отсутствует!

Для того чтобы установить статус выезда для выделенных адресов:

1.Выберите пункт Закрепить за оператором.

2.В открывшемся списке выберите нужного оператора.

3.Появится окно подтверждения. Нажмите Yes.

4.После выполнения действия будет показано информационное окно с сообщением о результатах. Нажмите OK.

Печать выездных листов

Для того чтобы напечатать выездные листы для выбранных адресов:

1.Выберите пункт Печать выездных листов.

2.Появится окно подтверждения. Нажмите Yes.

3.По выбранным долгам будут сформированы выездные листы и открыты в режиме предварительного просмотра в приложении Fast Report.

Печать уведомлений

Для того чтобы напечатать уведомления для выделенных адресов:

1.Выберите пункт Печать уведомлений.

2.Появится окно подтверждения. Нажмите Yes.

3.В открывшейся форме заполните следующие поля:

·Тип письма — выберите одну из форм, настроенных в модуле Конструктор писем;

·Флаг По всем долгам — включите флаг, если вы хотите создать отдельные уведомления для каждого долга у выбранных должников или выключите флаг, чтобы создать по одному уведомлению на каждого должника.

4.По выбранным долгам будут сформированы уведомления и открыты в режиме предварительного просмотра в приложении Fast Report.
