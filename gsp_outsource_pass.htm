Передача на аутсорсинг

При осуществлении передачи долгов на аутсорсинг происходит включение выбранных долгов в исходящий портфель, находящийся в статусе "Новый". Операция доступна из модулей:

·Список должников — Работа с долгами в аутсорсинге;

·Группы свободного поиска — Аутсорсинг.

Доступы, влияющие на доступность пункта Передача на аутсорсинг:

№

Название доступа

Значение доступа

Описание доступа

363

OUTSOURCING_FILL

TRUE

Право на добавление/удаление долгов из портфелей направления "Исходящий" и со статусом "Новый"

619

DEBT_LIST_OUTSOURCING_PORTFOLIO_TYP

"all" или список номеров стадий через запятую

Список стадий исходящего портфеля по словарю, в который пользователь может переводить долги (1- soft, 2 - hard, 3 - legal, 4 - exec, 5 - bankrupt, all - доступны все стадии) (работает только при включенном доступе OUTSOURCING_FILL)

Проверка доступов осуществляется следующим образом:

1)Проверяется доступ OUTSOURCING_FILL (только для модуля Список должников). Если доступ OUTSOURCING_FILL отсутствует или равен FALSE, пункт Передача на аутсорсинг недоступен.

2)Проверяется доступ DEBT_LIST_OUTSOURCING_PORTFOLIO_TYP (для обоих модулей). Если доступ DEBT_LIST_OUTSOURCING_PORTFOLIO_TYP отсутствует или пуст, то пункт Передача на аутсорсинг недоступен.

3)Если доступ DEBT_LIST_OUTSOURCING_PORTFOLIO_TYP не пуст, то список доступных портфелей зависит от значения доступа следующим образом:

·если значение равно "all", то доступны все исходящие портфели в статусе "Новый";

·если значение не равно "all", то доступны только те портфели, в названии которых содержится текст, соответствующий кодам, перечисленным в доступе:

·1 — SOFT;

·2 — HARD;

·3 — LEGAL;

·4 — EXEC;

·5 — BANKRUPT.

Примечание. При запуске функционала из модуля Группы свободного поиска пункт меню Аутсорсинг кнопки Работа с долгом видно, если в РМ Группы должников было предоставлено право Работа с долгом. Аутсорсинг и на вкладке есть поле ID долга.

Для того чтобы передать долг на аутсорсинг, выполните следующие действия:

1.Выберите записи о долгах, которые требуется передать на аутсорсинг. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL

2.Из формы Группы свободного поиска нажмите кнопку Работа с долгом. В открывшемся меню выберите пункт Аутсорсинг, затем — Передать на аутсорсинг. Из формы Список должников в контекстном меню (вызывается правой кнопкой мыши) выберите пункт Аутсорсинг, затем Передать на аутсорсинг.

3.В открывшемся окне подтверждения нажмите кнопку Yes, чтобы подтвердить выполнение операции.

4.В диалоговом окне выберите исходящий портфель, в который требуется включить долги.

5.Нажмите кнопку ОК.

6.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить перевод долгов в выбранный портфель.

7.В результате будет выдано сообщение о переводе выбранных дел на аутсорсинг. Нажмите кнопку ОК, чтобы завершить операцию.
