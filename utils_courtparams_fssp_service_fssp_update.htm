Сервис обновления данных о ФССП и приставах

Реализована возможность автоматического обновления системных справочников ФССП и приставов в ФССП в соответствии с данными, которые публикует ФССП России.

Данный функционал реализован через отдельный java-сервис, который запрашивает данные о ФССП и приставах и при необходимости создает или обновляет записи в таблицах БД Системы.

Для включения и настройки сервиса обратитесь в техническую поддержку.

Описание работы сервиса

Обновление данных по ФССП

1.Сервис запрашивает данные по ФССП.

2.Далее, по каждой из полученной записи о ФССП осуществляются следующие проверки:

·Если полученной записи нет в БД Системы, то она добавляется.

·Для каждой записи о ФССП в БД Системы, для которой найдена соответствующая запись от сервиса, флаг "Не используется" выставляется в положение "Выключено".

·Для каждой записи о ФССП в БД Системы, для которой не найдена соответствующая запись от сервиса, флаг "Не используется" выставляется в положение "Включено".

Обновление данных по приставам

1.Сервис запрашивает данные по приставам.

2.Далее, по каждой из полученной записи о приставе осуществляются следующие проверки:

·Если полученной записи нет в БД Системы, то она добавляется.

·Для каждой записи о приставе в БД Системы, для которой найдена соответствующая запись от сервиса, флаг "Не используется" выставляется в положение "Выключено".

·Для каждой записи о приставе в БД Системы, для которой не найдена соответствующая запись от сервиса, флаг "Не используется" выставляется в положение "Включено".
