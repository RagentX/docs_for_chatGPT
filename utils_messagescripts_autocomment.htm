Автоматический комментарий

Автоматические комментарии используются в модуле Регистрация контактов для того, чтобы облегчить пользователю ввод комментария к результатам контакта с должником, в тех случаях, когда для определенного исхода контакта каждый раз требуются однотипные комментарии.

К каждому исходу контакта можно привязать автоматический комментарий в модуле Настройка дерева контактов (см. Добавление, редактирование и удаление результата контакта). Затем, при регистрации контакта с указанным исходом, комментарий сформируется автоматически и будет содержать, в том числе, данные, введенные оператором в процессе контакта (см. Подстановка переменных в текст сообщения).

Вкладка "Автоматический комментарий"

Описание вкладки "Автоматический комментарий"

·список сценариев автоматических комментариев, содержащий следующие поля:

·ID — идентификатор сценария, формируется автоматически;

·Номер — номер сценария;

·Название — название сценария;

·Текст — текст сценария;

·кнопки для добавления, редактирования, удаления сценариев.

Форма добавления/редактирования шаблона автоматического комментария

В текст шаблона могут быть добавлены переменные, как общие для всех вкладок модуля, так и доступные только для функционала автокомментария (см. Подстановка переменных в текст сообщения).

Описание формы добавления/редактирования шаблона автоматического комментария

·Название — название сценария. Поле обязательно для заполнения;

·Текст — текст сценария. Поле может содержать как статичный текст, так и переменные, вместо которых при выполнении сценария будут подставлены конкретные значения (см. Подстановка переменных в текст сообщения).

Действия с шаблонами автоматических комментариев

Создание шаблона автоматического комментария

Для того  чтобы создать шаблон автоматического комментария, выполните следующее:

1.В форме Сценарии сообщений для должников на вкладке Автоматический комментарий нажмите кнопку .

2.В открывшейся форме заполните нужные поля.

3.Нажмите кнопку ОК. Сценарий будет добавлен в список.

Редактирование шаблона автоматического комментария

Для того чтобы отредактировать шаблон автоматического комментария, выполните следующее:

1.В форме Сценарии сообщений для должников на вкладке Автоматический комментарий выберите нужную запись и нажмите кнопку или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.В форме добавления/редактирования сообщения измените необходимые атрибуты.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление шаблона автоматического комментария

Для того чтобы удалить шаблон автоматического комментария, выполните следующее:

1.В форме Сценарии сообщений для должников на вкладке Автоматический комментарий выберите нужную запись.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.
