Установка и удаление связи имущества с долгом

Связь имущества с долгом

Кнопка Установить связь с долгом отображается на вкладках модуля, содержащих поле ID имущества, только если в Системе включен функционал карточки имущества. Функционал включается технической константой GroupViewer.PropertyCard.Use.

Кнопка расположена в нижней части формы, на общей панели кнопок. По данной кнопке выполняется связь выбранных записей с указанным долгом.

Порядок действий

Для того, чтобы связать с долгом одну или более записей об имуществе, выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, содержащую поле ID имущества.

2.Сформируйте выборку. В полученном списке выберите одну или более записей.

3.Нажмите кнопку Установить связь с долгом.

4.Если хотя бы одна запись из выделенных уже имеет связь с долгом, будет показано сообщение с текстом:

Для <количество записей, у которых установлена связь с долгом> связь с долгом уже установлена. Вы действительно хотите изменить связь для всех выбранных записей?

Нажмите Yes. Тогда для всех выбранных записей будет установлена связь с долгом, выбранным на последующих шагах.

5.Откроется форма Поиск долга. Заполните поля в области Поиск и нажмите кнопку .

6.В области Должники отобразится список долгов, удовлетворяющих выбранным критериям. Выберите запись и нажмите кнопку .

7.По результатам выполнения будет выведено сообщение с текстом:

Изменена связь с долгом для <количество успешно измененных записей> записей

8.Нажмите ОК.

Всем выбранным записям об имуществе будет установлена связь с данным долгом.

Удаление связи имущества с долгом

Кнопка Удалить связь с долгом отображается на вкладках модуля, содержащих поле ID имущества, только если в Системе включен функционал карточки имущества. Функционал включается технической константой GroupViewer.PropertyCard.Use.

Кнопка расположена в нижней части формы, на общей панели кнопок рядом с кнопкой Установить связь с долгом. По данной кнопке выполняется удаление связей выбранных записей с долгами.

По результатам выполнения выводится сообщение:

Удалена связь с долгом для <количество успешно измененных записей> записей.
