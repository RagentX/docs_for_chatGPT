Кнопка "Подтверждение"

Подтверждение — Подтвердить

Условия доступности

·Не подтверждено.

Результирующий статус

·Подтверждено.

Порядок действий

1.Войдите в нужную карточку.

2.Активируйте вкладку Имущество.

3.Выберите запись об имуществе.

4.Нажмите кнопку Подтверждение. В открывшемся меню выберите пункт Подтвердить.

5.Появится окно выбора даты подтверждения. Выберите дату и нажмите ОК.

6.Имуществу будет присвоен статус "Подтверждено", а в поле Дата подтверждения будет указана выбранная на шаге № 5 дата.

Подтверждение — Снять подтверждение

Условия доступности

·Подтверждено.

Результирующий статус

·Не подтверждено.

Порядок действий

Для того чтобы снять подтверждение, выполните следующие действия:

1.Войдите в нужную карточку.

2.Активируйте вкладку Имущество.

3.Выберите запись об имуществе, находящемся в статусе "Подтверждено".

4.Нажмите кнопку Подтверждение. В открывшемся меню выберите пункт Снять подтверждение.

5.Появится окно с вопросом: "Вы действительно хотите присвоить имуществу статус "Не подтверждено?". Нажмите Yes.

6.Имуществу будет присвоен статус "Не подтверждено", поле Дата подтверждения будет очищено.

Кнопка "Регистр. действия"

Запрет на проведение регистрационных действий

Условие доступности

Данный пункт доступен для имущества, находящегося в любом статусе, кроме терминальных, на которое не наложен запрет на проведение регистрационных действий.

Результирующий статус

Не изменится.

Порядок действий

1.Войдите в модуль Карточка исполнительного производства.

2.Активируйте вкладку Имущество.

3.Выберите запись об имуществе.

4.Нажмите кнопку Регистр. действия. В открывшемся меню выберите пункт Запрет на проведение регистрационных действий.

5.Откроется окно ввода данных. Заполните следующие поля:

·Дата наложения запрета — выберите дату (поле обязательно для заполнения);

·Комментарий — дополнительная информация (поле не обязательно для заполнения).

6.Нажмите OK.

7.Выполните прикрепление документа.

8.Далее осуществляется формирование настроенных запланированных действий.

9.Статус имущества не изменится. Флаг Запрет на рег. д-ия будет переведен в положение Включено. В поле Дата наложения запрета будет указана дата, введенная в п. 5.

10.Нажмите OK на форме Имущество.

11.На вкладке Принятые меры появится соответствующая запись и документ.

12.На вкладке Протокол появится соответствующая запись и ссылка на документ.
