Изменение отправителя входящего документа

Show/Hide Hidden Text

Изменение отправителя документа доступно только пользователя, в роли которых доступ № 680 DOCFLOW_CARD_SENDER_CHANGE имеет значение TRUE.

Отправитель входящего документа и получатель исходящего документа может быть выбран из списка должников, судов, ФССП, а также из списка обычных физических и юридических лиц (не должников) с возможностью создания новых лиц.

При нажатии на кнопку в поле Отправитель открывается форма выбора отправителя входящего документа.

Описание формы выбора отправителя

Форма выбора отправителя входящего документа включает следующие элементы:

·Флаги, соответствующие подключенным для поиска справочникам:

·Справочник ФССП (при включенном чекбоксе отображать в гриде результаты по запросу select * from law_court.typ=1);

·Справочник Судов (при включенном чекбоксе отображать в гриде результаты по запросу select * from law_court.typ=2);

·Справочник Физ. лиц (при включенном чекбоксе отображать в гриде результаты по запросу select *, contract,  from person.typ=1);

·Справочник Юр. лиц /ИП (при включенном чекбоксе отображать в гриде результаты по запросу select * from person.typ in 2,3).

Каждый флаг указывает на то, какие справочники подключены к поиску отправителя. По умолчанию все флаги включены. Оставьте включенными только флаги, соответствующие нужным вам для поиска справочникам. Это существенно ускорит поиск нужной записи.

·Поле ввода;

·Кнопка для поиска отправителя;

·Кнопка Очистить — для очистки поля ввода;

·Кнопка Добавить со следующими подпунктами:

·Суд — для добавления суда. При выборе открывается форма добавления суда. После добавления суд будет доступен также в модуле Настройка судов и ФССП, а также в других модулях Системы (доступно только пользователям, в роли которых доступ № 219 LAW_JUDGE_SET_FULL имеет значение TRUE);

·ФССП — для добавления ФССП. При выборе открывается форма добавления ФССП. После добавления ФССП будет доступна также в модуле Настройка судов и ФССП, а также в других модулях Системы (доступно только пользователям, в роли которых доступ № 219 LAW_JUDGE_SET_FULL имеет значение TRUE);

·Физ. / Юр. лицо — для добавления персоны: физического или юридического лица, которое не является должником;

·Расшир.поиск физ. / юр. лица — для расширенного поиска персоны: должника или не должника, физического или юридического лица;

·Кнопка ОК — для добавления найденного отправителя.

Форма выбора отправителя.

Поиск отправителя

В соответствии с данными, введенными в поле ввода, Система осуществляет поиск возможного отправителя, полное имя или название которого включают введенные символы. Поиск ведется среди подключенных справочников (флаги которых включены на форме). Найденные записи отображаются в открывшейся под полем ввода таблицей.

Записи в таблице отображаются по следующим правилам:

·Найденные записи отсортированы по алфавиту;

·Для судов, ФССП столбец Договор пустой, в столбце ФИО / Название выводится соответствующее наименование суда или ФССП, а в столбце Комментарий ОД — соответствующий комментарий по суду или ФССП;

·Для физических лиц в столбце ФИО / Название выводится ФИО персоны, для юридических лиц — Название компании / Наименование юр.лица;

·Для физических и юридических лиц в столбце Комментарий ОД выводится соответствующий комментарий;

·Для физических и юридических лиц, не являющихся должниками, колонка Договор пуста;

·Если у должника несколько договоров, то для каждого номера договора в выпадающем списке должна быть отдельная запись.

Если вы выберете одну из найденных записей, то она будет введена в поле ввода формы выбора отправителя. После этого таблица под полем ввода будет скрыта.

Для добавления отправителя нажмите ОК.

Поиск отправителя с помощью поля ввода.

Добавление персоны

При выборе пункта Физ. / Юр. лицо в меню кнопки Добавить открывается форма Добавление лица. Для того чтобы добавить персону заполните поля формы и нажмите ОК.

Форма "Добавление лица".

Расширенный поиск физ. / юр. лица

При выборе пункта Расшир.поиск физ. / юр. лица в меню кнопки Добавить открывается форма Выбор объекта: Физ. / Юр. лица для расширенного поиска персоны: должника или не должника, физического или юридического лица.

Форма расширенного поиска персоны включает следующие элементы:

·Поля для поиска персоны:

·ID персоны;

·ID долга;

·Договор;

·ФИО / Название;

·Серия;

·Номер;

·ИНН;

·Комментарий ОД;

·Кнопка Найти для инициации поиска по введенным данным;

·Таблица для отображения результатов поиска;

·Кнопка Изменить для редактирования персоны.

Для осуществления расширенного поиска персоны введите данные в поля для поиска и нажмите кнопку Найти. В таблице справа будут отображены результаты поиска.

В список результатов поиска выводится не более 100 записей, удовлетворяющих заданным условиям поиска. При этом для каждого договора, связанного с должником, в списке выводится отдельная запись.

Для выбора отправителя выберите запись в таблице, а затем нажмите ОК.

После этого вы вернетесь в форму выбора отправителя входящего документа, при этом в поле ввода будет введена информация о выбранной персоне.

Для добавления отправителя нажмите ОК.

Форма "Выбор объекта: Физ. / Юр. лица" для расширенного поиска отправителя.

Редактирование персоны

В форме расширенного поиска персоны можно также отредактировать данные о персоне.

Право редактирования физических и юридических лиц определяется доступом DOCFLOW_CARD_PERSON_EDIT.

Для редактирования записи о персоне выполните поиск персоны. После того как запись о нужной персоне отобразилась в таблице, выберите эту запись и нажмите кнопку Изменить (или выполните двойной клик по выбранной записи). Откроется форма Добавление лица с заполненными полями. Измените нужные значения полей и нажмите ОК. После этого вы вернетесь в форму расширенного поиска персоны.
