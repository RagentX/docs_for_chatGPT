Вкладка "Суды и судьи"

Примечание. Если константа LawAct.Judge.Use имеет значение "Ложь", то список судов на вкладке не отображается.

Список судов и судей отображается в форме Настройка судов и ФССП на вкладке Суды и судьи. Вкладка содержит следующие элементы:

·список судов, содержащий следующие поля:

·ID — идентификатор суда в системе;

·Родовая подсудность — см. словарь № 14;

·Название — название суда;

·Тип суда — см. словарь № 15;

·Индекс, Телефон, Адрес, Email, Веб-сайт;

·Комментарий;

·Родительский суд (если используется иерархическая структура судов, UseHierarchyLawCourt = "Истина");

·кнопки для добавления, редактирования, удаления данных о судах;

·кнопка Загрузка судов для запуска утилиты загрузки судов из Excel-файла;

·список судей, относящихся к выбранному суду (отображается только если LawAct.Judge.Use="Истина"). Он содержит следующие поля:

·ID — порядковый номер судьи в системе;

·ФИО — фамилия, имя, отчество судьи;

·Должность, Телефон, Email, Кабинет;

·Комментарий;

·кнопки для добавления, редактирования, удаления данных о судьях.
