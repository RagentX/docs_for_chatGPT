Редактирование суммы обещания (админская)

 Внимание! Данный функционал настроен только для ЛетоБанка. Поэтому, если вы открыли админский HELP для ЛетоБанка, у вас будут две страницы с описанием этого ф-ла. Но вообще, его можно включить и кому-то еще

Функционал редактирования суммы обещания доступен если техническая константа DebtCard.PromSum.Edit имеет значение "Истина" для пользователей, в роли которых включен доступ № 624 DEBT_CARD_PROM_SUM_EDIT. В этом случае на вкладке Обещания в панели инструментов, относящихся к списку обещаний, добавлена кнопка  Изменить сумму обещания.

Порядок действий

Для того чтобы изменить сумму обещания:

1.В карточке должника активируйте вкладку Долги, выберите нужный долг.

2.Активируйте вкладку Обещания, расположенную в нижней части формы. 

3.Выберите нужное обещание и нажмите кнопку  Изменить сумму обещания.

4.В открывшейся форме Обещания измените значение суммы обещания.

5.Нажмите ОК.

 Примечание. Если форма Обещания открывается для редактирования суммы обещания, то на ней заблокированы все поля, кроме поля Сумма.

Логика процедуры

При изменении суммы обещания Система выполняет следующие изменения:

·Изменяется значение суммы обещания;

·Изменяется общая сумма всех обещаний в таблице, расположенной в нижней строке таблицы;

·Пересчитывается покрытая платежами сумма обещания, затем устанавливается новая связь подтвержденных платежей с данным обещанием;

·Если данное обещание не является последним обещанием по долгу, то пересчитывается сумма покрытия платежами по всем последующим обещаниям и устанавливается новую связь обещаний и платежей;

·Действие протоколируется.
