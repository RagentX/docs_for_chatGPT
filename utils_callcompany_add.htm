Добавление кампании обзвона

Для того чтобы добавить новую кампанию обзвона, выполните следующие действия:

1.В форме Кампании обзвона на вкладке Кампании нажмите кнопку , относящуюся к списку кампаний.

2.В открывшейся форме заполните следующие поля:

·Название — название кампании. Поле обязательно для заполнения;

·Тип кампании — см. словарь № 318 "Тип кампании обзвона";

·Группа должников — группа должников, обзвон которых будет производиться в рамках кампании. Группы должников формируются в модуле "Группы должников". Поле обязательно для заполнения;

·Комментарий — дополнительная информация;

·Учет GMT — если установлен данный флаг, то должники будут обрабатываться с востока на запад по часовым поясам.

3.Нажмите кнопку ОК. Новая кампания будет добавлена в список со статусом "Создана". Далее сформируйте группу участников кампании обзвона.
