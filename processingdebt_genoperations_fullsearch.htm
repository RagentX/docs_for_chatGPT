Расширенный поиск должника (Дополнительные данные)

В форме Обработка должников на вкладке Все доступна функция расширенного поиска должника. Для этого выполните следующие действия:

1.Нажмите кнопку Дополнительные данные. Откроется форма выбора данных для отображения.

2.В форме установите переключатель для тех данных, которые требуются для расширенного поиска, и нажмите кнопку Показать. На вкладке появятся дополнительные столбцы, по которым можно делать фильтрацию и группировку.

Примечание: в случае выбора вариантов Телефоны или Адреса если должник имеет несколько телефонов или адресов, в таблице для него будет отображаться количество записей, равное количеству этих телефонов/адресов (записи будут отличаться только дополнительными колонками).

 Примечание. При отображении произвольных атрибутов с типом "Словарь" реализована отдельная ветка построения отображения произвольных атрибутов для выключенного значения RandomValue.DictVariable.UseCodeStorage.

·Если константа RandomValue.DictVariable.UseCodeStorage выключена, то просто выводится value

·Если константа RandomValue.DictVariable.UseCodeStorage включена, то выводится текст термина, хранящегося в value, по настроенному словарю 95
