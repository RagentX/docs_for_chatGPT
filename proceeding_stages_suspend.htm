Приостановление

Приостановление судебных дел одинаково работает для следующих типов судебных дел:

·Исковое производство;

·Правопреемство;

·Банкротство.

Меню кнопки Приостановление включает следующие пункты:

·Приостановить;

·Отмена приостановления;

·Возобновление.

Кнопка Приостановление доступна, если доступен хотя бы один из ее подпунктов.

Приостановление

Условия доступности

Кнопка Приостановить доступна, если судебное дело (иск, правопреемство или банкротство) находится в одном из статусов из списка:

·Подан;

·Определение о принятии к произв.;

·Отказ в принятии;

·Определение о рассмотрении по сущ.;

·Отказ в удовлетворении иска;

·Подана апелляция;

·Апелляция отклонена;

·Оставление иска без движения;

·Определение об оставл. без рассмотрения;

·Не подготовлена апелляция;

·Запрос ГП апелляции;

·ГП апелляции оплачена;

·Подготовлена апелляция;

·Не подготовлена кассация;

·Запрос ГП кассации;

·ГП кассации оплачена;

·Подготовлена кассация;

·Подана кассация;

·Кассация отклонена;

·Не подготовлен надзор;

·Запрос ГП надзора;

·ГП надзора оплачена;

·Подготовлен надзор;

·Подан надзор;

·Надзор отклонен;

·Отказ в правопреемстве;

·Не подготовлена частная жалоба;

·Подготовлена частная жалоба;

·Подана частная жалоба;

·Частная жалоба удовлетворена;

·Частная жалоба отказ;

·Возвращение заявления;

·Частичное удовлетворение иска.

Порядок действий

Для приостановления дела выполните следующие действия:

1.Нажмите кнопку Приостановление и в раскрывшемся списке выберите Приостановить.

2.В открывшемся диалоговом окне нажмите кнопку Yes.

3.В открывшейся форме Приостановление заполните следующие поля:

·Дата приостановления — укажите дату. По умолчанию установлена текущая дата. Поле обязательно для заполнения.

·Причина — выберите причину из списка. В списке указаны термины словаря № 319, имеющие в качестве родительского термина "Приостановление". Поле обязательно для заполнения.

Примечание. Поле Причина на форме не отображается, если в словаре № 319 Причина перевода судебного дела в текущий статус нет ни одного термина, имеющего в качестве родительского термин Приостановлено.

·Комментарий — дополнительная информация, не обязательно для заполнения.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

Примечание. Для правопреемственных дел настроенные действия не формируются.

6.Дело будет переведено в статус "Приостановлено".

7.Действие протоколируется.

Отмена приостановления

Кнопки Отмена приостановления доступна, если дело находится в статусе "Приостановлено".

Для отмены приостановления дела в архив выполните следующие действия:

1.Нажмите кнопку Приостановление и в раскрывшемся списке выберите Отмена приостановления.

2.В открывшемся диалоговом окне нажмите кнопку Yes.

3.В открывшейся форме Отмена приостановления заполните следующие поля:

·Дата — укажите дату. По умолчанию установлена текущая дата. Поле обязательно для заполнения.

·Комментарий — дополнительная информация, не обязательно для заполнения.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

Примечание. Для правопреемственных дел настроенные действия не формируются.

6.Дело будет переведено в статус "Отмена приостановления".

7.Действие протоколируется.
