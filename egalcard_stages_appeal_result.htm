Результат обжалования

Примечание. Результат обжалования доступен только если включен функционал обжалования. Функционал обжалования используется в приказном производстве, если константа LawAct.Appeal.Use включает значение 1.

Меню кнопки Рез. обжалования включает следующие пункты:

·Решение суда обжаловано;

·Жалоба отклонена.

Отметка результата обжалования доступна для дел в статусах:

·Подана апелляция;

·Подана кассация;

·Подан надзор.

Решение суда обжаловано

Для отметки результата обжалования:

1.Нажмите кнопку Рез. обжалования.

2.Выберите результат обжалования Решение суда обжаловано.

3.Появится окно подтверждения. Нажмите Да.

4.При необходимости выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.Открывается окно для редактирования составляющих суммы по решению суда. Вид окна зависит от значения константы LawActIskDebtComponentList. Кроме составляющих суммы по решению суда в окне отображается утвержденная госпошлина. Введите необходимые данные и нажмите ОК.

Константа LawActIskDebtComponentList пуста

Константа LawActIskDebtComponentList не пуста

7.Дело принимает статус:

·Апелляция — "Апелляция удовлетворена";

·Кассация — "Кассация удовлетворена";

·Надзор — "Надзор удовлетворен".

Жалоба отклонена

Для отметки результата обжалования:

1.Нажмите кнопку Рез. обжалования.

2.Выберите результат обжалования Жалоба отклонена.

3.Появится окно подтверждения. Нажмите Да.

4.При необходимости выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.Дело принимает статусы:

·Апелляция — "Апелляция отклонена";

·Кассация — "Кассация отклонена";

·Надзор — "Надзор отклонен".
