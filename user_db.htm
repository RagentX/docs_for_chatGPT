Управление доступом к базам данных

Show/Hide Hidden Text

Система позволяет предоставлять пользователю доступ к одной или нескольким БД. При входе в систему пользователь будет иметь возможность выбрать БД из тех, к которым ему предоставлен доступ. Если пользователю не предоставлен доступ ни к одной БД, он не сможет выполнять какие-либо действия в системе.

Добавление доступа к базе данных

1.В форме Пользователи системы выберите нужного пользователя. При этом в списке Доступ к базам будут отображаться базы данных, к которым ему уже предоставлен доступ.

2.Нажмите кнопку Добавить, расположенную под списком доступов к базам данных. Откроется форма редактирования доступа к БД.

3.В форме заполните следующие поля:

·Имя сервера — имя сервера, на котором размещается БД, выбирается из списка зарегистрированных серверов;

·Название БД — имя нужной БД, выбирается из списка БД, зарегистрированных на сервере;

·Пользователь — имя пользователя, которому предоставляется доступ к БД, выбирается из списка зарегистрированных пользователей;

·Заголовок меню — заголовок, который будет иметь главное меню системы;

·Комментарий — дополнительная информация.

4.Нажмите кнопку ОК, чтобы сохранить запись.

Редактирование параметров доступа к базе данных

1.В форме Пользователи системы выберите нужного пользователя.

2.В списке Доступ к базам выберите БД.

3.Нажмите кнопку Изменить, относящуюся к списку доступов к БД, или выполните двойной щелчок левой клавишей мыши по выбранной записи.

4.В форме редактирования доступа к БД измените необходимые атрибуты.

5.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление доступа к базе данных

1.В форме Пользователи системы выберите нужного пользователя.

2.В списке Доступ к базам выберите БД.

3.Нажмите кнопку Удалить, относящуюся к списку доступов к БД.

4.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.
