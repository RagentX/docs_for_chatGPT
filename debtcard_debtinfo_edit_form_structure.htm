Ведение списка задолженностей по услугам для долгов по ЖКХ

Show/Hide Hidden Text

Внимание! Данный раздел описывает действия на форме "Информация о долге" только для долгов по ЖКХ!

Ведение списка задолженностей по услугам для долгов ЖКХ осуществляется на форме "Информация о долге" для долгов по ЖКХ в области данных Структура долга.

Ведение списка задолженностей по услугам регулируется следующими доступами:

№

Название доступа

Значение доступа

Описание доступа

821

DEBT_CARD_DDI_EDIT

TRUE

право на редактирование атрибутов на форме "Задолженность по услуге".

826

DEBT_CARD_DDI_ADD

TRUE

право на добавление новых периодов на форме "Информация о долге". Важно: работает только при включенном доступе № 821 DEBT_CARD_DDI_EDIT.

827

DEBT_CARD_DDI_DEL

TRUE

право на удаление периодов на форме "Информация о долге".

Примечание. Для того чтобы настроить отображение дополнительных полей в списке структура долга, обратитесь в техническую поддержку.

Для поддержки: В грид на форме «Информация о долге» (fmDebtDetailGKH) выводятся дополнительно те столбцы, user_name которых заполнен в таблице column_spec (parent_id=73). Названия столбцов выводятся по user_name из таблицы column_spec.

Доступны следующие действия:

Добавление записи о задолженности по услуге

Добавление записей о задолженности по услуге доступно только пользователям, в роли которых доступы № 821 DEBT_CARD_DDI_EDIT и № 826 DEBT_CARD_DDI_ADD имеют значение "TRUE".

Для того чтобы добавить запись о задолженности по услуге, выполните следующее:

1.В модуле Карточка должника на вкладке Долги выберите долг по ЖКХ.

2.Нажмите кнопку Изменить или выполните двойной щелчок левой клавишей мыши по выбранной записи.

3.Откроется форма Информация о долге. Нажмите кнопку Добавить, относящуюся к области данных Структура долга.

4.Откроется форма Задолженность по услуге. Заполните следующие поля:

·Тип услуги — см. словарь № 87;

·Составляющая — см. словарь № 62;

·С: — дата начала периода;

·По: — дата окончания периода;

·Статус задолженности — см. словарь № 10005 "Статусы задолженности";

·Плановая дата списания;

·Сумма;

·Сегмент счета (периода) — см. словарь № 10001;

·Комментарий — дополнительная информация.

5.Нажмите ОК. Запись будет добавлена в список Структура долга.

6.Нажмите ОК, чтобы закрыть форму Информация о долге.

Редактирование записи о задолженности по услуге

Редактирование полей формы Задолженность по услуге доступно пользователям, в роли которых доступ № 821 DEBT_CARD_DDI_EDIT имеет значение "TRUE".

Для того чтобы изменить запись о задолженности по услуге, выполните следующее:

1.В модуле Карточка должника на вкладке Долги выберите долг по ЖКХ.

2.Нажмите кнопку Изменить или выполните двойной щелчок левой клавишей мыши по выбранной записи.

3.Откроется форма Информация о долге. Выберите запись в списке Структура долга (Для выбора записи с нулевой суммой включите флаг Показывать нулевые составляющие).

4.Нажмите кнопку Изменить, относящуюся к области данных Структура долга.

5.Откроется форма Задолженность по услуге. Измените нужные значения полей.

6.Нажмите ОК. Запись в списке Структура долга будет изменена.

7.Нажмите ОК, чтобы закрыть форму Информация о долге.

Примечание. Если в роли пользователя доступ № 821 DEBT_CARD_DDI_EDIT имеет значение "FALSE", то все поля формы Задолженность по услуге заблокированы.

Запрещено редактирование полей формы.

Удаление записи о задолженности по услуге

Удаление записей о задолженности по услугам доступно пользователям, в роли которых доступ № 827 DEBT_CARD_DDI_DEL имеет значение "TRUE".

Для того чтобы удалить запись о задолженности по услуге, выполните следующее:

1.В модуле Карточка должника на вкладке Долги выберите долг по ЖКХ.

2.Нажмите кнопку Изменить или выполните двойной щелчок левой клавишей мыши по выбранной записи.

3.Откроется форма Информация о долге. Выберите запись в списке Структура долга (Для выбора записи с нулевой суммой включите флаг Показывать нулевые составляющие).

4.Нажмите кнопку Удалить, относящуюся к области данных Структура долга.

5.Запись будет удалена из списка Структура долга.

6.Нажмите ОК, чтобы закрыть форму Информация о долге.
