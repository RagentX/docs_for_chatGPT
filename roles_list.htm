Форма "Роли и доступы"

Форма Роли и доступы содержит следующие элементы:

·список Роли, содержащий перечень ролей пользователей;

·кнопки для добавления, изменения, удаления ролей;

·вкладки с настройками доступов для выбранной роли:

·Доступы, содержащая перечень доступов на выполнение действий в модулях (см. Управление доступами для роли);

·Элементы экранных форм;

·СМС, содержащая перечень шаблонов SMS-сообщений с указанием уровня доступа к каждому из них (см. Управление доступом к шаблонам SMS-сообщений);

·Email, содержащая перечень шаблонов Email-сообщений с указанием уровня доступа к каждому из них (см. Управление доступом к шаблонам Email-сообщений);

·Письма, содержащая перечень печатных форм с указанием уровня доступа к каждой из них (см. Управление доступом к формированию писем);

·Произвольные отчеты, содержащая перечень произвольных отчетов с указанием уровня доступа к каждому из них (см. Управление доступом к произвольным отчетам);

·Произвольные функции, содержащая перечень предварительно созданных пользовательских функций/операций (см. Управление доступом к произвольным функциям);

·Печатные формы (Суд), содержащая перечень документов судебной работы с указанием уровня доступа к каждому из них (см. Управление доступом к печатным формам (Суд));

·Печатные формы (ИП), содержащая перечень документов исполнительного производства с указанием уровня доступа к каждому из них (см. Управление доступом к печатным формам (ИП));

·Произвольные атрибуты, содержащая список произвольных атрибутов, доступных пользователю в выбранной роли в модуле Карточка должника на вкладке Произвольные атрибуты (см. Управление доступом к произвольным атрибутам).

·Типы вложений, содержащая список типов вложений, доступных пользователю в выюранной роли (см. Управление доступом к типам вложений).

·кнопки для добавления, изменения, удаления прав доступа на выбранной вкладке.

 

При мультиязычности в списке доступов добавлено поле Описание на альтернативном яззыке"
