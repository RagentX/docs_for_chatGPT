Карточка судебной работы в приказном производстве

Обновлено до 50 релиза 03.11.2020.

Модуль Карточка судебной работы предназначен для работы с информацией о должниках, находящихся на стадиях судебного производства. Карточка судебной работы в приказном производстве открывается из модуля Юридический департамент, вкладка Приказы (см. Переход к карточке судебной работы/исполнительного производства).

Создание приказного производства осуществляется:

·в модуле Список должников при выборе соответствующего пункта контекстного меню (вызывается правой кнопкой мыши) — см. Судебная работа;

·в модуле Карточка должника на вкладке Долги — см. Судебная работа;

·в модуле Юридический департамент, на вкладке Иски — см. Преобразование судебного иска в приказ.

Право для входа в модуль имеют только те пользователи, в роли которых доступ LAW_ACT_CARD_VIEW установлен в значение TRUE.
