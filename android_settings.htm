Настройки

Система поддерживает следующие настройки (Рисунок 15‑1).

·Адрес сервера;

·Количество записей в списке ближайших точек;

·Логирование действий;

·Массовая загрузка документов;

·Использование PIN-кода:

·Выбор языка;

·Принудительная перезагрузка информации.

Рисунок 15-1. Меню настроек системы.

Адрес сервера

При прикосновении к строке "Адрес сервера" открывается форма "Введите URL" (Рисунок 15‑2), предназначенная для ввода адреса сервера. Введите нужную информацию и нажмите ОК.

Рисунок 15‑2. Форма ввода адреса сервера.

Количество записей в списке ближайших точек

Для настройки количества отображаемых адресов:

1.В списке настроек выберите "Количество записей в списке ближайших точек" и нажмите на эту строку.

2.В открывшейся форме (Рисунок 15‑3) введите нужное число.

Примечание: Количество записей не может превышать 25.

3.Нажмите OK.

Рисунок 15‑3. Форма ввода количества записей в списке ближайших точек.

Логирование действий

Включение/выключение функции логирования действий осуществляется прикосновением к соответствующей строке списка настроек.

Поддерживаются следующие режимы логирования (Рисунок 15‑4):

·Отключить логирование;

·Логирование действий;

·Отладочное логирование.

Текущий режим отмечен переключателем (radio-button).

Для настройки режима логирования:

1.В списке настроек коснитесь ("tap") строки "Логирование действий".

2.В открывшейся форме выберите режим логирования из списка, коснувшись соответствующей строки.

3.Если вы не хотите менять текущий режим логирования, нажмите "Отмена".

Рисунок 15‑4. Выбор режима логирования действий пользователя.

Массовая загрузка документов

Массовая загрузка документов осуществляется при использовании подключения к сети интернет с использованием следующих технологий:

·Wi-Fi;

·Мобильный интернет.

Если устройство подключено к сети Интернет с использованием Wi-Fi, то массовая загрузка доступна.

Если подключение к сети Интернет осуществляется с использованием мобильного подключения, для разрешения функционала массовой загрузки включите флаг "Загрузка документов через мобильный интернет" (Рисунок 15‑5).

Рисунок 15‑5. Включение загрузки через мобильный интернет.

Использование PIN-кода

Пункт "Использовать PIN-код" в настройках виден только в тех случаях, когда значение константы Android.PinCodeLogin.Use равно "Истина".

Настройка пин-кода дает возможность одному из пользователей авторизовываться в Системе используя пин-код.

Использование пин-кода включено, когда переключатель в поле "Использовать PIN-код" находится в положении "On". Если переключатель находится в положении "Off", то использование пин-кода при авторизации отключено.

Пин-код представляет собой цифровой пароль, состоящий из фиксированного количества знаков. Длина пин-кода определяется значением константы Android.PinCodeLogin.DigitCnt.

Примечание. В ситуации, когда у пользователя не отключено использование пин-кода (переключатель стоит в положении "On"), но значение пин-кода не задано, после авторизации в приложении пользователю предлагается установить пин-код (Рисунок 15‑6). При выборе "Пропустить" переключатель будет установлен в положение "Off".

Настройка пин-кода

Настройка пин-кода может быть осуществлена при выключенном переключателе. Если пин-код настроен, но вы хотите его сменить, то сначала коснитесь строки "Использовать PIN-код" для перевода переключателя в положение "Off".

Для того чтобы настроить пин-код, выполните следующие действия:

1.В меню "Настройки" коснитесь строки "Использовать PIN-код". Откроется окно для первого ввода значения пин-кода (Рисунок 15‑7).

2.Введите пин-код, используя цифровую клавиатуру. По мере ввода цифры будут заменяться на экране символами "*" (Рисунок 15‑8). Для отмены введенного символа используйте кнопку .

3.После того, как пин-код введен, нажмите ОК. Откроется окно для подтверждения пин-кода (Рисунок 15‑9). Введите пин-код повторно. Если введенный пин-код не совпадет с введенным в первом окне, то процедура ввода пин-кода повторится — снова появится окно для первого ввода.

4.После успешного повтора пин-кода появится окно подтверждения (Рисунок 15‑10). Нажмите ОК.

5.Вы вернетесь в меню настроек. Переключатель будет находиться в положении "On".

Отключение пин-кода

Если переключатель пункта "Использовать PIN-код" находится в положении "On", коснитесь его. Переключатель перейдет в положение "Off".

Принудительная перезагрузка информации

При выборе пункта "Принудительная перезагрузка" осуществляется полная загрузка всех данных из БД в мобильное устройство.
