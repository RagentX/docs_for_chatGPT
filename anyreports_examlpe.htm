Пример создания произвольного отчета

В данном разделе рассматривается пример создания отчета, содержащего ФИО должника, остаток долга и номер договора для всех должников с остатком больше 10000 руб. и фамилией на букву "А".

1.В форме Произвольные отчеты нажмите кнопку , относящуюся к списку произвольных отчетов, и в открывшейся форме заполните поля, как показано на рисунке. В поле SQL-запрос введите текст SQL-запроса, представленный ниже.

 

select p.f + ' ' + p.i + ' ' + p.o fio, d.debt_sum, d.contract 

from debt d join person p on d.parent_id = p.id

where d.debt_sum > :param1 and d.currency = 1 and p.f like 'А%'

order by p.f, p.i, p.o

 

Данный запрос означает, что из таблицы персон (person p) будут выбраны столбцы с фамилией (p.f), именем (p.i), отчеством (p.o) должника и объединены в столбец ФИО (fio), из таблицы долгов (debt d) для этих должников будут выбраны столбцы с суммой долга (d.debt_sum) и номером договора (d.contract).

Отбор данных из таблиц будет производиться с учетом того, что сумма долга должна быть больше значения, заданного входным параметром param1 (d.debt_sum > :param1), код валюты долга равен единице (d.currency = 1) и фамилия должника начинается на букву "А" (p.f like 'А%').

Данные в выходном отчете будут отсортированы сначала по фамилии должника, потом по имени, потом по отчеству (order by p.f, p.i, p.o).

2.Нажмите кнопку ОК.

3.На вкладке Описание полей нажмите кнопку , в открывшейся форме введите данные о поле отчета fio и нажмите кнопку ОК. Повторите шаг, чтобы добавить данные о полях отчета debt_sum и contract.

   

4.На вкладке Входные параметры нажмите кнопку , в открывшейся форме введите данные о входном параметре отчета param1 и нажмите кнопку ОК.

5.В форме Произвольные отчеты нажмите кнопку Сформировать и выберите пункт Вывести на экран. Откроется форма для ввода значения входного параметра, созданного на шаге 4.

6.В форме введите значение параметра (например, 100000) и нажмите кнопку ОК. Отчет будет выведен на экран. В нем будут отображаться данные о должниках, сумма долга которых более 100 000 руб. и фамилия начинается на букву "А".
