Боковое меню

Для переключения между экранными формами, настройки и выхода из Системы используется боковое меню (Рисунок 3‑1). Боковое меню включает следующие пункты:

·Меню — переход на экранную форму "Главное меню";

·Выезды — переход на экранную форму "Выезды";

·Звонки — переход на экранную форму "Звонки";

·Платежи — переход на экранную форму "Платежи";

·Плановые платежи — переход на экранную форму "Плановые платежи";

·Упущенные платежи — переход на экранную форму "Упущенные платежи";

·Должники — переход на экранную форму "Должники";

·История контактов — переход на экранную форму "История контактов";

·Ближайшие — переход на экранную форму "Ближайшие";

·Настройки — переход на форму "Настройки";

·Выход.

Для доступа к боковому меню можно использовать следующие способы:

·кнопка , расположенная в левом верхнем углу рабочей области экрана;

·кнопка "Назад", расположенная непосредственно на устройстве;

·движение "свайп" (swipe) слева направо.

Для выхода из бокового меню можно использовать следующие способы:

·прикосновение к экрану в области, свободной от меню;

·движение "свайп" (swipe) справа налево.
