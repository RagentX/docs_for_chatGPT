Выгрузка реестра в формат MS Excel

В списке должников выгрузка реестра записей о должниках в формат MS Excel доступно только пользователям, в роли которых доступ № 690 DEBT_LIST_EXPORT_DIALER имеет значение TRUE.

Для выгрузки реестра выполните следующие действия:

1.В форме Список должников сформируйте выборку при помощи мультифильтров и/или сложного фильтра.

2.Выберите одну или более записей, при необходимости используя клавиши SHIFT и/или CTRL.

3.Активируйте контекстное меню, нажав на правую клавишу мыши, и выберите пункт Выгрузить–>Реестр.

4.В открывшемся диалоговом окне Windows укажите путь и имя файла, в который будут выгружены данные, и нажмите кнопку Save (Сохранить). По окончании выгрузки будет выдано сообщение о завершении создания файла.

5.Нажмите кнопку ОК, чтобы завершить процедуру.
