Формат файла данных

Для загрузки используются данные в формате файла MS Excel. Доступны как файлы Excel 97 - 2003 (с расширением ".xls"), так и старших версий (с расширением ".xlsx").

Файл должен содержать колонки с данными в следующем порядке:

·ФИО — фамилия, имя, отчество должника; поле обязательно, если не заполнены номер договора, номер счета и ID в системе клиента;

·Портфель — название портфеля;

·Банк — название банка, которому принадлежит портфель; обязательное поле;

·Номер договора — поле обязательно, если не заполнены ФИО, номер счета и ID в системе клиента;

·Номер счета — поле обязательно, если не заполнены ФИО, номер договора и ID в системе клиента;

·ID в системе клиента — поле обязательно, если не заполнены ФИО, номер договора и номер счета;

·Статус — код статуса долга (см. словарь № 6), обязательное поле. Статус долга должен совпадать с одним из допустимых значений:

·3 — Отказ;

·4 — Контакт установлен;

·5 — Обещание;

·6 — Завершено;

·9 — Проблемный;

·8 — Отозван клиентом;

·10 — К возврату Клиенту;

·11 — Возврат в работу;

·12 — Без перспектив;

·пользовательские статусы с кодом >100;

·Код причины — код причины перевода в выбранный статус; разным статусам соответствуют разные словари со списками причин — см. Списки причин для перевода долгов в определенные статусы;

·Комментарий — необязательная колонка.
