Судебная работа по поручителю

Общее описание функционала

По кнопке на форме "Поручитель доступно создание судебных дел и ИП, по которым ответчиком является поручитель.

Доступность кнопки на форме определяется доступностью пунктов ее меню. Если все пункты недоступны, то кнопка не видна. Если хотя бы один пункт доступен, то кнопка видна.

Меню кнопки включает следующие пункты:

·Создать суд. дело (предв.);

·Создать судебный приказ;

·Создать правопреемство;

·Создать судебный иск;

·Создать входящий иск;

·Исполнительное производство;

·Исполнительная надпись (только если включен функционал исполнительной надписи);

·Создать встречное суд. дело — данный пункт может отображаться только в карточке поручителя, открытой из карточки судебной работы. При выборе этого пункта выполняется создание встречного иска.

Примечание. Банкротство по поручителю не создается.

Порядок действий

Для того чтобы сформировать дело судебной работы или ИП, по которому ответчиком является поручитель, выполните следующие действия:

1.В карточке должника, карточке судебной работы или в карточке исполнительного производства активизируйте вкладку "Поручители".

2.В списке поручителей выберите нужную запись и нажмите кнопку или выполните двойной клик по записи.

3.В открывшейся форме "Поручитель" нажмите кнопку .

4.В открывшемся меню кнопки выберите нужный пункт:

·Создать суд. дело (предв.) — по текущему долгу будет создано неопределенное судебное дело с поручителем в качестве ответчика (далее см. Карточка судебной работы в судебном производстве);

·Создать судебный приказ — по текущему долгу будет создан судебный приказ с поручителем в качестве ответчика (далее см. Карточка судебной работы в приказном производстве);

·Создать правопреемство — по текущему долгу будет создано правопреемственное дело с поручителем в качестве ответчика (далее см. Карточка судебной работы в правопреемстве);

·Создать судебный иск — по текущему долгу будет создан судебный иск с поручителем в качестве ответчика (далее см. Карточка судебной работы в исковом производстве);

·Исполнительное производство — по текущему долгу будет создано исполнительное производство с поручителем в качестве ответчика (далее см. Карточка исполнительного производства).

·Исполнительная надпись — по текущему долгу будет создана исполнительная надпись с поручителем в качестве ответчика (далее см. Карточка исполнительной надписи).

5.Для каждого долга из списка осуществляется ряд проверок возможности создания судебного дела или исполнительного производства в зависимости от настроек Системы.

6.По результатам соответствующей проверки будет выдано сообщение. Если требуется решение пользователя о создании дела, то сообщение будет содержать вопрос.

Подробное описание возможностей по судебной работе с долгами приведено в описании модуля Юридический департамент.
