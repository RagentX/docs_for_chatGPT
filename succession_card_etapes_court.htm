Судебное разбирательство

При установке отметки об итогах судебного разбирательства возможны следующие варианты:

·Правопреемство установлено;

·Отказ в правопреемстве.

Правопреемство установлено

Для установки отметки об установлении правопреемства, выполните следующие действия:

1.В карточке судебной работы на вкладке Правопреемство в разделе Правопреемство заполните поля:

·Дата вынесения решения.

2.Нажмите кнопку Судебное разб.. В раскрывшемся списке выберите Правопреемство установлено.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить действие.

Если какие-либо обязательные поля не заполнены, будет выдано информационное сообщение. Нажмите в нем кнопку ОК, после чего произойдет возврат в карточку судебной работы для внесения данных.

4.Выполните прикрепление документа.

5.Правопреемство переводится в статус "Правопреемство установлено".

6.Действие протоколируется. В поле Действие записывается: "Правопреемство установлено".

Отказ в правопреемстве

Для установки отметки об отказе в правопреемстве, выполните следующие действия:

1.В карточке судебной работы на вкладке Правопреемство в разделе Правопреемство заполните поля:

·Дата вынесения решения.

2.Нажмите кнопку Судебное разб.. В раскрывшемся списке выберите Отказ в правопреемстве.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить действие.

Если какие-либо обязательные поля не заполнены, будет выдано информационное сообщение. Нажмите в нем кнопку ОК, после чего произойдет возврат в карточку судебной работы для внесения данных.

4.Выполните прикрепление документа.

5.Правопреемство переводится в статус "Отказ в правопреемстве".

6.Действие протоколируется. В поле Действие записывается: "Отказ в правопреемстве".

1.Далее возможно обжалование результатов решения Суда.
