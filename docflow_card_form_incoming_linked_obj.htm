Вкладка "Связанные объекты"

Вкладка Связанные объекты и действия на ней абсолютно идентичны для входящих и исходящих документов.

На вкладке Связанные объекты отображается список объектов Системы, для которых установлена связь с данным документом. Это могут быть:

·Долги;

·Дела судебной работы;

·Дела исполнительного производства.

Вообще, список объектов Системы, которые могут быть связаны с документом модуля "Документооборот", определяется по согласованию с вендором. При необходимости изменения обратитесь в техническую поддержку. Список объектов Системы, которые могут быть связаны с документом модуля "Документооборот", определяется значением технической константы DocflowCard.ObjectForDocument.

В списке связанных объектов выводятся все объекты (долги, судебные дела, ИП), связанные с документом:

·Для долгов поля ID дела, Тип дела пустые, в поле Ответственный выводится закреплённый за долгом оператор, в поле Статус — статус долга;

·Для судебного дела или ИП в поле Ответственный выводится ответственный оператор по делу/ИП. Для приказов в поле Статус выводится статус дела law_act.status по словарю 18. Для исков, банкротства, правопреемства в поле Статус выводится статус дела law_act.act_status по словарю 25. Для неопределенных судебных дел (предв.) в поле Статус выводится статус дела law_act.pre_act_status по словарю 252. Для ИП в поле Статус выводится статус ИП law_exec.state по словарю 77.

Поля вкладки "Связанные объекты"

Вкладка Связанные объекты входящего документа включает следующие элементы:

·Таблица со списком связанных объектов, содержащая следующие поля:

·ID Должника;

·ID долга;

·ID дела;

·Тип дела;

·ФИО/Название;

·Договор;

·Дата рождения;

·Серия;

·Номер;

·ИНН;

·Статус;

·Ответственный.

·Кнопки:

·Кнопка Добавить — для добавления связи с объектом. Доступность кнопки определяется доступом № 677 DOCFLOW_CARD_RELATIONS_EDIT.

·Кнопка Удалить — для удаления связи с объектами с выделенными объектами. Доступность кнопки определяется доступом № 677 DOCFLOW_CARD_RELATIONS_EDIT.
