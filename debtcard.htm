Карточка должника

Обновлено до 50 релиза 03.11.2020.

Карточка должника предназначена для работы с данными о должнике и его долгах.

Карточка должника открывается при создании или редактировании записи о должнике в модуле "Список должников", а также из других модулей системы.

Право для входа в модуль имеют только те пользователи, в роли которых доступ DEBT_CARD_VIEW установлен в значение TRUE, значение доступа DEBT_CARD_CNT больше или равно единице.

Полный перечень доступов для работы с карточкой должника приведен в таблице.

DEBT_CARD_REQUEST_REPORT_LIST

"ALL" или список

Содержит через запятую список отображаемых отчетов, доступных по кнопке "Отчеты" на вкладке "Долги"  (ALL – доступны все). Например, отчет виден в списке согласно настройке константы DebtCardRequestReport2. Для отображения видимости отчета согласно роли в доступ необходимо добавить значение 2.
