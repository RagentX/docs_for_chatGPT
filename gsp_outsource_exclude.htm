Исключение из аутсорсинга

Операция доступна из модулей:

·Список должников — Работа с долгами в аутсорсинге;

·Группы свободного поиска — Аутсорсинг.

Исключить долги из аутсорсинга можно в следующих случаях:

·если исходящий портфель находится в статусе "Новый", то пользователь может исключать долги при наличии доступа OUTSOURCING_FILL;

·если исходящий портфель находится в статусе "Сформирован", то пользователь может исключать долги при наличии у него доступа OUTSOURCING_DEL_FROM_FORMED.

Примечание. При запуске функционала из модуля Группы свободного поиска кнопку Работа с долгом видно, если в РМ Группы должников было предоставлено право Работа с долгом. Аутсорсинг и на вкладке есть поле ID долга.

Для исключения долгов из аутсорсинга выполните следующие действия:

1.Выберите записи о долгах, которые требуется исключить из аутсорсинга. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL

2.Из формы Группы свободного поиска нажмите кнопку Работа с долгом. В открывшемся меню выберите пункт Аутсорсинг, затем — Исключить из аутсорсинга. Из формы Список должников в контекстном меню (вызывается правой кнопкой мыши) выберите пункт Аутсорсинг, затем Исключить из аутсорсинга.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

4.В результате будет выдано сообщение об исключении выбранных долгов из аутсорсинга. Нажмите кнопку ОК, чтобы завершить операцию.

Если производится попытка исключить долг из исходящего пакета в статусе "Передан", то будет выдано информационное сообщение, и действие выполнено не будет.
