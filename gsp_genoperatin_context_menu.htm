Контекстное меню модуля

Контекстное меню вызывается щелчком правой кнопки мыши в области списка.

На вкладках модуля Группы свободного поиска в контекстном меню доступны следующие действия:

·Открыть карточку должника в новом окне;

·Копировать строки — осуществляется копирование выделенных строк в буфер обмена;

·Копировать ячейку — осуществляется копирование текста выделенной ячейки в буфер обмена;

·Выделить все;

·Обновить;

·Поиск по долгу — см. Поиск по долгу.

·Сохранить выборку — см. Сохранить выборку (Выгрузка данных в Excel, XML, CSV);

·Входящий звонок — см. Регистрация входящего звонка;

·Входящий звонок (текущий долг) — см. Регистрация входящего звонка;

·Установить значение — см. Групповое изменение атрибута;

·Распределение по списку;

·Объединение должников;

·Сформировать с подпунктами (см. Формирование писем, SMS, Email):

·Письма;

·SMS;

·Email;

·Судебная работа;

·Информация об операторе;

·Выгрузить — см. Выгрузка данных для автодайлера Genesys;

·Сформировать документы по шаблону — см. Формирование документов судебной работы.
