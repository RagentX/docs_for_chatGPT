Форма "Настройка дерева контактов"

Форма Настройка дерева контактов содержит следующие элементы:

·поле Тип контакта, предназначенное для выбора типа контактов;

·поле Тип дерева, предназначенное для выбора типа дерева контактов при использовании функционала независимых деревьев контакта;

·поле Стадия соответствует стадии взыскания долга, с которой работает оператор. Для каждой стадии настраивается отдельное дерево контакта типа "Звонок".

Примечание. Стадия используется только для основного дерева контактов. Дополнительные деревья контактов не имеют стадии.

·дерево возможных результатов выбранного типа контактов;

·кнопки для добавления, изменения, удаления результатов контактов.

Результат контакта может быть:

·промежуточным — если для него созданы подчиненные результаты контакта;

·окончательным, в котором определены параметры детальных данных результата контакта.

Если справа от результата стоит знак /, это значит, что данный результат контактов является промежуточным и содержит подчиненные результаты. Чтобы отобразить подчиненные результаты контактов, нажмите знак , чтобы скрыть — нажмите знак .

Пример. На рисунке ниже промежуточными являются, например, результаты контактов "Разговор с должником" и "Оплата позже уст. срока", а окончательными, детализирующими их — результаты контактов "Обещание и пользовательский статус", "Обещание с перераспределением" и т.д.

Выбор результата контактов в дереве осуществляется щелчком левой клавиши мыши. Выбранный результат выделяется темно-синим цветом.

Цвет остальных элементов зависит от настроек графического изображения (см. Добавление нового результата контакта).
