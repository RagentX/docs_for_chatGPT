Операция

Список возможных операций и функций перечислен в системном словаре № 273 "Операция/Функция в формулах" на языке пользователя.

В зависимости от текущего состояния дерева формулы раскрывающийся перечень операций может содержать не все, а только некоторое подмножество возможных типов операций.

При выбора в карточке операнда типа "Операция" необходимо указать для него:

·Тип — тип операции. Поле обязательно для заполнения.

Выбранный тип операции появится в текущем узле дерева формулы. При этом для каждого аргумента выбранной операции в дереве формулы автоматически формируется дочерний узел с признаком <ввод> (отсутствие данных).

Операции "Текущая дата" и "Пустая операция" не имеют аргументов. Операции "Округление числа", "Минус" имеют один аргумент. Конструкция "Если" имеет три аргумента, а конструкция "Выбор" — пять аргументов. Все остальные типы операций имеют по два аргумента.

Операции "Сложение чисел", "Сложение строк", "Умножение чисел" являются множественными, первоначальное количество аргументов (два) может быть увеличено до пяти. Для увеличения количества аргументов выберите в дереве узел с названием операции и нажмите кнопку Добавить. Аргумент будет добавлен в дерево формулы без раскрытия карточки операнда формулы.

При задании одного из значений в операции "Выбор" в дереве текущей формулы появится новое значение и автоматически будет сформирован дочерний узел типа "То" с признаком <ввод> (отсутствие данных).
