Правила расчета остатков

Правила расчета остатков применяются для цессий — долгов, проданных клиентами в данную организацию. Правило расчета определяет порядок списания составляющих долга при регистрации или загрузке платежей, который будет применяться в портфеле.

Если в портфеле правило расчета остатков отсутствует, платежи будут только отображаться, но не учитываться в расчете остатка долга.

При загрузке или регистрации одного или более подтвержденных платежей расчет остатка будет производиться только при наличии правила расчета в портфеле. Без правила расчета нет, расчет остатка производиться не будет.

Примечание. При регистрации подтверждённого платежа в историю изменения составляющих долга заносится ссылка на этот платеж. Если загрузка осуществляется через РМ Загрузка платежей, и пользователь вместе с платежами указал и остаток, то правило расчета не срабатывает, ссылка на платеж не загружается в историю изменения составляющих долга, составляющие долга не перерассчитываются.

Каждое правило содержит набор шагов правила (штраф, пени, основной долг и т. д.). Для каждого шага указан порядковый номер его выполнения. Каждый шаг может встречаться в правиле только один раз. Каждый номер также может встречаться в правиле только один раз.

Если создать пустое правило и прикрепить его к портфелю, то составляющие вычитаться не будут, а остаток к погашению будет. Если правила расчета нет, то ни остаток, ни составляющие автоматически вычитаться не будут!

Пример. Допустим, долг составляет 10000 руб., включая: 5000 руб. — основной долг, 2000 — пени, 3000 — проценты. Договором с банком определено, что сначала погашаются пени, затем проценты и только потом основной долг.

В этом случае правило расчета должно включать следующие шаги:

1. Пени

2. Проценты

3. Основной долг

При регистрации платежа на сумму 4000 руб. сначала будет списана сумма пени (станет 0 руб.), потом сумма по процентам (станет 1000 руб.), а основной долг так и останется 5000 руб., т.к. суммы платежа достаточно только на оплату пени и частично на оплату процентов. Таким образом, остаток по долгу составит 6000 руб.

Примечание. Если долг не является цессией, то вычитать суммы автоматически некорректно. Это связано с тем, что банк ежедневно начисляет различные штрафы и пени, и информация по этим формулам есть не всегда. Для цессии обычно все суммы фиксированы, поэтому правило расчета удобно для организации.

Примечание. Необходимо понимать, что сумма составляющих долга не обязательно является остатком долга. Клиент, передавая долг в работу организации, может не предоставлять всей информации, а передавать только основные суммы.

Примечание. При отмене платежа, при условии что в истории изменения составляющих долга есть ссылка на этот платеж, происходит перерасчет составляющих долга в сторону увеличения, но не больше чем было до отмены этого платежа.

Пример. При отмене платежа из примера выше, сумма  составляющих восстановится там все просто. А вот если после платежа из пример поступит еще один платеж на 2000 руб., то сумма процентов станет 0 руб., сумма основного долга станет равна 4000 руб. Далее если мы отменяем этот платеж, то составляющие долга восстановятся до уровня который был перед внесением платежа, т.е. пени = 0 руб. , проценты = 1000 руб., основной долг = 5000 руб.

При этом алгоритм расчет суммы по каждому составляющему долга следующий – берется текущая сумма составляющего долга к ней прибавляется сумма  составляющего которая была до момента внесения платежа и вычитается сумма составляющего после внесения платежа. Если на примере,

- Пени = "0+0-0" = 0

- Проценты = "0+1000-0" = 1000

- Основной долг = "4000+5000-4000" = 5000

Список правил расчета остатков

Для того чтобы открыть список правил расчета, в нижней части формы Клиенты и портфели нажмите кнопку Правило расчета. Форма списка содержит следующие элементы:

·раскрывающийся список правил расчета;

·кнопки для добавления, редактирования, удаления правил расчета;

·список шагов, входящих в состав выбранного правила расчета;

·кнопки для добавления, редактирования, удаления шагов в правиле расчета.

Создание правила расчета остатков

1.В списке правил расчетов нажмите кнопку , относящуюся к списку правил. Откроется форма редактирования нового правила.

2.В форме заполните следующие поля:

·Название — название правила расчета. Поле обязательно для заполнения;

·Комментарий — дополнительная информация.

3.Нажмите кнопку ОК, чтобы сохранить правило.

4.Сформируйте список шагов правила расчета (см. Формирование списка шагов правила расчета).

Формирование списка шагов правила расчета

1.В списке правил расчета выберите нужное правило в раскрывающемся списке.

2.Нажмите кнопку , относящуюся к списку шагов правила. Откроется форма редактирования нового шага правила.

3.В форме заполните следующие поля:

·Название — название шага правила. Если выбранный шаг уже есть в правиле расчета, при сохранении будет выдано соответствующее сообщение. Такой шаг сохранить нельзя;

·Порядок — номер выполнения шага в правиле. Если выбранный номер шага уже есть в правиле расчета, при сохранении будет выдано соответствующее сообщение. Такой шаг сохранить нельзя.

4.Нажмите кнопку ОК, чтобы сохранить шаг правила в списке.

5.Повторите шаги 1-4, чтобы добавить все шаги в правило.

Если требуется отредактировать шаг правила, выберите его в списке и нажмите относящуюся к списку кнопку . Откроется форма редактирования записи правила, в которой измените необходимые атрибуты и нажмите кнопку ОК, чтобы сохранить изменения.

Если требуется удалить шаг из правила, выберите его в списке и нажмите относящуюся к списку кнопку . В открывшемся диалоговом окне подтвердите удаление записи, нажав кнопку Yes.

Редактирование правила расчета остатков

1.В списке правил расчета выберите нужное правило в раскрывающемся списке и нажмите относящуюся к нему кнопку .

2.В форме редактирования правила расчета измените необходимые атрибуты.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Для того чтобы отредактировать шаги правила, см. Формирование списка шагов правила расчета.

Удаление правила расчета остатков

Удалить правило расчета можно только в том случае, если оно не используется ни в одном портфеле, а также не назначено ни одному долгу в каком-либо портфеле.

Для того чтобы удалить правило расчета, выполните следующие действия:

1.В списке правил расчета выберите нужное правило в раскрывающемся списке.

2.Нажмите кнопку , относящуюся к списку правил.

3.В открывшемся диалоговом окне нажмите Yes, чтобы подтвердить удаление правила расчета.

Если правило расчета используется в каких-либо портфелях, будет выдано сообщение о невозможности выполнения операции. Нажмите кнопку ОК, чтобы закрыть его.

Назначение правила расчета портфелю

Выбор правила расчета выполняется при создании/редактировании портфеля в форме редактирования портфеля.

Дополнительно доступна настройка правил расчета остатоков для части долгов портфеля — см. Правило расчета остатков по долгу.
