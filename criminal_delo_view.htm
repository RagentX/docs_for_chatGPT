Просмотр дел по уголовному преследованию

Данные о делах по уголовному преследованию отображаются в форме Уголовное преследование на вкладке Уголовные преследования. Вкладка содержит следующие поля:

·№ дела — идентификатор, порядковый номер дела в системе;

·ФИО должника — фамилия, имя, отчество должника;

·ID должника — идентификатор должника, связанного с делом;

·ID долга — идентификатор долга, связанного с делом;

·№ договора — номер кредитного договора с должником;

·Ответственный — оператор, ответственный за работу с долгом;

·Сумма ущерба;

·Статус — статус дела по уголовному преследованию, см. словарь № 145;

·Номер УД/ОВУД — номер уголовного дела/отказа в возбуждении уголовного дела;

·Дата УД — дата возбуждения уголовного дела;

·Дата талона уведомления;

·Номер талона уведомления;

·Причина уголовного преследования — см. словарь № 146;

·Статья УК;

·Филиал — филиал банка, к которому относится долг (см. словарь № 61);

·Дата передачи в суд;

·Комментарий;

·Указание руководителя.
