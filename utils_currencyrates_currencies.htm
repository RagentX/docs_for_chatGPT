Управление списком валют

Управление списком валют доступно в форме Список валют, которая откроется, если в форме Курсы валют нажать кнопку Валюты. Форма доступна только тем операторам, в роли которых доступ CUR_EXCH_FULL имеет значение TRUE. В форме доступны операции добавления, редактирования, удаления валют.

Первым номером в списке валют указана основная валюта Системы, в которую могут конвертироваться суммы в других валютах.

Список валют

Добавление валюты

Для того чтобы добавить валюту в список, выполните следующие действия:

1.В форме Список валют нажмите кнопку .

2.В открывшейся форме заполните следующие поля:

·Наименование — название валюты. Поле обязательно для заполнения;

·Краткое обозначение — поле обязательно для заполнения;

·Код ЦБР — код Центрального банка России.

3.Нажмите кнопку ОК. Валюта будет добавлена в список.

В результате добавления валюты автоматически добавляются соответствующие записи в словарях № 280 "Валюта (Полное наименование)" и № 277 "Валюта (Краткое обозначение)".

При поддержке мультиязычности термин добавляется для всех языков, которые перечислены в константе MultiLanguage.SupportedLanguageList. После успешного добавления пользователю выводится сообщение:

ВНИМАНИЕ! Не забудьте отредактировать наименование новой валюты на каждом языке в словарях  № 280 "Валюта (Полное наименование)" и № 277 "Валюта (Краткое обозначение)".

Редактирование валюты

Для того чтобы отредактировать данные о валюте, выполните следующие действия:

1.В форме Список валют выберите нужную запись и нажмите кнопку .

2.В форме редактирования данных о валюте измените необходимые данные.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

При редактировании валюты автоматически изменяются соответствующие термины в словарях № 280 "Валюта (Полное наименование)" и № 277 "Валюта (Краткое обозначение)". Код терминов = currency.id.

Если включена поддержка мультиязычности, то изменяются только соответствующие термины на языке пользователя, а не на всех языках.

При редактировании валюты изменения в таблице currency по полям name и short_name осуществляются только, если язык пользователя = основному языку системы, установленному в константе MultiLanguage.SupportedLanguageList.

Удаление валюты

Для удаления доступны только те валюты, в которых нет долгов. При попытке удалить валюту при наличии выраженных в ней долгов будет показано предупреждение и валюта не будет удалена.

Для того чтобы удалить данные о валюте, выполните следующие действия:

1.В форме Список валют выберите нужную запись.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.

В результате удаления валюты автоматически удаляются соответствующие записи в словарях № 280 "Валюта (Полное наименование)" и № 277 "Валюта (Краткое обозначение)".
