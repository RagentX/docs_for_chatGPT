Корректировка нестандартных данных

В процессе преобразования может потребоваться уточнение нестандартных данных в следующих случаях:

·число элементов в ячейке исходного столбца превышает заданное в формате;

·в адресе не идентифицирован город;

·в адресе не идентифицирована улица (только для города);

·в адресе не идентифицирован дом (только для города);

·в регистрационном, фактическом или дополнительном адресе не идентифицирована квартира (только для города);

·наличие неопознанных реквизитов в адресе;

·число элементов адреса меньше заданного в формате;

·число элементов неполного адреса без разбивки столбцов меньше минимально допустимого значения;

·в неполном адресе без разбивки не найден почтовый индекс;

·число элементов неполного адреса без разбивки столбцов, не считая индекса, меньше минимально допустимого значения;

·в адресе отсутствует признак реквизита "Город";

·в адресе отсутствует признак реквизита "Улица";

·в адресе отсутствует признак реквизита "Дом";

·в адресе отсутствует признак реквизита "Квартира";

·в адресе отсутствует признак реквизита "Корпус".

В указанных случаях откроется форма, содержащая:

·сведения о номере записи и причине уточнения данных;

·исходный текст ячейки таблицы до преобразования (в режиме чтения);

·тексты преобразованных элементов ячейки (в режиме чтения и записи).

В форме уточните текст преобразованных элементов и нажмите кнопку OK. На экран будет выведена форма для уточнения очередной нестандартной записи. После уточнения всех нестандартных записей процесс преобразования заканчивается. Все уточненные данные будут внесены в преобразованную таблицу.

Примечание. Корректировка нестандартных реквизитов адреса описаны в разделе Особенности корректировки реквизитов адреса.

В процессе обработки нестандартных данных в форме можно установить флаг Пропустить все. В этом случае после нажатия OK диалог с пользователем по всем остальным нестандартным записям будет отменен. Данный режим следует применять в исключительных случаях, так как качество преобразованных автоматически нестандартных данных не гарантируется.

При нажатии кнопки Отмена в форме уточнения нестандартной записи будет выдано предупреждение:

При нажатии кнопки Yes работа с файлом прекращается, форма с данными будет очищена. Если закрыть форму уточнения нестандартной записи нажатием кнопки в правом верхнем углу, форма с данными будет очищена без дополнительного предупреждения.
