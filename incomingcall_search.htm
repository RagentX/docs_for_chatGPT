Поиск должника (ответственного лица)

Для поиска должника выполните следующие действия:

1.В форме Входящий звонок в разделе Поиск задайте условия поиска, заполнив одно или несколько полей. При этом действуют следующие правила:

·если значение в поле не задано, оно не будет учитываться при поиске;

·для числовых полей (ID долга, Серия и Номер (паспорта)) поиск будет выполняться по точному совпадению значений;

·для строковых полей поиск выполняется по вхождению заданных символов в искомое значение;

·для строковых значений регистр букв не учитывается;

·в строковых значениях можно использовать подстановочные символы:

·"%" — означает "любая (в том числе пустая) строка символов";

·"_" — означает один любой символ.

Пример. Если в поле ФИО/Название указать значение:

· Ива% — в результате поиска будут возвращены все должники, фамилия или название которых начинаются с "Ива";

·%Ива% — в результате поиска будут возвращены все должники, фамилия, имя, отчество или название которых содержит (в том числе начинается или заканчивается) на "Ива";

·%Ива%ва% — в результате поиска будут возвращены все должники, фамилия, имя, отчество или название которых содержит "Ива" и "ва".

2.Выберите значения для флагов По должникам и По поручителям.

При последующем нажатии кнопки Найти в зависимости от состояния флагов осуществляются следующие действия:

·если выключены оба флага — выдается предупреждение, поиск не осуществляется;

·если включен флаг По должникам и выключен флаг По поручителям — поиск осуществляется только по должникам. При выводе результатов в таблицу в области Должники в поле Тип ответственного лица указывается Должник;

·если выключен флаг По должникам и включен флаг По поручителям — осуществляется поиск долгов по введённым данным поручителей. При выводе результатов в таблицу в области Должники в поле Тип ответственного лица указывается Поручитель;

·если включены оба флага — осуществляются оба вида поиска. При выводе результатов в таблицу в поле Тип ответственного лица для найденных должников указывается Должник, а для найденных поручителей указывается Поручитель.

3.В разделе Поиск нажмите кнопку Найти. Общее количество найденных записей будет выведено внизу раздела, а сами найденные записи будут отображаться в списке Должники.

Для выполнения нового поиска можно очистить значения в разделе Поиск, нажав кнопку .

Примечание. Поле для поиска по дате рождения становится доступно при включении константы InnerCall.SearchByBirthDate.Use.
