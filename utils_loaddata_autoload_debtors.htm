Для словарей семейного положения (12), образования (38), причины возникновения задолженности (13), регионов (39), типов объекта (85) реализовано автозаполнение.

По результатам автоматической проверки в файле статистических данных для каждой некорректной строки входного файла должников фиксируется информация о первой выявленной ошибке: "<текущая дата и время> Строка <номер строки> пропущена, <сведения об ошибке>".

В зависимости от типа ошибки сведения об ошибке могут иметь вид:

·неверное количество атрибутов <количество> (меньше <требуемое количество>);

·неверное количество атрибутов <количество> (больше <требуемое количество>);

·неверный формат даты, атрибут <название атрибута>, значение <значение>;

·неверный формат числа, атрибут <название атрибута>, значение <значение>;

·недопустимое значение словарного термина, атрибут <название атрибута>, значение <значение>;

·неверный формат телефона, атрибут <название атрибута>, значение <значение>;

·пустое значение для обязательного атрибута <название атрибута>;

·неверное значение названия портфеля, атрибут <название атрибута>, значение <значение>.

Некорректные строки входного файла должников не загружаются. По результатам загрузки данных в файле статистических данных фиксируется информация:

·о формировании новых записей или обновлении записей в таблицах персон, паспортов и долгов;

·о результатах попыток корректировки данных в таблицах телефонов, адресов и произвольных атрибутов;

·по итогам загрузки из файла должников.

Информация по итогам загрузки из файла должников имеет вид: "<текущая дата и время> Завершение загрузки из файла должников. Всего строк <число строк>, успешно обработанных строк <число строк>, откинутых строк <число строк>".

Примечание. В файле статистических данных не фиксируется информация об обновлении записей в таблице задач на обзвон для операторов.
