Форма "Платежи"

Форма Платежи содержит следующие элементы:

·поля мультифильтров и кнопка Найти для применения условий мультифильтров;

·поля С и По для фильтрации платежей по дате и времени (первоначально настроены отображать данные за текущий месяц);

·список платежей (подробнее см. Просмотр данных о платежах);

·общее количество записей, общая сумма платежей и комиссии, которые отображаются в списке;

·количество выбранных в списке записей и их общая сумма;

·кнопки для выполнения действий с платежами.

Для работы со списком платежей также используется контекстное меню, которое вызывается щелчком правой клавиши мыши в области списка.

 Примечание.

Если техническая константа PaymentsUseOperatorScope включена, то в модуле Платежи в списке отображаются только те платежи, к которым у текущего пользователя настроен доступ в модуле Настройка списка должников.

Если техническая константа PaymentsUseOperatorScope выключена, ограничения не накладываются.
