Словари

Обновлено до 50 релиза 03.11.2020.

Модуль Словари предназначен для ведения словарей системы. Словарь представляет собой объект системы, содержащий множество текстовых значений (терминов), которое применяется в качестве множества значений атрибута некоторого объекта. Такой атрибут называется словарным, и он не может принимать значения, не описанные в соответствующем словаре.

Словари делятся на две группы:

·системные — словари данной группы определяют работу Системы;

·общие (пользовательские) — словари данной группы настраиваются сопровождением при внедрении, они могут отличаться для разных компаний, использующих Систему, в соответствии с их бизнес-процессами.

Внимание! Важно! Пользователям запрещено удалять, добавлять или редактировать словари!

Для настройки схем коммуникации используется Вкладка "Схемы коммуникации".

При работе со словарями осуществляется автоматический контроль прав пользователей. Все действия со словарями регистрируются в логе действий операторов.

Для того чтобы открыть модуль, выберите в главном меню пункт АдминистрированиеаСловари.
