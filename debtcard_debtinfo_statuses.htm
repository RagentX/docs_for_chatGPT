Описание системных статусов долга

1.Не распределен — данный статус долг приобретает после регистрации в системе, снятии оператора или переводе в другое подразделение. Статус означает, что ни один оператор в данный момент не работает с долгом.

2.Не обработан — статус устанавливается после распределения долга одному из операторов. Наличие этого статуса означает отсутствие контактов по этому долгу, в частности, отсутствие платежей и обещаний.

3.Отказ — статус означает, что оператор зарегистрировал контакт с должником, в результате которого должник сообщил о своем отказе производить выплаты. Выход из этого статуса возможен только в случае получения платежа или регистрации обещания должника после отказа. Также возможен перевод долга в статус "Проблемный".

4.Контакт установлен — статус устанавливается после того, как оператор зарегистрировал контакт с должником с успешным результатом. Кроме того, долг переходит в статус 4 "Контакт установлен" из статуса "Обещание", если истекла обещанная дата и количество дней после нее, заданное константой PromGracePeriod.

5.Обещание — статус устанавливается, если в результате контакта с должником по долгу зафиксировано обещание оплаты.

6.Завершено — статус означает, что должник погасил задолженность перед кредитором. Статус устанавливается вручную (см. Завершение работы с долгом, выполнить эту операцию можно только при наличии специального доступа) или автоматически при загрузке платежей, если остаток к погашению стал равен 0. Выход из этого статуса возможен только при возврате долга в работу.

7.Аннулировано — статус устанавливается вручную (см. Аннулирование долга, выполнить эту операцию можно только при наличии специального доступа) в случае необходимости удаления долгов при ошибках ввода. Выход из этого статуса возможен только при возврате долга в работу.

8.Отозван Клиентом — статус устанавливается для долгов, которые были возвращены клиенту по причине завершения работы с реестром, невозможности взыскания и т. п. Выход из этого статуса возможен только при возврате долга в работу.

9.Проблемный — статус означает, что оператор не имеет возможности связаться с должником (неверные контакты) или ему не удается достичь результатов ввиду сложности общения с должником (должник нарушает обещания, передает фальшивые платежки и т. п.). Статус устанавливается непосредственно оператором (см. Перевод долга в проблемные), после чего долг поступает на рассмотрение супервизору, который принимает решение о дальнейшей работе с должником. Кроме того, статус устанавливается автоматически с причиной "Отсутствуют или неверны контактные телефоны", если у должника отсутствуют телефоны в статусе, отличном от "Неверный номер".

10.К возврату Клиенту — статус является промежуточным перед статусом "Отозван Клиентом". Все долги, которые необходимо вернуть в банк, сначала устанавливаются в "К возврату Клиенту", а после подписания всех документов переводятся в статус "Отозван Клиентом".

11.Возврат в работу — статус устанавливается вручную, если необходимо возобновить работу с долгом, переведенным ранее в один из конечных статусов (см. Возврат долга в работу).

12.Без перспектив — статус устанавливается для долгов, отозвать которые в настоящий момент нет возможности, но обрабатывать их не имеет смысла.

13.Контакт не установлен — статус устанавливается после того, как оператор зарегистрирует контакт с должником с неуспешным результатом. При регистрации последующего результативного контакта долг перейдет в статус "Контакт установлен".
Кроме того долг переходит в этот статус: а) автоматически из статуса "Контакт установлен", если по всем телефонам должника, имеющим статус "Результат" или "Последний результат", нет ни одного исходящего или входящего звонка с успешным результатом более чем количество дней, заданное константой ContactSuccessPeriod; б) автоматически из статуса "Проблемный" с причиной "Отсутствуют или неверны контактные телефоны", если у должника имеются телефоны, в статусе отличном от "Неверный номер".

14.Аутсорсинг — долг передан на аутсорсинг.

15.Поиск информации — выполняется поиск данных по долгу.

16.Возврат из аутсорсинга — долг возвращен из аутсорсинга.

17.Цессия — долг передан на цессию.
