Форма "Постановка задач"

Форма Постановка задач содержит следующие элементы:

·вкладки, на которых отображаются списки должников и их долгов, отобранных по различным признакам (см. Просмотр данных о должниках);

·поля мультифильтров и кнопка Найти для применения условий мультифильтров;

·кнопки для выполнения различных действий с долгами;

·индикатор, показывающий общее количество записей, которые отображаются в списке;

·поле, в котором отображается информация о датах, способах и результатах контактов с должником, информация о переводе долга на аутсорсинг и возврата из аутсорсинга (отображается, если константа WorkTaskContactHistory имеет значение "Истина").

Примечание. В списке отображаются только те долги, к которым у текущего пользователя настроен доступ в модуле Настройка списка должников.

Отдельные области формы можно скрыть/отобразить с помощью специальных элементов управления.

Записи о долгах могут быть окрашены, цвет записи о долге задается в модуле Карточка должника.

Для работы со списком должников также используется контекстное меню, которое вызывается щелчком правой клавиши мыши в области списка.

Примечание. Пункт контекстного меню Установить значение активен только при наличии в роли пользователя доступа WORK_TASK_GROUP_UPDATE.

Примечание. Если одновременно в системе включена константа WorkTaskAllShowClose и у пользователя в роли включен доступ WORK_TASK_ALL_SHOW_CLOSE, то на вкладке Все должники заблокированы следующие кнопки для работы со списком:

·Назначить оператора;

·Перевод в проблемные;

·Вернуть Клиенту;
