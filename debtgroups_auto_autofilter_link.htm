Связывание групп с автофильтрами

Show/Hide Hidden Text

Для работы функционала автогрупп, с каждой автогруппой рекомендуется связывать не менее одного автофильтра. Имеется возможность создавать новые автофильтры уже связанными с группой (см. Создание связанного автофильтра) или связывать с группой один из уже созданных автофильтров (см. Добавление связи).

Связывание автофильтров с группой или удаление такой связи доступно пользователям, в роли которых доступ № 539 GROUP_DEBT_SQL_EDIT имеет значение TRUE.

Форма добавления или редактирования связи автофильтров

       Типы связей автофильтров

Типы связей автофильтров задаются в словаре № 136 Тип связи автофильтров. Словарь № 136 является системным и заполняется вендором при установке системы.

Термины словаря № 136 обрабатываются в зависимости от их кода, а не названия:

1."Добавление" — данные, удовлетворяющие условиям автофильтра, добавляются к данным, полученным с использованием предыдущих автофильтров;

2."Исключение по условию" — данные, удовлетворяющие условиям автофильтра, исключаются из  данных, полученных с использованием предыдущих автофильтров;

3."Выбор из найденного" — данные, удовлетворяющие условиям автофильтра, выбираются из данных, полученных с использованием предыдущих автофильтров, остальные удаляются.

Если с группой связан один автофильтр, в качестве вида связи всегда используется только термин № 1.

Если с группой связано более одного автофильтра, то для заполнения группы они будут применяться по очереди в соответствии с номерами, указанными в поле Порядок выполнения. Первый фильтр связывается термином № 1, а остальные могут иметь любую связь.

       Создание связанного автофильтра

Для создания связанного с группой автофильтра:

1.Войдите в форму Группы долгов и активируйте вкладку Автогруппы.

2.Выберите группу в списке автогрупп.

3.В нижней части окна активируйте вкладку Связи автофильтров.

4.Нажмите кнопку , относящуюся к связям автофильтров.

5.Далее действуйте аналогично созданию автофильтра.

6.Автофильтр будет одновременно создан и связан с выбранной группой. Он появится в списке связей указанной группы во вкладке Связи автофильтров. Параметры связи будут заполнены значениями по умолчанию. Для изменения параметров связи воспользуйтесь редактированием (Редактирование связи).

7.Созданный автофильтр появится также в списке автофильтров во вкладке Все автофильтры. Его можно будет использовать для связи с другими автогруппами.

       Добавление связи с автофильтра с группой

Имеется возможность связать с группой ранее созданный автофильтр.

Для этого:

1.Войдите в форму Группы долгов и активируйте вкладку Автогруппы.

2.Выберите группу в списке автогрупп.

3.В нижней части окна активируйте вкладку Все автофильтры. Выберите нужный автофильтр в списке автофильтров.

4.Нажмите кнопку , относящуюся к списку автогрупп. Появится окно для добавления связи. В верхней части окна указаны номер и наименование выбранной группы и выбранного автофильтра.

5.Задайте:

·Порядок выполнения — номер в последовательности запуска автофильтров (по умолчанию — 1);

·Тип связи — см. Типы связей автофильтров (по умолчанию — термин № 1).

·Флаг Связь активна — если его выключить, то связь не будет обрабатываться (по умолчанию включен).

6.Нажмите OK. Связь добавится в список связей автофильтров для указанной группы.

       Быстрое добавление связи автофильтра с группой

Для быстрой привязки автофильтра к группе:

1.На вкладке Все автофильтры выберите автофильтр.

2.Зажмите левую кнопку мыши и перетащите выбранный фильтр к нужной автогруппе, затем отпустите кнопку мыши.

Примечание. Для поиска нужной строки в списке групп подведите курсор (не отпуская левую кнопку мыши) к верхней или нижней границе списка групп, тогда список начнет автоматически прокручиваться в выбранном направлении.

3.Фильтр будет добавлен выбранной группе. При этом атрибуты связи будут заполнены автоматически следующими значениями:

·Порядок выполнения — если у группы еще не было связанных фильтров, то установится № 1, иначе — число, следующее за максимальным номером;

·Тип связи — термин № 1;

·Флаг Связь активна — по умолчанию включен.

       Редактирование связи

Для редактирования связи:

1.Войдите в форму Группы долгов и активируйте вкладку Автогруппы.

2.Выберите группу в списке автогрупп.

3.В нижней части окна активируйте вкладку Связи автофильтров. Выберите нужную строку в списке связей автофильтров.

4.Нажмите кнопку , относящуюся к связям автофильтров.

5.Далее действуйте, как описано в разделе Добавление связи.

       Удаление связи

Для удаления связи:

1.Войдите в форму Группы долгов и активируйте вкладку Автогруппы.

2.Выберите группу в списке автогрупп.

3.В нижней части окна активируйте вкладку Связи автофильтров. Выберите нужную строку в списке связей автофильтров.

4.Нажмите кнопку , относящуюся к связям автофильтров.

5.Подтвердите удаление.
