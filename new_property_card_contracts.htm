Договоры

Договоры по объектам имущества могут относиться к доходам либо расходам. Типы договоров перечислены в словаре № 376 "Тип договора по объекту имущества". Одни из типов договоров относятся доходам, другие  — к расходам.

Примечание. Чтобы определить, какие типы договоров относятся к расходам, а какие — к доходам, обратитесь в техническую поддержку.

Договоры, относящиеся к расходам, доступны на вкладке "Процессы", подчиненная вкладка "Сопровождение" по кнопке "Договоры (расходы)".

Договоры, относящиеся к доходам, доступны на вкладке "Процессы", подчиненная вкладка "Монетизация" по кнопке "Договоры (доходы)".

Все действия с договорами аналогичны, вне зависимости от того, относятся ли они к доходам либо расходам.

В карточке имущества доступны единичные действия с договорами. Массовое добавление договора (один договор добавляется к одному или более объектам имущества) доступно в моделу "Группы свободного поиска".

Описание формы "Договоры (расходы) / Договоры (доходы)"

Формы "Договоры (расходы)" и "Договоры (доходы)" аналогичны.

Форма "Договоры (расходы)" / Договоры (доходы)" включает следующие элементы:

·Область данных о договорах:

·Список договоров (расходов или доходов соответственно) с полями:

·ID договора.

·Название договора.

·Тип договора — см. словарь № 376 "Тип договора по объекту имущества". Для договоров по расходам (форма "Договоры (расходы)") доступны только типы договоров, относящиеся к расходам, а для договоров по доходам (форма "Договоры (расходы)") — только типы договоров, относящиеся к расходам.

·Кнопка "Добавить" — Добавление договора.

Примечание. Кнопка "Добавить" активна только в случае, если при открытии формы "Договоры (расходы)" / Договоры (доходы)" по кнопке "Договоры (расходы)" / Договоры (доходы)" в списке задач была выделена задача. Если в списке задач нет задач, то кнопка "Добавить" не активна.

·Кнопка "Изменить" — Редактирование договора.

·Кнопка "Удалить" — Удаление договора.

·Область данных "Атрибуты" об атрибутах выбранного договора:

·Список атрибутов выбранного договора с полями:

·ID.

·Название атрибута — см. словарь № 377 "Атрибут договора по объекту имущества".

·Значение атрибута.

·Кнопка "Добавить" — Добавление атрибута договора.

·Кнопка "Изменить" — Редактирование атрибута договора.

·Кнопка "Удалить" — Удаление атрибута договора.

·Область данных "Контракты" о контрактах выбранного договора:

·Список контрактов выбранного договора с полями:

·ID.

·Наименование контракта.

·Сумма контракта.

·Дата контракта.

·Кнопка "Добавить" — Добавление контракта договора.

·Кнопка "Изменить" — Редактирование контракта договора.

·Кнопка "Удалить" — Удаление контракта договора.

·Кнопка "ОК" для выхода из формы.

Область данных о договорах является основной. Данные в областях "Атрибуты" и "Контракты" относятся к выбранному договору.

При двойном нажатии на строку в области данных о договорах открывается документ, который связан с договором. Если связанный с договором документ отсутствует, то отображается сообщение с текстом:

Не найдено связанных с договором документов
