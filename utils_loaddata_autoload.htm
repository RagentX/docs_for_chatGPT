Проверка и загрузка данных в автоматическом режиме

Проверка и загрузка данных в автоматическом режиме производится ежедневно по расписанию. Для этого необходимы соответствующие настройки сопровождения LuxBase в Планировщике заданий.

Загрузка данных в автоматическом режиме обеспечивает:

·загрузку из файлов должников;

·загрузку из файла платежей.

Для того, чтобы  автоматическая загрузка могла осуществиться, необходимо дополнительно выполнение следующих условий:

·в Системе должен быть зарегистрирован пользователь с ID пользователя, равным 1;

·пользователь с ID, равным 1, должен иметь в своей роли включенный доступ № 6 LOAD_DATA_VIEW.

Данные в файлах должников, платежей, графиков платежей, поручителей и залогов хранятся в формате CSV, в качестве разделителя значений атрибутов используется символ "точка с запятой". Описание форматов приведено в соответствующих разделах с описанием загрузки данных.

Информация о директориях и именах входных файлов должников, платежей, графиков платежей, поручителей и залогодателей задается в файле CSV.INI, который находится в текущей директории модуля загрузки данных. Входные файлы формируются заранее другими комплексами программ (например, в системе по обслуживанию клиентов банка). Имена файлов должников, поручителей и залогодателей задаются с помощью маски, позволяющей выбирать два или более файлов нужного типа из заданной директории.

В процессе проверки и загрузки данных формируется файл статистических данных log_csv <дата выполнения>.log. Информация о директории файла статистических данных задается в файле CSV.INI. Первая строка файла содержит сведения о версии модуля загрузки данных, пользователе и запуске конвертера системы "Долг-Контроль". В последней строке фиксируется завершение работы конвертера. Содержание файла статистических данных зависит от загружаемых файлов и приведено в соответствующих разделах с описанием загрузки данных.

Если в форме Загрузка данных доступна кнопка Загрузка из CSV, то можно вручную запустить проверку и загрузку данных. Для этого нажмите кнопку Загрузка из CSV. В процессе работы последовательно будут выводиться сообщения:

·"Информация о должниках. Успешно загружено записей <число записей>. Не загружено записей <число записей>."

·"Информация о платежах. Успешно загружено записей <число записей>. Не загружено записей <число записей>."

Пример файла CSV.INI

[main]

in1=C:\CSV\import_debt*.csv

in2=C:\CSV\import_calc*.csv

outlog=C:\CSV\log

Примечание. Кнопка Загрузка из CSV доступна только тогда, когда в папке с модулями Системы создан корректный файл "CSV.ini". Если файл отсутствует, кнопка не доступна. Загрузка данных по кнопке Загрузка из CSV может быть осуществлена только тогда, когда по адресам, указанным в файле "CSV.ini", расположены соответствующие входные файлы.
