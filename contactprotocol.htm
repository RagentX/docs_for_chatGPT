Протокол контактов

Обновлено до 50 релиза 03.11.2020.

Модуль Протокол контактов предназначен для просмотра всех контактов операторов с должниками, к которым пользователь имеет доступ. Для того чтобы открыть протокол контактов, выберите в главном меню пункт Рабочие места а Протокол контактов.

Право на вход в модуль имеют только те пользователи, в роли которых доступ CONTACT_LOG_VIEW установлен в значение TRUE.
Право на выполнение любых действий в модуле имеют те пользователи, в роли которых доступ CONTACT_LOG_FULL установлен в значение TRUE.
