Формирование списка документов портфеля

К записи о портфеле можно прикрепить файлы документов. Для этого выполните следующие действия:

1. В форме Клиенты и портфели выберите нужную запись в списке портфелей и нажмите кнопку Документы. Откроется форма редактирования списка документов.

2. В форме выполните необходимые действия по формированию списка файлов документов:

·присоединение файла документа. Для этого нажмите кнопку , выберите файл в диалоговом окне Windows и нажмите кнопку Open (Открыть);

·просмотр файла документа. Для этого выберите документ в списке и нажмите кнопку . Файл документа будет открыт в соответствующем ему приложении;

·сохранение файла на диск. Для этого выберите документ в списке, нажмите кнопку , в диалоговом окне Windows укажите путь и название сохраняемого файла и нажмите кнопку Save (Сохранить). Файл документа будет сохранен в указанном месте;

·удаление файла из списка. Для этого выберите документ в списке, нажмите кнопку и подтвердите удаление в диалоговом окне.

3. Нажмите кнопку ОК, чтобы сохранить изменения в списке файлов документов.

Список документов, добавленных портфелю, будет отображаться для долгов, принадлежащих данному портфелю, в следующих модулях:

·Карточка должника;

·Карточка судебной работы;

·Карточка исполнительного производства.
