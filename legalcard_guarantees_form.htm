Форма "Поручитель"

Форма Поручитель включает следующие элементы:

·Поля:

·ФИО поручителя — фамилия, имя, отчество поручителя;

·Тип — тип поручителя (физическое лицо/юридическое лицо/ИП — см. словарь № 54);

·Роль — роль поручителя (см. словарь № 155);

·ID Клиента — идентификатор клиента;

·Область данных Общая информация — раздел, в котором вводится общая информация о поручителе. Состав атрибутов различается в зависимости от типа поручителя:

·Физическое лицо — для физических лиц с полями:

oДата рождения;

oПол;

oСемейное положение;

o№ уд. личн;

oEmail;

oГражданство;

oКомпания;

oДолжность;

oОбразование;

oРНН;

oИНН;

oМесто рождения;

oПаспортные данные;

Примечание.

·Если константа DebtCard.GuarantorCard.PassportInfo.Mode = 1, то паспортные данные поручителя отображаются в карточке поручителя единым блоком:

·Если константа DebtCard.GuarantorCard.PassportInfo.Mode = 2, то паспортные данные поручителя отображаются в карточке поручителя так же, как в карточке должника:

·Юридическое лицо — для юридических лиц. Состав атрибутов различается в зависимости от значения константы GuarantorPersonPropertyPayInfoMode:

oзначение константы равно 1:

§ИНН;

§РНН;

§КПП;

§БИН;

§ФИО ген. директора;

§Платежные реквизиты;

oзначение константы равно 2:

§ИНН;

§РНН;

§КПП;

§ОГРН;

§ОКВЭД;

§ОКПО;

§Рас.счет;

§Кор.счет;

§Банк;

§БИК;

§БИН;

§ФИО ген. директора;

oзначение константы равно 3:

§ИНН;

§РНН;

§КПП;

§ОГРН;

§ОКВЭД;

§ОКПО;

§Рас.счет;

§Кор.счет;

§Банк;

§БИК;

§БИН;

§ФИО ген. директора;

§Платежные реквизиты;

Примечание. Если константа GuarantorPersonPropertyPayInfoMode имеет значение, отличное от 1, 2 или 3, то считается, что ее значение считается равным 1.

·Юридическое лицо — для ИП с полями:

oИНН;

oФИО;

oПлатежные реквизиты;

·Область данных Финансовая информация — раздел, в котором вводится финансовая информация о поручителе:

Примечание. Если константа DebtCard.GuarantorCard.FinancialInfo.Use имеет значение "Ложь", то блок финансовой информации о поручителе заблокирован.

·№ договора обеспечения;

·Дата заключения — дата заключения договора;

·Дата окончания — дата окончания действия договора;

·Дата завершения;

·Залоговая стоимость — залоговая стоимость имущества. Поле обязательно для заполнения;

·Рыночная стоимость;

·Справедливая стоимость — реальная стоимость имущества;

·Валюта — валюта, в которой выражена стоимость имущества;

·Коэфф. кач-ва — коэффициент качества;

·Доход — доход поручителя;

·Является плательщиком;

·Кнопка — Судебная работа по поручителю, кнопка доступна только на форме редактирования (не создания) поручителя при наличии у пользователя соответствующих доступов (подробнее см. описание доступности судебной работы по поручителю).

·Телефоны — телефоны поручителя. Для работы со списком телефонов поручителя используется те же функции, что и для списка телефонов должника (см. Контактные телефоны поручителя и работа с ними);

·Адреса — адреса поручителя. Для работы со списком адресов поручителя используется те же функции, что и для списка адресов должника (см. Адреса поручителя и работа с ними);

·Email — адреса электронной почты поручителя. Для работы со списком адресов электронной почты поручителя используется те же функции, что и для списка адресов электронной почты должника (см. Адреса электронной почты поручителя и работа с ними);

·Комментарий — дополнительная информация.
