Добавление и удаление таблицы или представления из структуры

Show/Hide Hidden Text

Внимание! Изменение данных в структуре может повлечь за собой сбои в работе запросов! Изменения разрешено осуществлять только администратору Системы.

       Добавление таблицы или представления

Для того, чтобы создать шаблон структуры на основе представления, выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Таблицы.

2.Нажмите кнопку Добавить таблицу.

3.Откроется форма Таблицы/представления. Заполните следующие поля:

·База данных — выберите базу данных;

·Таблицы/представления — выберите предварительно созданное представление;

·Флаг Автоматически добавить поля — включите флаг, чтобы в структуру были автоматически добавлены все поля выбранного представления.

4.Нажмите OK.

Форма добавления таблицы/представления

5.Таблица будет добавлена на форму Конструктор структур на вкладку Таблицы.

6.Для сохранения изменений нажмите кнопку Сохранить изменения. Для того чтобы отменить последние действия нажмите кнопку Отменить изменения.

       Удаление таблицы или представления

Для того чтобы удалить таблицу или представление, выполните следующие действия:

1.В форме Конструктор структур на вкладке Таблицы выберите таблицу.

2.Нажмите кнопку Удалить таблицу.

3.Таблица будет удалена.
