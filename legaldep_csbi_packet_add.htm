Добавление пакета CSBI

Для того чтобы добавить пакет CSBI для определения подсудности / подведомственности, выполните следующее:

1.В модуле Юридический департамент активируйте вкладку Пакеты CSBI.

2.Выберите вложенную вкладку Определение подсудности или Определение подведомственности соответственно.

3.Нажмите кнопку Добавить пакет.

4.Откроется форма Добавление пакета.

5.Укажите название пакета. По умолчанию в поле указано значение константы CSBI.PackageName.Default. Поле обязательно для заполнения.

6.Настройте приоритет.

Блок Приоритет включает следующие пункты:

·Использовать адрес подачи;

·Выбирать адрес по типу, подпунктами которого являются термины словаря № 5 "Тип адреса".

Включите нужные флаги и настройте порядок. Выбор хотя бы одного флага является обязательным.

Вы можете менять местами пункты, используя перетаскивание мышью. Вы также можете менять положения типов адреса. Таким образом настраивается порядок приоритета.

7.Выберите статусы адреса. При необходимости измените порядок статусов, используя перетаскивание мышью. Выбор хотя бы одного статуса является обязательным.

8.Нажмите ОК.

Будет создан пакет CSBI.

·Полное название пакета формируется следующим образом:

<Название пакета>_ <Дата>;

где:

·Название пакета — значение из поля Название пакета;

·Дата — текущая дата в формате DDMMYYYY.

·Тип пакета: "Определение подсудности" или "Определение подведомственности" соответственно.

·Состояние пакета — "Формирование".

·Настройки выбора адресов сохраняются в xml формате.
