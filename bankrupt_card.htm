Карточка судебной работы в банкротстве

Обновлено до 49 релиза 08.09.2020.

Карточка судебной работы в банкротстве (далее — карточка банкротства) открывается для судебных дел типа "Банкротство". Банкротство является отдельным типом судебной работы и создается по должнику, а не по долгу.

Карточка банкротства открывается из модуля Юридический департамент, вкладка Банкротство (см. Переход к карточке судебной работы/исполнительного производства).

Создание банкротства может быть осуществлено из модулей:

·Список должников — при выборе соответствующего пункта контекстного меню (вызывается правой кнопкой мыши) — см. Судебная работа;

·ГСП — при выборе соответствующего пункта контекстного меню — см. Судебная работа;

·Расписание — мероприятие Создание банкротства;

·Карточка должника — см. Создание банкротства.

Право для входа в модуль имеют только те пользователи, в роли которых доступ LAW_ACT_CARD_VIEW установлен в значение TRUE, а значение константы LawActUseBankrupt = TRUE.

Для карточки судебной работы в банкротстве действуют те же доступы, что и для карточки судебной работы в приказном производстве.
