Конструктор фильтров и работа с ним

Конструктор фильтров открывается при нажатии кнопки Условие в строке краткого описания фильтра в таблице с данными. В нем можно добавлять или изменять условия (фильтры) или группы условий, а также сохранять и открывать созданные фильтры. Группа условий представляет собой набор условий, объединенных логическими операторами И, ИЛИ, НЕ И, НЕ ИЛИ.

 

Формирование условий фильтра в конструкторе фильтров выполняется следующим образом:

1.Для добавления условия или группы условий нажмите кнопку (или кнопку Фильтр для добавления в корень фильтра) и выберите в меню соответственно пункт Добавить условие или Добавить группу. Для удаления выбранной записи выберите пункт Удалить запись. Для добавления условия можно также нажать кнопку Нажмите кнопку для добавления условия.

2.Группа условий по умолчанию объединена оператором И. Чтобы изменить оператор, щелкните по нему левой клавишей мыши и выберите новый оператор в списке.

3.Для изменения условия:

a.В строке условия щелкните левой клавишей мыши по названию поля таблицы (выделено зеленым цветом) и выберите нужное поле в списке.

b.Щелкните левой клавишей мыши по названию оператора сравнения (выделено коричневым цветом) и выберите нужный оператор в списке.

c.Щелкните левой клавишей мыши по значению (выделено синим цветом) и введите значение вручную или выберите в списке (способ ввода зависит от поля таблицы). Для облегчения поиска в списке предусмотрена возможность сортировки щелчком левой клавиши мыши по заголовку списка.

4.Повторите шаги 1-3, чтобы добавить все условия в фильтр. Например, на рисунке выше фильтр позволяет отобрать в модуле "Лог действий оператора" записи о действиях операторов Оглоблина Е. П. и Иванова Д. С., которые они выполняли в модулях "Карточка должника" и "Входящий звонок".

5.Чтобы применить условия фильтра:

·нажмите кнопку Применить, чтобы применить условия фильтра, не закрывая окно конструктора;

·нажмите кнопку ОК, чтобы применить условия фильтра и закрыть окно конструктора.

Для сохранения расширенного фильтра в форме конструктора фильтров нажмите кнопку Сохранить, укажите путь в диалоговом окне Windows и нажмите кнопку Save (Сохранить).

Для того чтобы открыть сохраненный ранее фильтр, в форме конструктора фильтров нажмите кнопку Открыть, в диалоговом окне Windows выберите файл фильтра и нажмите кнопку Open (Открыть).
