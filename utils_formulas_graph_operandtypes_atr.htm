Атрибут системного объекта

Список возможных атрибутов перечислен в системном словаре № 272 "Атрибут системного объекта в формулах" на языке пользователя.

При выбора в карточке операнда типа "Атрибут системного объекта" необходимо указать для него имя атрибута.  Раскрывающийся перечень атрибутов содержит подмножество имен атрибутов системного объекта с учетом нужного типа данных. Поле имени обязательно для заполнения.

Примечание. Перечень имен атрибутов может уточняться и пополняться с использованием средств БД без изменений в программном модуле Произвольная формула.

Выбранное пользователем имя атрибута отображается в текущем узле дерева формулы.
