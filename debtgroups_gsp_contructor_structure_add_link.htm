Связи между полями таблиц или представлений

Show/Hide Hidden Text

Внимание! Изменение данных в структуре может повлечь за собой сбои в работе запросов! Изменения разрешено осуществлять только администратору Системы.

       Создание связи

Для того, чтобы создать связь выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Таблицы.

2.Прежде чем добавлять связь между полями таблиц, убедитесь в том, что на форме присутствует не менее двух таблиц.

3.Нажмите кнопку Добавить связь. Выберите поле в одной из таблиц и щелкните левой клавишей мыши. Затем переведите курсор на другую таблицу (связь будет отображаться на экране в виде линии)) и выберите поле и щелкните левой клавишей мыши на нем.

4.Откроется окно Редактор связей. Связь будет задана в области данных Объединяемые поля как равенство между выбранными значениями.

5.В поле Тип объединения выберите тип связи:

·Inner — один к одному;

·Outer Left — многие к одному;

·Outer Right — один ко многим;

·Outer Full — многие ко многим.

6.Нажмите OK.

Форма редактирования связи

       Редактирование связи

Для того, чтобы создать связь выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Таблицы.

2.Нажмите кнопку Редактировать связь. Откроется окно Редактор связей.

3.Измените данные и нажмите OK.

       Удаление связи

Для того, чтобы создать связь выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Таблицы.

2.Нажмите кнопку Удалить связь. Связи удаляются начиная с той, которая была добавлена последней.

3.В окне подтверждения будут указаны поля, связь между которыми планируется удалить. Нажмите Да для удаления.
