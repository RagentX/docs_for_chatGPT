Внимание! Изменение данных в структуре может повлечь за собой сбои в работе запросов! Изменения разрешено осуществлять только администратору Системы.

Если поля были добавлены автоматически

Если при добавлении таблицы флаг Автоматически добавить поля был включен, то все поля таблицы будут добавлены в список полей. Каждая таблица будет представлять собой группу полей.

Поля были добавлены автоматически

Если поля не были добавлены

Если при добавлении таблицы флаг Автоматически добавить поля был выключен, то полей не будет добавлено. Список полей будет пуст.

       Добавить группу полей

Для того чтобы добавить группу полей, выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Поля.

2.Нажмите кнопку Добавить поле.

3.К существующим группам полей будет добавлена новая группа.

4.В правой области расположено единственное свойство группы полей — Имя. По умолчанию группа называется "Новая группа". Введите имя группы полей.

Добавлена группа полей

       Удалить группу полей

Для того чтобы удалить группу полей, выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Поля.

2.Выберите группу полей.

3.Нажмите кнопку Удалить поле.

4.Подтвердите удаление.

       Добавить поле

Для того чтобы добавить поле из таблицы, выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Поля.

2.Выберите группу полей.

3.Нажмите кнопку Добавить атрибут данных.

4.Откроется форма "Поля". Выберите нужные поля и нажмите OK.

5.Поля будут добавлены в выбранную группу полей в качестве дочерних элементов.

       Удалить поле

Для того чтобы удалить поле, выполните следующие действия:

1.В форме Конструктор структур активируйте вкладку Поля.

2.Выберите группу полей и разверните ее , нажав на .

3.В группе полей выберите поле.

4.Нажмите кнопку Удалить атрибут.

5.Подтвердите удаление.
