Версионность произвольных атрибутов долга

Show/Hide Hidden Text

Все нижеизложенное верно, если ваша конфигурация Системы поддерживает функционал версионности произвольных атрибутов долга. В технической константе RandomValue.Version.CodeList перечислены атрибуты, для которых поддерживается версионность. Если значение константы равно нулю, то версионность произвольных атрибутов не поддерживается.

В Системе поддерживается версионность некоторых атрибутов долга. Это дает возможность хранить в системе величины, значения которых меняется с течением времени. Сохраняются различные версии значения атрибута, привязанные к различным временным отрезкам, как в прошлом, так и в будущем. Таким образом, для каждого момента времени можно найти соответствующую версию значения атрибута.

Форма "История версий"

При нажатии на кнопку История версий, расположенную на вкладке Произвольные атрибуты модуля Карточка должника, открывается форма История версий, которая содержит записи обо всех версиях атрибута.

Вид формы "История версий"

       Описание формы

Форма История версий содержит следующие элементы:

·Список версий атрибута. Таблица включает следующие поля:

·Атрибут;

·Значение;

·Дата начала жизни версии;

·Дата окончания жизни версии.

Примечание. Дата окончания жизни версии соответствует дате начала жизни следующей версии. Дата окончания жизни последней версии равна 01.01.9999.

·Кнопка Добавить версию;

·Кнопка Изменить версию;

·Кнопка Закрыть — для выхода из формы.

       Добавление версии

Для того чтобы добавить новую версию, выполните следующие действия:

1.На форме История версий нажмите кнопку Добавить версию.

2.В открывшейся форме заполните следующие поля:

·Дата начала жизни версии;

·Значение.

3.Нажмите Сохранить.

Введенная версия будет добавлена между двумя соседними версиями, чьи даты начала жизни меньше и больше введенной. Если более ранних версий нет, то введенная версия будет добавлена первой. Если более старшей версии нет, то введенная версия будет последней, и дата окончания ее жизни будет 01.01.9999.

Если введенная дата начала жизни совпадает с началом жизни уже существующей версии, будет показано предупреждение. Если вы нажмете Yes, значение версии будет обновлено.

       Редактирование версии

Для того чтобы изменить значение версии, выполните следующие действия:

1.В форме История версий нажмите кнопку Изменить версию.

2.В открывшейся форме введите новое значение версии.

3.Нажмите Сохранить.

Управление версиями

Добавление новой версии атрибута может осуществляться двумя способами:

·Редактирование значения атрибута на форме "Произвольные атрибуты".

·Добавление версии на форме "История версий".
