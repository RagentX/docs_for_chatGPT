Экранная форма "Должники"

Описание формы "Должники"

Экранная форма "Должники" (Рисунок 6‑1) содержит следующие элементы:

·Кнопка — переход на Боковое меню;

·Список должников;

·Кнопка — Обновление;

·Кнопка — Передать контакты (синхронизация);

·Кнопка — Инструменты;

По умолчанию список должников отсортирован по полю ФИО по возрастанию.

Каждая запись списка должников содержит ФИО должника. При нажатии на строку списка ("tap") запись разворачивается, и открывается детализация записи, включающая следующую информацию:

·ФИО должника;

·Сумма долга с указанием валюты долга;

·Статус долга;

·Банк;

·Скоринговые баллы.

По умолчанию все записи списка свернуты.

В нижней части формы отображается количество записей.

Окраска записей о должниках в зависимости от статуса долга

Есть возможность установить различную цветовую окрасу записей о должниках в зависимости от статуса долга. Для этого настройте константу Android.DebtStatus.Color. ФИО должника и развернутая детализация по долгу будут окрашены в соответствии с назначенным цветом для данного статуса долга.

Примечание. Если должник имеет больше одного долга, то детализация каждого долга будет окрашена в свой цвет в соответствии со значением статуса долга, а ФИО должника будут окрашены по цвету первого долга в списке (с наименьшим ID).

Инструменты формы "Должники"

Для доступа к инструментам нажмите кнопку , расположенную в правом нижнем углу формы. Кнопка примет вид , а над ней появится список инструментов (Рисунок 4‑2).

На форме доступны следующие инструменты:

Местоположение

При прикосновении ("tap") к кнопке "Местоположение" открывается карта, на которой отображены адреса, соответствующие всем записям на форме с учетом результатов примененных фильтрации и поиска (Рисунок 4‑3).

Фильтр

Фильтр позволяет вывести записи, отвечающие иным критериям, чем заданные на форме по умолчанию.

См. описание настраиваемых фильтров (Фильтр формы "Должники").

Поиск

Поиск — см. описание поиска на форме "Выезды".

Сортировка

Форма сортировки выездов (Рисунок 6‑2) содержит следующие элементы:

·Кнопки:

·Для отмены фильтрации — ;

·Для подтверждения выбора — ;

·Критерии сортировки (после текста "Сортировать долги по:"):

·ФИО;

·Порядок сортировки:

·По возрастанию;

·По убыванию.

После задания критериев сортировки и порядка сортировки нажмите . На экране отобразятся записи, в порядке, соответствующем выбранным критериям. При этом рядом слева от кнопки "Инструменты" появится кнопка отмены сортировки: .

Действия на форме "Должники"

Переход к карточке должника

Для перехода к карточке должника с экранной формы "Должники" выберите запись в списке звонков и выполните на ней короткое нажатие ("tap").
