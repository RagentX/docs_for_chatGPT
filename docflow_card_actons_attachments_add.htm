Добавление вложения

Доступность действия определяется доступом № 681 DOCFLOW_CARD_DOC_ADD.

Для того чтобы добавить вложение, выполните следующее:

1.На вкладке Вложения нажмите кнопку Добавить.

2.Откроется форма добавления вложения. На форме заполните поля атрибутов вложения:

·Название — название документа; если поле не заполнено, то при прикреплении вложения в качестве названия будет указано название вложения;

·Тип документа — см. словарь № 100; поле обязательно для заполнения;

·Номер — номер документа;

·Версия — версия документа. По умолчанию документу присваивается версия 1.0.

3.Затем прикрепите вложение. Для этого нажмите кнопку . Откроется диалоговое окно Windows для выбора файла. Выберите файл и нажмите кнопку Открыть (Open).

4.Нажмите ОК.

5.Откроется уведомление о выполнении действия с текстом:

Вложение добавлено.

Нажмите ОК.
