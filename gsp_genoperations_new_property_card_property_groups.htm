Группы имуществ и действия с ними

Действия с группами имуществ доступны на вкладках модуля, содержащих поле ID имущества, только если в Системе включен функционал карточки имущества. Функционал включается технической константой GroupViewer.PropertyCard.Use.

Доступны следующие действия с группами имуществ:

·Добавление имущества в группу;

·Создание новой группы имуществ;

·Удаление имущества из группы имуществ;

·Создание задачи для группы имуществ.

Добавление имущества в группу

На вкладках модуля, на которых присутствует поле ID имущества, действие по добавлению в группу выполняется только для групп, имеющих в качестве объекта имущество. Действие выполняется по пункту контекстного меню (вызывается правой кнопкой мыши) Добавить в группу.

Дополнительно доступно создание новой группы имуществ.

Порядок действий

Для того, чтобы добавить запись об имуществе в группу имуществ, выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, содержащую ID имущества.

2.Сформируйте выборку.

3.Выберите одну или более записей.

4.Активируйте контекстное меню (правой кнопкой мыши) и выберите пункт Добавить в группу. Затем выберите режим выполнения действия: Все или Только выделенные.

5.Откроется форма Выберите группы, в которые добавить объекты. Выберите группу и нажмите ОК.

Создание новой группы имуществ

Создание новой группы имуществ доступно из формы Выберите группы, в которые добавить объекты по кнопке Создать группу объектов.

Порядок действий

Для того, чтобы добавить запись об имуществе в группу имуществ, выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, содержащую ID имущества.

2.Сформируйте выборку.

3.Выберите одну или более записей.

4.Активируйте контекстное меню (правой кнопкой мыши) и выберите пункт Добавить в группу. Затем выберите режим выполнения действия: Все или Только выделенные.

5.Откроется форма Выберите группы, в которые добавить объекты. Нажмите кнопку Создать группу объектов.

6.Откроется форма Группа объектов для создания новой группы имуществ. Заполните следующие поля:

·Название — введите название группы;

·Комментарий — дополнительная информация;

·Флаг очистки группы — см. словарь № 371 "Флаг очистки группы"; используется для очистки группы перед ее формированием — наполнением долгами;

Примечание. При переформировании группы, в зависимости от выбранного термина словаря № 371 "Флаг очистки группы", очистка данных в группе и ее наполнение долгами будут происходить следующим образом:

·0. Не очищать предыдущие данные — при переформировании в группу добавляются долги, удовлетворяющие условиям группы.

·1. Очищать предыдущие данные — группа полностью очищается, а затем в нее добавляются долги, удовлетворяющие условиям группы.

·2. Очищать только данные, не попавшие повторно — долги, которые не момент формирования группы не удовлетворяют условиям, удаляются. А для тех долгов, которые были в группе до ее переформирования и остались после ее переформирования, не изменяется дата попадания в группу.

·Флаг Разрешить добавление объектов в группу вручную — разрешает или запрещает ручное добавление долга в группу из списочных РМ.

7.Нажмите ОК.

Удаление имущества из группы имуществ

Действие по удалению группы доступно на вкладках модуля, на которых присутствует поле ID имущества, добавлено действие по созданию задачи для группы.

Действие по удалению из группы доступно в контекстном меню только в тех случаях, если на вкладке в мультифильтре "Группа имуществ" выбрана группа, и затем выполнен поиск. То есть, на вкладке в этот момент отображаются записи только об имуществе, входящем в выбранную группу. В контекстном меню для этого действия отображается пункт Удалить из группы, расположенный ниже пункта Добавить в группу.

Порядок действий

Для того, чтобы удалить одну или более записей об имуществе из группы, выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, содержащую ID имущества.

2.В мультифильтре Группа имуществ выберите группу, а затем нажмите кнопку Найти.

3.В списке отобразятся записи только об имуществе, включенном в выбранную группу. Выберите одну или более записей.

4.В контекстном меню (вызывается правой кнопкой мыши) выберите пункт Удалить из группы. Затем выберите режим выполнения действия: Все или Только выделенные.

5.Откроется форма подтверждения. Нажмите Yes.

Выбранные записи об имуществе будут удалены из выбранной группы.

Создание задачи для группы имуществ

На вкладках модуля, на которых присутствует поле ID имущества, добавлено действие по созданию задачи для группы. Действие включается кнопкой Создать задачу для группы, расположенной в нижней части рабочей области экрана.

Порядок действий

Для того чтобы создать задачу для группы, выполните следующее:

1.В модуле Группы свободного поиска активируйте вкладку, содержащую ID имущества.

2.Нажмите кнопку Создать задачу для группы.

3.Откроется форма Выберите группу. Для выбора доступны только группы, имеющие в качестве объекта имущество. Выберите группу и нажмите ОК.

4.Далее откроется стандартная форма для добавления задачи. Добавьте задачу (см. описание функционала "Планирование (Задачи)").

5.Задача будет добавлена для каждого объекта имущества, включенного в выбранную группу.
