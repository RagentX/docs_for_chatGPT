Управление узлами дерева контактов

Важно! Не рекомендуется редактировать дерево контактов в рабочее время. Это может вызвать сбой в работе операторов при регистрации контактов.

Создание нового результата контакта

Для того чтобы создать новый результат контакта:

1.Добавьте новый узел к дереву контактов.

2.При необходимости переместите его в нужную позицию.

3.Затем настройте исход контакта для созданного узла.

Создание нового узла дерева контактов

·Если вы создаете первый результат контакта для нового дерева, то выполните следующие действия:

1.В дереве результатов контактов Нажмите кнопку Добавить. Откроется форма добавления результата контакта.

2.В форме заполните необходимые поля.

3.Нажмите кнопку ОК, чтобы добавить результат контакта к дереву результатов.

·Если вам нужно создать подчиненный результат контакта, то в дереве результатов контактов выберите результат, для которого требуется создать подчиненный результат, а затем выполните пункты 1 — 3.

·Если вы хотите создать новый узел на самом верхнем уровне, то сначала создайте подчиненный узел, а затем переместите его в нужную позицию.

Копирование узла (с потомками) дерева контактов (в том числе меду разными типами деревьев)

Возможно копирование узла дерева контактов, как одного, так и вместе со всеми потомками, и добавление в качестве потомка любому узлу любого дерева контактов.

При копировании узлов осуществляется копирование привязанных к ним задач и групп задач (см. Автоматическая постановка задач).

Важно! Все копируемые узлы должны иметь заполненное значение кода! Иначе при копировании узла привязанные к нему задачи не будут скопированы.

Копирование узла

Для того чтобы скопировать узел дерева контактов:

1.В форме Настройка дерева контактов выберите нужное дерево контактов.

2.Выберите нужный узел дерева контактов. Узел может включать подчиненные узлы.

3.В контекстном меню (вызывается правой кнопкой мыши) выберите пункт Копировать. Затем:

·Если вы хотите скопировать только один узел, выберите подпункт Выбранный узел;

·Если вы хотите скопировать узел вместе с подчиненными узлами, выберите подпункт Со всеми потомками.

Вставка скопированного узла

Для того чтобы вставить скопированный узел дерева контактов:

1.В форме Настройка дерева контактов выберите нужное дерево контактов.

2.Выберите узел, подчиненным которому должен стать скопированный узел.

3.В контекстном меню (вызывается правой кнопкой мыши) выберите пункт Вставить.

Редактирование результата контакта

Для того чтобы отредактировать результат контакта

1.В дереве результатов контактов выберите нужный результат.

2.Нажмите кнопку Изменить или выполните клик (двойной щелчок левой клавишей мыши) на выбранном элементе.

3.Откроется форма настройки исхода контакта.

4.Заполните нужные поля и нажмите OK.

Удаление результата контакта

Удалить можно только те результаты контакта, которые не имеют подчиненных результатов. Для того чтобы удалить результат контакта из дерева результатов, выполните следующие действия:

1.В дереве результатов контактов выберите нужный результат.

2.Нажмите кнопку Удалить.

3.В открывшемся диалоговом окне нажмите Yes, чтобы подтвердить удаление результата.

Если выбранный результат имеет подчиненные результаты, будет выдано сообщение о невозможности выполнения операции. Нажмите кнопку ОК, чтобы закрыть его.
