Долги

Список долгов должника отображается в карточке должника на вкладке Долги. Каждый долг в списке находится в определенном статусе. Изменение статуса долга происходит по результатам регистрации контакта с должником и при выполнении определенных действий с долгом (подробнее см. Статусы долга).

Поля списка долгов

·ID долга — идентификатор долга;

·Название продукта — указывается название продукта, по которому возник долг;

·Портфель — портфель, в который входит долг;

·Статус — статус долга, см. словарь № 6;

·Договор — номер договора с банком;

·Дата выдачи — дата выдачи кредита;

·Город;

·Валюта — валюта долга (см. Список валют);

·Размер кредита — полный размер кредита, выводится в валюте долга, указанной в поле Валюта;

·Ставка;

·Ост. осн. долга — остаток основного долга, выводится в валюте долга, указанной в поле Валюта;

·Начальный долг;

·Остаток — остаток суммы долга на текущую дату, выводится в валюте долга, указанной в поле Валюта;

·Дата просрочки — дата выхода на просрочку;

·Дней просрочки;

·Крайний срок — Крайний срок погашения;

·Причина — Причина возникновения задолженности (см. словарь № 13);

·Лицевой счет — лицевой счет должника;

·ID Клиента;

·Балл № 1 — см. словарь № 101, Скоринговые баллы;

·Балл № 2 — см. словарь № 102, Скоринговые баллы;

·Дата возврата;

·Балл № 3 — см. словарь № 244, Скоринговые баллы;

·Балл № 4 — см. словарь № 245, Скоринговые баллы;

·Основной долг;

·Проценты — сумма процентов по кредиту;

·Штрафы;

·Пени.

Можно добавлять дополнительные колонки по всем абстрактным колонкам из таблицы debt_detail — см. Добавление полей в список долгов.
