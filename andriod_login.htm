Вход и авторизация

Запуск приложения на мобильном устройстве

Запуск системы выполняется стандартными для ОС Android способами.

1.Найдите ярлык Системы "Контакт" на рабочем экране устройства или в списке приложений (Рисунок 1‑1) и коснитесь его ("tap").

2.После запуска системы откроется окно авторизации (Рисунок 1‑2).

В окне авторизации также доступны кнопки: "Настройка" и "Информация".

Если в Системе настроено использование пин-кода, то появится кнопка "PIN-соde" для авторизации по пин-коду (Рисунок 1‑4). В этом случае для того пользователя, у которого настроен пин-код, будет возможность выбора одного из двух способов авторизации — либо по пин-коду, либо по логину и паролю. Кнопка "Информация" будет доступна в меню кнопки .

При входе в Систему осуществляется Обновление данных.

В зависимости от настройки Системы при запуске приложения может осуществляться проверка, включена ли на устройстве GPS.

Авторизация с использованием логина и пароля

1.В окне авторизации введите логин и пароль пользователя и нажмите на экранной клавиатуре кнопку "Далее". Клавиатура закроется (Рисунок 1‑3).

2.Если вы работаете локально, то есть с вашего android-устройства в данный момент времени нет доступа в сеть, включите флаг "Локальная авторизация".

Примечание. Наличие возможности произвольной локальной авторизации определяется константой Android.LocalAuth.Use.

3.Затем нажмите кнопку "Вход" (Рисунок 1‑3).

При загрузке системы сначала будет осуществлено Обновление данных. Затем откроется Главное меню системы.

Авторизация по пин-коду

Примечание. Авторизация по пин-коду возможна только если константа Android.PinCodeLogin.Use имеет значение "True".

1.Если вы хотите работать локально, то есть без подключения к интернету, включите флаг "Локальная авторизация".

Примечание. Наличие возможности произвольной локальной авторизации определяется константой Android.LocalAuth.Use.

2.Нажмите кнопку "PIN-соde" (Рисунок 1‑4). Откроется окно для ввода пин-кода (Рисунок 1‑5).

3.После ввода нажмите ОК. Если пин-код был введен правильно, вы будете авторизованы в Системе.

4.При загрузке системы сначала будет осуществлено Обновление данных. Затем откроется Главное меню системы.

Примечание. Авторизация по пин-коду возможна только для одного пользователя мобильного приложения.
