Прикрепление файлов и ссылок на файлы

Если для исхода контакта настроено прикрепление файла или ссылки на файл, то в форме регистрации контакта соотвественно отображаются панели для прикрепления файла или ссылки.

Прикрепление файла документа

Карточка документа

В модуле "Регистрация контакта" можно прикреплять файлы документов, размер которых не превышает величину, заданную константой DocAttachMaxSize.

Если в вашей версии поддерживается механизм сканирования, то доступно также Прикрепление документов с использованием сканера (Вопросы о возможности поддержки механизма сканирования решайте путем обращения в техническую поддержку).

Для того чтобы прикрепить файл документа, выполните следующие действия:

1.На панели для прикрепления файла нажмите кнопку . Откроется форма "Документ".

2.Заполните следующие поля:

·Название — название документа. Поле обязательно для заполнения. См. Примечание ниже.

Для выбора документа нажмите кнопку . В открывшемся диалоговом окне Windows выберите файл и нажмите кнопку Open (Открыть).

·Прикрепить к — выберите объект, к которому будет прикреплен документ. Данное поле отображается, если включен ф-л прикрепления файлов к объектам Системы. Подробнее — см. Логика прикрепления вложений и ссылок к объектам Системы.

·Тип документа — см. словарь № 100 (для выбора доступны только те типы вложений, доступ к которым настроен для роли пользователя в модуле Роли и доступы на вкладке Типы вложений). Если значение константы Системное сообщение: ВНИМАНИЕ!!! ОШИБКА!!! Пропущена переменная COND_TEXT_1!!! равно "Истина", то поле является обязательным для заполнения;

·Номер — номер документа;

·Техническое поле типа "Словарь" 1 — дополнительное поле типа "Словарь". Данное поле отображается по согласованию с технической поддержкой, так же, как и словарь, который соответствует данному полю. Наименование поля также согласовывается с поддержкой.

·Техническое поле типа "Дата" 1 — дополнительное поле типа "Дата". Данное поле отображается по согласованию с технической поддержкой. Наименование поля также согласовывается с поддержкой.

3.Нажмите кнопку ОК. Документ будет загружен в систему и добавлен в список.

Примечание. Названия для прикрепляемых файлов.

1.Если в Системе настроено автоматическое именование файлов вложений, то поле Название не доступно для редактирования. Подробнее см. Логика прикрепления вложений и ссылок к объектам Системы, раздел Правило именования файлов.

2.Если автоматическое именование файлов не настроено, то:

·Если при нажатии на кнопку "Добавить документ" поле Название пусто, то в него автоматически вводится название файла.

·Если при нажатии на кнопку "Добавить документ" поле Название заполнено, то введенное название сохраняется.

Прикрепление ссылки на файл

Для того чтобы прикрепить ссылку файл, выполните следующие действия:

1.На панели для прикрепления ссылки нажмите кнопку . Откроется форма "Ссылка на документ".

2.В форме заполните следующие поля:

·Название — название документа, поле обязательно для заполнения. См. Примечание ниже.

·Прикрепить к — выберите объект, к которому будет прикреплена ссылка. Данное поле отображается, если включен ф-л прикрепления файлов к объектам Системы. Подробнее — см. Логика прикрепления вложений и ссылок к объектам Системы.

·Тип документа — значение выбирается из словаря № 99. Если значение константы DebtCardUseDocAttachMandatoryType равно TRUE, то поле обязательно для заполнения.

·Ссылка на файл — для заполнения поля нажмите кнопку , в открывшемся диалоговом окне выберите файл и нажмите кнопку Open (Открыть).

·Техническое поле типа "Словарь" 1 — дополнительное поле типа "Словарь". Данное поле отображается по согласованию с технической поддержкой, так же, как и словарь, который соответствует данному полю. Наименование поля также согласовывается с поддержкой.

·Техническое поле типа "Дата" 1 — дополнительное поле типа "Дата". Данное поле отображается по согласованию с технической поддержкой. Наименование поля также согласовывается с поддержкой.

3.Нажмите кнопку ОК. Ссылка на документ будет добавлена в список.

Примечание. Названия для прикрепляемых ссылок.

1.Если в Системе настроено автоматическое именование ссылок, то поле Название не доступно для редактирования. Подробнее см. Логика прикрепления вложений и ссылок к объектам Системы, раздел Правило именования ссылок.

2.Если автоматическое именование ссылок не настроено, то:

·Если при нажатии на кнопку "Добавить документ" поле Название пусто, то в него автоматически вводится название файла.

·Если при нажатии на кнопку "Добавить документ" поле Название заполнено, то введенное название сохраняется.

Ссылки на папки с документами

Для того чтобы добавить ссылку на папку с файлами, нажмите кнопку Выбрать папку. Откроется стандартное окно операционной системы для выбора папки.
