Ведение списка параметров выбранного имущества

Параметры имущества, заданные при добавлении имущества, отображаются на вкладке Имущество в списке параметров.

Пример формы редактирования параметров имущества (групповой параметр)

Добавление параметра имущества

Для того чтобы добавить новое значение параметра имущества, выполните следующие действия:

1.В соответствующем модуле на вкладке Имущество выберите в списке запись об имуществе, для которого надо добавить новое значение параметра.

2.Нажмите кнопку , относящуюся к списку параметров имущества. Откроется форма добавления нового параметра.

3.В форме выберите нужный параметр. Общий список параметров зависит от типа имущества и настраивается в модуле Параметры имущества. В списке отображаются те параметры, значения которых не заданы. После того как параметр выбран, в форме появится поле для ввода его значения. Если для параметра включен признак группового атрибута, то на форму добавляется также поле "Группа" (см. словарь № 369 "Группы параметров имущества"), которое является обязательным для заполнения.

4.Укажите значение параметра.

5.Нажмите кнопку ОК, чтобы сохранить данные. Параметр будет добавлен в список.

Редактирование параметра имущества

Для того чтобы изменить значение параметра имущества, выполните следующие действия:

1.В соответствующем модуле на вкладке Имущество выберите в списке запись об имуществе, для которого надо изменить значение параметра.

2.В списке параметров имущества выберите нужную запись и нажмите кнопку , относящуюся к списку параметров имущества, или выполните двойной щелчок левой клавишей мыши по выбранной записи. Откроется форма редактирования параметра.

3.В форме укажите нужное значение параметра.

4.Нажмите кнопку ОК, чтобы сохранить данные. Параметр будет изменен.

Удаление параметра имущества

Для того чтобы удалить параметр имущества из списка, выполните следующие действия:

1.В соответствующем модуле на вкладке Имущество выберите в списке запись об имуществе.

2.В списке параметров имущества выберите нужную запись.

3.Нажмите кнопку , относящуюся к списку параметров имущества.

4.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.

Особенности добавления словарных параметров имущества

При добавлении имуществу словарного параметра выполняется проверка, есть ли у данного имущества прежде добавленные другие словарные параметры, а также, есть ли в словаре для добавляемого параметра родительский словарь. Если родительский словарь добавляемого параметра является основным словарем для уже добавленного параметра имущества, то возможными значениями для нового параметра могут быть только дочерние термины выбранного значения прежде добавленного словарного параметра.

Например:

1.Для выбранного типа имущества настроены два словарных параметра:

·Параметр 1;

·Параметр 2.

2.Словарь для параметра "Параметр 1" не имеет родительского словаря, поэтому при добавлении параметра "Параметр 1" пользователю доступны все термины соответствующего словаря.

3.Словарь для параметра "Параметр 2" имеет в качестве родительского словарь для параметра "Параметр 1". Для терминов словаря "Параметр 2" указаны родительские термины из словаря "Параметр 1". Для такого параметра действует следующая логика:

·Если мы добавляем параметр имущества "Параметр 2" для имущества, у которого нет параметра "Параметр 1", то нам в качестве значения доступны все термины словаря "Параметр 2".

·Если мы добавляем параметр имущества "Параметр 2" для имущества, которому уже добавлен параметр "Параметр 1", то для выбора значения параметра "Параметр 2" доступны только термины словаря "Параметр 2", являющиеся дочерними по отношению к настроенному значению параметра "Параметр 1". Причем, если соответствующих дочерних терминов в словаре нет, для выбора нет доступных значений.
