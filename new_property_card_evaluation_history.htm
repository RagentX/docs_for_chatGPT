Отображение истории изменений

Для того чтобы просмотреть историю изменения оценки, выполните следующее:

1.Откройте форму "Карточка имущества" для нужного имущества.

2.Активируйте вкладку "Процессы", подчиненную вкладку "Оценка".

3.Выберите нужную запись об оценке.

4.Нажмите кнопку "История изменения". Откроется форма "История изменения".

5.По окончании просмотра нажмите ОК.

Список полей формы "История изменения":

·Процесс (property_estimate_log.r_task_id).

·Сумма оценки (property_estimate_log.estimate_sum).

·Номер оценки (property_estimate_log.estimate_num).

·Оценщик (property_estimate_log.estimator).

·Дата оценки  (property_estimate_log.dt).

·Дата действия отчета об оценке (property_estimate_log.valid_estimate_dt).

·Дата составления отчета (property_estimate_log.create_dt).

·НДС (property_estimate_log.is_VAT Если значение в поле 1, то в интерфейсе выводится «Да», если значение 0, то в интерфейсе выводится «Нет»).

·Категория оценки (property_estimate_log.estimate_typ, вывод терминов из словаря «Категория оценки»).

·ИНН оценщика (property_estimate_log.estimator_inn).

·Назначение оценки (property_estimate_log.assigning).

·Ссылка на отчет (property_estimate_log.doc_link).

·Цель оценки (property_estimate_log.estimate_goal, вывод терминов из словаря «Цель оценки»).

·Номер рецензии на отчет об оценке (property_estimate_log.estimate_review_num).

·Дата рецензии на отчет об оценке (property_estimate_log.estimate_review_dt).

·Сумма, определенная рецензией (property_estimate_log.estimate_review_sum).

·Рецензент (property_estimate_log.reviewer, вывод терминов из словаря «Рецензент»).

·Вывод о корректности рыночной стоимости (property_estimate_log.conclusion, вывод терминов из словаря «Вывод о корректности рыночной стоимости»).

·Ликвидационная стоимость (property_estimate_log.liquid_sum).

·НДС по ликвид стоимости (property_estimate_log.is_liquid_VAT. Если значение в поле 1, то в интерфейсе выводится «Да», если значение 0, то в интерфейсе выводится «Нет»).

·Стоимость аренды (руб./кв.м. в год) (property_estimate_log.rent_sum).

·НДС по стоим. Аренды (property_estimate_log.is_rent_VAT. Если значение в поле 1, то в интерфейсе выводится «Да», если значение 0, то в интерфейсе выводится «Нет»).

·Дата изменения (property_estimate_log.update_date).

·Пользователь (property_estimate_log.contact_user).
