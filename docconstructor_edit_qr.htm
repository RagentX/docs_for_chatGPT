Добавление QR-кода в текст письма

Системные требования

Добавление QR-кода доступно начиная версии FastReport VCL 4.

Данный функционал доступен в версиях Системы, начиная с 45 релиза.

Доступность функционала в Системе

Добавление QR-кода в шаблон документа доступно в следующих модулях Системы:

·Конструктор писем

·Конструктор документов судебной работы

·Письмо должнику (выездной лист)

·Документ по судебной работе

·Произвольные отчеты

Данные для формирования QR-кода

Вы можете добавлять QR-код в документ в одном из перечисленных выше модулей. Данные для формирования QR_кода должны храниться в одном из полей БД, доступных в данном модуле.

Порядок действий

Поскольку данное руководство рассчитано на опытных пользователей, мы не будем останавливаться на том, как создать документ в формате FR3, изменить, сохранить, как добавлять объекты и пр.

Для того чтобы добавить QR-код в документ, выполните следующее:

1.Откройте в Системе нужный шаблон документа.

2.В окне редактора Fast Report убедитесь, что подключен нужный набор данных.

3.В компонент шаблона TfrxMasterData добавьте компонент TfrxPictureView.

См. Рисунок 1 — в компоненте MasterData1 с набором данных frxDBRep1 добавлен рисунок Picture1).

4.Выберите компонент TfrxMasterData.

5.Откройте вкладку "События", расположенную в левой части рабочей области экрана, ниже названия компонента TfrxMasterData, и выберите на ней событие OnBeforePrint. Откроется вкладка "Код". Автоматически будет создана процедура для данного события OnBeforePrint.

См. Рисунок 2 — в компоненте MasterData1 выбрано событие MasterData1OnBeforePrint. Открыто окно с программным кодом (вкладка «Код»). Мы видим название процедуры и операторы begin и end;.

6.Перейдите на вкладку «Код» в левой верхней части рабочей области экрана.

7.Введите следующий текст:

Компонент.ConvertToQRCode(<Данные>);

где:

·Компонент — название компонента TfrxPictureView (в примере — Picture1);

·Данные — текст или поле для кодирования в QR-код.

См. Рисунок 3 — введена строка:

Picture1.ConvertToQRCode(<frxDBRep1."account">);

8.Нажмите кнопку "Предварительный просмотр" в панели инструментов редактора Fast Report, расположенной в верхней части рабочей области экрана для просмотра полученного QR-кода — см. Рисунок 4.
