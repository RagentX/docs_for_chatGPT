Обновлено до 49 релиза 08.09.2020.

Модуль Группы свободного поиска содержит набор вкладок с данными, отобранными по каким-либо критериям с помощью запросов. Для того чтобы открыть главную форму модуля, выберите в главном меню пункт Рабочие места/Группа свободного поиска.

Доступ пользователей к вкладкам модуля свободного поиска назначается в модуле Группы должников. Полный перечень доступов к модулю приведен в таблице.
