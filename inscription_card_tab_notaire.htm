Вкладка "Информация о нотариусе"

В карточке исполнительной надписи на вкладке Информация о нотариусе отображается информация о нотариусе, который проставил данную надпись.  

Вкладка содержит следующие элементы:

·Поля:

·Тип нотариуса — тип нотариуса, выпадающий список выбора типа по словарю № 342 "Тип нотариуса";

·ФИО нотариуса — выпадающий список выбора нотариуса. Список выпадает только в случае, если был выбран тип, и зависит от выбранного типа, т.е. в выпадающем списке должны быть ФИО только тех нотариусов, тип которых совпадает с выбранным. Данные в выпадающем списке выводятся табличкой со столбцами ФИО, Индекс, Телефон.

Остальные поля не доступны для редактирования, в них отображаются данные о выбранном нотариусе:

·Индекс;

·Телефон;

·Адрес;

·Нотариальный орган;

·ИНН;

·Расчетный счет;

·Корреспондентский счет;

·БИК;

·кнопка для добавления нотариуса.
