Блокирование адреса

Доступ на блокирование адреса — 142 DEBT_CARD_ADDR_BLOCK.

Доступ на блокирование адреса, по которому был зафиксирован один или более выездов — 521 DEBT_CARD_CONTACT_ADDR_BLOCK.

Для того чтобы заблокировать адрес, выполните следующие действия:

1.Выберите нужный адрес в списке адресов.

2.Активируйте контекстное меню с помощью правой кнопки мыши.

3.В открывшемся списке выберите пункт Заблокировать адрес.

4.Если по данному адресу были зафиксированы выезды, а пользователь не имеет соответствующих полномочий (доступ № 521), то будет показано предупреждение, и блокировка не осуществится. Иначе процесс движется дальше.

5.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить блокирование.

6.Появится форма ввода причины блокировки адреса. Список возможных причин хранится в словаре № 314 "Причина блокировки адреса". Если константа Address.Block.Reason.Mandatory имеет значение "Истина", то ввод причины обязателен. Выберите причину блокировки адреса (если необходимо).

7.Нажмите ОК.

8.Запись о заблокированном адресе будет выделена серым шрифтом.

Примечание. Удалить адрес можно только из модуля Постановка задач на вкладке Подготовка выездов (см. Удаление адресов должников). При этом удаление доступно только для тех  адреса, по которым не было зафиксированных выездов или сформированных писем. Осуществить удаление могут только пользователи, наделенные соответствующими полномочиями.
