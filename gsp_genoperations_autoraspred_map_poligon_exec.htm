Выполнение автораспределения по многоугольникам

1.Войдите в модуль Группы свободного поиска.

2.Активируйте нужную вкладку.

3.Создайте выборку, используя фильтрацию.

4.При необходимости выберите записи, используя клавиши SHIFT и/или CTRL.

5.Нажмите кнопку Работа с оператором.

6.В открывшемся меню включенный флаг Только выделенные означает выполнение операции только для выделенных записей, выключенный — для всей выборки.

7.Нажмите кнопку Автораспределение.

8.В выпадающем меню выберите пункт По карте.

9.В раскрывшемся списке выберите пункт По многоугольникам.

10.Далее выберите пункт Выполнить.

11.Система попытается автоматически распределить долги по операторам в соответствии с заданными настройками. Затем будет показано информационное окно с указанием количества распределенных долгов.
