Доступ к вкладке Лог действий

Вкладка Лог действий в карточке должника содержит список действий, совершённых с карточкой должника и в модуле Входящий звонок и из модуля Загрузка данных для данного пользователя. На вкладке доступен только просмотр данных.

В случае, если значение константы PartitioningActionsHistoryDefaultPeriod больше нуля, на форме вкладки Лог действий отображается фильтр "Выводить действия с". Дата, указанная в этом поле, вычисляется как текущая дата минус число дней, заданное в константе PartitioningActionsHistoryDefaultPeriod. При открытии формы осуществляется фильтрация: выводятся записи, начиная с указанной даты и до текущей. Во время работы с формой пользователь может устанавливать в этом поле любую дату, при этом в списке будут выводится записи начиная с выбранной даты.
