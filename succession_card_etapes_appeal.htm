Обжалование

Примечание. Функционал обжалования используется в правопреемстве, если константа LawAct.Appeal.Use включает значение 3.

Для обжалования правопреемственных дел доступны следующие типы жалоб:

·Частная жалоба;

·Апелляция;

·Кассация.

Обжалование доступно для дел в статусах:

·Отказ в правопреемстве;

·Частная жалоба отклонена;

·Апелляция отклонена;

·Кассация отклонена.

Жизненный цикл жалобы

·Если используется полный цикл жалобы значение технической константы LawActUsePrepareAppeal равно "Истина":

·Обжалование (кнопка Обжалование) — дело переводится в статус "Не подготовлена частная жалоба/апелляция/кассация";

·Подготовка жалобы (кнопка Подготовить) — дело переводится в статус "Подготовлена частная жалоба/апелляция/кассация";

·Подача жалобы (кнопка Подать) — дело переводится в статус "Подана частная жалоба/апелляция/кассация";

·Если используется сокращенный цикл жалобы значение технической константы LawActUsePrepareAppeal равно "Ложь":

·По кнопке Обжалование дело переводится в статус "Подана частная жалоба/апелляция/кассация".

Отметка результата обжалования становится доступна для дел в статусах "Подана частная жалоба/апелляция/кассация".

Продвижение жалобы по этапам обжалования

Обжалование

Для обжалования правопреемства, выполните следующие действия:

1.Нажмите кнопку Обжалование. В раскрывшемся списке выберите тип жалобы:

·Частная жалоба;

·Апелляция;

·Кассация.

2. В открывшемся диалоговом окне нажмите кнопку Yes на вопрос "Подтверждаете подачу жалобы?".

3.Выполните прикрепление документа.

4.В зависимости от типа жалобы дело переводится в статус:

·Полный цикл жалобы:

·Не подготовлена частная жалоба;

·Не подготовлена апелляция;

·Не подготовлена кассация;

·Сокращенный цикл жалобы:

·Подана частная жалоба;

·Подана апелляция;

·Подана кассация.

5.Действие протоколируется. В поле Действие записывается: "Обжалование", в поле Комментарий в зависимости от типа жалобы записывается:

·"Частная жалоба";

·"Апелляция";

·"Кассация".

Подготовка жалобы

Подготовка жалобы доступна, если используется полный цикл жалобы значение технической константы LawActUsePrepareAppeal равно "True".

Подготовка жалобы доступна для дел, находящихся в статусах:

·Не подготовлена частная жалоба;

·Не подготовлена апелляция;

·Не подготовлена кассация.

Для того чтобы подготовить жалобу, выполните следующие действия:

1.Нажмите кнопку Подготовить.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить действие.

3.Выполните прикрепление документа.

4.В зависимости от типа жалобы дело переводится в статус:

·Подготовлена частная жалоба;

·Подготовлена апелляция;

·Подготовлена кассация;

5.Действие протоколируется. В поле Действие записывается: "Подготовлена жалоба", в поле Комментарий в зависимости от типа жалобы записывается:

·"Частная жалоба. Подготовлена жалоба";

·"Апелляция. Подготовлена жалоба";

·"Кассация. Подготовлена жалоба".

Подача жалобы

Подача жалобы доступна, если используется полный цикл жалобы значение технической константы LawActUsePrepareAppeal равно "True".

Подача жалобы доступна для дел, находящихся в статусах:

·Подготовлена частная жалоба;

·Подготовлена апелляция;

·Подготовлена кассация;

1.Укажите адрес подачи, если это необходимо (см. значение константы LawAddressLinkUseMode).

2.Нажмите кнопку Подать.

Если какие-либо обязательные поля не заполнены, будет выдано информационное сообщение. Нажмите в нем кнопку ОК, после чего произойдет возврат в карточку судебной работы для внесения данных.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить действие.

4.Выполните прикрепление документа.

5.В зависимости от типа жалобы дело переводится в статус:

·Подана частная жалоба;

·Подана апелляция;

·Подана кассация.

6.Действие протоколируется. В поле Действие записывается: "Подана жалоба", в поле Комментарий в зависимости от типа жалобы записывается:

·"Частная жалоба. Подана жалоба";

·"Апелляция. Подана жалоба";

·"Кассация. Подана жалоба".
