Закрепление адресов за исп. органами

Вкладка Вкладка "Закрепление адресов за исп. органами" доступна пользователям, в роли которых доступ № 701 LAW_JUDGE_ADDR_SET_COURT_VIEW имеет значение TRUE.

Вкладка предназначена для закрепления адресов за судами и последующего автоматического выбора суда в карточке судебной работы или автоматический выбор исполнительного органа в карточке исполнительного производства.

Закрепление адресов за исполнительными органами осуществляется путем добавления, редактирования и удаления записей о связи исполнительного органа с адресами.

Добавление записи о связи исполнительного органа с адресами

Для того чтобы добавить запись о связи исполнительного органа с адресами, выполните следующее:

1.В модуле Настройка судов и ФССП активируйте вкладку Закрепление адресов за исп. органами.

2.Нажмите кнопку Добавить. Откроется форма Создание правила.

3.Заполните поля формы в соответствии с Правилами заполнения формы "Создание правила"/"Редактирование правила".

4.Нажмите ОК.

Запись будет добавлена в список на вкладке Закрепление адресов за судами.

Редактирование записи о связи исполнительного органа с адресами

Для того чтобы отредактировать запись о связи исполнительного органа с адресами, выполните следующее:

1.В модуле Настройка судов и ФССП активируйте вкладку Закрепление адресов за исп. органами.

2.Выберите запись из списка.

3.Нажмите кнопку Редактировать. Откроется форма Редактирование правила, аналогичная форме Создание правила, поля которой заполнены.

4.Измените поля формы в соответствии с Правилами заполнения формы "Создание правила"/"Редактирование правила".

5.Нажмите ОК.

Запись будет изменена.

Удаление записи о связи исполнительного органа с адресами

Для того чтобы удалить запись о связи исполнительного органа с адресами, выполните следующее:

1.В модуле Настройка судов и ФССП активируйте вкладку Закрепление адресов за исп. органами.

2.Выберите запись из списка.

3.Нажмите кнопку Удалить.

4.Откроется форма подтверждения. Нажмите Yes.

Запись будет удалена.
