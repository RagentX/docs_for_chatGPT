Просмотр судебных дел по долгу

Для того чтобы просмотреть карточку должника, нажмите кнопку Карточка должника в верхней части карточки судебной работы, а затем выберите пункт "Судебные дела по долгу".

Данный пункт доступен пользователям, в роли которых доступ LAW_ACT_CARD_LAW_ACT_LIST_VIEW имеет значение TRUE.

При выборе данного пункта открывается форма для просмотра судебных дел, созданных по текущему долгу.

На форме доступны следующие действия:

·Отображение только активных дел / отображение всех дел — включите флаг Показывать активные записи, чтобы в списке отобразились также и архивные дела;

·Открытие карточки судебной работы — выберите запись в списке и выполните двойной клик левой кнопкой мыши.

В колонке Вид списка судебных дел по долгу отображается следующая информация:

·Объединенное судебное дело — для объединенных дел;

·Входящий иск — для входящих исков.
