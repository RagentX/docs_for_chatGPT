Редактирование печатной формы письма

Внимание! Создавать и редактировать печатные формы писем в модуле Конструктор писем рекомендуется только пользователям, имеющим опыт работы с приложением FastReport.

Примечание. Поддержана возможность работы с документами в формате А3.

Для того чтобы отредактировать печатную форму письма, выполните следующие действия:

1.В форме Конструктор писем выполните двойной щелчок (клик) левой клавишей мыши по выбранной записи. Откроется окно редактора FastReport, содержащее форму письма.

2.В открывшемся окне выберите объект для редактирования и выполните клик (двойной щелчок левой клавишей мыши) на нем.

3.В открывшемся окне измените текст. Можно изменять как статичный текст (который будет одинаковым для всех должников, например "Требование о полном погашении задолженности по кредиту" или "Уважаемый (ая)" т. п.), так и динамичный текст (текст который будет различаться в зависимости от выбранного должника, например фамилия должника или его ИНН и т. п.).

4.После окончания редактирования текста нажмите кнопку OK, чтобы сохранить изменения. Для отмены нажмите Cancel.

5.Если вам требуется сохранить измененную форму письма, чтобы использовать ее при создании новой печатной формы, сохраните измененный файл, иначе пропустите этот пункт. Для того чтобы сохранить форму:

a.В окне FastReport выберите в меню пункт Файл а Сохранить как… (File а Save as…).

b.В диалоговом окне Windows выберите путь, куда необходимо сохранить отредактированную форму, и укажите имя файла. Имя файла рекомендуется выбирать таким образом, чтобы было понятно, к какому типу формы будет относиться данный файл. Например для формы "Форма 1" — form1.fr3, для формы "Физ.лица" — FizL.fr3 и т. п.

c.Нажмите кнопку Save (Сохранить).

6.Закройте окно приложения FastReport.

7.Для сохранения изменений в шаблоне письма нажмите Yes в окне подтверждения.

8.При необходимости выполните Редактирование атрибутов печатной формы письма.
