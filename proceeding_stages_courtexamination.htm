Судебное разбирательство

При установке отметки об итогах судебного разбирательства возможны следующие варианты:

·удовлетворение иска;

·отказ в удовлетворении иска;

·перенос судебного заседания;

·определение о прекращении;

·определение об оставлении без рассмотрения;

·Частичное удовлетворение иска.

Удовлетворение иска

Для установки отметки об удовлетворении иска выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство в разделе Судебный иск заполните поле Дата вступления в силу.

2.Нажмите кнопку Судебное разб. и в раскрывшемся списке выберите Удовлетворение иска.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.После этого иск будет переведен в статус "Удовлетворение иска". Данный статус является конечным.

7.Действие протоколируется.

Примечание. Если константа LawAct.Decision.Components.Compare имеет значение "Истина", то при выполнении действия Удовлетворение иска дополнительно сравниваются текущие значения составляющих на момент подачи в суд со значениями составляющих по решению суда (составляющие для приказа перечислены в константе LawActIskDebtComponentList), а также значение размера пошлины с утвержденным размером пошлины.

Если хотя бы одна пара значений не совпадает, пользователю отображается окно с сообщением:

Поданные и присужденные суммы не совпадают. Продолжить?

Нажмите ОК чтобы продолжить или Отмена чтобы прервать действие.

Примечание для поддержки

Если было выбрано Удовлетворение иска, то производится дополнительная проверка обязательности заполнения следующих полей карточки приказного производства, согласно значению технической константы LawAct.PositiveResult.Fields.Mandatory:

·Утвержденная пошлина — если в скобке с двойкой есть значение 741;

·Сумма по решению суда — если в скобке с двойкой есть значение 740;

·Номер дела в суде — если в скобке с двойкой есть значение 526;

·Дата вступления в силу — если в скобке с двойкой есть значение 525;

·Дата вынесения решения — если в скобке с двойкой есть значение 420.

3.Выполните прикрепление документа.

4.Далее осуществляется формирование настроенных запланированных действий.

5.После этого иск будет переведен в статус "Отказ в удовлетворении иска".

6.Действие протоколируется.

7.Далее возможно обжалование результатов решения Суда.

Примечание. Если константа LawAct.Decision.Components.Compare имеет значение "Истина", то при выполнении действия Отказ в удовлетворении иска дополнительно сравниваются текущие значения составляющих на момент подачи в суд со значениями составляющих по решению суда (составляющие для приказа перечислены в константе LawActIskDebtComponentList), а также значение размера пошлины с утвержденным размером пошлины.

Если хотя бы одна пара значений не совпадает, пользователю отображается окно с сообщением:

Поданные и присужденные суммы не совпадают. Продолжить?

Нажмите ОК чтобы продолжить или Отмена чтобы прервать действие.

Примечание для поддержки

·Номер дела в суде — если в скобке с двойкой есть значение 526;

·Дата вынесения решения — если в скобке с двойкой есть значение 420.

Перенос судебного заседания

Для установки отметки о переносе судебного заседания выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Судебное разб. и в раскрывшемся списке выберите Перенос заседания.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.В открывшейся форме укажите новую дату заседания и причину переноса (оба поля обязательны для заполнения) и нажмите кнопку ОК.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.Действие протоколируется.

7.В данном случае иск будет оставлен в прежнем статусе до появления итогов проведения перенесенного заседания.

Определение о прекращении дела

Действие доступно для дел в следующих статусах:

·Определение о принятии к производству;

·Определение о рассмотрении по существу.

Для установки отметки о прекращении дела выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Судебное разб. и в раскрывшемся списке выберите Определение о прекращении.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.Откроется форма Определение о прекращении. Заполните следующие поля:

·Дата — выберите дату. Поле обязательно для заполнения.

·Причина — выберите причину, доступны термины словаря № 319 "Причина перевода судебного дела в текущий статус", имеющие в качестве родительского термин "Определение о прекращении". Поле не обязательно к заполнению.

·Комментарий — дополнительная информация. Поле не обязательно к заполнению.

Примечание. Поле Причина отображается только в том случае, если в словаре № 319 есть хотя бы один термин, имеющий в качестве родительского термин Определение о прекращении.

4.Нажмите ОК.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.Действие протоколируется.

8.После этого иск будет переведен в статус "Определение о прекращении". Данный статус является конечным. Просмотр дела доступен в Архиве судебной работы.

7.Для продолжения работы необходимо будет заново подать иск в суд. Также доступно Принятие судебного иска к производству.
