Архив

Меню кнопки Архив включает следующие пункты:

·Перевод в архив;

·Возобновить.

Кнопка Архив доступна, если доступен хотя бы один из ее подпунктов.

Перевод в архив

Условия доступности

Кнопка Перевод в архив доступна, если:

·В роли пользователя доступ № 589 LAW_ACT_BANKRUPT_ARCHIVE имеет значение TRUE;

·Дело находится в любом статусе, кроме: "Архив", "Аннулировано", "Определение о прекращении", "Перевод на исполнительное производство" или "Мировое соглашение".

Порядок действий

Для перевода дела в архив выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Архив и в раскрывшемся списке выберите Перевод в архив.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить перевод.

3.В открывшейся форме Перевод в архив заполните следующие поля:

·Причина — выберите причину из списка; для  для выбора доступны термины словаря № 319, имеющие в качестве родительского термина "Архив"; если таких терминов нет, то поле не отображается; если поле отображается, то оно обязательно для заполнения;

·Комментарий — дополнительная информация, не обязательно для заполнения.

4.По окончании перевода будет выдано сообщение о результатах выполнения операции. Нажмите кнопку ОК, чтобы закрыть его. Дело будет переведено в статус "Архив" и перенесено в соответствующий архив.

5.Действие протоколируется.

Возобновить

Условия доступности

Кнопка Возобновить доступна, если:

·В роли пользователя доступ № 496 LAW_ACT_RETURN_FROM_ARCHIVE имеет значение TRUE.

·Дело находится в одном из архивных статусов или "Мировое соглашение".

Порядок действий

Для перевода дела в архив выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Архив и в раскрывшемся списке выберите Возобновить.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить перевод.

3.Далее осуществляется формирование настроенных запланированных действий.

4.Дело получит статус "Возобновление" и будет перемещено из архива.

5.Действие протоколируется.
