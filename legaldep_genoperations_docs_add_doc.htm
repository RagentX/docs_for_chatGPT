Загрузить документ

Show/Hide Hidden Text

В модуле вы можете прикрепить документ как к одному, так и к нескольким должникам (долгам) одновременно.

       Прикрепление документов к долгу или к должнику определяется значением константы DebtCardAttachDocsToDebt:

Прикрепление документов к долгу или должнику:

Прикрепление документов к долгу или к должнику определяется значением константы DebtCardAttachDocsToDebt:

·Если значение  равно "Истина", то в форме добавления документа по умолчанию выбран объект "Долги" — активный долг.

·Если значение  равно "Ложь", то в форме добавления документа по умолчанию выбран объект "Должники" — данный должник.

 

Для того чтобы загрузить документ в одну или несколько записей о долгах

1.Войдите в модуль Юридический департамент, активируйте вкладку Приказы, Правопреемство, Иски, Банкротство или Судебные дела (предв.).

2.Сформируйте выборку при помощи мультифильтров. В полученном списке выберите записи при помощи клавиш SHIFT и/или CTRL.

3.Активируйте контекстное меню, щелкнув правой кнопкой мыши, и выберите пункт Добавить документ.

4.В открывшемся меню выберите пункт Загрузить документ. Откроется стандартное окно загрузки файла.

5.Выберите файл и нажмите Открыть (Open). Откроется окно загрузки описания документа.

6.Заполните поля:

·Название — введите название документа (обязательно для заполнения, при выборе файла заполняется автоматически);

·Тип документа — см. словарь № 100 "Тип вложения для файла" (не обязательно для заполнения);

·Номер — введите номер документа (не обязательно для заполнения);

·Версия — введите номер версии, состоящий из двух натуральных чисел (не обязательно для заполнения).

7.Нажмите OK.

Примечание. Значение атрибута Стадия при прикреплении документа или ссылки на документ заполняется автоматически терминами словаря № 132 Стадия взыскания в зависимости от того, в каком модуле было осуществлено прикрепление документа:

·Карточка должника — термин № 1 "Досудебная";

·Юридический департамент, вкладки судебной работы — термин № 2 "Судебная работа";

·Карточка судебной работы —  термин № 2 "Судебная работа";

·Карточка исполнительного производства — термин № 3 "Исполнительное производство".
