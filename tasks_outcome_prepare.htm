Подготовка выездов и печать соответствующих документов

Show/Hide Hidden Text

Подготовка выездов к должникам может включать следующие операции:

·печать уведомлений;

·печать выездных листов;

·установка отметки подготовки выезда.

Подготовить выезд можно:

·для всех записей из выборки, сформированной с учетом фильтров;

·для всех записей, выделенных в текущей выборке с помощью клавиш SHIFT и/или CTRL.

       Учет ограничений коммуникации

При массовой подготовке выездов проверка ограничений коммуникации осуществляется на этапе создания списка адресов.

Осуществляется проверка, все ли выезды могут быть подготовлены с учетом ограничений.

Если все выезды могут быть подготовлены, они подготавливаются.

Если есть выезды, которые не могут быть подготовлены, то в зависимости от значения в роли пользователя доступа, разрешающего превышение лимита, осуществляется следующее:

·Если доступ выключен или отсутствует, то пользователю предлагается выбор:

·Подготовить только разрешенные выезды;

·Не подготавливать выезды вообще.

·Если доступ включен, то пользователю предлагается выбор:

·Подготовить все выбранные выезды;

·Подготовить только разрешенные выезды;

·Не подготавливать выезды вообще.

Примечание. При подготовке всех доступных выезды осуществляется проверка возможности формирования каждого выезда.

       Возможность осуществления коммуникации при превышении лимита

Существует ряд доступов, дающих пользователю возможность осуществить коммуникацию сверх лимита.

Если в роли пользователя для данного типа коммуникации такой доступ включен, то при превышении лимита ему выдается соответствующее сообщение:

Формирование сообщения превысит допустимый лимит коммуникации. Вы действительно хотите <название действия для данного типа коммуникации>?

Выберите:

·Yes — если вы хотите совершить коммуникацию сверх лимита;

·No — если вы не хотите совершить коммуникацию сверх лимита.

Доступы на превышение лимитов коммуникации разных типов

·Доступ № 633 COMMUNICATION_SMS_EXCEED_LIMIT — дает право на формирование SMS – сообщений при превышении лимита согласно схеме коммуникации;

·Доступ № 634 COMMUNICATION_EMAIL_EXCEED_LIMIT — дает право на формирование EMAIL – сообщений при превышении лимита согласно схеме коммуникации;

·Доступ № 635 COMMUNICATION_CALL_EXCEED_LIMIT — дает право на совершение исходящего вызова при превышении лимита коммуникации;

·Доступ № 636 COMMUNICATION_VISIT_EXCEED_LIMIT — дает право на подготовку выезда при превышении лимита коммуникации.

 

Для этого выполните следующие действия:

1.В форме Постановка задач перейдите на вкладку Подготовка выездов (доступна подготовка выезда, находящегося в любом статусе) или на вкладку Подготовка выездов 2 (доступна подготовка выезда, находящегося в статусе "Требуется").

2.При необходимости сформируйте выборку записей с помощью фильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Подготовка выездов и в раскрывающемся списке выберите вариант выполнения операции: По всей выборке или Только выделенные.

4.В открывшейся форме установите флаги для тех операций, которые должны быть выполнены, и нажмите кнопку ОК.

5.Если был установлен флаг выполнения операции Отметка подготовки выезда, то откроется форма для указания даты выезда. В этом случае:

a.В поле Дата выезда выберите дату выезда, а также укажите цели вызова, установив соответствующие флаги. Дата выезда должна быть больше текущей даты. Перечень возможных целей формируется в словаре № 90. Информация о планируемой дате выезда будет сохранена и доступна в модуле "Обработка должников" на вкладке Календарь (см. Календарь).

b.Нажмите кнопку ОК. Откроется форма выбора оператора, для которого планируется выезд.

c.В открывшейся форме выберите оператора и нажмите кнопку ОК. Выезду будет присвоен статус "Подготовлен". Информация о подготовленном выезде будет отображаться в модуле "Обработка должников" для указанного оператора. При установке выезда по адресу в статус "Подготовлен" фиксируются:

·дата подготовки выезда;

·цель выезда;

·оператор, который наметил выезд;

·оператор, для которого намечен выезд.

6.Если был установлен флаг выполнения операции Печатать уведомления, то откроется форма выбора типа письма. В этом случае:

a.В открывшейся форме в поле Тип письма выберите форму уведомления (см. словарь № 36 и модуль "Конструктор писем"). Если требуется включить в письмо информацию по всем долгам должника, установите флаг По всем долгам.

b.Нажмите кнопку ОК. По окончании выполнения процедуры будет сформирован отчет, каждая страница которого будет содержать уведомление о выезде к должнику. Сформированный отчет откроется в приложении Fast Report в режиме предварительного просмотра.

Примечание:

·Право на печать писем (уведомлений) имеют пользователи, в роли которых включен доступ № 43 REP_DEBT_LETTER.

·Пользователю доступны только те печатные формы писем, к которым в его роли настроен доступ.

·В зависимости от банка, которому принадлежит долг, работает ограничение на формирование писем. Если константа InfoDebtLetterBankRestriction имеет значение "Ложь", это ограничение не применяется.

7.Если был установлен флаг выполнения операции Печатать выездные листы, то по окончании выполнения процедуры будет сформирован отчет, каждая страница которого содержит выездной лист по должнику. Сформированный отчет откроется в приложении Fast Report в режиме предварительного просмотра.

Примечание. Право на печать выездных листов имеют пользователи, в роли которых включен доступ № 30 REP_MAIN_INFO.

Предварительная настройка констант для выездного листа, вид выездного листа и особенности отображения данных в нем описаны в разделе Отчет "Версия для печати" (Выездной лист) для модуля Список должников.

Учет статистики по выездам

Зарегистрированный контакт с типом "Выезд" всегда учитывается в статистике "Количество контактов всего".

Зарегистрированный контакт с типом "Выезд" учитывается в статистике "Количество контактов, учитываемых в ограничении", если у узла дерева контактов проставлен атрибут "Учитывать в ограничениях" и "Перевести выезд в статус выполнения" и независимо от значения параметра "Считать результат успешным"
