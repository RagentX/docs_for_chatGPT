Вкладка "Долги в группе"

Show/Hide Hidden Text

Вкладка Долги в группе предназначена для просмотра списка долгов, входящих в  выбранную группу, и формирования групп долгов.

Вкладка содержит следующие элементы:

·поле Группа для выбора группы, долги которой будут отображаться в списке;

Внимание! В списке отображаются только группы, имеющие объект Долги!

·список долгов, входящих в выбранную группу. Список содержит атрибуты долга, включая ID должника, ID долга, ФИО должника, Портфель, Остаток, Статус долга, Договор, Фактическая просрочка и др. Слева внизу блока отображается общее количество записей в группе (список полей — см. далее);

·кнопки для работы со списком долгов, входящих в группу. Кнопки доступны только для обычных, сформированных вручную групп долгов.

·кнопка Упорядочить — после выбора варианта По всей выборке или Только выделенные осуществляет упорядочивание записей в группе. Доступна только для простых групп.

Примечание. Если осуществить клик (двойной щелчок левой клавишей мыши) на одной из записей в таблице, то откроется Карточка должника.

На вкладке Долги в группе доступны следующие действия:

·Просмотр долгов в группе — действие доступно для любых групп;

·Добавление долгов в простую группу — доступно только для простых групп;

·Удаление долгов из простой группы — доступно только для простых групп;

·Упорядочивание записей в простой группе — доступно только для простых групп.

       Список полей таблицы "Долги в группе"

·ID — идентификатор записи;

·ID должника;

·ID долга;

·Номер п/п — порядковый номер записи в группе;

·ФИО — фамилия, имя и отчество должника;

·Банк — название банка, которому принадлежит портфель;

·Портфель — входящий портфель, в состав которого входит долг;

·Название — продукт, по которому возник долг;

·Валюта — валюта долга (см. Список валют);

·Остаток — остаток суммы долга на текущую дату, выводится в валюте долга, указанной в поле Валюта;

·Оператор — ФИО оператора, закрепленного за данным долгом должника;

·Основной фильтр — наименование автофильтра;

·Статус долга;

·Ближайший звонок — дата и время для ближайшего звонка должнику (операторы устанавливают эту дату при регистрации контакта с результатом "Необходимо перезвонить");

·Обещ. дата — обещанная дата (не отображается для неверифицированных обещаний);

·Статус обещания — см. Статусы обещаний;

·GMT — см. словарь № 51 "Часовые пояса";

·Группа GMT — см. словарь № 315;

·Дата обработки;

·Балл № 1 — см. словарь № 101, Скоринговые баллы;

·Балл № 2 — см. словарь № 102, Скоринговые баллы;

·Гос. пошлина — сумма государственной пошлины;

·Остаток основного долга — выводится в валюте долга, указанной в поле Валюта;

·Договор — номер договора с банком;

·Фактическая просрочка — если значение поля flag_preventive таблицы debt = 1 (Признак "Профилактика"), то в колонке отображается 0; иначе значение рассчитывается как "Текущая дата – Дата выхода долга на просрочку (debt.start_date)". См. также Дней просрочки;

·Филиал — филиал организации, к которому относится долг (словарь № 61).
