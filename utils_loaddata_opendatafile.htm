Выбор формата и открытие файла с данными

Для того чтобы открыть файл MS Excel с данными для загрузки, выполните следующие действия:

1.В правой верхней части формы Загрузка данных выберите формат загружаемых данных в списке Название формата. После этого станут доступны кнопки Открыть файл и Отменить.

Примечание 1. Первые двенадцать строк перечня форматов в списке Название формата соответствуют специальным форматам дозагрузки (контактных лиц, расчетных лиц, телефонов, адресов и текстов адресов). Шестая строка соответствует так называемому "универсальному" формату. Файл для универсального формата формируется с помощью меню пользователя на этапе проверки данных в таблице с целью сохранения промежуточных данных и последующей их обработки без повторного преобразования данных. Остальные форматы настраиваются пользователями с использованием модуля "Настройка шаблонов загрузки данных".

Примечание 2. В случае отсутствия в БД констант, используемых при функционировании модуля (см. Предварительная установка констант), при открытии формы будет выдано предупреждение/предупреждения "Отсутствует константа <имя константы>". Функционирование модуля будет продолжено.

Примечание 3. В случае некорректных значений констант LoadDataOblast и LoadDataOblast2 при открытии формы будет выдано сообщение "Константа <название константы, LoadDataOblast или LoadDataOblast2> имеет некорректное значение <значение>".

Примечание 4. В случае отсутствия в БД словаря (словарей) для словарных реквизитов при открытии формы будет выдано предупреждение (предупреждения) "Отсутствует <номер словаря> для реквизита <условный номер реквизита в БД>". Функционирование модуля будет продолжено.

2.Нажмите кнопку Открыть файл.

3.В открывшемся стандартном диалоговом окне выберите файл с данными и нажмите кнопку Open (Открыть). В процессе открытия файла в нижней строке окна формы будет отображаться тип операции (открытие файла), информация о проценте обработанных записей, директория и имя файла.

Примечание 5. Если выбран формат "Досудебная работа (обновление)", первая колонка файла MS Excel должна содержать идентификаторы обновляемых долгов в БД, остальные колонки должны соответствовать описанию формата "Досудебная работа (обновление)". Колонка идентификаторов долгов в формате обновления не описывается и формируется автоматически при открытии файла MS Excel.

По окончании открытия файла данные будут выведены в таблицу формы. Наименования столбцов в таблице данных формируются автоматически в соответствии с выбранным форматом загрузки. Нумерация строк таблицы и установка ширины каждого столбца с учетом объема данных также производится автоматически. Данные в каждой ячейке таблицы идентичны данным в соответствующем элементе файла MS Excel. Доступна возможность корректировки данных в любой ячейке таблицы и ручное изменение ширины любого столбца.

Примечание 6. Данные в файле должны начинаться с первой строки (не должно быть "шапки"); в файле должен быть один лист с данными.

Если файл открыт успешно, то станет доступна кнопка Преобразовать (см. Преобразование данных). Если был выбран специальный формата дозагрузки (контактных лиц, расчетных лиц, телефонов или адресов) или универсальный формат, преобразование данных осуществляется автоматически. Кнопка Преобразовать остается блокированной, а кнопки Проверить, Сохранить в БД становятся доступными (см. Проверка данных, Загрузка данных в базу данных).

Если количество столбцов в открытом файле MS Excel отличается от количества столбцов, заданного в настройках формата, будет выдано сообщение "Файл <наименование файла> не соответствует выбранному формату <наименование формата>". В этом случае кнопка Открыть файл останется доступной, а кнопка Преобразовать — недоступной.

Проверить крайнюю ячейку в файле можно сочетанием клавиш Ctrl+End.
