Логика проверки доступов

Для проверки доступности возврата дел из архива осуществляется проверка доступов:

·LAW_EXEC_RETURN_FROM_ARCHIVE

·620 LAW_EXEC_RETURN_FROM_ARCHIVE_STATUS_LIST.

Последовательность проверки доступов показана на рисунке ниже.

Информационные сообщения

Если возобновление недоступно, то пользователю выводится сообщение:

У вас нет доступа на выполнение указанной операции.

Если возобновление доступно, то пользователю выводится сообщение:

Вы действительно желаете вернуть выделенные дела в работу?

Если все дела не доступны для возобновления из архива, то пользователю выводится следующее сообщение:

Все исполнительные производства находятся в статусах, недоступных для возврата из архива. Доступных для обработки исполнительных производств нет. Операция прервана.

Если не все дела доступны для возобновления из архива, то после выполнения всех действий пользователю выводится сообщение с указанием количества обработанных дел, статуса, в который они были переведены, и количества возникших ошибок.

Соответственно если все дела были возвращены из архива, сообщение выводить пользователю не требуется.

Порядок действий

Для того чтобы вернуть дела из архива исполнительного производства в работу:

1.Войдите в модуль Юридический департамент, активируйте вкладку Архив ИП.

2.Создайте выборку при помощи мультифильтров или сложного фильтра. Выберите одну или несколько записей.

3.Вызовите контекстное меню нажатием правой кнопки мыши, выберите пункт Возврат из архива.

4.В появившемся окне предупреждения нажмите OK.

5.Откроется форма "Возобновление". Заполните следующие поля:

·Комментарий — введите комментарий; поле не обязательно для заполнения.

6.Нажмите кнопку ОК. Выбранные дела будут возвращены в работу. С делами будут призведены следующие действия:

·если константа UseLawExecDoc имеет значение "Истина", то дела получат статус 13 "Не создано" и будут доступны из вкладки Исполнительные документы;

·если константа UseLawExecDoc имеет значение "Ложь", то дела получат статус 11 "Возобновление" и будут доступны из вкладки Исполнительное производство.

7.По окончании перевода будет выдано сообщение о результатах выполнения операции. Нажмите кнопку ОК, чтобы закрыть его.

8.Действие протоколируется.

9.Затем выполняется формирование настроенных запланированных действий.
