Настройка информационных блоков для вкладки "История работы (полная)"

Существует возможность настройки информационных блоков для отображения на вкладке История работы (полная) в следующих модулях:

·Карточка должника;

·Карточка судебной работы в приказном производстве;

·Карточка судебной работы в исковом производстве;

·Карточка исполнительного производства.

Для этого на форме есть специальный фильтр (см. Вид вкладки "История работы (полная)").

Список информационных блоков задается в словаре № 243 "Блоки информации для отображения истории по долгу/должнику".

Для задания первоначального состояния фильтров формы созданы следующие константы:

·DebtHistoryFullDataListDefaultFilter

·Тип – строка;

·Первоначальное значение – пусто;

·Константа предназначена для отображения блоков данных, выбираемых по умолчанию при открытии формы. Для того чтобы задать нужные блоки, запишите номера этих блоков из словаря № 243, через запятую и без пробелов. Если значение пусто, то отображается вся информация.

·DebtHistoryFullDataLevel

·Тип – число;

·Первоначальное значение 1;

·Константа задает первоначальное значение фильтра "Все записи по должнику/Записи по текущему долгу" во вкладке "История работы (полная)". Возможные значения:

o1 – по должнику;

o2 – по долгу.

·_UseOldDebtHistory

·Тип – логический;

·Первоначальное значение FALSE;

·При значении константы TRUE используется старый вариант истории по долгу (для обратной совместимости); при значении FALSE используется новый функционал.

 Примечание. Если константа _UseOldDebtHistory включена, то информационные блоки не отображаются.

 Пример (Рисунок 1) показывает, какие именно данные отобразились на форме вкладки "История работы (полная)" при значениях констант:

DebtHistoryFullDataListDefaultFilter = 1,2 

DebtHistoryFullDataLevel = 2

 Примечание. Если в выпадающем списке отметить флаги других блоков информации, не перечисленных в значении константы DebtHistoryFullDataListDefaultFilter, то они также отобразятся в таблице. Однако при повторном открытии карточки должника отображаться будут только те блоки, которые перечислены в значении константы DebtHistoryFullDataListDefaultFilter.
