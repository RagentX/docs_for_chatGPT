Формирование письма

Для того чтобы сформировать письмо для отправки на адрес должника, выполните следующие действия:

1.На вкладке Общая перейдите на вложенную вкладку Адреса и выберите нужный адрес в списке.

2.Нажмите кнопку и в раскрывшемся списке выберите пункт Сформировать письмо.

3.В открывшейся форме заполните следующие поля:

·Тип письма — список доступных форм настраивается в модуле "Конструктор писем" (при формировании писем должнику доступны только письма, у которых в поле Получатель указано Должник, при формировании писем поручителю — Поручитель; залогодателю — Залогодатель);

Примечание. Если значение константы DebtCardLetterBankRestriction равно "Истина", то будут доступны только формы, настроенные для банка, которому принадлежит долг, в модуле Настройка ограничений на формирование писем.

·Регистрировать письмо — флаг регистрации письма в системе. Если флаг не установлен, информация о письме сохранена не будет;

·Сохранять копию письма — флаг работает только при включенном флаге Регистрировать письмо. Если этот флаг включен, то копия письма будет сохранена в БД.

Примечание. Если флаг Сохранять копию письма включен, то при открытии письма на вкладке Контакты будет открываться сохраненная копия письма; если флаг выключен, то при открытии письма на вкладке Контакты будет открываться шаблон письма с подставленными в него текущими данными из БД (например, текущая дата, текущий пользователь и т.д.).

·По всем долгам — если флаг установлен, в письмо будут включена информация по всем долгам должника.

4.Нажмите кнопку ОК. Сформированное письмо откроется в приложении Fast Report.
