Фильтрация записей лога

Система позволяет фильтровать записи лога действий оператора по следующим параметрам:

·период времени выполнения действия;

·тип действия;

·оператор, выполнивший действия.

Общий порядок действий для фильтрации записей лога

Для того чтобы отфильтровать записи лога, выполните следующие действия:

1.В форме Лог действий операторов укажите период времени, если требуется.

2.Укажите типы действий, если требуется.

3.Укажите операторов, если требуется.

4.Нажмите кнопку Найти, чтобы применить заданные условия фильтра. В логе будут отображаться только те записи, которые соответствуют всем заданным условиям фильтра.

Выбор периода времени

Для того чтобы задать период времени, записи за который должны отображаться в логе, выполните следующие действия:

1.В форме Лог действий операторов в поле С: нажмите значок . Откроется системный календарь и часы.

2.Укажите дату и время начала периода.

3.В поле По: нажмите значок . Откроется системный календарь и часы.

4.Укажите дату и время окончания периода.

Выбор типов действий

Для того чтобы указать типы действий, записи о выполнении которых должны отображаться в логе, выполните следующие действия:

1.В форме Лог действий операторов в поле Тип действия нажмите значок . Откроется форма для выбора типа действия. В левой части формы находится общий список действий, в правой — список выбранных действий.

Примечание. В общем списке действий доступны функции фильтрации и сортировки. При необходимости их можно использовать, чтобы найти нужные записи.

2.Перенесите записи о нужных типах действия из правого списка в левый. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL. Для переноса записей между списками используйте следующие кнопки:

· — переместить выбранные записи из правого списка в левый;

· — переместить все записи из правого списка левый;

· — переместить выбранные записи из левого списка в правый;

· — переместить все записи из левого списка в правый.

Перенести одну выбранную запись между списками можно с помощью двойного щелчка левой клавишей мыши.

3.Нажмите кнопку ОК. Выбранные типы действий будут добавлены в условия фильтра.

Выбор операторов

Для того чтобы указать операторов, записи о действиях которых должны отображаться в логе, выполните следующие действия:

1.В форме Лог действий операторов в поле Оператор нажмите значок . Откроется форма для выбора операторов. В левой части формы находится общий список операторов, в правой — список выбранных операторов.

Примечание. В общем списке операторов доступны функции фильтрации и сортировки. При необходимости их можно использовать, чтобы найти нужные записи.

2.Перенесите записи о нужных операторах из правого списка в левый. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL. Для переноса записей между списками используйте следующие кнопки:

· — переместить выбранные записи из правого списка в левый;

· — переместить все записи из правого списка левый;

· — переместить выбранные записи из левого списка в правый;

· — переместить все записи из левого списка в правый.

Перенести одну выбранную запись между списками можно с помощью двойного щелчка левой клавишей мыши.

3.Нажмите кнопку ОК. Выбранные операторы будут добавлены в условия фильтра.
