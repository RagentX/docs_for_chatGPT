По продукту и просрочке

Автоматическое распределение долгов по ответственным операторам в зависимости от продукта и количества дней просрочки может быть выполнено:

·для всех долгов из выборки, сформированной с учетом фильтров;

·для всех долгов, выделенных в текущей выборке с помощью клавиш SHIFT и/или CTRL.

Для того чтобы автоматически распределить операторов, выполните следующие действия:

1.В форме Постановка задач перейдите на вкладку Новые должники.

2.При необходимости сформируйте выборку долгов с помощью фильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Автораспределение и в раскрывающемся списке выберите По продукту и просрочке -> По всей выборке или Только выделенные.

4.В открывшемся диалоговом окне нажмите Yes, если все долги одного должника требуется закреплять за одним оператором. Если долги одного должника могут быть закреплены за разными операторами, нажмите No. Система попытается автоматически распределить долги по операторам в соответствии с заданными настройками. При этом:

·В первом варианте распределение будет выполняться по следующему алгоритму: система выберет самый "старый" по дням просрочки долг и в зависимости от настроек закрепит его за определенным оператором. Остальные долги этого должника автоматически будут закреплены за этим же оператором.

·Во втором варианте каждый долг обрабатывается независимо и закрепляется за тем или иным оператором в соответствии с заданными настройками. Если за сегментом закреплено более одного оператора, то долги будут распределены между ними поровну.

По окончании распределения будет выдано сообщение с информацией о количестве должников, которых удалось распределить.

5.Нажмите кнопку ОК, чтобы завершить процедуру.

Примечание. Операторам, на которых распределены долги, и операторам, с которых были сняты долги, будут направлены соответствующие уведомления по внутренней почте системы.

См. далее:

ØНастройка автоматического распределения в зависимости от продукта;

ØМассовая настройка ответственного по нескольким продуктам;

ØРаспределение по типу продукта и просрочке на основании файла.
