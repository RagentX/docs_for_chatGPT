Инструкция по настройке шаблонов для писем

1.Создание шаблона для писем

Для создания нового шаблона для писем необходимо выбрать меню «Файл» -> «Новая страница»

2.Настройка параметров страницы

Необходимо выбрать страницу («Page1»)  и далее в контекстном меню выбрать «Настройки страницы…»

Для формирования писем (обычно всегда) выбирается формат A4, затем расстояния поля с право, с лево, с низу и с верху.

3.Добавления данных

Чтобы добавить данные, необходимо выбрать меню «Отчет» -> «Данные…»

Затем включить данные frxDBRep1 и нажать кнопку OK

4.Добавления компонентов для вывода данных

Главным компонентом для всех писем (и отчетов) является «Данные 1 уровня» (MasterData), он предназначен для вывода данных. Для этого необходимо выбрать раздел «Вставить бэнд» -> «Данные 1 уровня», после этого при открытии окна «Источник данных» необходимо выбрать данные frxDBRep1 и нажать кнопку OK

 

После этого (в этом компоненте) настроить свойства StartNewPage = True и Stretched = True

5.Использование компонента «Объект "Текст"» (TfrxMemoView)

При добавлении компонента открывается «Редактор текста», для ввода необходимого текста

ВАЖНО!!! Данный компонент обязательно должен быть внутри компонента «Данные 1 уровня»

После ввода текста нажмите кнопку OK

Для настройки шрифта, размера, цвет текста и т.д. текста можно выбрать в панели

При выводе поля (из данных) в тексте, очень часто бывает так, что письмо становится длиннее, чем обычно в этом случае необходимо настроить свойство StretchMode = smActualHeight

ВАЖНО!!! Это свойство связано с свойством Stretched = True в компоненте «Данные 1 уровня»

Для использования вывода с красной строки, необходимо указать значение свойства ParagraphGap (например, ParagraphGap = 30)

Если в компоненте «Объект "Текст"» (TfrxMemoView) должн быть текст обычного типа, а так же текст жирным (и/или курсивным, подчеркнутым и т.д.), то для этого необходимо включить свойство AllowHTMLTags = True, и затем в «Редактор текста» указать <b>чтобы текст был жирным</b>, <i>чтобы текст был курсивным</i>, <u>чтобы текст был подчеркнутым</u>, <font color=red>чтобы текст был красным</font>. Цвета можно менять, например, для вывода «Маркеры» и/или «Нумерации» можно, в «Редактор текста» указать

1.<font color=white>_</font>Выдать судебный приказ…

Тогда при формировании символ «_» будет белым цветом (не видным)

1._Выдать судебный приказ…

6.Дополнительный компонент для вывода данных – «Дочерний бэнд»

При просмотре письма текст (компонента «Объект "Текст"» (TfrxMemoView)) иногда сдвигается, чтобы устранить эту проблему необходимо добавить компонент «Вставить бэнд» -> «Дочерний бэнд» (Child1), после этого в уже добавленном компоненте «Данные 1 уровня» (MasterData) необходимо указать связь в свойстве Child = Child1 (Child1 – это имя дочернего бэнда). В Дочернем бэнде (Child1) так же можно указать свойства Stretched = True

Затем в «Дочерний бэнд» (Child1) необходимо добавить компонент «Объект "Текст"» (TfrxMemoView), точно так же, как было указано выше

При добавлении компонента «Вставить бэнд» -> «Дочерний бэнд» (Child2), следует выбрать предыдущий компонент Дочерний бэнд (Child1) и указать свойство Child = Child2, точно так же, как было указано выше
