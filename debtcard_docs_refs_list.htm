Ссылки на файлы документов

Функционал ccылок работает только при включенной константе DocLink.Use.

Добавление и редактирование ссылок доступно пользователям, в роли которых доступ № 220 DEBT_CARD_LINK_FULL имеет значение "Истина".

Если в роли пользователя доступ № 220 DEBT_CARD_LINK_FULL имеет значение "Ложь", то право на добавление и редактирование ссылок определяется следующими доступами:

·Добавление ссылок разрешено пользователям, в роли которых доступ № 831 DEBT_CARD_LINK_ADD имеет значение "Истина".

·Редактирование ссылок разрешено пользователям, в роли которых доступ № 832 DEBT_CARD_LINK_EDIT имеет значение "Истина".

Если в роли пользователя доступ № 824 DEBT_CARD_LINK_ADD_TO_CLOSED имеет значение "Истина", то пользователь может добавить ссылку к долгу, который находится в закрытом статусе 6,7 или 8. Если доступ имеет значение "Ложь", то добавление ссылок к закрытым долгам запрещено.

Удаление ссылок доступно пользователям, в роли которых доступ № 268 DEBT_CARD_LINK_DEL имеет значение "Истина".

Форма добавления/редактирования ссылки на файл

Прикрепление ссылок к долгу или должнику

Прикрепление ссылок на документы к долгу или к должнику определяется значением константы DebtCardAttachLinksToDebt:

·Если значение  равно "Истина",то в форме добавления ссылки по умолчанию выбран объект "Долги" — активный долг.

·Если значение  равно "Ложь", то в форме добавления ссылки по умолчанию выбран объект "Должники" — данный должник.

Добавление ссылки на файл документов

Ссылки на документы добавляются в карточке должника на вкладке Вложения. Для этого выполните следующие действия:

1.Нажмите кнопку , относящуюся к списку ссылок на документы. Откроется форма "Ссылка на документ".

2.В форме заполните следующие поля:

·Название — название документа, поле обязательно для заполнения. См. Примечание ниже.

·Прикрепить к — выберите объект, к которому будет прикреплена ссылка. Данное поле отображается, если включен ф-л прикрепления файлов к объектам Системы. Подробнее — см. Логика прикрепления вложений и ссылок к объектам Системы.

·Тип документа — значение выбирается из словаря № 99. Если значение константы DebtCardUseDocAttachMandatoryType равно TRUE, то поле обязательно для заполнения.

·Ссылка на файл — для заполнения поля нажмите кнопку , в открывшемся диалоговом окне выберите файл и нажмите кнопку Open (Открыть).

·Техническое поле типа "Словарь" 1 — дополнительное поле типа "Словарь". Данное поле отображается по согласованию с технической поддержкой, так же, как и словарь, который соответствует данному полю. Наименование поля также согласовывается с поддержкой.

·Техническое поле типа "Дата" 1 — дополнительное поле типа "Дата". Данное поле отображается по согласованию с технической поддержкой. Наименование поля также согласовывается с поддержкой.

3.Нажмите кнопку ОК. Ссылка на документ будет добавлена в список.

Примечание. Названия для прикрепляемых ссылок.

1.Если в Системе настроено автоматическое именование ссылок, то поле Название не доступно для редактирования. Подробнее см. Логика прикрепления вложений и ссылок к объектам Системы, раздел Правило именования ссылок.

2.Если автоматическое именование ссылок не настроено, то:

·Если при нажатии на кнопку "Добавить документ" поле Название пусто, то в него автоматически вводится название файла.

·Если при нажатии на кнопку "Добавить документ" поле Название заполнено, то введенное название сохраняется.

Примечание. Значение атрибута Стадия при прикреплении документа или ссылки на документ заполняется автоматически терминами словаря № 132 Стадия взыскания в зависимости от того, в каком модуле было осуществлено прикрепление документа:

·Карточка должника — термин № 1 "Досудебная";

·Юридический департамент, вкладки судебной работы — термин № 2 "Судебная работа";

·Карточка судебной работы —  термин № 2 "Судебная работа";

·Карточка исполнительного производства — термин № 3 "Исполнительное производство".

Просмотр и редактирование файла документа по ссылке

1.В карточке должника на вкладке Вложения в списке ссылок на файлы документов выполните двойной щелчок левой клавишей мыши по нужной записи. Откроется файл документа.

2.При необходимости отредактируйте файл документа, сохраните и закройте его средствами соответствующего приложения.

Редактирование ссылки на файл документа

1.В карточке должника на вкладке Вложения выберите нужную запись в списке ссылок на документы и нажмите относящуюся к нему кнопку .

Примечание: двойной щелчок мыши по выбранной записи приведет к открытию файла документа.

2.В форме редактирования атрибутов ссылки на документ измените необходимые данные.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление ссылки на файл документа

1.В карточке должника на вкладке Вложения выберите нужную запись в списке ссылок на документы.

2.Нажмите кнопку , относящуюся к списку ссылок.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.

Ссылки на папки с документами

Для того чтобы добавить ссылку на папку с файлами, нажмите кнопку Выбрать папку. Откроется стандартное окно операционной системы для выбора папки.
