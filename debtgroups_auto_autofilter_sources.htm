Настройка источников данных

Show/Hide Hidden Text

Источником данных для автофильтров может быть любая таблица или представление базы данных.

Единственное условие состоит в том, что в представлении должен присутствовать столбец, указывающий на колонку debt.id - идентификатор для выборки данных. В качестве debt.id может быть выбрана любая колонка из набора данных.

Для доступа к источникам данных нажмите кнопку Источники в форме Группы долгов на вкладке Автогруппы.

Откроется форма для работы с источниками данных. Она состоит из двух окон. В левой части формы расположено окно для работы со списком источников, а в правой части – окно для работы с полями выбранного источника.

Форма Источники данных

Источники данных хранятся в таблице AUTO_FILTER_SRC. Для задания текста заголовков при настройке условий фильтров используется таблица AUTO_FILTER_SRC_FIELDS. Заголовки могут быть загружены из описаний колонок представлений из служебных таблиц базы данных.

       Добавление источника

Для добавления источника:

1.Войдите в форму Источники данных.

2.Нажмите кнопку , относящуюся к списку источников данных, или произведите импорт источника данных.

3.Откроется форма для добавления источника данных. Заполните следующие поля:

·Название — вводится название существующей таблицы или представления в базе данных. Если ввести название несуществующей таблицы – в дальнейшем невозможно будет загрузить ее поля. Поле обязательно для заполнения.

·Объект группы — выберите объект группы. Поле обязательно для заполнения. Настройка списка объектов групп: в технической константе GroupDebtObjectForGroup перечислить идентификаторы соответствующих терминов словаря № 117.

·Примечание — в свободной форме.

4.Нажмите OK. Окно закроется. В форме Источники данных появится новая строка в списке добавленных источников. Строка будет активна. В правой части окна список полей будет пуст.

5.Переходите к загрузке и редактированию полей источника.

       Настройка полей источника

Для настройки полей источника:

1.Войдите в форму Источники данных и выберите источник.

2.Нажмите кнопку Загрузить все поля. Если список полей указанного источника не пуст, появится окно с предложение очистить предыдущие поля. Вы можете либо очистить, либо не очищать поля.

3.После этого вы можете настроить список полей с помощью кнопок удаления и редактирования, относящихся к списку полей.

4.Для редактирования поля: выберите нужное поле и нажмите кнопку , относящуюся к списку полей. Откроется форма редактирования поля, в которой вы можете изменить все параметры поля, назначить поле специальным полем debt.id - идентификатором для выборки данных, а также использовать SQL-запрос для указанного поля. По окончании редактирования поля нажмите OK.

5.Для удаления одного или нескольких полей выделите их с помощью мыши, затем нажмите кнопку , относящуюся к списку полей в форме Источники данных. Подтвердите удаление.

6.После настройки полей источника есть возможность переименовать его. Вы можете выбрать название другой таблицы базы данных и добавить некоторые ее поля в ваш источник, повторив действия с п. 2 (без очистки предыдущих полей). Затем вы можете задать источнику любое название и произвести его экспорт.

       Редактирование источника

Для редактирования источника данных:

1.Войдите в форму Источники данных и выберите источник.
2.Нажмите кнопку , относящуюся к списку источников. Откроется форма редактирования источника.
3.Измените значения ее полей и нажмите OK.

       Удаление источника

Для удаления источника данных:

1.Войдите в форму Источники данных и выберите источник.
2.Нажмите кнопку , относящуюся к списку источников.
3.Введите подтверждение.

       Экспорт источника данных

1.Войдите в форму Источники данных.

2.Выберите источник с помощью курсора мыши.

3.Настройте поля источника.

4.Нажмите кнопку Экспорт в XML.

5.Откроется стандартное окно сохранения файла. Выберите путь и задайте имя для сохраняемого файла и нажмите Сохранить (Save).

       Импорт источника данных

1.Войдите в форму Источники данных.

2.Нажмите кнопку Импорт из XML.

3.Откроется стандартное окно выбора файла. Выберите файл источника и нажмите Открыть (Open).
