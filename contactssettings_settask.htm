Автоматическая постановка задач

Автоматическая постановка задач по результату контакта включает в себя как настройку задач, так и настройку групп задач.

Группа задач может включать одну или более задач. Группа задач имеет множественный или одиночный выбор задач, входящих в нее. Кроме того, выбор задач может быть как обязательным, так и необязательным. Подробное описание см. Описание формы добавления/редактирования группы задач (ниже). Это позволяет более гибко настраивать список задач, добавляемых долгу по результату контакта. После регистрации контакта для пользователя открывается форма выбора задач, если настроена возможность выбора. В противном случае задачи добавляются автоматически без уведомления.

Порядок действий

1.Настройте задачи в области данных Задачи.

2.Настройте группы. Укажите признаки обязательности и единичного/множественного выбора, видимость группы.

3.Добавьте задачи в группы.

Как будут добавляться задачи при регистрации контакта такой настройки — см. Выбор автоматических задач.

Описание формы "Настройка задач"

Форма Настройка задач включает следующие элементы:

·Область данных Задачи, предназначенная для настройки задач и включающая:

·Флаг Актуализация состояния задач по факту регистрации — если флаг включен, то при регистрации контакта запускается функция проверки выполнения задачи (например, стояла задача дозвониться, а результатом контакта как раз и стал звонок — задача завершается);

·Список задач, добавленных по результату контакта;

·Описание выбранной задачи;

·Кнопка Добавить автоматическую задачу. Откроется форма "Задача" для добавления задачи;

·Кнопка Изменить автоматическую задачу. Откроется форма "Задача" для редактирования выбранной задачи;

·Флаг "Показывать только активные задачи" — если флаг включен, то показываются только те задачи, при создании которых был включен флаг "Включено". Если флаг выключен, то показываются все задачи;

·Область данных "Группы", предназначенная для настройки групп задач и включающая:

·Кнопка Установить связь для добавления задачи в группу;

·Список созданных групп задач;

·Кнопки для работы с группами задач:

o Добавить группу — для создания новой группы задач;

o Изменить группу — для изменения атрибутов группы задач;

o Удалить группу — для удаления группы задач;

·Список задач, входящих в группу.

Описание формы "Задача"

Атрибуты формы "Задача"

·Название задачи — обязательное поле, заполняется при создании задачи;

·Описание — необязательное поле, заполняется при создании задачи;

·Тип задачи — обязательное поле, заполняется при создании задачи. Значения задаются в словаре № 229;

·Подтип задачи — обязательное поле, заполняется при создании задачи. Значения задаются в словаре № 230;

·Условие автоматического завершения задачи — извините, этот функционал находится в разработке;

·Приоритет задачи — обязательное поле, заполняется при создании задачи. Значения задаются в словаре № 235;

·Крайняя дата завершения (в днях)  — указывается количество дней;

Примечание. При создании задачи выполняется проверка, настроен ли для задачи данного типа и подтипа крайний срок в модуле Словари на Вкладке "Настройка крайнего срока для задачи". Если такой срок настроен, и в форме создания задачи крайняя дата завершения не указана, то для такой задачи крайняя дата завершения устанавливается как дата создания задачи плюс количество дней, указанное для задач данного типа и подтипа в модуле Словари на Вкладке "Настройка крайнего срока для задачи".

Важно отметить, что если при создании задачи вы укажете крайнюю дату завершения, то будет сохранена эта дата.

 

·Флаг Проверять наличие задачи. Если включен, то для текущего долга (дела) будет проверяться наличие задачи такого же типа. Это сделано для защиты от дублирования задач;

·Флаг Прибавлять только рабочие дни — Если флаг включен, то в тот момент, когда задача будет поставлена, для установки крайней даты ее завершения будет выбрана дата, равная текущей плюс количество рабочих дней, указанное в поле Крайняя дата завершения (в днях), а если флаг выключен, то к текущей дате будет прибавляться указанное количество календарных дней;

·Флаг Включено. Если его выключить, задача не будет ставиться. Выключение флага применяется в тех случаях, когда требуется удалить задачу.

Описание формы добавления/редактирования группы задач

Атрибуты группы задач

Группа задач имеет следующие параметры:

·Название группы — наименование группы, обязательное поле;

·Описание — дополнительная информация, необязательное поле;

·Признак обязательности выбора — если флаг включен, то при регистрации контакта с данным исходом выбрать хотя бы одну задачу в группе обязательно, если флаг выключен, то можно не выбирать ни одной задачи в группе;

·Множественный выбор — если флаг включен, то при регистрации контакта с данным исходом можно выбрать более одной задачи в группе, если флаг выключен, можно выбрать только одну задачу в группе;

·Не показывать группу — если флаг включен, то при регистрации контакта с данным исходом группа не отображается, и задачи из нее не выбираются.

Настройка задачи

Добавление задачи

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.Нажмите кнопку Добавить автоматическую задачу, относящуюся к области данных "Задачи". Откроется форма "Задача" добавления автоматической задачи.

4.Заполните поля и нажмите OK. Если поля заполнены правильно, форма "Задача" закроется, и вы вернетесь в форму "Результат контакта".

5.Созданная задача появится в списке задач.

Редактирование задачи

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.В списке задач в области данных "Задачи" выберите задачу.

4.Нажмите кнопку Изменить автоматическую задачу, относящуюся к области данных "Задачи". Откроется форма "Задача" редактирования автоматической задачи.

5.Измените нужные поля и нажмите OK. Если поля заполнены правильно, форма "Задача" закроется, и вы вернетесь в форму "Результат контакта".

6.Задача будет изменена.

Настройка группы задач

Добавление группы задач

Для того чтобы добавить новую группу задач, выполните следующие действия:

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.В области данных "Группы" нажмите кнопку Добавить группу. Откроется форма добавления группы задач.

4.Заполните поля и нажмите OK.

5.Новая группа будет добавлена в список групп.

Редактирование группы задач

Для редактирования группы задач выполните следующее:

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.В списке групп задач выберите нужную группу.

4.В области данных "Группы" нажмите кнопку Изменить группу. Откроется форма редактирования группы задач

5.Измените нужные поля и нажмите OK.

6.Группа будет изменена.

Удаление группы задач

Для того чтобы уделить группу задач, выполните следующее:

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.В списке групп задач выберите нужную группу.

4.В области данных "Группы" нажмите кнопку Удалить группу.

5.Группа будет удалена без предупреждения. Если в группу входили задачи, их связь с группой будет разорвана.

Добавление задачи в группу

Для того чтобы добавить задачу в группу, выполните следующее:

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.В списке групп задач выберите нужную группу.

4.В списке задач в области данных "Задачи" выберите нужную задачу.

5.Нажмите кнопку Установить связь.

6.Задача будет добавлена, если она не была ранее добавлена ни в одну группу.

Удаление задачи из группы

Для того чтобы удалить задачу из группы

1.В модуле Настройка дерева контактов выберите тип дерева контактов, тип контакта, исход контакта.

2.В форме "Результат контакта" активируйте вкладку Настройка задач.

3.В списке "Задачи в группе", выберите задачу, предназначенную для удаления.

4.Когда задача выбрана, на месте кнопок для действий с группами отобразится кнопка для действия "Удалить связь". Нажмите на нее.

5.Задача пропадет из группы (список "Задачи в группе"), но не пропадет из списка задач, относящихся к текущему исходу контакта (область данных "Задачи").

Пример

На приведенном примере для исхода контакта создано шесть задач и три группы.

·Группа задач "Взаимодействие с должником" включает две задачи, имеет признак обязательности выбора и не имеет признака множественности.

·Группа "Дополнительно" включает две задачи, не имеет признака обязательности и не имеет признака множественности.

·Группа "Судебная стадия" включает четыре задачи, не имеет признака обязательности и имеет признак множественности.

·Задача № 260 "Обязательная задача" не включена ни в одну группу.

Как будут отображаться и обрабатываться задачи — см. Выбор автоматических задач — Пример.
