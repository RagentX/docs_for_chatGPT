Удаление вложения

Доступность действия определяется доступами № 684 DOCFLOW_CARD_DOC_DELETE.

Удалить можно только текущую версию документа. При этом для нее отключается флажок Текущий, но физически файл документа из системы не удаляется. Такие документы не доступны для редактирования, но доступны для просмотра. Чтобы в списке документов можно было видеть удаленные версии, включите флаг Показывать удаленные вложения.

При выполнении действия будет разорвана связь между данным файлом и текущим документом. Кроме того, будет разорвана связь между данным файлом и всеми объектами, связанными с текущим документом.

Для того чтобы удалить вложение, выполните следующее

1.На вкладке Вложения выберите одну или более записей. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL,

2.Нажмите кнопку Удалить, расположенную под списком вложений.

3.Откроется окно подтверждения с текстом:

Вы действительно хотите удалить вложения?

Нажмите Yes.
