Групповое изменение атрибута

Система позволяет выполнить групповое изменение значений атрибутов для записей, отображающихся в списке, в том числе:

·Тип должника — см. словарь № 54;

·Название продукта — продукт, по которому возник долг;

·Портфель (пункт виден только если доступ № 536 DEBT_CHANGE_PORTFOLIO включен) ;

·Филиал;

·Регион;

·Часовой пояс;

·Важность;

·Порядочность;

·Стадия (пункт виден только при включенном доступе DEBT_STAGE; при смене стадии может осуществляться перевод долгов в соответствующий портфель);

·Тип продукта — см. словарь № 11;

·Стратегия;

·Балл № 1 — список значений определяется словарем № 101 и доступом № 557 DEBT_MARK1_SET_LIST;

·Балл № 2 — список значений определяется словарем № 102 и доступом № 558 DEBT_MARK2_SET_LIST;

·Балл № 3 — список значений определяется словарем № 244 и доступами № 492 DEBT_MARK3_SET № 559 DEBT_MARK3_SET_LIST;

·Балл № 4 — список значений определяется словарем № 245 и доступами № 493 DEBT_MARK4_SET № 560 DEBT_MARK4_SET_LIST;

Примечание. Подробнее — см. Скоринговые баллы, соответствия скоринговых баллов.

·Вознаграждение (%) —  доступно при наличии у пользователя доступа AGENCY_RATE_EDIT;

·Дата пролонгации.

Порядок действий для группового изменения атрибута

Для того чтобы изменить значение атрибута, выполните следующие действия:

1.В нужной вкладке формы при помощи фильтров сформируйте выборку записей, в которых требуется изменить значение атрибута.

2.В контекстном меню (вызывается правой кнопкой мыши) выберите пункт Установить значение (столбца) -> <Название атрибута>. Откроется окно для ввода нового значения атрибута.

3.Введите новое значение атрибута и нажмите кнопку ОК. Это значение атрибута будет установлено во всех записях, входящих в выборку.

Массовое изменение атрибутов протоколируется.
