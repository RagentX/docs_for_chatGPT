Ведение списка адресов электронной почты должника

Атрибуты адреса электронной почты

Доступ № 529 DEBT_CARD_EMAIL_BLOCK = TRUE

   

Доступ № 529 DEBT_CARD_EMAIL_BLOCK = FALSE

Примечание. Адреса электронной почты поручителей и залогодателей не отображаются в списке адресов электронной почты должника на вкладке Общая на вложенной вкладке Email. Адреса электронной почты поручителя отображаются в аналогичной области на вкладке Поручители, а адреса электронной почты залогодателя отображаются в аналогичной области на вкладке Имущество.

Добавление адреса электронной почты

Добавление адреса электронной почты доступно пользователям, в роли которых доступ № 644 DEBT_CARD_EMAIL_ADD имеет значение TRUE.

Для того чтобы добавить в список адрес электронной почты, выполните следующие действия:

1.В форме со списком адресов электронной почты нажмите кнопку .

2.В открывшейся форме заполните следующие поля:

·Адрес — адрес электронной почты;

·Комментарий — дополнительная информация;

·Флаг Верифицирован — признак верификации адреса электронной почты, см. Верификация контактных данных;

·Флаг Заблокирован — признак блокировки адреса электронной почты. Флаг доступен только при включенном доступе № 529 DEBT_CARD_EMAIL_BLOCK. При добавлении нового адреса электронной почты значение флага по умолчанию — "Выключено".

3.Нажмите кнопку ОК, чтобы сохранить данные.

Редактирование адреса электронной почты

Редактирование адреса электронной почты доступно пользователям, в роли которых доступ № 645 DEBT_CARD_EMAIL_EDIT имеет значение TRUE.

Для того чтобы отредактировать адрес электронной почты, выполните следующие действия:

1. В списке адресов электронной почты выберите нужную запись и нажмите кнопку .

2. В форме редактирования атрибутов адреса электронной почты измените необходимые данные.

3. Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление адреса электронной почты

Удаление адреса электронной почты доступно пользователям, в роли которых доступ № 646 DEBT_CARD_EMAIL_DEL имеет значение TRUE.

Для того чтобы удалить адрес электронной почты, выполните следующие действия:

1.В списке адресов электронной почты выберите нужную запись.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.

Блокировка адреса электронной почты

Блокирование адреса электронной почты доступно пользователям, в роли которых доступ № 529 DEBT_CARD_EMAIL_BLOCK имеет значение TRUE.

Для того чтобы заблокировать адрес электронной почты, выполните следующие действия:

1.В списке адресов электронной почты выберите нужную запись.

2.В открывшейся форме переведите флаг "Заблокирован" в положение "Включено".

3.Нажмите кнопку ОК, чтобы сохранить данные.
