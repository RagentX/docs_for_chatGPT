Ограничения доступа к данным на основании структуры предприятия

Возможно настроить для каждого пользователя нужный вариант доступа к объектам Системы  (долгам, платежам или делам) в следующих модулях:

·Список должников;

·Протокол контактов;

·Постановка задач;

·Исходящее информирование;

·Платежи;

·Юридический департамент;

·Уголовное преследование.

Доступ только к своим долгам

Для того чтобы оператор мог работать в только с объектами, закрепленными за ним, необходимо, чтобы в соответствующем отделе у него была должность Сотрудник.

При этом в мультифильтре Операторы у него будет доступ только к одному оператору — самому себе.

Список должников

123 DEBT_LIST_FULL

Сотрудник

•  Доступ только к долгам, которые закреплены за данным пользователем.

Куратор

• Доступ к долгам, которые закреплены за данным пользователем.

• Доступ к долгам, которые закреплены за пользователями, включенными в подразделение, в котором пользователь является куратором.

• Доступ к долгам, которые не закреплены за оператором, непустой филиал которых совпадает с непустым филиалом подразделения, в котором пользователь является куратором.

Руководитель/Заместитель

• Доступ к долгам, которые закреплены за данным пользователем или его подчиненными.

• Доступ к долгам, которые закреплены за пользователями, включенными в дочерние подразделения.

• Доступ к долгам, которые не закреплены за оператором, непустой филиал которых совпадает с непустым филиалом подразделения пользователя или одного из дочерних подразделений.

Примечания.

·К данным параллельных отделов у руководителей и их заместителей доступа нет.

·В мультифильтре Операторы для пользователя будут доступны только разрешенные операторы.

·Если у пользователя включен доступ ко всем долгам для модуля, то в этом модуле в мультифильтре Операторы для него доступны все операторы.

·Если филиал долга пуст, то доступ к таким долгам есть только у пользователя, в роли которого включен соответствующий доступ ко всем долгам.

·Если филиал подразделения пуст, значит для такого подразделения нет долгов, филиал которых совпадает с ним.

·Если оператор не включен в структуру предприятия, то он имеет доступ только к своим долгам/делам/контактам/платежам.
