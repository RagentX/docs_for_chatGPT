Отчет по должникам

Отчет "Отчет по должникам" выводит агрегированные данные о портфеле за период в разрезе должников в Excel.

Отчет "Отчет по должникам" доступен только пользователям, в роли которых доступ REP_DEBT имеет значение "Истина".

Порядок действий

Для того чтобы сформировать отчет, выполните следующее:

1.Выберите в главном меню пункт Отчеты, затем выберите отчет "Отчет по должникам".

2.Откроется форма "Отчет по должникам". Заполните следующие поля:

·С: — выберите начала периода (по умолчанию в поле указано первое число текущего мес яца).

·По: — выберите окончания периода (по умолчанию в поле указано последнее число текущего месяца).

·Портфели — откроется форма для множественного выбора портфелей.

·Подразделение — выберите подразделение (Soft, Hard или Legal) или пункт "Все". при выборе подразделения в отчет можно выводить только долги должников относящиеся к выбранному подразделению

·Оператор — выберите оператора из списка или пункт "Все операторы".

·Флаг "Не показывать закрытых" и поле "С:" (для выбора даты, доступно толко при включенном флаге) — если флаг включен, и дата выбрана, то в отчет попадут только долги, которые не были закрыты в указанную дату.

·Флаг "Сортировка по ФИО" — в отчет выводятся отсортированные данные.

3.Нажмите кнопку "Запустить". Начнется формирование отчета. В процессе будет отображаться индикатор выполнения. Затем откроется стандартное окно операционной системы для указания расположения и названия файла с последующим сохранением.

4.Сохраните полученный файл. По окончании сохранения файла будет показано соответствующее системное сообщение.

Логика формирования отчета

Для каждого должника, удовлетворяющего условиям отбора по оператору, работающему с ним на данный момент, кредитору и подразделению, отображаются все контакты, платежи и обещания, попавшие в период отчета.

Примечание: должники в статусе "Аннулировано" не попадают в отчет, других ограничений на статус нет.

Далее в таблице отображается количество контактов по каждому возможному результату контакта, то есть в таблице добавлено столько полей, сколько возможных результатов контактов (узлов дерева контактов) настроено в Системе (см. модуль Настройка дерева контактов).

Затем в таблице отображаются значения произвольных атрибутов, то есть в таблицу добавлено столько полей, сколько произвольных атрибутов настроено в Системе — все термины словаря № 53.
