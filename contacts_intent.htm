Ввод намерения исполнения обещания или подтверждения оплаты при регистрации контакта

Ввод намерения исполнения обещания

Если в модуле Настройка дерева контактов в поле Ввод подтверждения обещания было выбрано одно из следующих значений:

·Ввод подтверждения намерения;

·Ввод подтверждения намерения. Не обязательно.

то при регистрации контакта с этим исходом на форме регистрации контакта будет отображаться панель ввода намерения исполнения обещания.

Если по долгу, по которому регистрируется контакт, есть активное обещание (такое, дата которого плюс значение константы PromGracePeriod (в днях) больше текущей даты), то поля по умолчанию заполнены данными этого обещания:

·Дата намерения — дата последнего активного обещания, в поле возможно установить значение только большее или равное текущей дате.

·Сумма намерения — сумма последнего активного обещания, в поле возможно установить значение только больше нуля и меньшее или равное значению остатка. При этом, если намерение создано для обещания по персоне, то данное ограничение не действует.

Ввод подтверждения оплаты

Если в модуле Настройка дерева контактов в поле Ввод подтверждения обещания было выбрано одно из следующих значений:

·Ввод подтверждения оплаты;

·Ввод подтверждения оплаты. Не обязательно.

то при регистрации контакта с этим исходом на форме регистрации контакта будет отображаться панель ввода подтверждения оплаты.

Если по долгу, по которому регистрируется контакт, есть активное обещание (такое, дата которого плюс значение константы PromGracePeriod (в днях) больше текущей даты), то поля по умолчанию заполнены данными этого обещания:

·Дата подтверждения оплаты — дата последнего активного обещания, в поле возможно установить значение только большее или равное текущей дате.

·Сумма подтверждения оплаты — сумма последнего активного обещания, в поле возможно установить значение только больше нуля и меньшее или равное значению остатка. При этом, если намерение создано для обещания по персоне, то данное ограничение не действует.

Примечание. Данный функционал включается по согласованию с технической поддержкой (Техническая константа ConfirmIntent.Use).
