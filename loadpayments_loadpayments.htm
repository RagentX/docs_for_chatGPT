Формат "Загрузка платежей"

Формат "Загрузка платежей" предназначен для загрузки платежей и остатков к погашению. Возможно использовать этот формат и для загрузки только платежей либо только остатков к погашению.

Файл MS Excel, содержащий данные о платежах и/или остатках к погашению, должен иметь следующий формат:

·данные в файле должны начинаться с первой строки (не должно быть "шапки");

·в файле должен быть один лист с данными;

·данные в файле должны содержаться в следующем порядке:

1.ФИО — ФИО должника (всегда обязательно для заполнения).

2.Банк — название банка (всегда обязательно для заполнения).

3.Портфель — название портфеля.

4.Номер договора.

5.Номер счета.

6.ID в системе клиента.

7.Дата внесения платежа.

8.Сумма платежа.

9.Валюта.

10.Дата учета информации.

11.Остаток погашения.

12.Дата расчета.

13.Дата актуальности остатка.

14.Комиссия (комиссионные выплаты).

15.Назначение платежа (необязательный столбец).

16.Комментарий — комментарий к строке таблицы платежей (необязательный столбец).

17.№ договора к оплате (необязательный столбец).

18.Заданный id долга (необязательный столбец).

Если вы осуществляете загрузку платежей без загрузки остатков, обязательными для заполнения являются колонки:

·7. Дата внесения платежа.

·8. Сумма платежа.

Если вы осуществляете загрузку остатков без загрузки платежей, то обязательными для заполнения являются колонки:

·11. Остаток погашения.

·12. Дата расчета.

Необязательные колонки (15. Назначение платежа, 16. Комментарий, 17. № договора к оплате и 18. Заданный id долга) могут отсутствовать либо присутствовать в файле MS Excel одновременно. То есть в исходном файле при загрузке платежей должно быть либо 14 колонок (обязательных) либо 18 колонок (14 обязательных и 4 необязательных).
