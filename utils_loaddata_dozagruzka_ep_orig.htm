Особенности загрузки файлов с типом формата "Исполнительное производство"

Особенности загрузки файлов с типом формата "Исполнительное производство"

Доработка логики загрузки

·Если в шаблоне есть поле «ссылка на debt» по таблице «Исполнительное производство», то поля «Фамилия», «Имя», «Отчество» (ФИО), «Исполнительное производство — Общая сумма долга» не обязательно должны присутствовать в шаблоне.

·При этом либо «ФИО» и «Общая сумма долга», либо «ссылка на debt» должны быть заполнены при загрузке файла.

·Если в файле загрузки указана «ссылка на debt»:

·Если долг (debt.id) найден по указанному значению, то создаваемое исполнительное производство прикрепляется к этому долгу.  Если в файле загрузки по таблице Исполнительное производство не указано поле или не передается значение поля «Общая сумма долга» (law_exec.total_sum), то значение копируется из прикрепляемого долга (debt.total_sum = law_exec.total_sum). Данные по долгу и должнику в этом случае обновляются.

·Если в файле загрузки указано значение поля «ссылка на law_act» и по нему найдена запись в law_act принадлежащая тому же самому долгу (law_act.r_debt_id = law_exec.r_debt_id), то в обновляемой записи law_exec заполняется поле law_exec.r_act_id.

·Если в файле загрузки указано значение поля «ссылка на law_act» и по нему НЕ найдена запись в law_act принадлежащая тому же самому долгу (law_act.r_debt_id = law_exec.r_debt_id), то значение law_exec.r_act_id не обновляется. При проверке поле отмечается желтым цветом.

·Если поле «ссылка на law_act» не указано или не заполнено, дополнительных действий не требуется.

·Если долг не найден по указанному значению, то при проверке поле должно отмечаться желтым цветом. Запись не должна загружаться.

·Если в файле не указана «ссылка на debt», то проверяется наличие и заполнение полей «ФИО». В этом случае при загрузке по заполненным данным добавляется новый должник и к нему прикрепляется исполнительное производство.

·При этом прикрепление к судебному делу по полю «ссылка на law_act» не происходит.
