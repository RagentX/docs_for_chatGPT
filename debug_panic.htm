Снятие паники модулей

Если в при определенных условиях неверно отрабатывается какой-либо функционал, причем вы можете воспроизвести последовательность действий, приводящих к ошибке, то вы можете создать специальный лог-файл, который зафиксирует все действия модуля, с тем чтобы разработчик смог увидеть, что именно привело к возникновению ошибки или неверной работе. Для этого используется процедура под названием "снятие паники модуля".

Для снятия паники модуля выполните следующие действия

1.Откройте модуль, в котором происходит ошибка.

2.Нажмите комбинацию клавиш "Ctrl" + "Shift" + "Alt" + "F11"

3.В открывшемся окне выберите

·Уровень отладки — "Паника";

·Для сохранения выберите текстовый формат "txt";

·Нажмите OK.

4.Повторите действия, приводящие к ошибке.

5.Полученный файл отправьте разработчикам.

Примечание. Если в ходе работы из одного модуля вызывается другой (например, из модуля Протокол контактов вызывается Карточка должника), то вызванный модуль записывает свои действия в тот же файл.
