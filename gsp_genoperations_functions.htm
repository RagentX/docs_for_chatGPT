Дополнительные операции

По кнопке Дополнительные операции доступны следующие пункты меню (если в модуле Группы должников настроены соответствующие доступы):

·Только выделенные — если флаг включен, то выбранная операция применяется только к выделенным записям списка, а если флаг включен, то операция применяется ко всей выборке;

·Операции по долгу — активен, когда в таблице на вкладке присутствует идентификатор долга;

·Операции по должнику — активен, когда в таблице на вкладке присутствует идентификатор должника;

·Операции по имуществу — активен, когда в таблице на вкладке присутствует идентификатор имущества;

·Операции по Исполнительному производству — активен, когда в таблице на вкладке присутствует идентификатор ИП;

·Операции по судебному производству — активен, когда в таблице на вкладке присутствует идентификатор судебного дела;

·Процедуры — операция, которая запускается с параметрами, не привязанная к записям списка.

Доступ для пользователей к дополнительным операциям и процедурам настраивается в модуле Роли и доступы. Пункты меню доступны только для тех пользователей, в роли которых есть доступ хотя бы к одной операции или процедуре соответственно.

 

Примечание. Для дополнительных операций по долгу и по должнику добавлена возможность использовать в качестве входного параметра временную таблицу, содержащую список ID. Таким образом, дополнительная операция будет выполняться один раз для всего списка объектов, переданных в качестве входного параметра.
