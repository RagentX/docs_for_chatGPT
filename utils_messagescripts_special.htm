Специальное

Вкладка Специальное предназначена для:

·Создания информационных блоков, отображающихся в модулях:

·Карточка должника — Основные сведения о должнике — Краткая информация;

·Карточка судебной работы в приказном производстве — Вкладка "Приказное производство" — Доп. информация;

·Карточка судебной работы в правопреемстве — Вкладка "Правопреемство" — Доп. информация;

·Карточка судебной работы в исковом производстве — Вкладка "Исковое производство" — Доп. информация;

·Карточка судебной работы в банкротстве — Вкладка "Банкротство" — блок "Краткая информация";

·Карточка исполнительного производства — Вкладка "Общая информация" — Доп. информация.

Назначение конкретного блока для каждого из модулей осуществляется с помощью константы DebtMainInfoFormat.

Записи на этой вкладке имеют тип сценария сообщений — Специальное (см. Словарь № 64 Тип сценария сообщения).

Вкладка "Специальное"

Описание вкладки "Специальное"

·список сценариев информационных блоков, содержащий следующие поля:

·ID — идентификатор сценария, формируется автоматически;

·Номер — номер сценария;

·Название — название сценария;

·Текст — текст сценария;

·кнопки для добавления, редактирования, удаления сценариев.

Форма добавления/редактирования информационного блока

Описание формы добавления/редактирования информационного блока

·Название — название сценария. Поле обязательно для заполнения;

·Номер — номер сценария, который может использоваться для привязки сценария к модулю (см. описание константы DebtMainInfoFormat.);

·Текст — текст информационного блока. Поле может содержать как статичный текст, так и переменные, вместо которых при выполнении сценария будут подставлены конкретные значения (см. Подстановка переменных в текст сообщения).

В нижней части формы расположены кнопки управления внешним представлением информационного блока. Например, для того чтобы изменить цвет определенного фрагмента текста, выделите его в поле Текст, выберите цвет в раскрывающемся списке, а затем нажмите кнопку . Так же можно изменить тип шрифта (жирный — кнопка , курсив — кнопка , подчеркнутый — кнопка ).

Действия с информационными блоками

Создание информационного блока

Для того чтобы создать информационный блок, выполните следующее:

1.В форме Сценарии сообщений для должников на вкладке Специальное нажмите кнопку .

2.В открывшейся форме добавления сценария разговора заполните нужные поля настройте внешнее представление сценария.

3.Нажмите кнопку ОК. Сценарий будет добавлен в список.

Редактирование информационного блока

Для того чтобы отредактировать информационный блок, выполните следующее:

1.В форме Сценарии сообщений для должников на вкладке Специальное выберите нужную запись и нажмите кнопку или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.В форме редактирования атрибутов сценария измените необходимые атрибуты.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление информационного блока

Для того чтобы удалить информационный блок, выполните следующее:

1.В форме Сценарии сообщений для должников на вкладке Специальное выберите нужную запись.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.
