Вкладка "Жалобы"

В карточке исполнительного производства на вкладке Жалобы отображается список жалоб. Вкладка содержит следующие элементы:

·Область данных Жалобы, которая включает:

·список жалоб со следующими полями:

§№ — номер жалобы;

§Дата подачи;

§Фактич. дата жалобы — фактическая дата жалобы;

§Вид жалобы — см. словарь № 30;

§Тип жалобы — см. словарь № 31;

§Статус — см. словарь № 32;

§Дата рассмотрения — дата принятия решения об удовлетворении или отказе в удовлетворении жалобы;

§Описание;

§Результат;

·кнопки для работы со списком жалоб;

·Область данных Документы, которая включает список документов, относящихся к выбранной жалобе и кнопки для их добавления и удаления.

Примечание. По согласованию с технической поддержкой доступно переименование поля "Фактич. дата жалобы".

Стринг:

·Фактич. дата жалобы — law_exec_appel.appel_dt.
