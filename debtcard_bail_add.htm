Добавление, редактирование и удаление данных о поручителе

Форма Поручитель, которая открывается для добавления или редактирования записи о поручителе, имеет одинаковый вид и функционал при вызове из всех карточек:

·Карточка должника;

·Карточка судебной работы;

·Карточка исполнительного производства.

Описание формы — см. Форма "Поручитель".

Добавление данных о поручителе

Право на добавление поручителя имеют пользователи, в роли которых доступ № 192 DEBT_CARD_GUARANTOR_ADD имеет значение TRUE.

Для того чтобы добавить информацию о поручителе в список, выполните следующие действия:

1.В карточке должника на вкладке Поручители нажмите кнопку . Откроется форма "Поручитель".

2.В форме заполните нужные поля.

3.Нажмите кнопку ОК, чтобы сохранить данные. Информация о поручителе будет добавлена в список.

Редактирование данных о поручителе

Право на редактирование данных о поручителе (кроме комментария) имеют пользователи, в роли которых доступ № 193 DEBT_CARD_GUARANTOR_EDIT имеет значение TRUE.

Право на редактирование комментария имеют пользователи, в роли которых доступ № 643 DEBT_CARD_GUARANTOR_DSC_EDIT имеет значение TRUE.

Если оба указанных доступа у пользователя отсутствуют, карточка поручителя открывается только на просмотр.

Для того чтобы отредактировать данные о поручителе, выполните следующие действия:

1.В карточке должника на вкладке Поручители выберите нужную запись и нажмите кнопку или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.В форме "Поручитель" измените необходимые атрибуты.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Удаление данных о поручителе

Право на удаление поручителя имеют пользователи, в роли которых доступ № 286 DEBT_CARD_GUARANTOR_DELETE имеет значение TRUE.

Для того чтобы удалить данные о поручителе, выполните следующие действия:

1.В карточке должника на вкладке Поручители выберите нужную запись.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить удаление.
