       Добавление сотрудника в подразделение

Перед тем как включить сотрудника в то или иное подразделение предприятия, необходимо добавить его в список пользователей системы (см. Создание записи о пользователе). Каждый сотрудник может входить только в одно подразделение.

Для того чтобы добавить сотрудника в подразделение, выполните следующие действия:

1.В иерархической структуре предприятия выберите подразделение, в состав которого требуется включить сотрудника.

2.Нажмите кнопку Добавить, относящуюся к списку сотрудников. Откроется форма со списком сотрудников, которые не входят ни в одно подразделение предприятия. Если требуется отобразить в списке заблокированных сотрудников, установите флаг Показывать блокированных пользователей.

3.В форме выберите сотрудника.

4.Нажмите кнопку Добавить в подразделение, чтобы включить сотрудника в выбранное подразделение.

       Информация о сотруднике и его статус

Для того чтобы просмотреть данные о сотруднике подразделения и при необходимости изменить его статус, выполните следующие действия:

1.В иерархической структуре предприятия выберите нужное подразделение.

2.В списке сотрудников выберите нужного сотрудника.

3.Нажмите относящуюся к списку сотрудников кнопку Детали. Откроется форма просмотра данных о выбранном сотруднике.

4.При необходимости измените статус сотрудника, установив соответствующий переключатель (Руководитель/Заместитель/Сотрудник).

5.Нажмите кнопку ОК, чтобы закрыть форму.

       Перемещение сотрудника между подразделениями

1.В иерархической структуре предприятия выберите нужное подразделение.

2.В списке сотрудников выберите нужного сотрудника.

3.Нажмите относящуюся к списку сотрудников кнопку Переместить. Откроется форма перемещения сотрудника в организационной структуре.

4.В левой части формы выберите подразделение, в которое требуется переместить сотрудника.

5.Нажмите кнопку ОК, чтобы закрыть форму. Сотрудник будет перенесен в выбранное подразделение.

       Исключение сотрудника из подразделения

1.В иерархической структуре предприятия выберите подразделение, в состав которого входит сотрудник.

2.В списке сотрудников подразделения выберите нужного сотрудника.

3.Нажмите кнопку Удалить, относящуюся к списку сотрудников.

4.В открывшемся диалоговом окне нажмите Yes, чтобы подтвердить удаление. Сотрудник будет исключен из состава выбранного подразделения и попадет в список сотрудников, не включенных ни в одно подразделение.
