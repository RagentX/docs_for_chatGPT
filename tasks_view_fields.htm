Просмотр данных о должниках на вкладках модуля

·ID должника;

·ID долга;

·ФИО — фамилия, имя и отчество должника;

·Дата рождения;

·Тип должника — см. словарь № 54;

·Кол-во долгов — общее количество долгов должника;

·Кол-во активных долгов — количество активных долгов должника;

·Наличие тел. — признак наличия телефона у должника;

Примечание. Флаг Наличие тел. включен при наличии у должника хотя бы одного актуального телефонного номера.

Флаг Наличие тел. выключен в следующих случаях:

·По долгу не зарегистрировано ни одного телефонного номера;

·По долгу зарегистрированы номера, но они имеют следующие атрибуты:

·Статус имеет одно из значений, перечисленных в константе WorkTask.PhoneCnt.Status.NotUse;

·Словарный атрибут телефона 1 имеет одно из значений, перечисленных в константе WorkTask.PhoneCnt.DictValue1.NotUse;

·Словарный атрибут телефона 2 имеет одно из значений, перечисленных в константе WorkTask.PhoneCnt.DictValue2.NotUse.

·Тип продукта — см. словарь № 11;

·Название продукта — указывается название продукта, по которому возник долг;

·Банк — название банка, которому принадлежит портфель;

·Портфель — входящий портфель, в состав которого входит долг;

·Статус — статус долга, см. словарь № 6;

·Оператор — ФИО оператора, закрепленного за данным долгом должника;

·Отдел — подразделение, к которому относится оператор в модуле Структура предприятия;

·Важность;

·Порядочность;

·Подразделение — подразделение соответствует стадии должника;

·Дата выхода на просрочку — дата, с которой долг считается просроченным;

·Банковская просрочка (от даты) — значение рассчитывается как "Дата подписания реестра (portfolio.sign_date) – Дата выхода на просрочку (debt.start_date)"

·Фактическая просрочка — если значение поля flag_preventive таблицы debt = 1 (Признак "Профилактика"), то в колонке отображается 0; иначе значение рассчитывается как "Текущая дата – Дата выхода долга на просрочку (debt.start_date)". См. также Дней просрочки;

·Валюта — валюта долга (см. Список валют);

·Размер кредита в руб. — полный размер кредита в рублях; выводится сумма значений по выборке;

·Дата выдачи — дата выдачи кредита;

·Основной долг;

·Остаток — остаток суммы долга на текущую дату; выводится сумма значений по выборке;

·Гос. пошлина — сумма государственной пошлины;

·Дата расчета остатка;

·Нач. долг в руб. — начальный долг в рублях; выводится сумма значений по выборке;

·Филиал — филиал организации, к которому относится долг (словарь № 61);

·Регион — регион, к которому относится долг (см. словарь № 39);

·Часовой пояс — см. словарь № 51;

·Обещ. дата — обещанная дата (не отображается для неверифицированных обещаний);

·Обещ. сумма — обещанная сумма (не отображается для неверифицированных обещаний);

·Ближайший звонок — дата и время для ближайшего звонка должнику (операторы устанавливают эту дату при регистрации контакта с результатом "Необходимо перезвонить");

·Последний платеж — дата последнего платежа;

·Проц. погашения — процент погашения;

·Последний звонок — дата последнего звонка;

·Оператор, сов. посл. звонок — ФИО оператора, совершившего последний звонок;

·Последний выезд — дата последнего выезда;

·Дней в работе — количество дней в работе;

·Договор — номер договора с банком;

·Номер лицевого счета — номер лицевого счета должника;

·ID Клиента;

·Результат посл. звонка — название результата (исхода) контакта для последнего контакта типа "Звонок";

·Результат посл. выезда — название результата (исхода) контакта для последнего контакта типа "Выезд";

·Дата подписания — дата подписания портфеля;

·Цвет;

·Причина возникновения задолженности — см. словарь № 13;

·Балл № 1 — см. словарь № 101;

·Балл № 2 — см. словарь № 102;

·Дата закрепления — дата закрепления оператора за долгом;

·Город;

·Вознаграждение;

·Торговая точка — место выдачи кредита (адрес торговой точки);

·Количество дней просрочки;

·Реестр КА — исходящий портфель, в состав которых входит долг;

·Балл № 3 — см. словарь № 244;

·Балл № 4 — см. словарь № 245.
