Акции

Обновлено до 14 релиза 20.10.2015.

Модуль Акции предназначен для хранения данных об акциях по списанию задолженностей, которые могут быть применены к должникам. Данные модуля используются:

·при включении долгов в акцию в модуле Постановка задач;

·при загрузке данных о долгах с помощью модуля Загрузка данных;

·при загрузке платежей с помощью модуля Загрузка платежей;

·при добавлении подтвержденного или подтверждении имеющегося платежа в модуле "Платежи" для долгов, участвующих в акциях, выполняется пересчет остатка суммы по акции, а также автоматическое закрытие цессионных долгов и перевод агентских долгов в статус 9 "Проблемный" с причиной "Долг погашен по акции" при нулевом остатке по акции;

·для предложения акции при регистрации контакта с помощью модуля Регистрация контакта.

Для того чтобы перейти к модулю, выберите в главном меню пункт Утилиты а Акции.

Функционал акций включается коммерческой константой @Discounts.
