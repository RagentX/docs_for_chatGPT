Вкладка "Соответствия БАЛЛ № 1 - БАЛЛ № 2..."

Вкладка Соответствия БАЛЛ № 1 — Балл № 2… используется для настройки соответствия между значениями скоринговых баллов.

Условия доступности

Вкладка Соответствия БАЛЛ № 1 — Балл № 2… отображается в модуле Словари в случае, когда выполняются одновременно следующие условия:

·для текущего пользователя доступ № 657 DICT_MARKS_RELATION_VIEW имеет значение TRUE;

·в константе Debt.Marks.UseRelation.List перечислены два или более номеров атрибутов, и при этом функционал использования этих скоринговых баллов включен (соответствующие им константы @UseMarkX включены);

·одно из двух условий:

·скоринговый балл № 1 не используется в качестве стадии долга (константа @DepartUseMark1 имеет значение "Ложь");

·скоринговый балл № 1 используется в качестве стадии долга (константа @DepartUseMark1 имеет значение "Истина"), но значение "1" в константе Debt.Marks.UseRelation.List не указано.

Описание вкладки

Вкладка Соответствия БАЛЛ № 1 — Балл № 2…  имеет название в зависимости от количества атрибутов, указанных в значении константы Debt.Marks.UseRelation.List:

·Соответствия БАЛЛ № 1 — БАЛЛ № 2 — для значения "1,2";

·Соответствия БАЛЛ № 1 — БАЛЛ № 2 — Балл № 3 — БАЛЛ № 4 — для значения "1,2,3,4";

и так далее.

Вкладка включает следующие элементы:

·Список соответствий скоринговых баллов; Список включает столько колонок, сколько значений перечислено в константе Debt.Marks.UseRelation.List (от двух до четырех);

·Кнопки для добавления и удаления соответствия скоринговых баллов.

Действия на вкладке

Добавление и удаление соответствия доступны для пользователей, в роли которых доступ № 658 DICT_MARKS_RELATION_EDIT имеет значение TRUE.

Добавление соответствия

Для того чтобы добавить соответствие, выполните следующие действия:

1.На вкладке Соответствия БАЛЛ № 1 — БАЛЛ № 2... модуля Словари нажмите кнопку Добавить.

2.Откроется форма добавления соответствия. В ней будет от двух до четырех полей, в зависимости от количества указанных в константе Debt.Marks.UseRelation.List значений. Каждое поле соответствует одному из скоринговых баллов и представляет собой список для выбора значения Любое значение и значений соответствующих словарей:

·Балл № 1 — словарь № 101;

·Балл № 2 — словарь № 102;

·Балл № 3 — словарь № 244;

·Балл № 4 — словарь № 245.

3.Заполните поля формы добавления соответствия. Каждое поле должно быть заполнено, иначе будет показано сообщение об ошибке, и соответствие не будет добавлено.

4.Нажмите ОК. Если данное соответствие не было добавлено ранее, то оно будет добавлено.

Удаление соответствия

Для того чтобы удалить запись о соответствии:

1.На вкладке Соответствия БАЛЛ № 1 — БАЛЛ № 2... модуля Словари нажмите кнопку Удалить.

2.Будет показано окно подтверждения. Нажмите Yes.
