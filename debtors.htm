Список должников

Обновлено до 41 релиза 21.05.2019.

Модуль Список должников предназначен для выполнения основных действий с данными о должниках: просмотра списка всех должников, к которым пользователь имеет доступ, просмотра списков должников, отобранных по различным критериям, создание и редактирование данных о должниках и др. Модуль Список должников является рабочим местом супервизора или старшего коллектора.

Для того чтобы открыть список должников, выберите в главном меню пункт Рабочие места а Список должников.

Право для входа в модуль имеют только те пользователи, в роли которых доступ DEBT_LIST_VIEW установлен в значение TRUE. Полный перечень доступов к модулю приведен в таблице.
