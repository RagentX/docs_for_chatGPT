Карточка судебной работы в судебном производстве

Обновлено до 45 релиза 28.01.2020.

Карточка открывается для неопределенных судебных дел (не приказ и не иск). Доступна только при значении константы LawActUseIndefiniteAct равном "Истина".

Модуль "Карточка судебной работы" предназначен для работы с информацией о должниках, находящихся на стадиях судебного производства. Карточка судебной работы в судебном производстве открывается из модуля "Юридический департамент", вкладка Судебные дела (предв.) (см. Переход к карточке судебной работы/исполнительного производства).

Для карточки судебной работы в судебном производстве действуют те же доступы, что и для карточки судебной работы в приказном производстве, кроме того дополнительные доступы для неопределенного судебного дела:

№

Название доступа

Значение доступа

Описание доступа

502

LAW_ACT_INDEFINITE_DELETE

TRUE

Юридический департамент. Доступ на аннулирование «неопределенного» суд. дела.

505

LAW_ACT_INDEFINITE_FLOW_FULL

TRUE

Судебная стадия. Доступ на управление жизненным циклом «неопределенного» суд. дела.

504

LAW_ACT_INDEFINITE_SET_PORTFOLIO

TRUE

Юридический департамент. Доступ на смену портфеля «неопределенного» суд. дела.

503

LAW_ACT_INDEFINITE_SET_USER

TRUE

Юридический департамент. Доступ на смену ответственного «неопределенного» суд. дела.
