Отчет о проделанных выездах

Отчет "Отчет о проделанных выездах" отображает информацию о выполненных выездах к должникам за период в приложении Fast Report.

Отчет "Отчет о проделанных выездах" доступен только пользователям, в роли которых доступ REP_PERF_VISIT имеет значение "Истина".

Порядок действий

Для того чтобы сформировать отчет, выполните следующее:

1.Выберите в главном меню пункт Отчеты, затем выберите отчет "Отчет о проделанных выездах".

2.Откроется форма "Отчет о проделанных выездах" для ввода параметров данного отчета. Заполните следующие поля:

·С: — выберите начала периода (по умолчанию в поле указано первое число текущего месяца).

·По: — выберите окончания периода (по умолчанию в поле указано последнее число текущего месяца).

·Банк и портфель — панель состоит из двух полей. Левое поле предназначено для выбора банка, в нем доступны все банки, а также пункт "Все". Правое поле предназначено для выбора портфеля.

·Если в правом поле выбран банк, то выбор портфеля обязателен. В отчете будут отображены только долги из выбранного портфеля.

·Если в правом поле выбран пункт "Все", то правое поле заблокировано. В отчете будут отображены долги из всех портфелей.

·Флаг "Показывать нулевые":

·Если флаг включен, то  в отчет попадают все должники: те должники, по которым выездов не было, попадают в отчет с нулевыми показателями количества выездов.

·Если флаг выключен, то в отчет попадают только те должники, по которым за период был хотя бы один выезд.

Примечание: должник, который ни разу не был распределен, в отчет не попадает в любом случае.

3.Нажмите кнопку "Запустить". Сформированный отчет откроется в приложении Fast Report.

Описание отчета

В шапке отчета отображается выбранный период и выбранные портфели.

Отчет представляет собой таблицу со следующими полями:

·№ п/п.

·ФИО — фамилия, имя и отчество должника.

·Дата рождения — дата рождения должника.

·Продукт.

·Кредитор — входящий портфель

·Размер кредита.

·Сумма к погашению.

·Дата выхода на просрочку.

·Оператор — оператор, ответственный за долг.

·Объем оказанных услуг — в поле отображаются следующие значения по совершенным выездам:

·выездов — общее количество выездов по данному долгу;

·удачных — количество успешных;

·обещаний — количество зарегистрированных обещаний;

·отказов — количество отказов.

·Комментарий — комментарий к долгу.
