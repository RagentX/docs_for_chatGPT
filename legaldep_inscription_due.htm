Массовый расчет госпошлины

Массовый расчет госпошлины доступен только тем пользователям, в роли которых доступ № 713 LAW_ACT_INSCRIPTION_SET_DUE имеет значение TRUE (а также включен доступ  № 716 LAW_ACT_INSCRIPTION_FLOW_FULL).

Расчет госпошлины доступен для исполнительной надписи в статусах (см. словарь № 25):

·1002 "Запрос документов";

·1002 "Запрос документов";

·1003 "Расчет госпошлины";

·1011 "Возобновление".

Для того чтобы осуществить массовый запрос документов, выполните следующее:

1.В форме Юридический департамент на вкладке Исполнительная надпись выберите одно или более дел. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL.

2.Активируйте контекстное меню правой кнопкой мыши и выберите пункт Жизненный цикл, затем в открывшемся списке выберите Расчет госпошлины.

3.Для действия Расчет госпошлины доступны исполнительные надписи только в статусах, указанных выше, поэтому действие может быть совершено только если в выборке есть хотя бы одно дело в одном из нужных статусов. Иначе будут показаны следующие сообщения о невозможности выполнения действия:

·Если ни одна запись не выбрана:

Список пуст. Операция невозможна.

·Если ни одна из выбранных записей не находится в разрешенном статусе:

Для всех выделенных записей операция недоступна.

4.Расчет госпошлины осуществляется по следующим правилам:

·Если размер требований ≤ 300 000 руб, то Расчетная госпошлина фиксированная = 1 500 руб. + значение, указанное в константе LawActExecInscriptionPTRSum;

·Если размер требований > 300 000 руб, то Расчетная госпошлина = Размер требований * 0,5% + значение, указанное в константе LawActExecInscriptionPTRSum.

5.В поле Госпошлина каждого из обработанных дел будет записано рассчитанное значение.

6.Обработанные дела будут переведены в статус 1003 "Расчет госпошлины".

7.По окончании исполнения будет показано информационное окно с указанием количества выделенных записей и количества переведенных в статус 1003. Нажмите ОК.

8.Действие протоколируется.
