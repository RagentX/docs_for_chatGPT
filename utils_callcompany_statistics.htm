Просмотр статистики

Статистика работы кампаний обзвона ведется по участникам кампании. Статистические данные отображаются в форме Кампании обзвона на вкладке Статистика.

При загрузке статистики по выбранной кампании обзвона учитывается константа Campaign.Stat.MaxDayCnt. Период отбора данных для подсчета статистики не может быть указанного количества дней.

Список содержит следующие поля:

·ФИО сотрудника — фамилия, имя, отчество оператора — участника кампании обзвона;

·Обработано — количество обработанных должников;

·Звонков — количество сделанных звонков;

·Успешных — количество сделанных успешных звонков;

·Количество обещаний;

·Сумма обещаний;

·Отказов — количество отказов.

Внизу под столбцами отображается суммарное значение статистических показателей по всем участникам.
