Удаление вложения

Удаление вложений доступно только пользователям,  роли которых доступ № 759 PROPERTY_CARD_DOC_DELETE имеет значение "TRUE".

Для того чтобы удалить вложение к имуществу, выполните следующее:

1.Откройте форму "Карточка имущества" для нужного имущества.

2.Активируйте вкладку "Вложения".

3.Выберите запись о вложении.

4.Нажмите кнопку Удалить.

5.Откроется форма подтверждения. Нажмите "Yes".

Вложение будет удалено.

Примечание. Если в Системе настроено хранение документов на сервере (то есть, если константа DocAttach.SaveMode имеет значение 2), то при удалении документа выполняется проверка, был ли этот документ прикреплен к более чем одной записей об имуществе. Если это так, то есть возможность удалить данный документ одновременной у всех записей об имуществе или удалить его только у текущей записи. Для этого открывается диалоговое окно с сообщением:

Выбранный документ связан с несколькими объектами имущества. Выберите вариант удаления:

Нажмите Yes чтобы удалить у всех

Нажмите No чтобы удалить у текущего

Нажмите Cancel для отмены

·Если вы хотите удалить данный документ у всех записей, к которым он прикреплен, нажмите кнопку Yes.

·Если вы хотите удалить данный документ только у текущей записи, нажмите кнопку No.
