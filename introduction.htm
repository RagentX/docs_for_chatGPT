Введение

В данном руководстве содержится информация о настройке и эксплуатации коллекторской системы "Контакт" (далее Система). Информация, представленная в данном руководстве, предназначена для администраторов и пользователей системы. Все пользователи должны обладать начальными навыками работы с персональным компьютером: быть знакомыми с интерфейсом операционной системы Microsoft Windows, владеть основными приемами работы в ней, уметь пользоваться компьютерными программами для работы с электронной почтой (Microsoft Office Outlook), программами пакета Microsoft Office.

Цель документа

Обеспечить быстрый поиск информации для решения вопросов и задач, связанных с настройкой и эксплуатацией системы.

Структура документа

Раздел Интерфейс пользователя описывает особенности интерфейса, общие для всех модулей, и особенности работы с табличными формами.

Разделы Администрирование, Утилиты, Рабочие места, Отчеты соответствуют разделам Системы и содержат описания модулей, включенных в эти разделы.

Раздел Мобильная версия содержит описание мобильной версии Системы (ANDROID).

Раздел Функциональные возможности содержит описание функционалов, работа которых затрагивает несколько различных модулей.

Раздел Отладочная информация поможет пользователю при столкновении с неверной или ошибочной работой модулей.
