Принятие иска к производству

При установке отметки о принятии иска к производству возможны следующие варианты:

·принятие судебного иска к производству;

·оставление судебного иска без движения до устранения недостатков;

·отказ в принятии судебного иска к производству;

·возвращение заявления;

·определение об оставлении без рассмотрения.

Принятие судебного иска к производству

Принятие иска к производству доступно в следующих статусах иска:

·"Подан";

·"Оставление иска без движения";

·"Определение об оставл. без рассмотрения".

Для установки отметки о принятии судебного иска к производству выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство при необходимости введите комментарий в поле Комментарий.

2.Нажмите кнопку Принятие иска и в раскрывшемся списке выберите Определение о принятии иска к  произв.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.После этого иск будет переведен в статус "Определение о принятии к производству".

7.Действие протоколируется.

8.Далее можно перейти к регистрации результатов предварительного заседания. Также становится доступна возможность заключения мирового соглашения.

Оставление судебного иска без движения

Пункт "Оставление иска без движения" доступен для исков, находящихся в статусе "Подан".

Если какие-либо документы, необходимые для принятия судебного иска к производству, содержат те или иные недостатки, то есть возможность проставить отметку об оставлении иска без движения до их устранения. Для этого выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство в поле Дата срока исполнения недостатков укажите соответствующую дату. Для заполнения поля нажмите кнопку , в открывшейся форме укажите дату, при необходимости введите комментарий и нажмите кнопку ОК.

2.Нажмите кнопку Принятие иска и в раскрывшемся списке выберите Оставление иска без движения.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.После этого иск будет переведен в статус "Оставление иска без движения".

7.Действие протоколируется.

8.Для продолжения работы необходимо будет заново подать иск в суд.

Отказ в принятии судебного иска к производству

Пункт "Отказ в принятии" доступен для исков, находящихся в статусе "Подан".

Для установки отметки об отказе в принятии судебного иска к производству выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство заполните поле Комментарий.

2.Нажмите кнопку Принятие иска и в раскрывшемся списке выберите Отказ в принятии.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.После этого иск будет переведен в статус "Отказ в принятии".

7.Действие протоколируется.

8.Для продолжения работы необходимо будет заново подготовить иск в суд.

Примечание. Если константа LawAct.Decision.Components.Compare имеет значение "Истина", то при выполнении действия Отказ в принятии иска дополнительно сравниваются текущие значения составляющих на момент подачи в суд со значениями составляющих по решению суда (составляющие для приказа перечислены в константе LawActIskDebtComponentList), а также значение размера пошлины с утвержденным размером пошлины.

Если хотя бы одна пара значений не совпадает, пользователю отображается окно с сообщением:

Поданные и присужденные суммы не совпадают. Продолжить?

Нажмите ОК чтобы продолжить или Отмена чтобы прервать действие.

Возвращение заявления

Возвращение заявления доступно в следующих статусах иска:

·"Подан";

·"Оставление иска без движения".

Для установки отметки о возвращении искового заявления выполните следующие действия:

1.Нажмите кнопку Принятие иска и в раскрывшемся списке выберите Возвращение заявления.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.Выполните прикрепление документа.

4.Далее осуществляется формирование настроенных запланированных действий.

5.После этого иск будет переведен в статус "Возвращение заявления".

6.Действие протоколируется.

7.Для продолжения работы необходимо будет заново подать иск в суд.

Определение об оставлении без рассмотрения

Действие доступно для дел в статусе "Подан".

Для установки отметки об оставлении иска без рассмотрения выполните следующие действия:

1.В карточке судебной работы на вкладке Исковое производство нажмите кнопку Принятие иска и в раскрывшемся списке выберите Определение об оставлении без рассмотрения.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.Выполните прикрепление документа.

4.Далее осуществляется формирование настроенных запланированных действий.

5.Действие протоколируется.

6.После этого иск будет переведен в статус "Определение об оставл. без рассмотрения". Данный статус является конечным. Просмотр дела доступен в Архиве судебной работы.

7.Для продолжения работы необходимо будет заново подать иск в суд. Также доступно Принятие судебного иска к производству.
