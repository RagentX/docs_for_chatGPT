Форма "Группы свободного поиска"

Форма Группы свободного поиска содержит следующие элементы:

·Вкладки с данными, отобранными по каким-либо критериям с помощью запросов. Перечень вкладок, к которым пользователь имеет доступ, настраивается в модуле "Группы должников";

·Кнопки для действий — см. Общие операции на вкладках формы "Группы свободного поиска".

·Блок истории работы — см. далее.

·Индикатор, показывающий общее количество записей, которые отображаются в списке.

По умолчанию вкладки открываются с таблицами, не заполненными данными. Для того чтобы заполнить таблицу данными, примените инструменты фильтрации, мультифильтры, сложный фильтр, а затем нажмите кнопку "Найти".

Примечание. Вы можете задать вкладки модуля, которые при открытии будут автоматически заполняться данными, без нажатия кнопки "Найти". Для этого свяжитесь с технической поддержкой. Это настраивается в технической константе GroupViewer.ShowLinesImmdediately.TabsList.

Максимальное количество записей в списке ограничивается значением константы GroupViewerMaxRecords. Если данная константа равна нулю, ограничение на количество записей не накладывается.

Примечание. В списке отображаются только те долги, к которым у текущего пользователя настроен доступ в модуле Настройка списка должников.

Для работы со списком должников также используется контекстное меню, которое вызывается щелчком правой клавиши мыши в области списка.

Отображение истории работы

Если в модуле Группы должников для модуля Группы свободного поиска настроено отображение истории работы, то на вкладках модуля открывается блок, в котором отображается информация о датах, способах и результатах контактов с должником, информация о переводе долга на аутсорсинг, возврата из аутсорсинга и т.д. по аналогии с отображением в модулях Список должников, Обработка должников и др.

Данный блок отображается, если для пользователя включен указанный элемент управления для данной вкладки и, при этом, на данной вкладке есть колонка ID должника.

Отображение итоговых сумм числовых полей

Доступна настройка отображения сумм любых числовых полей на вкладках модуля. Для этого обратитесь в техническую поддержку. См. доработку по заявке в Help Desk № 32057. ТЗ: http://repository/svn/debtcontrol/Documents/Технические задания/Центр-Профи/Заявки/32057_ТЗ_Подсчет_итогов_в_ГСП.docx.

Особенности фильтрации данных в таблицах модуля

К данным в таблицах модуля может быть применена Фильтрация данных в списке, как и к другим таблицам Системы. Отличие состоит в том, что на вкладках модуля "Группы свободного поиска" реализован принцип остаточного фильтра. То есть, после фильтрации в одном столбце, для других столбцов есть возможность фильтровать только по оставшимся значениям.
