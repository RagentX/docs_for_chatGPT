Экранная форма "Выезды"

Описание формы "Выезды"

На экранной форме "Выезды" отображается список выездов, находящихся в статусе "Подготовлен". В список попадают выезды, назначенные на сегодняшнюю дату и последующие дни. Количество дней, за которые отображаются выезды, зависит от константы Android.Statistic.VisitDayCnt. Если константа равна 1, то отображаются только выезды, назначенные на сегодня.

Экранная форма "Выезды" содержит следующие элементы (Рисунок 4‑1):

·Кнопка — переход на Боковое меню;

·Кнопка — Обновление;

·Кнопка — Передать контакты (синхронизация);

·Кнопка — Загрузка документов;

·Список выездов;

·Кнопка — Инструменты;

Каждая запись списка выездов содержит следующую информацию:

·Фамилию, имя, отчество должника;

·Адрес должника;

·Дата выезда.

·Скоринговые баллы;

В нижней части формы отображается количество записей.

По умолчанию выезды отсортированы по полю ФИО по возрастанию.

В зависимости от даты выезда записи подсвечиваются цветными рамками:

·Красный — дата выезда меньше начала текущего дня;

·Желтый — дата выезда больше начала текущего дня, меньше начала следующего дня;

·Зеленый — дата выезда больше начала следующего дня.

Инструменты формы "Выезды"

Для доступа к инструментам нажмите кнопку , расположенную в правом нижнем углу формы. Кнопка примет вид , а над ней появится список инструментов (Рисунок 4‑2). Инструменты на всех формах имеют одинаковый вид.

На форме доступны следующие инструменты:

·Местоположение;

·Фильтр;

·Поиск;

·Сортировка;

Местоположение

При прикосновении к кнопке "Местоположение" открывается карта, на которой отображены адреса, соответствующие всем записям на форме "Выезды" с учетом результатов примененных фильтрации и поиска (Рисунок 4‑3).

Фильтр

Фильтр позволяет вывести записи, отвечающие иным критериям, чем заданные на форме по умолчанию.

См. описание настраиваемых фильтров (Фильтр формы "Выезды").

Поиск

Форма поиска выездов (Рисунок 4‑4) содержит следующие элементы;

·Кнопки:

·Для отмены фильтрации — ;

·Для подтверждения выбора — ;

·Критерии поиска:

·Поля ФИО содержат;

·Номер договора;

·Номер телефона.

Для ввода текстовых полей ниже формы поиска открывается текстовая клавиатура, для ввода цифровых полей — цифровая.

После задания критериев поиска нажмите . На экране отобразятся записи, соответствующие критериям поиска. При этом рядом слева от кнопки "Инструменты" появится кнопка отмены поиска: .

Сортировка

Форма сортировки выездов (Рисунок 4‑5) содержит следующие элементы:

·Кнопки:

·Для отмены фильтрации — ;

·Для подтверждения выбора — ;

·Критерии сортировки (после текста "Сортировать выезды по:"):

·ФИО;

·Адрес выезда;

·Дата выезда;

·Срок просрочки;

·Порядок сортировки:

·По возрастанию;

·По убыванию.

После задания критериев сортировки и порядка сортировки нажмите . На экране отобразятся записи, в порядке, соответствующем выбранным критериям. При этом рядом слева от кнопки "Инструменты" появится кнопка отмены сортировки: .

Действия на форме "Выезды"

Переход к карточке должника

Для перехода к карточке должника с экранной формы "Выезды" выберите запись в списке выездов и выполните на ней короткое нажатие ("tap").

Переход к форме "Детали задачи"

Для перехода на форму "Детали задачи" выберите запись в списке выездов и выполните на ней длинное нажатие ("long tap").
