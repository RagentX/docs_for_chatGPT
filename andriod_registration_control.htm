Контроль регистрации координат по адресу выезда

Контроль регистрации координат по адресу выезда осуществляется в основной версии Системы, в модуле Протокол контактов, на вкладке Android.

Доступность полей

Если значение константы Android.VisitContactAddress.AllowableDeviationRadius имеет значение 0, то данные поля скрыты.

Если значение константы Android.VisitContactAddress.AllowableDeviationRadius имеет значение больше 0, то данные поля доступны.

Логика заполнения поля "Расстояние между контактом и адресом (м)"

Если у выезда есть координата адреса (флаг Наличие коорд. адреса включен) и есть координата выезда (флаг Наличие коорд. регистрация включен), то в поле Расстояние между контактом и адресом (м) указывается расстояние между этими точками.

Если у данной записи нет координат адреса и/или координат выезда, то это расстояние указывается равным 0.

Логика заполнения поля "Попадание выезда в радиус адреса"

Если у выезда нет координат адреса (флаг Наличие коорд. адреса выключен) и/или нет координат выезда (флаг Наличие коорд. регистрация выключен), то флаг Попадание выезда в радиус адреса всегда выключен, независимо от значения константы Android.VisitContactAddress.AllowableDeviationRadius.

Если у данной записи есть координаты адреса и есть координаты выезда, то значение флага Попадание выезда в радиус адреса зависит от значения константы Android.VisitContactAddress.AllowableDeviationRadius. Если расстояние между адресом и точкой регистрации выезда больше, чем значение константы, то флаг Попадание выезда в радиус адреса выключен. Если расстояние между контактом и адресом меньше, чем значение константы, то флаг Попадание выезда в радиус адреса включен.

Контекстное меню "Работа с картами"

Пункт контекстного меню (вызывается правой кнопкой мыши) Работа с картами включает следующие подпункты:

·Посмотреть место регистрации на карте — место регистрации контакта и адрес находятся в центре кругов, радиус которых равен погрешности определения координат;

·Посмотреть адреса на карте — адрес находится в центре круга, радиус которого равен значению константы Android.VisitContactAddress.AllowableDeviationRadius;

·Посмотреть выезды за весь день.

Пункт "Просмотреть место регистрации" - Место регистрации и адрес

Пункт "Посмотреть адреса на карте" - Место регистрации контакта попало в круг радиусом Android.VisitContactAddress.AllowableDeviationRadius
