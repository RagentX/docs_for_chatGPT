Вкладка "Характеристики"

Вкладка "Характеристики" включает в себя следующие подчиненные вкладки:

·Технические характеристики;

·Финансовые характеристики;

·Правовые характеристики;

·Обременения;

·Дополнительно.

Подчиненная вкладка "Технические характеристики"

Подчиненная вкладка "Технические характеристики" включает следующие элементы:

Таблица параметров имущества, для которых классификация равна "Технические" (см. словарь № 351). Таблица включает следующие поля:

·Параметр — наименование параметра имущества;

·Значение — значение параметра имущества.

·Кнопки для следующих действий:

· Добавить — Добавление параметра имущества (доступна пользователям, в роли которых доступ № 737 PROPERTY_CARD_TECHNICAL_SPECIFICATION_ADD имеет значение "TRUE");

· Редактировать — Редактирование параметра имущества (доступна пользователям, в роли которых доступ № 738 PROPERTY_CARD_TECHNICAL_SPECIFICATION_EDIT имеет значение "TRUE");

· Удалить — Удаление параметра имущества (доступна пользователям, в роли которых доступ № 739 PROPERTY_CARD_TECHNICAL_SPECIFICATION_DELETE имеет значение "TRUE");

·История изменения параметров — Просмотр истории изменения параметров имущества (доступна пользователям, в роли которых доступ № 740 PROPERTY_CARD_TECHNICAL_SPECIFICATION_LOG_VIEW имеет значение "TRUE").

Подчиненная вкладка "Правовые характеристики"

Подчиненная вкладка "Правовые характеристики" включает следующие элементы:

·Таблица правовых характеристик имущества, включающая следующие поля:

·ID права — идентификатор права

·Вид права — (см. словарь № 352 "Вид права на объект имущества");

·Дата начала действия права;

·Регистрационный номер права;

·Владелец права — (см. словарь № 353 "Владелец права на объект имущества");

·Дата действия связи, с;

·Дата действия связи, по;

·Размер доли в праве;

·Дата действия права, с — дата, с которой действует право;

·Дата действия права, по — дата, по которую действует право;

·Кнопки для следующих действий:

· Добавить — Добавление правовой характеристики имущества (доступна пользователям, в роли которых доступ № 745 PROPERTY_CARD_LEGAL_SPECIFICATION_ADD имеет значение "TRUE");

· Редактировать — Редактирование правовой характеристики имущества (доступна пользователям, в роли которых доступ № 746 PROPERTY_CARD_LEGAL_SPECIFICATION_EDIT имеет значение "TRUE");

· Удалить — Удаление правовой характеристики имущества (доступна пользователям, в роли которых доступ № 747 PROPERTY_CARD_LEGAL_SPECIFICATION_DELETE имеет значение "TRUE");

·История изменения — Просмотр истории изменения правовых характеристик имущества (доступна пользователям, в роли которых доступ № 748 PROPERTY_CARD_LEGAL_SPECIFICATION_LOG_VIEW имеет значение "TRUE").

Подчиненная вкладка "Обременения"

Подчиненная вкладка "Обременения" включает следующие элементы:

·Таблица записей о видах обременения данного имущества (см. словарь № 360 "Вид обременения"). Таблица включает следующие поля:

·ID обременения — идентификатор записи;

·Вид обременения — см. словарь № 360 "Вид обременения";

·ID права — код термина словаря № 352 "Вид права на объект имущества";

·Вид права — название термина словаря № 352 "Вид права на объект имущества";

·Дата установления — дата установления обременения;

·Дата снятия — дата снятия обременения;

·Дата действия обременения, с;

·Дата действия обременения, по.

·Кнопки для следующих действий:

· Добавить — Добавление обременения (доступна пользователям, в роли которых доступ № 749 PROPERTY_CARD_BURDEN_ADD имеет значение "TRUE");

· Редактировать — Редактирование обременения (доступна пользователям, в роли которых доступ № 750 PROPERTY_CARD_BURDEN_EDIT имеет значение "TRUE");

· Удалить — Удаление обременения (доступна пользователям, в роли которых доступ № 751 PROPERTY_CARD_BURDEN_DELETE имеет значение "TRUE");

·История изменения — Просмотр истории изменения обременения (доступна пользователям, в роли которых доступ № 752 PROPERTY_CARD_BURDEN_LOG_VIEW имеет значение "TRUE").

· Редактировать — Редактирование дополнительной характеристики имущества (см. Редактирование параметра имущества; доступна пользователям, в роли которых доступ № 754 PROPERTY_CARD_OTHER_SPECIFICATION_EDIT имеет значение "TRUE");

· Удалить — Удаление дополнительной характеристики имущества (см. Удаление параметра имущества; доступна пользователям, в роли которых доступ № 755 PROPERTY_CARD_OTHER_SPECIFICATION_DELETE имеет значение "TRUE");

·История изменения параметров — Просмотр истории изменения параметров имущества (доступна пользователям, в роли которых доступ № 756 PROPERTY_CARD_OTHER_SPECIFICATION_LOG_VIEW имеет значение "TRUE").
