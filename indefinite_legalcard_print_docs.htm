Печать документов судебной работы

Из карточки судебной работы доступна печать документов судебной работы, созданных в модуле Конструктор документов судебной работы и относящихся к форме Судебная работа (см. словарь № 79). Документ будет создаваться по текущему делу судебной работы.

Пользователю доступны для выбора только те печатные формы документов судебной работы, доступ к которым настроен в его роли в модуле Роли и доступы на вкладке Печатные формы (Суд).

Для того чтобы напечатать документ:

1.Нажмите кнопку Печать документов.

2.Откроется форма выбора документа. Заполните следующие поля:

·Тип документа — выберите тип заявления (см. словарь № 33);

·Флаг Регистрировать письмо — если флаг включен, документ будет зарегистрирован в базе данных, запись о нем будет доступна на вкладке Контакты текущего модуля, а также в модуле Карточка должника на вкладке Контакты;

Примечание. Значение данного флага по умолчанию определяется константой LawActRegLetterDefaultCheckState.

·Флаг Сохранять копию письма — если этот флаг включен, то копия письма будет сохранена в БД.

3.Нажмите OK. Будет сформирован документ выбранного типа и открыт в режиме предварительного просмотра в приложении Fast Report.

См. также Выбор банковских реквизитов при формировании письма или документа судебной работы.
