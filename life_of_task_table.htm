Таблица для настройки ЖЦ задач

Настройка жизненного цикла задач с помощью таблицы

Заявка в Help Desk № 29156, задача № 33467

Информация для разработчиков

Данная доработка дает возможность задать правила переходов по жизненным циклам задач при помощи таблицы.

Область применения доработки

В следующих модулях Системы доработаны переходы между этапами жизненного цикла задач при помощи кнопок жизненного цикла:

·Рабочая вкладка "Задачи" в модулях:

·Карточка должника;

·Карточка судебной работы;

·Карточка исполнительного производства.

·Вложенная вкладка "Задачи" на вкладке "Имущество" в следующих модулях:

·Карточка должника;

·Карточка судебной работы;

·Карточка исполнительного производства.

В следующих модулях доработан процесс массовой смены статуса задач при использовании пункта контекстного меню "Смена статуса":

·Информационная вкладка "Задачи" в модулях:

·Список должников;

·Постановка задач;

·Обработка должников;

·Юридический департамент;

·Группы свободного поиска.

Описание доработки

В указанных модулях доработаны переходы между этапами жизненного цикла задач. В настоящей версии Системы правила переходов задаются в таблице debt_task_status_flow.

Описание таблицы debt_task_status_flow

Каждая запись данной таблицы описывает возможность перевести задачу из одного статуса в другой для пользователя, имеющего определенный доступ к задаче (оператор или постановщик).

Таким образом, в Системе возможны только те переходы по этапам жизненного цикла задач, которые явно указаны в данной таблице.
