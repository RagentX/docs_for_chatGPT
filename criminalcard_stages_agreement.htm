Достижение договоренности

Если доступ № 837 PROSECUTION_CARD_FULL_WORKFLOW имеет значение "TRUE", то кнопка "Договоренность" доступна всегда, вне зависимости от текущего статуса дела (включая статус 7 "Получено решение суда"). При нажатии на данную кнопку осуществляется перевод дела в статус 8 "Договоренность достигнута", независимо от наличия связанных с делом долгов и ограничений, настраиваемых константой Prosecution.Agreement.Ban.DebtStatusList и доступом № 687 PROSECUTION_CARD_AVAILABLE_STATUSES_FOR_CHANGE.

Если доступ № 837 PROSECUTION_CARD_FULL_WORKFLOW имеет значение "FALSE", то:

1.Принятие решения о договоренности с должником может выполняться на любом этапе работы с делом по уголовному преследованию, кроме статуса 7 "Получено решение суда" (см. словарь № 145).

2.Для статуса 2 "Достигнута договоренность" доступность кнопки Договоренность определеяется значением  доступа № 687 PROSECUTION_CARD_AVAILABLE_STATUSES_FOR_CHANGE — см. соответствующее описание.

3.Для следующих типов дел договоренность не может быть достигнута (статус дела на изменяется):

·дел, в которых отсутствует связь с долгом. Для таких дел при нажатии на кнопку Договоренность появляется сообщение:

Договоренность невозможна, договор отсутствует!

·дел, связанных с долгами, которые находятся в статусах, не перечисленных в значении константы Prosecution.Agreement.Ban.DebtStatusList. Для таких дел при нажатии на кнопку Договоренность появляется сообщение:

Договоренность невозможна, договор не закрыт!

Порядок действий

Для того чтобы зафиксировать факт договоренности, выполните следующие действия:

1.На вкладке Уголовное преследование нажмите кнопку Договоренность.

2.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

3.В правом верхнем углу нажмите кнопку сохранения изменений, затем подтвердите сохранение, нажав кнопку Yes. Дело будет переведено в конечный статус 8 "Достигнута договоренность" и затем автоматически перенесено в архив.

Действие логируется.
