Добавление долгов в простую группу

Show/Hide Hidden Text

Добавление долгов в группу в модуле Группы должников разрешено только для простых групп. Если список уже содержит записи, новые долги будут добавлены к нему.

Если у автогруппы включен флаг Разрешить добавление долгов в группу вручную, то добавление долгов в такую группу может быть реализовано с использованием функционала Добавление долга в группу из списочных РМ.

Для добавления долгов в простую группу:

1.В форме Группы долгов активируйте вкладку Долги в группе.

2.Выберите группу с помощью фильтра, расположенного в верхней части формы.

3.Нажмите кнопку Найти. Откроется список долгов в группе.

4.Нажмите кнопку Добавить список долгов.

5.В открывшейся форме заполните следующие поля:

·Портфели — портфели, содержащие долги, которые должны быть включены в группу. Для заполнения поля используется мультифильтр по портфелям. Если поле не заполнено, выбор записей будет осуществляться по всем портфелям;

·Операторы — операторы, которым принадлежат долги, которые должны быть включены в группу. Для заполнения поля используется мультифильтр по операторам. Если поле не заполнено, выбор записей будет осуществляться по всем операторам.

6.Нажмите кнопку ОК. Откроется форма со списком долгов, отобранных в соответствии с заданными фильтрами.

7.При необходимости задайте условия отбора долгов в группу, как описано в разделе Фильтрация данных в списке.  

8.Если требуется задать дополнительные условия отбора, нажмите кнопку Дополнительно. В открывшейся форме задайте дополнительные условия отбора.

Примечание. Если по долгу имеется неверифицированное обещание, его сумма не учитывается при поиске.

9.Нажмите кнопку ОК. Будет выполнен возврат в форму со списком добавляемых долгов.

10.При необходимости выполните сортировку долгов в списке (см. Сортировка данных в списке).

11.В форме со списком отобранных долгов нажмите кнопку Добавить.

12.Затем задайте порядок нумерации записей в группе.

13.Нажмите кнопку ОК. Будет выдано сообщение о том, что список сформирован. Отобранные записи в выбранном порядке добавятся к прежним записям в группе.

       Список полей таблицы "Данные о добавляемых долгах"

·ID должника;

·ID долга;

·ФИО — фамилия, имя и отчество должника;

·Пол — пол должника;

·Возраст — возраст должника;

·Банк — название банка, которому принадлежит портфель;

·Портфель — входящий портфель, в состав которого входит долг;

·Название — продукт, по которому возник долг;

·Статус — статус долга (см. Статусы долга);

·Тип продукта — указывается тип продукта (см. словарь № 11);

·Валюта — валюта долга (см. Список валют);

·Размер кредита — полный размер кредита, выводится в валюте долга, указанной в поле Валюта;

·Остаток — остаток суммы долга на текущую дату, выводится в валюте долга, указанной в поле Валюта;

·Нач. долг — начальный долг, выводится в валюте долга, указанной в поле Валюта;

·Осн. долг — основной долг, выводится в валюте долга, указанной в поле Валюта;

·Проценты — сумма процентов по кредиту;

·Штрафы и пени;

·Гос. пошлина — сумма государственной пошлины, выводится в основной валюте Системы;

·% осн. долга в нач. долге;

·% процетов в нач. долге;

·% штрафов в нач. долге;

·Банковская просрочка (от даты) — значение рассчитывается как "Дата подписания реестра (portfolio.sign_date) – Дата выхода на просрочку (debt.start_date)"

·Фактическая просрочка — если значение поля flag_preventive таблицы debt = 1 (Признак "Профилактика"), то в колонке отображается 0; иначе значение рассчитывается как "Текущая дата – Дата выхода долга на просрочку (debt.start_date)". См. также Дней просрочки;

·Дней в работе агенства;

·Дней без просрочки;

·Возраст долга;

·Посл. звонок — дата последнего звонка;

·Обещ. дата — обещанная дата (не отображается для неверифицированных обещаний);

·Ближ. звонок — дата и время для ближайшего звонка должнику (операторы устанавливают эту дату при регистрации контакта с результатом "Необходимо перезвонить");

·Посл. письмо долж. — дата последнего письма должнику;

·Посл. SMS — дата последнего SMS должнику;

·Проц. погашения;

·Оператор — ФИО оператора, закрепленного за данным долгом должника;

·Подразделение — стадия;

·Важность;

·Порядочность;

·Текущее состояние — состояние долга, см. словарь № 29;

·Причина возникновения — причина возникновения долга (см. словарь № 13);

·Регион — регион, к которому относится долг (см. словарь № 39);

·Часовой пояс — см. словарь № 51 "Часовые пояса";

·Кол-во обработок должника;

·Балл № 1 — см. словарь № 101, Скоринговые баллы;

·Балл № 2 — см. словарь № 102, Скоринговые баллы;

·Остаток основного долга — выводится в валюте долга, указанной в поле Валюта.

·Количество дней просрочки — см. Глоссарий
