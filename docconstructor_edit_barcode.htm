Добавление штрихкода в текст письма

Для того чтобы добавить штрихкод в текст письма, выполните следующие действия:

1.В форме Конструктор писем выполните двойной щелчок (клик) левой клавишей мыши по выбранной записи. Откроется окно редактора FastReport, содержащее форму письма.

2.Нажмите кнопку Объект "Штрихкод".

3.Поместите объект в нужное место в теле письма.

4.Откроется форма Штрихкод.

5.Заполните нужные поля и нажмите ОК.
