Загрузка данных из внешних источников

Обновлено до 45 релиза 28.01.2020.

Модуль Загрузка данных обеспечивает поддержку работы операторов при пополнении базы данных на основе файлов формата MS Excel, содержащих данные о персонах (должниках), поручителях должников, залогодателях, контактных лицах, расчетных лицах, паспортных данных, адресах, телефонах, долгах, портфелях долгов, произвольных атрибутах должников, судебной работе, исполнительном производстве.

Для того чтобы перейти к работе в модуле загрузки данных, выберите в главном меню пункт Утилиты а Загрузка данных.
