Статусы обещаний

Обещание по долгу может находиться в одном следующих статусов (см. словарь № 259):

·Выполнено — данный статус присваивается обещанию, по которому имеется подтвержденный платеж, если дата платежа больше даты регистрации обещания и меньше даты обещания + количество дней, заданного в константе PromGracePeriod. Обещание считается выполненным, если сумма платежа составляет более того количества процентов, которое задано константой PromCoverRate;

·Част. выполнено — данный статус присваивается обещанию, по которому имеется подтвержденный платеж, если дата платежа больше даты регистрации обещания и меньше даты обещания + количество дней, заданного в константе PromGracePeriod. Обещание считается частично выполненным, если сумма платежа составляет менее того количества процентов, которое задано константой PromCoverRate;

·Не выполнено — данный статус присваивается обещанию, по которому нет платежей, по истечении количества дней, заданного константой PromGracePeriod, после даты обещания;

·Истекает — данный статус присваивается обещанию, по которому нет платежей, за то количество дней до даты обещания, которое задано константой PromExpireDays;

·Ожидание — данный статус присваивается обещанию, по которому нет подтвержденных платежей, до наступления даты обещания и в течение количества дней, заданного константой PromGracePeriod, после даты обещания;

·Не верифицировано — данный статус автоматически присваивается обещаниям, нарушающим регламент работы организации, если отношение обещанной суммы к остатку долга меньше суммы, заданной соответствующей константой PromMinSumRateHard или PromMinSumRateSoft. Для таких обещаний требуется подтверждение супервизором (см. Подтверждение обещания по долгу или Подтверждение обещаний). После подтверждения супервизором обещание получает один из вышеописанных статусов, а в случае отказа — удаляется. Правило определения обещаний, требующих подтверждения — см. Ограничения при регистрации контактов.

Примечание. Обещания в статусе "Не верифицировано" не отображаются в модуле Обработка должников, на вкладке Обещают заплатить модуля Список должников и др.
