Освобождение ответственного оператора

Операция освобождения ответственного оператора по долгам доступна в форме Постановка задач на вкладках Все должники, Проблемные, Поиск информации. Ответственный оператор может быть освобожден:

·для всех долгов из выборки, сформированной с учетом фильтров;

·для всех долгов, выделенных в текущей выборке с помощью клавиш SHIFT и/или CTRL.

Период работы оператора с долгом заканчивается в момент его освобождения. Во всех модулях и отчетах в атрибут "Оператор" для данного долга будет иметь пустое значение. Долг переводится в статус 1 "Не распределен". Все платежи и обещания, зарегистрированные по долгу позднее данного момента, не относятся ни к одному оператору до момента назначения нового оператора.

Для того чтобы освободить ответственного оператора, выполните следующие действия:

1.В форме Постановка задач перейдите на вкладку Все должники, Проблемные, Поиск информации.

2.При необходимости сформируйте выборку долгов с помощью фильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Освободить оператора и в раскрывающемся списке выберите вариант выполнения операции: По всей выборке или Только выделенные.

Примечание: если константа WorkTaskAllMode имеет значение "Ложь", то возможность освободить оператора по всей выборке долгов недоступна.

4.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить освобождение.

5.Если константа WorkTask.Operator.Change.Reason.Use имеет значение "Истина", то откроется форма для выбора причины снятия долгов (см. словарь № 326 "Причина назначения/снятия долга"). Выберите причину и нажмите ОК.

Примечание. Если в роли оператора, которого освобождают от долгов, доступ № 321 WORK_TASK_SEND_NOTIFICATION имеет значение TRUE, то ему приходит уведомление о снятии долгов.
