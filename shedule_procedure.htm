Вкладка "Процедуры"

На вкладке Процедуры находится список процедур, которые настраиваются разработчиком специально для клиента. Все процедуры на данной вкладке предназначены только для ручного запуска клиентом. Добавить или удалить данные на вкладке нельзя.

 

Доступны следующие действия (См. также):

ØЗапуск процедуры, настроенной для клиента
