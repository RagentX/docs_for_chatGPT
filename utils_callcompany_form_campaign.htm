Вкладка "Кампании"

Список кампаний обзвона отображается в форме Кампании обзвона на вкладке Кампании.

Вкладка содержит следующие элементы:

·список кампаний, который содержит следующие поля:

·ID кампании — порядковый номер кампании, формируется автоматически;

·Название — название кампании;

·Список — название группы должников, которые будут обрабатываться в рамках данной кампании (см. Группы должников);

·Статус — статус кампании (см. словарь № 40);

·Учет GMT — признак того, что должники должны обрабатываться с востока на запад по часовым поясам;

·Запущена — дата и время последнего запуска кампании (в формате "дд.мм.гггг чч.мм.сс");

·Остановлена — дата и время последней остановки кампании (в формате "дд.мм.гггг чч.мм.сс");

·Комментарий;

·Тип кампании — см. словарь № 318 "Тип кампании обзвона".

·кнопки для совершения действий с кампаниями обзвона:

· — Добавление кампании обзвона;

· — Изменение кампании обзвона;

· — Удаление кампании обзвона;

·Запустить — Запуск кампании обзвона;

·Остановить — Остановка кампании обзвона;

·Лог запуска — Просмотр лога запуска кампании;

·вкладка Участники, содержащая список операторов — участников выбранной кампании обзвона и кнопки для добавления и удаления участников в список. По каждому участнику отображается единственный атрибут ФИО — ФИО оператора. Внизу под столбцом ФИО отображается общее количество участников кампании;

·кнопки для совершения действий с участниками кампании обзвона:

· — Добавление участника кампании обзвона;

· — Удаление участника кампании обзвона;

·вкладка Статистика, содержащая статистику по участникам кампании (подробнее см. Просмотр статистики).
