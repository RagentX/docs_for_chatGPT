Вкладка "Простые Группы"

Вкладка Простые группы предназначена для работы с группами долгов, не использующими автофильтры, и содержит следующие элементы:

·список обычных групп долгов, содержащий следующие поля:

·ID группы — идентификатор простой группы долгов;

·Название группы — все автоматически созданные группы имеют название $auto_group_№;

·Владелец — оператор, создавший группу;

·Дата формирования — дата формирования автогруппы;

·Комментарий;

·кнопки для добавления, изменения, удаления групп долгов;

·кнопка История изменений — для просмотра истории изменений выбранной группы.

На вкладке Простые Группы доступны следующие действия:

·Создание простой группы;

·Редактирование простой группы;

·Превращение простой группы в автогруппу;

·Удаление группы;

·Просмотр истории изменений выбранной группы.
