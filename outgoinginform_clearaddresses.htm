Верификация адресов должников через интегрированный сервис DaData

DaData — это внешний сервис, интегрированный в систему.

Примечание. Для подключения данного функционала, свяжитесь с нашей техподдержкой.

Верификация адресов должника через интегрированный сервис DaData работает по следующему алгоритму. В очередь поступают заявки на проверку адресов через DaData. В ответ внешняя сервис возвращает этот же адрес, но с исправленными ошибками, привязкой к KLADR и статусом DaData (см. словарь № 72). Статус DaData показывает, на какие адреса не следует отправлять письма.

Для того отправить заявки на проверку адресов с помощью сервиса DaData, выполните следующие действия:

1.В форме Исходящее информирование перейдите на вкладку Должники.

2.При необходимости сформируйте выборку должников с помощью мультифильтров и/или выделите записи с помощью клавиш SHIFT и CTRL.

3.Нажмите кнопку Формирование ->Чистка адресов и в раскрывающемся списке выберите вариант выполнения операции: По всей выборке или Только выделенные. Откроется форма с параметрами запуска процесса.

4.В списке Тип адреса установите флаги для тех типов адресов, которые требуется проверить. Если не требуется проверять уже проверенные адреса, установите флаг Не выгружать уже проверенные по DaData.

5.Нажмите кнопку ОК. Заявки на проверку адресов с помощью сервиса DaData будут помещены в очередь. По мере обработки заявок результаты будут возвращены в систему. Информация о результатах работы сервиса DaData по адресам должников будет отображаться в форме Исходящее информирование на вкладке Должники, вкладка DaData в нижней части формы.
