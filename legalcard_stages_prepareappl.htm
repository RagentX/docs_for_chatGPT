Подготовка заявления в суд

Подготовка заявления в суд доступна для дел, которые находятся в следующих статусах:

·Не оформлен (если LawActUseRequestDueSum имеет значение "Ложь");

·Госпошлина оплачена (если LawActUseRequestDueSum имеет значение "Истина");

·Отказ в выдаче приказа (если LawAct.OrderReturn.Use имеет значение "Ложь");

·Возврат приказа (если LawAct.OrderReturn.Use имеет значение "Истина").

Операция подготовки заявления в суд может быть выполнена только после внесения всей необходимой информации:

·поле Расчетная гос. пошлина (см. Расчет госпошлины);

·поле Размер требования;

·информация о суде (см. Внесение данных о суде по делу в приказном производстве);

·В поле Укомплектованность введено значение "Полностью".

Для подготовки заявления в суд выполните следующие действия:

1.В карточке судебной работы на вкладке Приказное производство в разделе Общая информация в поле Укомплектованность укажите значение "Полностью".

2.Нажмите кнопку Подготовить.

3.В открывшемся диалоговом окне нажмите кнопку Yes, чтобы подтвердить выполнение операции.

Если какие-либо обязательные данные отсутствуют, будет выдано информационное сообщение. Нажмите в нем кнопку ОК, после чего произойдет возврат в карточку судебной работы для внесения данных.

4.Выполните прикрепление документа.

5.Далее осуществляется формирование настроенных запланированных действий.

6.Приказ переводится в статус "Подготовлен".

7.Действие протоколируется.
