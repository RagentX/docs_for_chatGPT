Расчет комиссии

Обновлено до 13 релиза 08.09.2015, MS SQL only.

Модуль Расчет комиссии предназначен для формирования правил расчета комиссий по долгам, назначения этих правил портфелям, в которые входят долги, и расчета комиссий в выбранном портфеле. Информация о размере комиссии по каждому долгу отображается в форме Платежи в столбце Комиссия в руб.

Для того чтобы перейти к модулю, в форме Платежи нажмите кнопку Комиссия.

Право для входа в модуль имеют только те пользователи, в роли которых доступ CALC_COM_VIEW имеет значение TRUE. Право для выполнения любых действий в модуле имеют те пользователи, в роли которых доступ CALC_COM_FULL имеет значение TRUE. Полный перечень доступов к модулю приведен в таблице.
