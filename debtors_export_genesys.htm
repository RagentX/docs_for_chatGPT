Выгрузка данных для автодайлера Genesys

В списке должников выгрузка данных в автодайлер доступна только пользователям, в роли которых доступ № 690 DEBT_LIST_EXPORT_DIALER имеет значение TRUE.

 

Возможность выгрузки данных для автодайлера Genesys используется для формирования кампаний обзвона и заданий на автоинформирование. Функция доступна только в том случае, если реализована интеграция с системой IP телефонии Genesys. Запуск кампаний обзвона и заданий на автоинформирование производится из модуля "Кампании обзвона" (см. Управление кампаниями для автодайлера и автоинформатора).

Выгрузка в автодайлер Genesys может осуществляться в двух вариантах:

·Новая кампания;

·Дозагрузка.

При создании новой кампании список номеров в call-list, соответствующий кампании, записывается заново. При выборе варианта дозагрузки порядок заполнения call-list может быть разным — от добавления номеров к уже сформированному списку до полной перезаписи списка.

Внимание! Для того чтобы выгрузка номеров телефонов могла осуществиться, необходимо, чтобы выбранная кампания существовала и была выгружена. Это можно видеть на вкладке "Genesys" модуля Кампании обзвона.

 

В таблицу dialer_setup добавлена новая запись:

name

dsc

dialer_get_calllist_version

Версия используемой процедуры ur_dialer_get_calllist_sql, от которой зависит форма "Описание задания".

У всех клиентов, кроме Летобанка значение dialer_get_calllist_version равно 1.

У ЛетоБанка в данную переменную требуется установлено значение 2.
