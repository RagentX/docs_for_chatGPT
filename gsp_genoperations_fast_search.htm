Быстрый поиск должника

Функционал быстрого поиска должника доступен из всех списочных модулей Системы. В модуле Группы свободного поиска данный функционал имеет некоторые отличия.

Быстрый поиск доступен на вкладках модуля по пункту контекстного меню Поиск по долгу или при нажатии комбинации клавишей CTRL + F.

Внимание! Важно! Константа FastSearch.InfoDrop.Mode в модуле "Группы свободного поиска" не работает!

Поля, по которым может осуществляться поиск

Быстрый поиск доступен, если на вкладке есть поля, по которым может выполняться поиск. При этом поля в источнике должны называться:

·ID долга.

·p_fio — для "ФИО должника".

·person_id,r_person_id,person.id — для "ID клиента".

·contract — для "Номер договора".

·account — для "Лицевой счет".

·total_rest — для "Остаток основного долга".

·debt.status, d_status, status — для "Статус долга".

Названиями полей на форме будут описания соответствующих полей источника.

Особенности работы функционала быстрого поиска

·При переключении вкладки открытое окно поиска автоматически закрывается.

·Значения, введенные на одной вкладке, не сохраняются при открытии формы поиска на другой вкладке.

·Поиск осуществляется только по отобранным и отфильтрованным записям (выборке).

·Если на вкладке отсутствуют некоторые поля, указанные в списке выше, поиск по ним не выполняется. В форме Поиск такие поля не отображаются.

·Если на вкладке отсутствуют все поля, указанные в списке выше, поиск на ней не доступен.

Порядок действий для осуществления поиска

1.Откройте модуль "Группы свободного поиска". Активируйте нужную вкладку.

2.Создайте выборку, используя мультифильтры, сложный фильтр, фильтрацию по полям таблицы. Дальнейший поиск будет происходить внутри сформированной выборки.

3.Активируйте контекстное меню, нажав на правую кнопку мыши, а затем в открывшемся списке выберите пункт "Поиск по долгу" (или нажмите комбинацию клавиш CTRL + F). Откроется форма Поиск.

4.В окне задайте один или несколько параметров поиска. В текстовых полях формы (таких, ка: ФИО, Договор, Лицевой счет) можно указать несколько символов, которые должны содержаться в соответствующем атрибуте искомого должника. Регистр символов при поиске не учитывается.

5.Нажмите кнопку Поиск. Будет выполнен переход к первой найденной записи списка, которая удовлетворяет заданным условиям.

6.Для продолжения поиска нажимайте повторно кнопку Поиск. Каждый раз в списке будет выделяться следующая по порядку запись, удовлетворяющая условиям.

7.Для того, чтобы стереть введенные в поля формы Поиск критерии, нажмите кнопку Отмена.
