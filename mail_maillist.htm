Группы рассылки

Show/Hide Hidden Text

Операции с группами рассылки доступны в адресной книге. Пользователь может создавать, изменять и удалять группы рассылки.

Форма редактирования атрибутов рассылки

       Создание группы рассылки

Для того чтобы создать группу рассылки, выполните следующие действия:

1.В адресной книге нажмите кнопку .

2.В открывшейся форме введите название группы рассылки в поле Название и при необходимости дополнительную информацию в поле Комментарий.

3.Нажмите кнопку ОК.

4.В адресной книге добавьте операторов в новую группу. Для этого выполните следующие действия:

a.В поле Группа рассылки выберите группу Все.

b.В левом списке выберите нужных операторов. Для выбора нескольких операторов используйте клавиши SHIFT и/или CTRL.

c.Нажмите кнопку , чтобы переместить операторов в правый список.

d.В поле Группа рассылки выберите созданную группу.

e.Нажмите кнопку Сохранить. Новая группа рассылки создана.

       Изменение группы рассылки

Для того чтобы изменить название группы рассылки или комментарий, выполните следующие действия:

1.В адресной книге нажмите кнопку .

2.В форме редактирования атрибутов рассылки измените название группы рассылки в поле Название и при необходимости дополнительную информацию в поле Комментарий.

3.Нажмите кнопку ОК, чтобы сохранить изменения.

Изменение состава группы рассылки выполняется, как описано в разделе Создание группы рассылки, шаг 4.

       Удаление группы рассылки

Для того чтобы удалить группу рассылки, выполните следующие действия:

1.В адресной книге в поле Группа рассылки выберите группу в раскрывающемся списке. Группу Все удалить нельзя.

2.Нажмите кнопку .

3.В открывшемся диалоговом окне подтвердите удаление, нажав кнопку Yes. Выбранная группа рассылки будет удалена.
