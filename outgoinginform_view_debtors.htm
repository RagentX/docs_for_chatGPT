Вкладки "Должники", "Поручители", "Залогодатели"

Show/Hide Hidden Text

Вкладки Должники, Поручители, Залогодатели содержат:

·Списки должников, поручителей и залогодателей соответственно:

·на вкладке Должники отображаются записи о должниках и принадлежащих им долгах, отобранных в соответствии с условиями мультифильтров. Если у одного должника имеется несколько долгов, то по каждому долгу будет отображаться отдельная запись;

·на вкладке Поручители отображаются записи о поручителях по долгам, отобранным в соответствии с условиями мультифильтров. Если по одному долгу имеется несколько поручителей, то по каждому из них будет отображаться отдельная запись;

·на вкладке Залогодатели отображаются записи о залогодателях по долгам, отобранным в соответствии с условиями мультифильтров. Если по одному долгу имеется несколько залогодателей, то по каждому из них будет отображаться отдельная запись;

·Кнопки для выполнения действий с выборкой. Набор кнопок и возможных действий различается на каждой вкладке;

·Набор вкладок, в которых отображаются результаты выполнения действий и кнопки для работы с ними.

Вкладки с результатами

1.Вкладка Письма доступна на вкладках Должники, Поручители, Залогодатели и содержит данные о письмах, сформированных на адреса выбранного контактного лица:

·№ письма — номер письма в системе;

·Адрес — адрес, на который сформировано письмо;

·Тип — тип письма;

·Заказное — признак заказного письма;

·Статус — статус письма (см. словарь № 35);

·Дата статуса — дата установки статуса.

2.Вкладка SMS доступна на вкладках Должники, Поручители, Залогодатели и содержит данные об SMS-сообщениях, отправленных на адреса выбранного контактного лица:

·Номер телефона;

·Статус — статус сообщения (см. словарь № 50);

·Дата изменения статуса СМС провайдером — дата изменения статуса провайдером (по согласованию с технической поддержкой данное поле может не отображаться);

·Дата статуса — дата установки статуса (по согласованию с технической поддержкой данное поле может не отображаться).

Примечание. На вкладке SMS не выводятся SMS в следующих статусах (см. словарь № 50):

·10 "Подготовка заявки";

·11 "Заявка";

·12 "Подтвержденная заявка".

3.Вкладка Email доступна на вкладках Должники, Поручители, Залогодатели и содержит данные о сообщениях электронной почты, отправленных на адреса выбранного контактного лица:

·Email — адрес электронной почты;

·Статус — статус сообщения (см. словарь № 65);

·Дата статуса — дата установки статуса.

4.Вкладка DaData доступна на вкладках Должники и содержит данные о результатах проверки адресов выбранного должника с помощью интегрированного сервиса DaData:

·Статус заявки — см. словарь № 84;

·Старое значение — проверяемый адрес должника;

·Текущее значение — адрес, полученный из сервиса DaData;

·Статус DaData — см. словарь № 72;

·Текст ошибки — значение возвращается сервисом DaData.
