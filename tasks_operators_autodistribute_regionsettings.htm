Настройка автоматического распределения по региону

Процесс распределения долгов по региону заключается в том, что районы субъектов РФ закрепляются за операторами.

Переход к форме настройки автоматического распределения по региону

1.Войдите в модуль Постановка задач.

2.Активируйте вкладку "Новые должники" или "Подготовка выездов".

3.Нажмите кнопку "Автораспределение".

4.В раскрывшемся списке выберите пункт По региону.

5.Далее выберите пункт Настройка.

6.Откроется форма Список настроенных районов по операторам.

Форма настройки списка районов по операторам

Настройка списка регионов для оператора

Для этого выполните следующие действия:

1.В поле Тип зоны выберите регион в раскрывающемся списке.

2.Нажмите кнопку . Откроется форма назначения ответственного оператора по районам выбранной зоны.

3.В форме выберите оператора в поле Коллектор, а затем выберите и переместите из правого списка в левый районы,  для которых он будет ответственным.

Примечание. Для каждого района есть возможность назначить одного или более операторов.

4.Нажмите кнопку ОК.

5.Повторите шаги 2-4 для добавления ответственных операторов по всем необходимым районам, входящим в регион.

6.При необходимости повторите шаги 1-5 для назначения ответственных операторов по другим регионам.

7.Закройте форму, нажав значок . Настройки выполнены.

Редактирование настроек автоматического распределения по региону

Если необходимо изменить ответственного оператора для какого-либо района или, наоборот, назначить оператору другой район, выполните следующие действия:

1.В форме настройки списка районов по операторам выберите нужную запись и нажмите кнопку или выполните двойной щелчок левой клавишей мыши по выбранной записи.

2.В открывшейся форме выберите нужный район и/или нужного оператора и нажмите кнопку ОК. Изменения будут внесены в список.

Для того чтобы просмотреть список всех районов, для которых назначены операторы, нажмите кнопку "Общий просмотр настройки".
