Простое создание входящего документа

Создание входящего документа доступно пользователям, в роли которых доступ № 668 DOCFLOW_INCOMING_CREATE имеет значение TRUE, а доступ № 676 DOCFLOW_CARD_ONLY_READ имеет значение FALSE.

При создании нового документа открывается карточка нового документа. См. также Первичное сохранение входящего документа.

Простое создание входящего документа осуществляется при выборе на вкладке Входящие документы в меню кнопки Создать документ пункта Простое создание.

Автоматическое заполнение полей

При простом создании входящего документа в карточке нового документа автоматически заполнены следующие поля:

·Входящий номер

Если константе Docflow.DocumentType.Default указан тип документа, предполагающий нумерацию (согласно текущему значению константы Docflow.DocumentType.WithoutNumber), то номер проставляется автоматически.

Если константе Docflow.DocumentType.Default указан тип документа, не предполагающий нумерацию (согласно текущему значению константы Docflow.DocumentType.WithoutNumber), или константа Docflow.DocumentType.Default равна 0 то номер проставляется только после ручного выбора типа документа, предполагающего нумерацию. Если выбран тип документа, не предполагающий нумерацию, поле "Входящий номер" пустое и недоступно для редактирования.

·Регистратор

В поле автоматически проставляется текущий оператор, поле недоступно для редактирования.

·Дата регистрации

В поле автоматически проставляется текущая дата, доступность редактирования определяется доступом № 674 DOCFLOW_MAIN_INFO_CHANGE.

·Тип документа

В поле автоматически проставляется значение, согласно значению константы Docflow.DocumentType.Default, иначе пустое.

·Статус

В поле автоматически проставляется термин с кодом 1 по словарю "Статус документа".

·Форма документа

Значение в это поле автоматически проставляется согласно значению константы Docflow.DocumentForm.Default, иначе пустое.

·Приоритет

Значение в это поле автоматически проставляется согласно значению константы Docflow.DocumentPriority.Default, иначе пустое.

·Способ доставки

Значение в это поле автоматически проставляется согласно значению константы Docflow.DocumentDeliveryMethod.Default, иначе пустое.

·Флаг "Без конверта"

По умолчанию выключен.

Порядок действий для простого создания входящего документа

Для того чтобы создать входящий документ, выполните следующие действия:

1.В модуле Документооборот активируйте вкладку Входящие.

2.Нажмите кнопку Создать документ.

3.В открывшемся меню выберите пункт Простое создание.

4.Откроется форма Карточка документа для нового документа (она включает только одну вкладку — Основная). Некоторые поля будут заполнены по умолчанию, согласно настройкам Системы. Заполните необходимые поля.

Примечание. Поле Тип документа обязательно для заполнения.

5.Заполнение поля Получатели — см. Изменение получателей входящего документа.

6.Заполнение поля Отправитель — см. Изменение отправителя входящего документа.

7.Заполнение поля Исполнители — см. Изменение исполнителей входящего документа.

8.При необходимости выполните обновление номера входящего документа.

9.Нажмите кнопку Сохранить. Будет осуществлено первичное сохранение документа. Карточка документа примет вид полноценной карточки входящего документа. Далее — см. Карточка входящего документа.
