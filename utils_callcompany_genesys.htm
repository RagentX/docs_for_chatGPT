Управление кампаниями автодайлера и автоинформатора

Show/Hide Hidden Text

Кампании автодайлера и автоинформатора создаются в модулях:

·Группы должников (см. Выгрузка данных для автодайлера Genesys);

·Список должников (см. Выгрузка данных для автодайлера Genesys);

Они подразумевают автоматический обзвон должников с участием (кампании автодайлера) или без участия (автоинформирование) оператора. Для того чтобы начать обзвон должников, необходимо выполнить запуск кампании автодайлера или задания для автоинформатора.

       Запуск и остановка кампании автодайлера

Перед запуском новой кампании необходимо удостовериться, что отсутствуют активные кампании в статусе, отличном от "Не загружена" и "Выгружена". Если такие кампании есть, то запустить новую кампанию не получится. Предварительно необходимо остановить и выгрузить активные кампании.

Для того чтобы запустить кампанию автодайлера, выполните следующие действия:

1.Загрузите сформированный список телефонов на сервер Genesys. Для этого в форме Кампании обзвона на вкладке Genesys выберите кампанию и нажмите кнопку Загрузить. Кампания изменит статус на "Загружена".

2.При необходимости перед запуском кампании скорректируйте критерий оптимизации и значение процентного показателя, которые влияют на интенсивность работы автодайлера. Для установки новых значений нажмите кнопку Установить.

3.Для запуска кампании нажмите кнопку Запустить. Кампания изменит статус на "Запущена", и начнется обзвон должников. Во время обзвона система будет дозваниваться до должника и соединять его с оператором, находящимся в статусе "Автодайлер" (зеленый цвет). При поднятии трубки будет автоматически открываться соответствующая карточка с указанием телефонного номера, по которому произошло соединение.

·Если  константа InnerCallUseOpenGuarantorCard выключена, то осуществляется открытие карточки должника на вкладке Общая.

·Если  константа InnerCallUseOpenGuarantorCard включена, то при поступлении входящего звонка с номера, для которого установлена связь с поручителем открывается карточка должника на вкладке с поручителями и поверх нее открывается карточку поручителя.

Для корректировки интенсивности работы автодайлера можно менять критерий оптимизации и значение процентного показателя в "горячем" режиме без остановки кампании. Для применения новых значений необходимо нажать кнопку Установить.

Для того чтобы остановить работу кампании обзвона, выполните следующие действия:

1.В форме Кампании обзвона на вкладке Genesys выберите кампанию и нажмите кнопку Остановить. Кампания изменит статус на "Остановлена".

2.Выгрузите кампанию с сервера Genesys. Для этого нажмите кнопку Выгрузить. Кампания изменит статус на "Выгружена".

       Запуск и остановка заданий для автоинформатора

Перед запуском нового задания для автоинформатора необходимо удостовериться, что отсутствуют активные задания в статусе, отличном от "Не загружено" и "Выгружено". Если такие задания есть, то запустить новое задание не получится. Предварительно необходимо остановить и выгрузить активные задания.

Для того чтобы запустить задание для автоинформатора, выполните следующие действия:

1.Загрузите сформированный список телефонов на сервер Genesys. Для этого в форме Кампании обзвона на вкладке Genesys выберите задание и нажмите кнопку Загрузить. Задание изменит статус на "Загружено".

2.Для запуска задания нажмите кнопку Запустить. Задание изменит статус на "Запущено", и начнется обзвон должников. Во время обзвона система будет дозваниваться до должника и указанное в задании количество раз до тех пор, пока должник не прослушает сообщение до конца.

Для того чтобы остановить работу задания, выполните следующие действия:

1.В форме Кампании обзвона на вкладке Genesys выберите задание и нажмите кнопку Остановить. Задание изменит статус на "Остановлено".

2.Выгрузите задание с сервера Genesys. Для этого нажмите кнопку Выгрузить. Задание изменит статус на "Выгружено".
