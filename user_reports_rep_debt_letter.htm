Письмо должнику

Отчет "Письмо должнику" предназначен для формирования письма должнику без открытия карточки должника и имеет аналогичный функционал (см. Формирование письма).

Отчет "Письмо должнику" доступен только пользователям, в роли которых доступ REP_DEBT_LETTER имеет значение "Истина".

Порядок действий

Для того чтобы сформировать отчет "Письмо должнику", выполните следующее:

1.Выберите в главном меню пункт Отчеты, затем выберите отчет "Письмо должнику".

2.Откроется форма "Письмо должнику" для ввода параметров данного отчета. Заполните следующие поля:

·ФИО — введите ФИО должника или выберите должника. При вводе символов в это поле Система автоматически начинает поиск. Поле обязательно для заполнения.

·Продукт — выберите продукт.

·Адрес — выберите адрес. Поле обязательно для заполнения.

·Форма — выберите форму письма. Список доступных форм настраивается в модуле "Конструктор писем" (при формировании писем должнику доступны только письма, у которых в поле Получатель указано Должник, при формировании писем поручителю — Поручитель; залогодателю — Залогодатель);

·Флаг "Сохранять копию письма" — флаг работает только при включенном флаге Регистрировать письмо. Если этот флаг включен, то копия письма будет сохранена в БД.

3.Нажмите кнопку "Запустить". Сформированное письмо откроется в приложении Fast Report.
