Ручная постановка задач (кнопка "Поставить задачу")

1.Войдите в модуль. Активируйте Рабочую вкладку "Задачи".

2.Нажмите кнопку Поставить задачу. Откроется форма Задача.

3.Заполните поля:

·Название задачи — обязательное поле, заполняется при создании задачи;

·Описание — необязательное поле, заполняется при создании задачи;

·Тип задачи — обязательное поле, заполняется при создании задачи. Значения задаются в словаре № 229;

·Подтип задачи — обязательное поле, заполняется при создании задачи. Значения задаются в словаре № 230;

·Приоритет задачи — обязательное поле, заполняется при создании задачи. Может быть изменено при действии Изменить приоритет задачи. Значения задаются в словаре № 235;

·Крайняя дата завершения — дата выбирается из календаря или вводится вручную. Заполняется при создании задачи. Также доступно для изменения при переходе в статус Отложена;

Примечание. При создании задачи выполняется проверка, настроен ли для задачи данного типа и подтипа крайний срок в модуле Словари на Вкладке "Настройка крайнего срока для задачи". Если такой срок настроен, и в форме создания задачи крайняя дата завершения не указана, то для такой задачи крайняя дата завершения устанавливается как дата создания задачи плюс количество дней, указанное для задач данного типа и подтипа в модуле Словари на Вкладке "Настройка крайнего срока для задачи".

Важно отметить, что если при создании задачи вы укажете крайнюю дату завершения, то будет сохранена эта дата.

 

·Исполнитель задачи — плановый исполнитель задачи; необязательное поле. Может быть изменено при действии Изменить исполнителя или при автоматическом изменении исполнителя задачи. Заполнено по умолчанию (см. Выбор исполнителя для задачи по умолчанию). Выбирается из списка доступных пользователей;

·Фактический результат — не доступно для редактирования при таком статусе. Это поле будет доступно при действиях Отметить выполнение задачи и Принять выполнение задачи.

·Дата постановки задачи — по умолчанию указан текущий момент времени. Доступно для изменения при создании задачи;

·Постановщик задачи — недоступно для редактирования, заполняется автоматически. Указывается пользователь, который поставил задачу;

·Статус задачи — недоступно для редактирования, заполняется автоматически. Изменяется при смене статусов задачи;

·Выполнивший задачу — фактический исполнитель задачи; недоступно для редактирования, заполняется автоматически при действии Отметить выполнение задачи;

·Дата факт. выполнения — дата выполнения задачи;

·Причина перехода задачи в текущий статус — доступна только для некоторых статусов и определяется словарями:

·Причина отклонения задачи — словарь № 231 (для действия Отклонить задачу);

·Причина невыполнения задачи — словарь № 234 (для действия Отметить невыполнение задачи).

5.Нажмите кнопку OK. Если все поля заполнены правильно, форма Задача закроется, и вы окажетесь во вкладке Задачи.

6.В списке задач появится строка с описанием задачи.

7.Статус новой задачи будет Поставлена.

В зависимости от стадии взыскания и настроек Системы задача будет доступна на Информационной вкладке "Задачи" и Рабочей вкладке "Задачи" соответствующих модулей.
