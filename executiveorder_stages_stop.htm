Приостановление ИП

Если дело находится в статусе "Возбуждено", есть возможность его приостановить. Приостановленному делу можно вернуть статус "Возбуждено".

Приостановление ИП

Для того чтобы приостановить ИП, выполните следующие действия:

1.Откройте карточку исполнительного производства для дела, находящегося в статусе "Возбуждено".

2.Нажмите кнопку Приостановление, в выпадающем меню выберите пункт Приостановить ИП.

3.Откроется форма Приостановить ИП. Заполните следующие поля:

·Дата приостановления — поле обязательно для заполнения; по умолчанию поле заполнено значением даты приостановления из текущей карточки исполнительного производства;

·Причина приостановления — выберите из выпадающего списка причину приостановления (обязательное поле). Полный список причин — см. словарь № 266 "Причины приостановления ИП";

·Комментарий — дополнительная информация (необязательное поле).

4.Нажмите OK.

5.Выполните прикрепление документа.

6.Далее осуществляется формирование настроенных запланированных действий.

7.Действие будет запротоколировано.

8.Дело будет переведено в статус "Приостановлено", в карточке исполнительного производства поля Дата приостановления и Причина приостановления будут заполнены данными, введенными в п. 3.

Для дел, находящихся в статусе "Приостановлено" доступен только пункт Отмена приостановления кнопки Приостановление. Никакие другие кнопки смены статуса недоступны.

Отмена приостановления ИП

Для того чтобы отменить приостановление ИП, выполните следующие действия:

1.Откройте карточку исполнительного производства для дела, находящегося в статусе "Приостановлено".

2.Нажмите кнопку Приостановление, в выпадающем меню выберите пункт Отмена приостановления.

3.Откроется окно подтверждения. Выберите Yes.

4.Откроется форма Отмена приостановления ИП. Заполните следующие поля:

·Дата отмены приостановления — поле обязательно для заполнения; по умолчанию поле заполнено значением даты отмены приостановления из текущей карточки исполнительного производства;

·Комментарий — дополнительная информация (необязательное поле).

5.Нажмите кнопку ОК.

6.Выполните прикрепление документа.

7.Далее осуществляется формирование настроенных запланированных действий.

8.Дело будет переведено в статус "Возбуждено".

9.В зависимости от значения константы LawExec.PauseFields.ClearByCancel в карточке исполнительного производства:

a)Если "Истина", то значения полей Дата приостановления и Причина приостановления будут очищены;

b)Если "Ложь", то значения данных полей останутся без изменения.

10.Действие протоколируется.
