Адреса

Раздел Адреса находится в карточке должника на вкладке Общая, вложенная вкладка Адреса, и содержит следующие элементы:

·Список адресов должника с полями:

·Тип — тип адреса (см. словарь № 5);

Примечание: если включена константа CardAllAddressShow, отображается информация по адресам поручителей и залогодателей.

·Адрес — адрес должника;

·Статус — статус адреса (см. ниже);

·Статус выезда — статус выезда по данному адресу (см. Выезды по адресам должника);

·Проживает — признак того, что должник проживает по данному адресу;

·Возвращено письмо — признак наличия возвращенного письма, направленного на данный адрес;

·Комментарий — дополнительная информация;

·Статус по DaData — статус проверки адреса с помощью сервиса DaData (см. Верификация адресов должников через интегрированный сервис DaData);

·Блокирован;

·Верифицирован — см. Верификация контактных данных;

·Дата блокировки — заполняется только для заблокированных адресов;

·Причина блокировки — заполняется только для заблокированных адресов.

·Кнопки:

· Добавить — добавление адреса;

· Изменить — изменение адреса;

· Отметить необходимость выезда — см. Отметка о необходимости выезда;

· Запланировать визит в офис — см. Планирование визита должника в офис;

· Письмо должнику — с подпунктами:

§Сформировать письмо;

§Сформировать заявку;

§Выездной лист;

· Установить связь с долгом — см. Связь адресов с долгом (долгами).

·Флаг Верификация контактных данных — см. Ручная верификация данных.

Окраска адресов в зависимости от статуса

Строки в списке адресов выделены цветом фона в зависимости от статуса адреса. (см. Статусы адресов):

·Не выезжали ни разу — серый цвет;

·Результата не было — белый цвет;

·Неверный адрес — красный цвет.

·Результат — зеленый цвет;

·Последний результат — голубой цвет.

Если у пользователя есть доступ 280 DEBT_CARD_ALL_ADR, то в карточке должника он может видеть заблокированные адреса. Такие записи в списке адресов выделены серым шрифтом на белом фоне.

Доступные действия с адресами

·Добавление, редактирование адреса — см. Ведение списка адресов должника;

·Блокирование / разблокирование адреса — см. Блокирование адреса должника, Разблокирование адреса должника.

·Просмотр, добавление или удаление связи адреса с долгом — см. Связь адресов с долгом (долгами).

·Копирование адреса или строки списка адресов

·Для копирования выбранного адреса в буфер обмена выберите в контекстном меню (вызывается правой кнопкой мыши) пункт Копировать адрес;

·Для копирования всей выбранной строки в буфер обмена выберите в контекстном меню (вызывается правой кнопкой мыши) пункт Копировать строку;
