Форма "Ближайшие"

Описание формы "Ближайшие"

На экранной форме "Ближайшие" (Рисунок 13‑1) отображаются адреса, ближайшие к местонахождению мобильного устройства. Адреса упорядочиваются по возрастанию расстояния до устройства. Количество записей в списке ближайших точек настраивается в пункте Количество записей в списке ближайших точек.

Форма содержит следующие элементы:

·Кнопка — переход на Боковое меню;

·Кнопка — Обновление;

·Кнопка — Передать контакты (синхронизация);

·Кнопка "Сбросить" — для сбора примененных фильтров;

·Кнопка "На карте" — для просмотра адресов на карте (Рисунок 13‑2);

·Фильтрация:

·Долги или выезды;

·Тип адреса;

·Классификаторы;

·Список адресов;

·Кнопка "Обновить список" — Обновление;

·Кнопка "Передать контакты" — Передать контакты (синхронизация);

Настройка списка ближайших адресов

Для настройки списка ближайших адресов:

1.Выберите тип фильтрации (по умолчанию выбрана фильтрация "Долги"):

·Долги;

·Выезды;

2.Если выбрана фильтрация "Долги", выберите тип адреса. По умолчанию отображаются все адреса.
