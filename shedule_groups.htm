Мероприятия для групп должников

Форма добавления мероприятия

Добавление нового мероприятия для групп должников

Для того чтобы добавить новое мероприятие в расписание, выполните следующее:

1.Войдите в Модуль "Расписание" и активируйте вкладку "Мероприятия для групп".

2.Нажмите кнопку Добавить.

3.Откроется форма для добавления мероприятия. Выберите тип мероприятия.

4.Настройте параметры мероприятия. Для этого нажмите на в поле Настройка параметров. Откроется форма для ввода параметров Параметры запуска. Каждому типу мероприятия соответствуют свои параметры (см. Описание типов мероприятий).

5.При необходимости в поле Группа выберите группу долгов, для обработки которых предназначено мероприятие. Если для запуска мероприятия выбора группы не требуется, это поле будет блокировано.

6.Настройте График запуска мероприятия.

7.Заполните поля:

·Последовательность — при необходимости введите числовое значение последовательности. В случае одновременного запуска двух и более мероприятий первым будет выполняться задание с наименьшим значением в поле.

·Флаг "Вкл." — если флаг находится в положении Включено, мероприятие запускается, если флаг находится в положении Выключено, мероприятие не запускается. Использование флага дает возможность отменить запуск мероприятия, не удаляя его.

·Флаг "Проверять формирование групп". При включенном флаге мероприятие выполняется только для тех групп, которые были сформированы или переформированы в соответствии со следующими условиями:

·Если значение константы Scheduler.ReformGroupCheck.Mode равно 2, то группа считается переформированной, если дата формирования больше или равна текущей дате.

·Если значение константы Scheduler.ReformGroupCheck.Mode равно 1, то группа считается переформированной, если дата формирования больше текущей даты и времени минус 1 сутки (текущая логика).

·Если значение константы Scheduler.ReformGroupCheck.Mode не равно 1 или 2, то используется логика режима 1.

Внимание! Если мероприятие в модуле Расписание не предполагает использование автогруппы, флаг Проверять формирование группы должен находиться в положении Выключено.

·Комментарий — дополнительная информация.

8.Нажмите OK.

Изменение мероприятия для групп должников

Для того чтобы изменить мероприятие в расписании, выполните следующее:

1.Войдите в Модуль "Расписание" и активируйте вкладку "Мероприятия для групп".

2.Выберите нужную строку и нажмите кнопку Изменить.

3.Откроется форма настройки соответствующего мероприятия с уже заполненными полями. Более подробное описание см. Описание типов мероприятий.

4.Отредактируйте значения полей и нажмите OK.

Удаление мероприятия для групп должников

Для того чтобы удалить мероприятие из расписания, выполните следующее:

1.Войдите в Модуль "Расписание" и активируйте вкладку "Мероприятия для групп".

2.Выберите нужную строку и нажмите кнопку Удалить.

3.Подтвердите удаление.

Обновление списка мероприятий

В модуле можно настроить автоматическое обновление списка мероприятий и данных об их выполнении. Для этого в контекстном меню (вызывается правой кнопкой мыши) вкладки "Мероприятия для групп" выберите пункт Таймер, затем выберите частоту обновления. По умолчанию обновление отключено.

Обновить список также можно, нажав кнопку Обновить или клавишу F5.

Примечание. Если вы обновите список в момент начала выполнения какого-либо мероприятия, вы сможете увидеть, как включается флаг Признак выполнения.

Просмотр лога выполнения мероприятий

Право на просмотр лога запуска мероприятия имеют пользователи, в роли которых доступ SCHEDULE_LOG_VIEW имеет значение TRUE. Если доступ отсутствует или выключен, кнопка История на форме отсутствует.

Для того чтобы просмотреть список последних запусков выбранного мероприятия:

1.Войдите в Модуль "Расписание".

2.Активируйте вкладку "Мероприятия для групп".

3.Выберите нужное мероприятие в списке.

4.Нажмите кнопку История.

5.В открывшемся меню выберите пункт Лог запуска.

6.По окончании просмотра нажмите кнопку Закрыть.

Просмотр истории изменений

Право на просмотр истории изменений мероприятия имеют пользователи, в роли которых доступ SCHEDULE_LOG_VIEW имеет значение TRUE. Если доступ отсутствует или выключен, кнопка История на форме отсутствует.

Для того чтобы просмотреть историю изменений выбранного мероприятия:

1.Войдите в Модуль "Расписание".

2.Активируйте вкладку "Мероприятия для групп".

3.Выберите нужное мероприятие в списке.

4.Нажмите кнопку История.

5.В открывшемся меню выберите пункт История изменений.

6.По окончании просмотра нажмите кнопку Закрыть.

 

Форма История изменений содержит следующие элементы:

·Область данных Версии, содержащая список всех редакций данной группы;

·Область данных Изменения, содержащая подробности выбранной редакции.

Типы изменений:

·I (insert) — присваивается при первой редакции группы, то есть ее создании;

·U (update) — присваивается при изменениях атрибутов группы;

·D (delete) — присваивается при удалении группы (недоступно для просмотра в данной форме).

Для просмотра выбранного поля "Старое значение" или "Новое значение" в области данных Изменения щелчком мыши можно открыть область, содержащую значение этого поля, размеры которой доступны для изменения. Это используется в тех случаях, когда, например, значение поля имеет длину, превышающую ширина столбца.
