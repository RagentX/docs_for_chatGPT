Просмотр данных о судебных заседаниях

Данные о планируемых и состоявшихся судебных заседаниях по делам, находящимся на стадиях искового и исполнительного производства, отображаются в форме Юридический департамент на вкладке Заседания. В нижней части вкладки отображается информация о решении суда для выбранного заседания.

На вкладке доступны:

·мультифильтр по ответственному оператору);

·возможность выбора интервала времени, за который нужно просмотреть информацию о заседаниях. Для выбора интервала укажите дату и время в полях С и По. По умолчанию начало интервала — это первый день текущего месяца (время 00:00:00), конец интервала — последний день текущего месяца (время 23:59:59);

·мультифильтр по судам, которые проводили заседания;

·флаг Показывать только последнее заседание, при установке которого  списке будут отображаться только последние заседания по каждому долгу, найденному с учетом условий мультифильтров.  

Вкладка Заседания содержит следующие поля:

·ID должника — идентификатор должника, связанного с делом;

·ID долга — идентификатор долга, связанный с делом;

·ID дела — уникальный идентификатор (порядковый номер) дела в системе;

·Тип — тип судебного дела (иск/приказ/исполнительное);

·ФИО — фамилия, имя, отчество ответчика по делу (должника или поручителя);

·Номер договора;

·Размер требования — сумма долга;

·Оператор — оператор, ответственный за работу с делом;

·Дата заседания;

·Тип суд. заседания — если UseLawSessionTyp="Истина", то значение указывается согласно словарю № 254 "Тип судебного заседания";

·Суд — суд, который проводил заседание;

·Судья — ФИО судьи;

·Статус заседания — см. словарь № 73;

·Результат заседания — см. словарь № 74;

·Решение — решение суда;

·Филиал  — филиал организации, к которому относится долг (см. словарь № 61).
