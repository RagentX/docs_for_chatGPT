Вкладка "Android"

Описание вкладки "Android"

На вкладке Android отображаются только выезды, зарегистрированные с помощью мобильной версии Системы (то есть те, у которых модулем регистрации является "Android").

На вкладке может осуществляться Контроль регистрации координат по адресу выезда.

Доступна возможность Просмотра маршрутов операторов мобильных устройств (в меню кнопки ).

Список полей вкладки Android

·Должник — ФИО должника или наименование организации;

·Тип должника — см. словарь № 54;

·Банк — краткое название банка;

·Название продукта — указывается название продукта, по которому возник долг;

·Портфель — портфель, в который входит долг;

·Дата;

·Оператор — оператор, зарегистрировавший контакт;

·Отдел — подразделение, котором работает оператор (см. Структура предприятия);

·Конт. — признак того, что контакт установлен;

·Результат — результат контакта (см. Настройка дерева контактов);

·Дата обещания — обещанная дата оплаты долга;

·Сумма обещания — обещанная сумма оплаты долга;

·Направление — номер телефона, по которому был установлен контакт;

·Тип направления — тип телефона;

·Комментарий — дополнительная информация о контакте;

·Валюта — валюта долга (см. Список валют);

·Остаток — остаток суммы долга на текущую дату, выводится в валюте долга, указанной в поле Валюта;

·Статус долга — см. словарь № 6;

·Договор — номер договора с должником из системы/базы данных клиента;

·Лицевой счет — лицевой счет должника из системы/базы данных клиента;

·ID Клиента — идентификатор должника из системы/базы данных клиента;

·Часовой пояс — часовой пояс должника;

·Важность;

·Порядочность;

·Флаг Поручитель — включен, если контакт состоялся с поручителем;

·Флаг Залогодатель — включен, если контакт состоялся с залогодателем;

·ID выезда — идентификатор данного контакта в Системе;

·Флаг Запись — этот флаг включен для записей о контактах типа "Звонок", для которых в БД есть запись телефонного разговора (для таких контактов доступно Прослушивание и сохранение записей телефонных разговоров, совершенных с помощью мобильной версии).

·Балл № 1 — см. словарь № 101;

·Балл № 2 — см. словарь № 102;

·Балл № 3 — см. словарь № 244;

·Балл № 4 — см. словарь № 245;

·Дата регистрации — дата регистрации контакта;

·Гос. пошлина — сумма государственной пошлины;

·Остаток основного долга — сумма остатка основного долга, выводится в валюте долга, указанной в поле Валюта;

·Филиал — если организация, использующая систему, имеет филиальную структуру, в данном поле указывается филиал, который должен работать с этим долгом. Значение выбирается из словаря № 61;

·Регион — регион, в котором выдан кредит, выбирается из словаря № 39;

·Реестр КА — исходящий портфель долга;

·Флаг "Наличие коорд. Адрес" — включен, если в базе данных заполнено значение географических координат адреса должника, по которому должен осуществиться данный выезд.

·Флаг "Наличие коорд. Регистрация" — включен, если в базе данных присутствует значение географических координат, соответствующее реальному местоположению оператора, совершившего выезд.

·Расстояние до адреса (м);

·Адрес — адрес, по которому был зарегистрирован выезд;

·Тип адреса — см. словарь № 5;

·Флаг Попадание выезда в радиус адреса — включен, если расстояние между местом контакта и адресом меньше значения константы Android.VisitContactAddress.AllowableDeviationRadius;

·Расстояние между контактом и адресом, м — расстояние в метрах между местом контакта и адресом.
