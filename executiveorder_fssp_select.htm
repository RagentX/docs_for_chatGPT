Выбор ФССП по делу в исполнительном производстве

Выбор ФССП по делу исполнительного производства разрешено пользователям, в роли которых доступ № 476 LAW_EXEC_COURT_SET имеет значение TRUE.

Для выбора ФССП в карточке исполнительного производства на вкладке Инф. о ФССП в разделе Информация о ФССП выберите значение в поле ФССП.  Поля Телефон и Адрес будут заполнены автоматически.

Примечание. Для выбора доступны записи о ФССП, у которых выключен флаг "Не используется". Это действует для любого способа выбора ФССП.

Если нужная ФССП отсутствует, ее можно добавить в список (см. Добавление данных о новой ФССП).

Также доступны:

·Выбор ФССП из списка;

·Поиск ФССП;

·Автоматический выбор исполнительного органа.

Выбор ФССП из списка

1.Войдите в модуль Карточка исполнительного производства. Активируйте вкладку Инф. о ФССП.

2.Нажмите кнопку "Выбрать из списка". Откроется окно выбора ФССП.

3.Выберите запись из списка. На правой части формы отобразятся данные по выбранному суду. Нажмите Выбрать.

4.Окно закроется, а в разделе "Информация о ФССП" карточки исполнительного производства будут отображена информация о выбранной ФССП. Действие протоколируется.

Поиск ФССП

5.Войдите в модуль Карточка исполнительного производства. Активируйте вкладку Инф. о ФССП.

6.Нажмите кнопку "Выбрать из списка". Откроется окно выбора ФССП.

·В верхней части формы расположено окно поиска. Поиск осуществляется по любой части, входящей в название. Введите фрагмент названия и нажмите .

·В левой части формы отображается список ФССП. Установите курсор на нужную строку списка.

·В правой части формы расположены атрибуты выбранной ФССП, соответствующие данным в форме Информация о ФССП в модуле Настройка судов и ФССП.

7.Выберите запись из списка или при помощи поиска. На правой части формы отобразятся данные по выбранному суду. Нажмите Выбрать.

8.Окно закроется, а в разделе "Информация о ФССП" карточки исполнительного производства будут отображена информация о выбранной ФССП. Действие протоколируется.

Автоматический выбор исполнительного органа

Автоматический выбор исполнительного органа, связанного с адресом подачи, осуществляется в карточке исполнительного производства на вкладке "Инф. о ФССП" по нажатию кнопки Автоматический выбор.

Данная кнопка видна и доступна, если константа LawExec.Fssp.AutoSelect имеет значение "Истина".

Для того, чтобы совершить автоматический выбор исполнительного органа, выполните следующее:

1.В модуле Карточка исполнительного производства в списке адресов убедитесь, что отмечен адрес подачи, а также что данный адрес заполнен корректно.

2.Активируйте вкладку "Инф. о ФССП".

3.Нажмите кнопку Автоматический выбор.

4.Если информация о суде уже заполнена, появится форма подтверждения, нажмите Yes.

5.Если найдено более одного исполнительного органа, соответствующих выбранному адресу подачи, то произойдет следующее:

a)Появится сообщение с текстом:

Найдено несколько исполнительных органов для выбранного адреса.

b)Нажмите ОК. Откроется форма для выбора исполнительного органа, на которой будут отображены записи об исполнительных органах, соответствующих выбранному адресу.

c)Выберите запись из списка, расположенного в левой части формы. При этом в правой части формы будут отображены подробные данные о выбранном исполнительном органе.

d)Нажмите Выбрать.

6.Будет произведена попытка поиска исполнительного органа, соответствующего адресу подачи. В случае успеха данные об исполнительном органе будут добавлены на поля формы Инф. о ФССП. Если исполнительный орган найти не удастся, будет показано сообщение об ошибке.
