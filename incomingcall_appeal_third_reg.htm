Зарегистрировать обращение третьих лиц

Регистрация обращения третьих лиц доступна из модуля Входящий звонок из выпадающего меню кнопки , пункт "Зарегистрировать обращение от третьих лиц".

Право на использование функционала имеют те пользователи, в роли которых доступ № 535 INNER_CALL_APPEAL_REG имеет значение TRUE.

Примечание. Для доступа к механизму обращений третьих лиц используется константа UseAppealFromThirdPerson логического типа. Если ее значение равно "Истина", работа с обращениями от третьих лиц доступна. Если ее значение равно "Ложь", обращения третьих лиц недоступны.

Для того чтобы зарегистрировать обращение от третьих лиц:

1.Нажмите кнопку , затем в выпадающем меню выберите пункт "Зарегистрировать обращение от третьих лиц".

2.Откроется Форма "Обращение 3-го лица". Измените значения полей:

·№ — введите номер обращения;

·ФИО — введите фамилию, имя и отчество обращающегося;

·Дата обращения — выберите дату (поле заполнено по умолчанию текущей датой и временем);

·Тип обращения — выберите тип обращения, см. словарь № 238 Тип обращения (поле обязательно для заполнения);

·Подтип обращения — выберите подтип обращения, см. словарь № 262 Подтип обращения (поле обязательно для заполнения);

·Входящий тел. — по умолчанию указан номер, с которого зарегистрирован входящий звонок;

·Контактный тел. — введите контактный номер телефона;

·Орган — выберите орган, см. словарь № 239 Орган.

·Характер обращения — введите дополнительную информацию, касающуюся характера обращения.

3.Нажмите OK.
