Просмотр данных о пакетах CSBI

Описание вкладки "Пакеты CSBI"

Вкладка Пакеты CSBI включает следующие вложенные вкладки:

·Определение подсудности;

·Определение подведомственности.

Каждая вложенная вкладка включает один и те же набор элементов и контекстное меню.

Элементы вкладки

Вложенная вкладка Определение подсудности / Определение подведомственности включает следующие элементы:

·Список пакетов, включающий следующие поля:

·ID пакета;

·Название пакета;

·Состояние пакета;

·Дата обновления состояния;

·Статус пакета;

·Дата обновления статуса;

·Внешний ID.

·Кнопки для действий со списком пакетов:

· Добавить пакет — для добавления пакета;

· Редактировать пакет — для редактирования пакета;

· Удалить пакет — для удаления пакета.

·Подчиненная вкладке Список заявок, включающая следующие элементы:

·Список заявок, включающий следующие поля:

oID заявки;

oID дела;

oАдрес;

oСтатус заявки;

oДата обновления статуса;

oОшибка.

Кнопка Удалить заявку — для удаления заявки.
