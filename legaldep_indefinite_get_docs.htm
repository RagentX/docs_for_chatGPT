Получение документов

1.Войдите в модуль Рабочие места а Юридический департамент. Активируйте вкладку Судебные дела (предв.).

2.Сформируйте выборку с помощью сложного фильтра и/или мультифильтров.

3.Выберите одну или более записей, используя при необходимости клавиши SHIFT и/или CTRL.

4.Активируйте контекстное меню, нажав на правую кнопку мыши. Выберите пункт Документы. В открывшемся меню выберите пункт Получить документы.

5.Появится окно подтверждения. Подтвердите свой выбор.

6.Затем будет осуществлена проверка статуса выбранных дел. Получение документов может осуществляться только для дел, находящихся в статусе "Запрос документов". Если ни одно из выбранных дел не находится в указанном статусе, появится соответствующее сообщение, и действие осуществлено не будет.

7.Если в списке дел одно или более находятся в статусе "Запрос документов", то алгоритм движется дальше, иначе появляется предупреждение "Ни одно из выделенных дел не находится в статусе “Запрос документов”", и процесс прекращается.

8.Если в системе настроены типы документов, то появится окно подтверждения получения документов с полями-флагами, по умолчанию значения из будут "Включено":

·КД;

·Расчет зад-ти;

·Выписка.

Если получены документы не всех указанных типов, выключите соответствующие флаги, а затем нажмите OK. Если для дела документы отмеченных типов еще не были получены, отмечается их получения, путем установки соответствующего поля-флага в положение "Включено". Статус "Документы получены" может быть установлен только при включении всех флагов, иначе статус остается "Запрос документов".

9.Устанавливается статус "Документы получены".

10.Действие протоколируется. В поле Действие записывается "Получение документов". В поле Комментарий указываются типы полученных документов.

11.В зависимости от настроек Системы возможно формирование настроенных запланированных действий.
