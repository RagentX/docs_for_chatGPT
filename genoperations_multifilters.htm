Мультифильтры

Show/Hide Hidden Text

Мультифильтры предназначены для формирования списка значений (операторов, портфелей и т. д.), в соответствии с которым будут отбираться записи для отображения в форме или выполнения каких-либо действий.

При использовании мультифильтров для настройки отображения данных поля мультифильтров находятся в верхней части формы. Для применения мультифильтров предназначена кнопка Найти.

Особенности применения мультифильтров в разных списочных модулях — см. Применение мультифильтров.

Кроме того, форма мультифильтра открывается при выполнении некоторых действий (например, см. Использование визуального конструктора запросов).

Для того чтобы задать условия мультифильтра, выполните следующие действия:

1.В поле мультифильтра нажмите значок .  Откроется форма для выбора значений. Данная форма также может открыться при выполнении некоторых операций (например, см. Использование визуального конструктора запросов). В левой части формы находится список еще не выбранных значений, в правой — список выбранных значений.

Примечание 1: в левом списке значений доступны функции отбора и сортировки записей. При необходимости их можно использовать, чтобы найти нужные записи.

Примечание 2: форма выбора портфелей дополнительно содержит флаг Отображать закрытые портфели, который позволяет скрывать/отображать в списке портфели в статусе "Закрыт".

2.Перенесите записи о нужных значениях из левого списка в правый. Для выбора нескольких записей используйте клавиши SHIFT и/или CTRL. Для переноса записей между списками используйте следующие кнопки:

· — переместить выбранные записи из левого списка в правый;

· — переместить все записи из левого списка в правый;

· — переместить выбранные записи из правого списка в левый;

· — переместить все записи из правого списка в левый.

Перенести одну выбранную запись между списками можно с помощью двойного щелчка левой клавишей мыши.

3.Нажмите кнопку ОК. Выбранные значения будут добавлены в условия фильтра.

4.Выполните шаги 1-3, чтобы задать условия других мультифильтров списка (если применимо).

5.Нажмите кнопку Найти, чтобы применить мультифильтры для отображения данных (если применимо).

       В случае очень длинных списков выбранных записей в мультифильтре

Следующая информация верна для мультифильтров:

·Портфели;

·Операторы.

В модулях:

·Список должников;

·Протокол контактов;

·Постановка задач;

·Исходящее информирование;

·Платежи;

·Юридический департамент;

·Группы свободного поиска;

·Выездное взыскание;

·Статистика.

Если длина строки выбранных значений не превышает 10000 символов

В настоящей версии Системы если длина строки выбранных в мультифильтре значений не превышает 10000 символов, то строка в поле мультифильтра отображается без изменений. Ее можно копировать. При повторном открытии мультифильтра в нем отображаются все выбранные значения.

Если длина строки выбранных значений превышает 10000 символов

Если длина строки превышает 10000 символов, то в поле мультифильтра отображается только первые 10000 символов, а затем добавляется многоточие. Для копирования доступна только отображенная часть строки, но при повторном открытии мультифильтра в нем отображаются все выбранные значения.
