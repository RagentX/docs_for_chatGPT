Создание новой записи с банковскими реквизитами

Для того чтобы добавить новую запись о получателе платежа и его банковские реквизиты, выполните следующие действия:

1.В форме Банковские реквизиты нажмите кнопку .

2.В открывшейся форме заполните следующие поля:

·Тип реквизита — тип реквизита определяет, где будут использоваться реквизиты (см. словарь № 115);

Примечание. При отсутствии доступа REQUISITES_PRINT нельзя выбрать тип реквизитов "Для печати". При отсутствии доступа REQUISITES_PAYMENT нельзя выбрать тип реквизитов "Для платежей".

·Название — краткое название получателя платежа. Поле обязательно для заполнения;

·Получатель — полное название получателя платежа. Поле обязательно для заполнения;

·ИНН — ИНН получателя платежа. Поле обязательно для заполнения;

·КПП — КПП получателя платежа;

·Банк получателя — название банка получателя платежа. Поле обязательно для заполнения;

·БИК — БИК банка. Поле обязательно для заполнения;

·К/с — корреспондентский счет получателя платежа;

·Р/с — расчетный счет получателя платежа;

·Назначение платежа — назначение платежа, которое будет указано по умолчанию при регистрации платежа для данного получателя;

·Юридический адрес банка — юридический адрес банка;

·Логотип — логотип банка. Доступны только файлы формата JPEG. Для добавления логотипа нажмите кнопку , в открывшемся диалоговом окне Windows  выберите файл с логотипом и нажмите кнопку Open (Открыть). Для удаления файла логотипа нажмите кнопку .

3.Нажмите кнопку ОК. Получатель платежа и его банковские реквизиты будут добавлены в список.
