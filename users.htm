Пользователи

Обновлено до 50 релиза 03.11.2020.

Модуль Пользователи предназначен для ведения списка учетных записей пользователей системы. Данные учетных записей используются для аутентификации и авторизации пользователей в системе. Если у пользователя нет учетной записи, он не сможет работать в системе.

Для того чтобы открыть модуль, выберите в главном меню пункт Администрирование а Пользователи.

Право для входа в модуль имеют только те пользователи, в роли которых доступ USERS_VIEW установлен в значение TRUE.

Право для выполнения любых действий в модуле имеют те пользователи, в роли которых доступ USERS_FULL установлен в значение TRUE.
