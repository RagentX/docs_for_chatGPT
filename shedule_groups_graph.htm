График запуска мероприятия

Вид и содержание поля "График" зависит от значения константы SchedulerUseExtendedScheduleMode.

Внимание! При изменении значения константы SchedulerUseExtendedScheduleMode для корректной работы системы необходимо перенастроить графики запуска всех существующих мероприятий! Переключение данной константы рекомендуется согласовывать с технической поддержкой.

Далее описано два режима графика запуска мероприятия:

·Расширенный (константа SchedulerUseExtendedScheduleMode = TRUE).

·Простой (константа SchedulerUseExtendedScheduleMode = FALSE).

Расширенный график запуска мероприятия

При нажатии на поле График открывается окно графика. Оно содержит следующие элементы:

·Область Период действия расписания, содержащая поля:

·Начало;

·Окончание.

·Область выбора периода запуска мероприятия, содержащая переключатели (radio button) с возможными значениями периода запуска:

·Ежедневно;

·Еженедельно;

·Ежемесячно;

·Однократно.

Примечание. По умолчанию выбран период Ежедневно.

·Флаг Учитывать рабочие дни — для запрета выполнения мероприятия в выходные или праздничные дни;

·Область описания выбранного периода. Ее вид зависит от значения периода.

·Поле Время запуска;

·Область Привязка к событию: — см. Привязка мероприятия к событию.

Настройки графика в зависимости от периода запуска

Ежедневно

Область описания периода содержит поле выбора Повторять каждые <количество дней> дней, дающее возможность задать количество дней от 1 до 100. Мероприятие будет выполняться в момент времени, указанный  в поле Время запуска.

Если флаг Учитывать рабочие дни включен, то каждый выбранный день проходит проверку на определение рабочий это день или нет. Если день нерабочий, то мероприятие будет выполняться на следующий рабочий день.

Еженедельно

Область описания периода содержит флажки (checkbox), соответствующие каждому дню недели. Есть возможность выбрать один или более дней недели, мероприятие будет выполняться в момент времени, указанный  в поле Время запуска.

Если флаг Учитывать рабочие дни включен, то выбор субботы и воскресения недоступен.

Если флаг Учитывать рабочие дни включен, то каждый выбранный день проходит проверку на определение рабочий это день или нет. Если день нерабочий, то высчитывается ближайший будущий рабочий день в рамках этой недели, когда данное мероприятие должно быть выполнено. Если такого дня в рамках этой недели не найдено, то мероприятие не выполняется.

Ежемесячно

Область описания периода содержит флажки (checkbox), соответствующие каждому числу месяца, а также дополнительный флажок Последний. Есть возможность выбрать одно или более чисел месяца, мероприятие будет выполняться в момент времени, указанный  в поле Время запуска.

Если флаг Учитывать рабочие дни включен, то каждый выбранный день проходит проверку на определение рабочий это день или нет. Если день нерабочий, то высчитывается ближайший будущий рабочий день в рамках этого месяца, когда данное мероприятие должно быть выполнено. Если такого дня в рамках этого месяца не найдено, то мероприятие не выполняется.

Если отмечен Последний день месяца, то мероприятие должно выполниться в последний рабочий день месяца.

Однократно

Область описания периода содержит следующие элементы:

·Поле для выбора даты запуска мероприятия.

·Кнопка Добавить, предназначенная для добавления выбранной даты в список дат запуска мероприятия.

·Кнопка Очистить, очищающая список выбранных дат.

·Список выбранных дат (для ознакомления).

С помощью поля выбора даты, кнопки Добавить и кнопки Очистить создайте список дат для запуска мероприятия. В указанные даты мероприятие будет выполняться однократно в момент времени, указанный  в поле Время запуска.

Флаг Учитывать рабочие дни не активен.

Простой график запуска мероприятия

При нажатии на поле График открывается окно графика. По умолчанию оно содержит два поля:

·Период запуска;

·Время запуска.

В зависимости от выбранного значения поля Период запуска вид окна графика может измениться.

Возможные значения периода запуска:

·Ежедневно;

·Еженедельно;

·Ежемесячно.

Настройки графика в зависимости от периода запуска

Ежедневно

Выберите время в поле Время запуска. Мероприятие будет запускаться каждый день в указанное время.

Еженедельно

Выберите значения следующих полей:

·День недели;

·Время запуска.

Мероприятие будет запускаться по указанным дням недели в указанное время.

Ежемесячно

Выберите значения следующих полей:

·Число месяца;

·Время запуска.

Мероприятие будет запускаться указанного числа каждого месяца в указанное время.
