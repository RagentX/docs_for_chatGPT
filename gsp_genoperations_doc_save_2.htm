Выгрузка документов и писем по долгу

В настоящей версии Системы реализована процедура выгрузки документов и писем по долгу. Функционал запускается в режиме для всей выборки либо для выделенных долгов. По каждому долгу выбираются типы прикрепленных документов и печатные формы писем, а также типы адресов.

В модуле Список должников функционал доступен из вкладки Все.

В модуле Группы свободного поиска функционала доступен для тех вкладок, на которых есть ID долга и только если есть доступ на эту операцию.

Осуществление выгрузки документов и писем

1.Войдите в модуль и активируйте нужную вкладку.

2.Сформируйте выборку при помощи сложного фильтра и/или мультифильтров.

3.Выберите одну или более записей, при необходимости используя клавиши SHIFT и/или CTRL.

4.Нажмите на кнопку "Документы по долгу". В выпадающем меню выберите пункт Выгрузка документов и писем по долгу.

5.Затем выберите режим выгрузки:

·Все;

·Только выделенные.

6.Откроется окно Выгрузка документов.

7.Выберите не менее одного типа документа или письма в одной (или более) области данных из следующих:

·Типы документов — см. словарь № 100 "Тип вложения для файла";

·Печатные формы досудебной работы — печатные формы, созданные в модуле Конструктор писем, см. словарь № 36 "Тип письма" (для выбора доступны только те термины, доступ к которым настроен для роли пользователя в модуле Роли и доступы на вкладке Письма);

·Печатные формы судебной работы — печатные формы, созданные в модуле Конструктор документов судебной работы, относящиеся к судебной работе, см. словарь № 33 "Тип заявления" (для выбора доступны только те термины, доступ к которым настроен для роли пользователя в модуле Роли и доступы на вкладке Печатные формы (Суд));

·Печатные формы исполнительного производства — печатные формы, созданные в модуле Конструктор документов судебной работы, относящиеся к исполнительному производству, см. словарь № 48 "Тип заявлений ИП" (для выбора доступны только те термины, доступ к которым настроен для роли пользователя в модуле Роли и доступы на вкладке Печатные формы (ИП)).

8.Выберите один или более типов адреса (словарь № 5).

9.При необходимости включите флаг Сформировать опись.

10.Нажмите OK.

11.Если ни одного типа документа или письма из п. 7 не было выбрано, будет показано окно предупреждения, и выгрузка не будет осуществляться.

12.Если типы документов или писем выбраны, пользователю предлагается стандартное окно для выбора пути сохранения. В указанную папку сохраняются документы выбранных типов, формируются и сохраняются письма и заявления выбранных типов, сгруппированные в подкаталоги по долгу. Название подкаталога формируется как <номер договора> - <ФИО должника>. Документы сохраняются с тем же названием, с которым сохранены в системе "Контакт"+<ID Документа> для исключения дублирования. Письма сохраняются с именем <Название шаблона документа>+<ID письма> для исключения дублирования названия письма. Если в системе несколько документов с одним типом, то выгружаются все документы указанного типа.

13.Если был включен флаг Сформировать опись, то в тот же каталог формируется Excel-файл содержащий опись выгруженных документов в формате. Шапка файла формируется из колонок:

·номер договора;

·ФИО должника;

·Все выбранные типы документов; если типы документов не указаны, то столбцы отсутствуют;

·Все выбранные типы печатных форм досудебной работы; если типы документов не указаны, то столбцы отсутствуют;

·Все выбранные типы печатных форм судебной работы; если типы документов не указаны, то столбцы отсутствуют;

·Все выбранные типы печатных форм работы исполнительного производства; если типы документов не указаны, то столбцы отсутствуют.

14.Для каждого долга выборки в файле описи заполняются поля Номер договора и ФИО должника, в остальных колонках при наличии документа или письма указывается название документа; если документов несколько, то они перечисляются через точку с запятой.

15.В процессе выгрузки пользователю демонстрируется модальное окно с индикатором хода загрузки. При этом доступна возможность прервать процесс, нажав на кнопку Отмена.

См. также Выбор банковских реквизитов при формировании письма или документа судебной работы.
