Работа с документами "По папкам"

Функционал работы с документами "По папкам" включается по согласованию с вендором! Для включения данного функционала обратитесь в поддержку. Функционал включается коммерческой константой @DocAttach.AttachByFolder.Use.

Область действия доработки

Если функционал не включен, работа с документами не изменяется. Дальнейшее описание относится к ситуации, когда функционал включен.

Данный функционал работает в следующих модулях Системы:

·Карточка должника — область данных "Файлы" на вкладке "Вложения".

·Карточка судебной работы — вкладка "Вложения".

·Карточка ИП — вкладка "Вложения".

·Карточка уголовного преследования — вкладка "Вложения".

·Карточка имущества — вкладка "Вложения".

·Карточка документа (Документооборот) — вкладка "Вложения".

В некоторых модулях кроме тех кнопок, которые будут описаны далее, есть дополнительный функционал и, соответственно, дополнительные кнопки. Они описаны в стандартной пользовательской документации HELP.

В модуле "Карточка документа" (Документооборот) кнопки имеют другой вид, но их функциональность такая же.

Общее описание функционала

Для работы с документами "по папкам" в Системе создаются папки для хранения документов.

Папки едины для всех персон и модулей. То есть, в разных карточках должника, в разных карточках судебной работы и т.д. — отображаются одни и те же папки.

При сохранении документа пользователь выбирает папку, куда будет сохранен данный документ.

Папки имеют иерархическую структуру. Есть возможность добавлять, редактировать и удалять папки, а также переносить документы из одной папки в другую.

Примечание. Папка верхнего уровня "default" является системной. Она не может быть удалена. Все остальные папки являются дочерними по отношению папке "default".

Для работы с документами "по папкам" на вкладке "Вложения" (или в области данных "Файлы" на вкладке "Вложения") добавлены дополнительные элементы и действия.

Область данных "Файлы" включает следующие элементы:

·Область данных, отображающая папки, которая включает следующие элементы:

·Список папок в их иерархической структуре.

·Кнопки для действий с папками:

· — Добавление папки.

· — Изменение папки.

· — Удаление папки.

·Область данных, отображающая файлы, которая включает следующие элементы:

·Список файлов, находящихся в выбранной папке.

·Кнопки для действий с файлами в папках:

· "Добавить" — Добавление документа.

· "Изменить" — Изменение документа.

· "Удалить" — Удаление документа.

· — Перенос документов в другую папку.

Действия с папками и документами "По папкам"

Добавление папки

Для того чтобы добавить новую папку для хранения документов, выполните следующее:

1.В нужном модуле активируйте вкладку "Вложения".

2.В области данных, содержащей список папок выберите папку, которая будет родительской.

3.Нажмите кнопку , относящуюся к области данных со списком папок.

4.Откроется форма "Новая запись" для добавления папки. В поле "Родительская папка" указана папка, которая будет родительской по отношению к добавляемой. Заполните следующие поля:

·Название папки — введите название.

·Комментарий — дополнительная информация.

5.Нажмите ОК.

Папка будет добавлена в качестве дочерней папки по отношению к выбранной.

Изменение папки

Редактирование папки заключается в изменении ее названия и/или комментария.

Для того чтобы отредактировать папку для хранения документов, выполните следующее:

1.В нужном модуле активируйте вкладку "Вложения".

2.В области данных, содержащей список папок выберите папку.

3.Нажмите кнопку , относящуюся к области данных со списком папок.

4.Откроется форма "Редактирование" для добавления папки. Измените нежные значения полей.

5.Нажмите ОК.

Папка будет изменена.

Удаление папки

Для удаления доступны только папки, не имеющие дочерних папок.

Все документы, всех персон, которые содержатся в удаляемой папке, будут перенесены в системную папку верхнего уровня "default".

Для того чтобы удалить папку для хранения документов, выполните следующее:

1.В нужном модуле активируйте вкладку "Вложения".

2.В области данных, содержащей список папок выберите папку.

3.Нажмите кнопку , относящуюся к области данных со списком папок.

4.Откроется форма подтверждения удаления. Нажмите "Yes".

Папка будет удалена.

Добавление документа

Для того чтобы добавить документ, выберите сначала в левой области папку, в которую вы будете добавлять документ.

Далее происходит стандартное действие по добавлению документа. В результате документ будет добавлен в выбранную папку.

Изменение документа

Изменение документа происходит стандартно.

Чтобы найти нужный документ, выберите в левой области нужную папку.

Удаление документа

Удаление документа происходит стандартно.

Чтобы найти нужный документ, выберите в левой области нужную папку.

Перенос документов в другую папку

Для того чтобы выполнить перенос одного или более документов в другую папку, выполните следующее:

1.В нужном модуле активируйте вкладку "Вложения".

2.В области данных, содержащей список папок выберите нужную папку.

3.В области данных, содержащей список документов выберите один или более документов.

4.Нажмите кнопку . Откроется форма "Выбор папки".

5.Выберите нужную папку.

6.Нажмите ОК.
