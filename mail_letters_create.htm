Создание письма

Для того чтобы создать письмо, выполните следующие действия:

1.В форме Почта нажмите кнопку или или комбинацию клавиш CTRL и "N".

2.В открывшейся форме в поле Тема введите тему письма, в нижнем блоке — текст письма. Если тема не указана, при отправке письма будет выдано сообщение с предложением указать ее. Если требуется получить от адресата ответ на письмо, установите флаг Требуется ответ.

3.Если требуется прикрепить вложения, выполните следующие действия:

a.Нажмите кнопку . Откроется форма для присоединения файлов.

b.В открывшейся форме нажмите кнопку .

c.В стандартном диалоговом окне выберите файл, который требуется прикрепить, и нажмите кнопку Open (Открыть). Файл будет добавлен в форму.

d.При необходимости повторите шаги a-c, чтобы добавить другие файлы.

e.Нажмите кнопку ОК, чтобы закрыть форму.

4.Укажите адресатов письма в поле Адресат . Для этого:

a.В поле нажмите значок . Откроется адресная книга, в которой содержатся все операторы, зарегистрированные в системе. В левой части формы находится список еще не выбранных значений, в правой — список выбранных значений.

В левом списке значений доступны функции отбора и сортировки записей. При необходимости их можно использовать, чтобы найти нужные записи.

b.При необходимости в поле Группа рассылки выберите группу рассылки, в которую входят нужные адресаты.

c.Перенесите записи о нужных адресатах из левого списка в правый. Для выбора нескольких адресатов используйте клавиши SHIFT и/или CTRL. Для переноса адресатов между списками используйте следующие кнопки:

· — переместить выбранных адресатов из левого списка в правый;

· — переместить всех адресатов из левого списка в правый;

· — переместить выбранных адресатов из правого списка в левый;

· — переместить всех адресатов из правого списка в левый.

Перенести одного выбранного адресата между списками можно с помощью двойного щелчка левой клавишей мыши.

d.Нажмите кнопку ОК. Выбранные адресаты будут добавлены в поле Адресат.

5.Нажмите кнопку Отправить. Письмо будет отправлено.

Отправленные письма хранятся в папке Исходящие -> Не прочитанные до тех пор, пока оно не будет прочитано адресатом. После этого оно автоматически переносится в папку Исходящие -> Прочитанные.
