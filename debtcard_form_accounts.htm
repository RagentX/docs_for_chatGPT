Вкладка "Счета"

Вкладка Счета в карточке должника предназначена для ведения списка расчетных счетов должника.

Вкладка Счета присутствует в карточке должника следующих типов:

·Юридические лица;

·ИП;

·Физические лица, если значение константы DebtCardAccountForPhysicalPerson равно "Истина".

Примечание. Если константа DebtCardAccountForPhysicalPerson имеет значение "Ложь", то для должника типа "Физическое лицо" вкладка Счета отсутствует.

Право на редактирование данных на  вкладке Счета

Вкладка Счета содержит следующие элементы:

·Список расчетных счетов должника;

·Кнопки для работы со списком расчетных счетов.

Список расчетных счетов содержит следующие поля:

·Банк — person_account.b_name;

·Расчетный счет — person_account.r_account;

·БИК — person_account.bik ;

·Кор. счет — person_account.k_account;

·Остаток на счете — person_account.sum_rest;

·Комментарий — person_account. dsc.
