Добавление нового должника

Для того чтобы добавить в новую запись о должнике, выполните следующие действия:

1.В форме Список должников нажмите кнопку . Откроется форма ввода основных данных о должнике (различается для физических и юридических лиц).

2.В форме заполните поля:

·Для физических лиц:

·Тип — тип должника (см. словарь № 54);

·Сумма задолженности — сумма долга;

·Дата выхода на просрочку — дата, с которой долг считается просроченным;

·Фамилия, Имя, Отчество;

·Дата рождения — дата рождения должника;

·Серия, № паспорта — серия и номер паспорта должника;

·Валюта — валюта долга (см. Список валют);

·Тип продукта — тип кредитного продукта (см. словарь № 11);

·Название продукта — кредитный продукт, по которому возник долг;

·Портфель — портфель, в который входит долг.

·Для юридических лиц:

·Тип — тип должника (см. словарь № 54);

·Сумма задолженности — сумма долга;

·Дата выхода на просрочку — дата, с которой долг считается просроченным;

·Компания — название компании-должника;

·ИНН — ИНН компании-должника;

·Валюта — валюта долга (см. Список валют);

·Тип продукта — тип кредитного продукта (см. словарь № 11);

·Название продукта — кредитный продукт, по которому возник долг;

·Портфель — портфель, в который входит долг.

·Для индивидуальных предпринимателей (ИП):

·Тип — тип должника (см. словарь № 54);

·Сумма задолженности — сумма долга;

·Дата выхода на просрочку — дата, с которой долг считается просроченным;

·Фамилия, Имя, Отчество;

·Дата рождения — дата рождения должника;

·Серия, № паспорта — серия и номер паспорта должника;

·Валюта — валюта долга (см. Список валют);

·Тип продукта — тип кредитного продукта (см. словарь № 11);

·Название продукта — кредитный продукт, по которому возник долг;

·Портфель — портфель, в который входит долг.

3.Нажать кнопку ОК. Откроется карточка должника, в которой будут заполнены основные сведения о должнике.

4.Заполните карточку должника. Подробнее о работе с карточкой должника см. Карточка должника.

5.Сохраните карточку одним из следующих способов:

·чтобы сохранить данные и оставить карточку открытой, нажмите кнопку . Если данные указаны корректно,  изменения будут сохранены;

·закройте карточку стандартным способом (). Будет выдано окно для подтверждения сохранения данных. Нажмите Yes, чтобы сохранить изменения и закрыть карточку; No — для отказа от сохранения и возврата к списку должников, Отмена — для возврата к редактированию карточки.
