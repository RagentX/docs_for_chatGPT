Конструктор SQL-запросов

Конструктор SQL-запросов предназначен для конструирования вкладок в модуль Группы свободного поиска.

Доступы для работы с запросами:

№

Название доступа

Значение доступа

Описание доступа

343

GROUP_DEBT_CONSTRUCTOR_FULL

TRUE

Дает полный доступ на работу с конструкторами структур БД и SQL-запросов

307

GROUP_DEBT_SQL_CONSTRUCTOR

TRUE

Дает право на работу с конструктором SQL-запросов

Чтобы вызвать конструктор SQL-запросов, в форме Группы долгов на вкладке Автогруппы нажмите кнопку ГСП и в выпадающем меню выберите пункт Конструктор запросов.

В соответствии с созданным запросом отбираются записи, отображаемые на вкладке модуля Группы свободного поиска.

Запросы создаются на основе шаблонов структур БД, созданных в Конструкторе структур. Для одного шаблона структуры может быть создано несколько запросов.
