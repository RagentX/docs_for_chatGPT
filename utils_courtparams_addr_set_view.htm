Вкладка "Закрепление адресов за исп. органами"

Вкладка Закрепление адресов за исп. органами доступна пользователям, в роли которых доступ № 701 LAW_JUDGE_ADDR_SET_COURT_VIEW имеет значение TRUE.

Вкладка предназначена для Закрепления адресов за судами и последующего автоматического выбора суда в карточке судебной работы.

Вкладка Закрепление адресов за исп. органами включает следующие элементы:

·Список судов, содержащий информацию об адресах, привязанных к этим судам. Колонки списка:

·ID — идентификатор суда, связанного с адресами;

·Тип органа — тип исполнительного органа, значение "Суд" или указывается тип исполнительного органа по словарю № 339;

·Тип суда — тип суда, связанного с адресами (для типа органа "Суд");

·Название — название исполнительного органа, связанного с адресами;

·Индекс — индекс для связанных с исполнительным органом адресов;

·Страна — страна для связанных с исполнительным органом адресов;

·Область — область для связанных с исполнительным органом адресов;

·Район — район для связанных с исполнительным органом адресов;

·Нас. пункт — населенный пункт для связанных с исполнительным органом адресов;

·Улица — улица для связанных с исполнительным органом адресов;

·Список домов — список домов для связанных с исполнительным органом адресов;

·Тип персоны — см. словарь № 54;

·Тип суд. дела:

·1 — Приказ;

·2 — Иск;

·3 — Правопреемство;

·4 — Банкротство;

·Список статусов:

·для приказа — см. словарь № 18;

·для иска, правопреемства или банкротства — см. словарь № 25.

·Кнопки для добавления, редактирования и удаления записей в списке.

Подробнее — см. Закрепление адресов за судами.
