Контекстное меню приказа

Контекстное меню вызывается щелчком правой кнопки мыши в области списка дел.

На вкладке Приказы в контекстном меню доступны следующие действия:

·Копировать;

·Выделить все;

·Обновить;

·Сохранить выборку;

·Установить значение столбца с подпунктами:

·Ответственный (определяется доступом LAW_ACT_ORDER_SET_USER);

·Портфель (определяется доступом LAW_ACT_ORDER_SET_PORTFOLIO);

·Примечание (определяется доступом LAW_ACT_ORDER_SET_DICT_VALUE);

·Перевод между стадиями с подпунктами:

·На исполнительное производство;

·Преобразовать в иск;

·Преобразовать в правопреемство;

·Объединение судебных дел (определеяется константой LawActList.UnionLawAct.Use и доступом LAW_ACT_LIST_UNION_LAW_ACT);

·Создать исполн. документ (определяется константой UseLawExecDoc, доступом LAW_EXEC_DOC_CREATE);

·Настройка запланированных действий;

·Назначить временного ответственного (определяется доступом LAW_ACT_SET_USER_UNTIL_DATE, включается по согласованию с вендором);

·Сформировать документы по шаблону с подпунктами:

·По всей выборке — осуществляется печать документов по всей выборке дел;

·Только выделенные — осуществляется печать документов только для выделенных дел;

·Конструктор документов;

·Настройка ограничений — см. Ограничения на формирование документов судебной работы в зависимости от банка и их настройка;

·Перевод в архив (определяется доступом LAW_ACT_ORDER_ARCHIVE);

·Госпошлина с подпунктами:

·Рассчитать госпошлину;

·Запросить госпошлину;

·Отменить оплату ГП;

·Подготовить;

·Подать;

·Приказ получен;

·Информация об операторе;

·Добавить документ с подпунктами:

·Загрузить документ;

·Добавить ссылку на файл;

·Просмотреть адреса на карте.
