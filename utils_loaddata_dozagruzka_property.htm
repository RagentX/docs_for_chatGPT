Особенности загрузки файлов с типом формата "Имущество"

1.Для каждой строки загружаемого файла выполняется поиск записи, совпадающей с указанным значением в поле "Идентификатор имущества" (id) ИЛИ "Внешний идентификатор имущества" (ext_id):

·Если по указанным данным однозначно определено имущество, то происходит обновление данной записи.

·Иначе, обновление не происходит. Запись помечается в файле ошибок с комментарием. "Не найдено имущество по ID2.

2.В рамках загрузки данных происходит обновление указанных в шаблоне полей:

·Если значение в поле файла загрузки совпадает со значением в поле БД для данной записи, то обновление поля не требуется.

·Если поле указано в шаблоне и значение = "" (пустое значение) или NULL, обновление данного поля не требуется.

· Для остальных полей шаблона происходит обновление данных согласно стандартной логике.
