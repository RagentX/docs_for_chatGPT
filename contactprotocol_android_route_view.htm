Просмотр маршрутов

Функционал дает возможность просматривать маршруты операторов мобильных устройств.

Форма просмотра маршрутов содержит следующие элементы:

·Область данных, содержащая карту;

·Область фильтров, содержащая следующие поля:

·Дата с — обязательное поле; по умолчанию заполнено текущей датой и временем 00:00:00;

·Дата по — обязательное поле; по умолчанию заполнено текущими датой и временем;

·Операторы — открывается стандартная форма выбора, какая используется в мультифильтрах, поддерживается множественный выбор; необходимо выбрать хотя бы одного оператора;

Примечание. Список операторов отображается в зависимости от значения доступа CONTACT_LOG_FULL:

·Если в роли пользователя доступ включен, то отображается полный список всех операторов;

·Если доступ отсутствует или выключен, отображаются только операторы, доступные пользователю согласно структуре предприятия.

·Флаги: "Звонки" и "Выезды" — обязательно выбрать хотя бы один флаг.

Порядок действий

Для того чтобы просмотреть маршруты, выполните следующие действия:

1.Войдите в модуль "Протокол контактов".

2.Активируйте вкладку "Android".

3.Нажмите кнопку "Информация о перемещениях" и в открывшемся меню выберите "Просмотр маршрутов".

4.В открывшейся форме заполните нужные поля.

5.Нажмите ОК.

Информация о маршрутах выбранных операторов за выбранный период будет отображена на карте.

Описание обозначений на карте

·Зеленая точка обозначает начало движения

·Красная точка обозначает окончание движения

·Большие круги обозначают кластеры контактов — близкие контакты, зарегистрированные рядом. Цвета в кластере:

·Зеленый — ближе чем выбранный диапазон;

·Красный — дальше чем выбранный диапазон;

·Серый — адреса без координат в базе.

Зарегистрированный выезд обозначается красным кругом.

Зарегистрированный звонок обозначается синим кругом.

Для просмотра информации по совершенному контакту (контактам) кликните на контакт или на кластер. Будет показано окно с подробной информацией.

Маршрут оператора обозначается сплошной линией. Точка регистрации контакта и адрес соединяются пунктирной линией.
